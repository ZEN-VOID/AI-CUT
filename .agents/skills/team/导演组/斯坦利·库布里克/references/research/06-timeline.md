# 06 · 人物时间线

## Timeline

| 时间 | 事件 | 方法论意义 |
|---|---|---|
| 1928 | 生于纽约布朗克斯。 | 城市观察、移民家庭、纽约公共空间成为早期视觉经验。 |
| 1945 | 17岁将FDR去世当天报摊照片卖给Look。 | 影像从新闻瞬间进入叙事构图。 |
| 1945-1950 | 为Look杂志摄影。 | 建立人物观察、城市空间、静态叙事和光影训练。 |
| 1951 | 拍摄拳击纪录短片 `Day of the Fight`。 | 从摄影转向剪辑、运动、纪录和低成本制作。 |
| 1953 | 完成首部长片 `Fear and Desire`。 | 战争、抽象寓言与低成本独立制片开始。 |
| 1955 | `Killer's Kiss`。 | 黑色电影、城市空间、拳击与运动调度继续成形。 |
| 1956 | `The Killing`。 | 非线性犯罪结构和制片伙伴体系成熟。 |
| 1957 | `Paths of Glory`。 | 战争制度、军队权力和空间调度形成早期高峰。 |
| 1960 | 接手 `Spartacus`。 | 大片经验与控制冲突并存，之后远离Hollywood常规模式。 |
| 1962 | `Lolita`。 | 审查、改编、黑色幽默和英国制作环境成形。 |
| 1964 | `Dr. Strangelove`。 | 核毁灭与制度荒诞以黑色喜剧承载。 |
| 1968 | `2001: A Space Odyssey`。 | 科幻、非语言叙事、特效系统和开放解释的代表作；获奥斯卡特殊视觉效果。 |
| 1971 | `A Clockwork Orange`。 | 暴力、自由意志、国家矫正与争议传播问题集中爆发。 |
| 1975 | `Barry Lyndon`。 | 自然光、绘画构图、历史质感和慢节奏叙事。 |
| 1980 | `The Shining`。 | Steadicam、空间恐怖、家庭崩坏和制作强度争议。 |
| 1987 | `Full Metal Jacket`。 | 越战、军事训练、语言暴力和战争主体分裂。 |
| 1999 | `Eyes Wide Shut`；同年3月去世。 | 性、婚姻、欲望制度、梦境逻辑和晚期长周期制作。 |
| 2007 | UAL开放/维护Stanley Kubrick Archive。 | 档案成为研究其工作法的重要真源。 |
| 2012-2019 | 大型回顾展在多地巡展，包括LACMA、Design Museum等。 | Kubrick作为档案型作者被持续再评估。 |
| 2022 | BFI/Sight and Sound导演投票中 `2001` 位列导演榜第一。 | 电影人影响力仍在增强。 |
| 2026 | Yale Film Archive排映完整Kubrick回顾（35mm/16mm胶片）。 | 档案放映和胶片呈现仍是当代接受的重要路径。 |

## Latest Dynamics as of 2026-04-15

- Kubrick本人已于1999年去世，人物观点不会更新；需要更新的是档案开放、修复、放映、学术出版和家族/档案机构活动。
- UAL Stanley Kubrick Archive仍是最关键的研究入口，可预约查阅职业全阶段材料。
- 2026年Yale Film Archive的 `Kubrick's Maze` 回顾展显示其作品仍以胶片放映和完整回顾方式进入新一轮接受。
- 2022 BFI/Sight and Sound导演投票把 `2001: A Space Odyssey` 推到导演榜第一，是近年影响力的重要证据。
