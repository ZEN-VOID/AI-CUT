# 特效 模块说明

## 定位

- 本叶子负责只在普通画面手段不够时，补必要的特效或主观感知强化。
- 它默认的最佳答案通常是“暂时不需要”。
- 它不是“风格加成器”，而是最后一道补救闸门。

## 使用方法

- 先判断这里是不是梦境、记忆、异象或主观感知场景。
- 再判断普通画面手段和保守转场为什么不够；若还够用，就不要进入特效。
- 若需要特效，必须说清它服务的是情绪、时空、世界观还是感知。
- 输出时同时给出克制边界，避免特效扩张成主角。

## 具体创作方法

### 1. 先证明“非特效不可”

- 在写任何特效前，先反问：
  - 普通构图、光影、色彩、运镜、剪切真的都不够了吗。
  - 这里的问题真的是“看不清”还是“感受不到”。
  - 删掉特效后，这段是否仍能成立。
- 若删掉后仍成立，默认不写。

### 2. 只允许四类正当进入理由

#### 梦境 / 记忆

- 适用：现实层与回忆层、梦境层之间需要明确区隔。
- 目标：帮助观众识别感知层切换，而不是单纯制造朦胧感。

#### 主观感知强化

- 适用：眩晕、失真、恐惧、恍惚、时间拉伸等主观体验。
- 目标：把人物感知传给观众，而不是把画面做得更花。

#### 异象 / 世界异常

- 适用：世界观确有异常现象、不可用普通镜头语言解释的变化。
- 目标：让异常被感知为叙事事实，而不是风格贴纸。

#### 时空桥接

- 适用：跨时空、跨层级切换仅靠普通剪切会误读。
- 目标：解决理解门槛，而不是展示技术。

### 3. 一旦进入，也要先选“最轻特效”

- 先想轻微感知提示，再想明确特效表现。
- 可接受的写法方向：
  - 极轻模糊或虚化，提示意识漂移。
  - 短促闪白、闪黑或拉空，提示记忆断层。
  - 局部失真、轻度拖影，提示主观晕眩。
  - 轻量叠映或残影，提示层级重叠。
- 禁止默认进入大幅炫技、长时间占画面、喧宾夺主的效果。

### 4. 每个特效都要带“收口规则”

- 说明它何时出现。
- 说明它服务什么。
- 说明它到哪里为止。
- 没有收口规则的特效最容易扩张成另一套表演。

## 思维·执行节点

### N1 非特效排除测试

- 先列出普通画面手段为什么不够。
- 如果理由含糊，停止进入本叶子。

### N2 锁特效目的

- 只允许锁一个主目的：
  - 情绪
  - 时空
  - 世界观
  - 主观感知
- 目的一多，特效就容易失控。

### N3 选择最低强度实现

- 在“极轻提示 -> 明确可感知强化 -> 较强异常显化”里选最低够用档。
- 不要一上来就写满。

### N4 写克制边界

- 需要同时写清：
  - 触发条件
  - 生效时段
  - 停止位置
  - 不得抢走的人物/动作主线

### N5 删除自嗨成分

- 任何不能回指叙事必要性的描述都删。
- 任何只证明“很酷”的描述都删。

## 节点延展

### 可复用的保守描述方向

- “仅做极轻感知失真，提示人物主观状态变化。”
- “以短促层级切换提示记忆/现实边界，不延长停留。”
- “只在异常显现的瞬间给出强化，随后立即收回。”
- “特效只服务识别和感受，不替代动作与表演。”

### 应立即回退的信号

- 特效描述比人物动作还醒目。
- 特效一删，段落意思完全没变。
- 特效只是延长氛围，没有承担识别或桥接任务。
- 特效开始替代摄影、运镜或表演本该完成的工作。

## 失真与修正

- 若特效只是为了让画面更炫，说明没有叙事必要性。
- 若普通画面手段已经足够，却仍引入特效，说明进入时机错误。
- 若删掉特效后场景仍成立，就不该保留它。
- 若特效抢走人物和动作主线，立即回退到普通画面手段。
