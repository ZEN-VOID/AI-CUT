# 2-插入 模块说明

## 定位

- 本叶子负责把 `[分镜N 第X秒-第Y秒]` 标记自然埋入 `1-水月` prose。
- 它不负责重新决定镜头数量，也不负责输出抽离 prose 的分镜清单。

## 使用方法

- 优先在动作转折、视线切点和节奏停顿处插入标记。
- 若原句不顺，可以轻改 prose，但只能服务镜头显化，不能改事实含义。
- 输出的标记必须像正文的一部分，而不是贴在句子上的标签。

## 失真与修正

- 若读起来像机械断句，说明插入点没有服从动作和节奏。
- 若标记脱离了 `1-水月` 原句承接，说明没有真正回指原始信息。
- 若为了插入而重写了一大段剧情，说明越权了。
