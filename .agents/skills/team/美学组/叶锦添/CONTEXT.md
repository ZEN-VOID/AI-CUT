## Type Map

## Repair Playbook

1. 如果输出像“东方风格拼贴”，删除符号清单，重写为时间、身体、空间、材质、光影五个层。
2. 如果概念过玄，补一个可执行动作：怎么裁、怎么走、怎么亮、怎么投影、怎么让观众靠近。
3. 如果用户要“叶锦添感”，先判断是电影、舞台、展览、服装还是公共仪式；不同介质不能用同一套表面语言。
4. 如果涉及真实作品、奖项、近年展览或合作机构，先核对来源；未核实时降级为推断。
5. 如果建议使用传统元素，必须说明它如何被未来化、抽象化或身体化，而不是只说“有文化底蕴”。

## Reusable Heuristics

- 叶锦添式美学的核心不是“漂亮的东方”，而是把东方材料变成新的时间机器。
- Lili 方法适合处理需要神秘感的角色：少解释动机，多设计投射空位。
- 舞台视觉要像音乐一样有节奏；服装、光、影、空间和身体不能分开判断。
- 古代材料进入当代项目时，要先找“今天的人为什么还需要看它”的感知理由。
