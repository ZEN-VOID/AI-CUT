# 06 · 人物时间线

| 时间 | 事件 | 对导演视角的影响 |
|---|---|---|
| 1930 | 深作欣二出生于日本水户市 | 地方出身、战前成长，后来视角常站在中心叙事之外 |
| 1945 前后 | 十五岁时被动员到军需工厂，经历空袭、同龄人死亡和尸体处理 | 形成对国家命令、战争、死亡和权威叙述的根本不信任 |
| 1953 | 日本大学艺术学部毕业，进入东映东京摄影所 | 在商业片工业和片场系统中学习导演，不是学院作者起步 |
| 1961 | 以《风来坊探侦 赤い谷の惨劇》导演出道 | 进入现代动作和类型片路线 |
| 1964 | 《狼、猪、人》 | 早期代表作，将家庭/底层/帮派与战后生存压力连接 |
| 1970 | 《如果你年轻：愤怒》 | 青年理想在高速经济增长下破灭，显示其对“经济奇迹”背面持续敏感 |
| 1972 | 《军旗はためく下に》 | 反战与国家责任问题进入调查式结构 |
| 1973 | 《仁义なき戦い》开启五部曲 | 实录 yakuza 路线确立，重塑日本黑帮片 |
| 1975 | 《仁义の墓场》《县警对组织暴力》获蓝丝带导演奖 | 巩固其社会暴力、警政腐败、组织崩坏题材 |
| 1978 | 《柳生一族の阴谋》《宇宙からのメッセージ》 | 进入时代剧和科幻/冒险大片，显示跨类型工业能力 |
| 1980 | 《复活之日》 | 参与角川全球灾难大片路线，扩大制作规模 |
| 1982 | 《蒲田行进曲》 | 获电影旬报、每日电影竞赛、蓝丝带等导演奖，证明其并非只限黑帮片 |
| 1986 | 《火宅の人》 | 获日本电影学院奖导演奖/脚本奖 |
| 1994 | 《忠臣藏外传 四谷怪谈》 | 获日本电影学院奖导演/作品/脚本等认可，继续改造传统题材 |
| 1997 | 获紫绶褒章 | 行业和国家层面对其电影贡献的正式确认 |
| 2000 | 《大逃杀》 | 晚年把战争记忆、代际失信和制度暴力转化为黑暗寓言 |
| 2003-01-12 | 因癌症去世，《大逃杀 II》开拍初期由深作健太接手 | 遗作意义被固定为父子协作、代际传递和未完成的战斗 |
| 2019 | 日本国立电影资料馆举办大型回顾展 | 日本机构层面对其 61 部剧场作品的系统性再评价 |
| 2026 | Japan Society 继续放映《新仁义なき戦い 组长の首》等相关作品 | 其类型片遗产仍在国际电影机构语境中被重新放映和解释 |

## 最新动态（截至 2026-04-15）

- 深作欣二本人无新动态；可更新的是机构放映、修复和研究。
- 2026 年 Japan Society 的 Meiko Kaji 回顾中继续放映深作作品，说明其后续 yakuza 片仍被作为日本类型片历史的一部分重新观看。
