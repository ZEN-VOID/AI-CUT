# 06 人物时间线

## 关键节点

| 时间 | 事件 | 对导演系统的影响 |
|---|---|---|
| 1962 | David Leo Fincher 生于美国科罗拉多州丹佛。 | 美国郊区、广告影像、电视和电影工业构成早期文化背景。 |
| 1970s-1980s | 进入影像制作和视觉效果相关工作，曾在 ILM 等环境中学习技术流程。 | 形成对模型、合成、后期、画面验证和技术 workflow 的敏感。 |
| 1980s | 拍摄广告和音乐视频，后与 Propaganda Films 等商业影像体系相关。 | 节奏、图像控制、短时信息密度和明星/产品影像能力进入长片方法。 |
| 1992 | 长片处女作 `Alien 3`。 | 与大制片厂控制冲突的经验加强其后来对前期、合同和最终控制的重视。 |
| 1995 | `Se7en` 上映。 | 确立暗黑犯罪、残酷结尾、城市腐败和省略式暴力的作者标签。 |
| 1997 | `The Game`。 | 阴谋、控制、身份操纵和高概念机制继续发展。 |
| 1999 | `Fight Club`。 | 消费社会、男性身份、反体制幻想和不可靠叙述成为文化争议核心。 |
| 2002 | `Panic Room`。 | 封闭空间、数字摄影/VFX 预演和空间机关能力强化。 |
| 2007 | `Zodiac`。 | 程序、档案、执念和无法结案成为成熟方向；数字 workflow 进入关键阶段。 |
| 2008 | `The Curious Case of Benjamin Button`。 | 时间、年龄、面孔和隐形 VFX 的大规模整合。 |
| 2010 | `The Social Network`。 | 将科技创业、诉讼叙事、友谊破裂和资本权力压缩成高密度现代寓言；获金球最佳导演并获奥斯卡提名。 |
| 2011 | `The Girl with the Dragon Tattoo`。 | 北欧犯罪、创伤、调查程序和寒冷视觉系统结合。 |
| 2013 | `House of Cards` 首播，芬奇执导前两集并任执行制作。 | 进入 Netflix 原创剧集与流媒体作者工业；获艾美导演奖。 |
| 2014 | `Gone Girl`。 | 婚姻、媒体、表演人格和公众叙事互相吞噬。 |
| 2017-2019 | `Mindhunter`。 | FBI 行为科学、访谈、犯罪心理和制度化知识生产成为核心。 |
| 2020 | `Mank`。 | 黑白时期质感、好莱坞政治、父亲剧本和电影史记忆整合；获奥斯卡最佳导演提名。 |
| 2023 | `The Killer` 在 Netflix 发行。 | 职业杀手题材被降温为程序、自律神话、失败和全球化服务网络。 |
| 2025 | `Se7en` 4K 修复版进入公开宣传/放映语境。 | 早期残酷结尾和暗黑犯罪遗产被重新讨论。 |
| 2025 | 多家 trade 报道 Fincher 与 Netflix、Brad Pitt、Quentin Tarantino 编剧的 Cliff Booth 项目相关。 | 属开发/报道状态，未作为稳定完成作品写入；后续需实时核对。 |
