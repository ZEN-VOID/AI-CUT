# 编导字段投影桥接规范

本文件吸收 `.agents/skills/aigc/2-编导` 中与漫画剧本改编兼容的规范：声画配对、字段纯度、可见/可听/可执行、场景锚点稳定与质量优先级。它服务 `漫画剧本改编`，不把漫画 1 号改造成 AIGC `2-编导`。

## Boundary

可吸收：

- 声音文本与画面承托就近配对。
- 声音字段只写可听内容，画面字段只写可见内容。
- 心理、判断、恐惧、规则和系统信息必须投影成可画、可听或可执行材料。
- 场景锚点服务下游拆页和九刀流，不用剧情 beat 冒充场景。
- 质量提升必须服务下游可消费性，不得牺牲事实边界或核心卖点。

不可照搬：

- 不要求输出影视 slugline 或 `第N集.md`。
- 不要求逐字冻结所有来源对白；只有用户要求保真、现实来源或明确引用原句时才执行对白冻结。
- 不要求把漫画稿改成完整影视编导稿；`第N组.md` 仍是唯一主真源。
- 不提前替下游摄影/分镜写完整镜头方案；只保留漫画拆页和九刀流需要的视觉锚点。

## Manga Audio-Visual Pairing

当组稿中出现对白、内心独白、旁白、音效或系统提示时，必须能在同一小段附近找到对应画面承托。

| 声音/文本条目 | 漫画侧画面承托 |
| --- | --- |
| `对白（角色）` | `对白画面` 或正文中的表情、动作、距离、对手反应 |
| `内心独白（角色）` | `内心独白画面` 或身体反应、手、眼神、空间压迫 |
| `旁白（主体）` | `旁白画面` 或可见物、环境变化、动作推进 |
| `音效（来源）` | `音效画面` 或可见声源、不可见声源的空间承托 |
| `系统提示 / 规则文字` | `系统画面 / 规则显影 / 道具特写` |

最低规则：

- 声音行不塞动作描写。
- 画面承托不复述声音文本，而是给出读者能看见的一格。
- 音效必须像能放进字效层的声音本体，不写事件说明。
- 旁白必须带可见物或状态变化，避免只做抽象总结。

## Field Purity

### 可见

漫画格里能看见的内容：

- 身体、表情、站位、手势、眼神、距离。
- 道具、屏幕字、黑板字、系统字、纸张、衣物、门、灯光。
- 空间、天气、影子、构图关系、群体行为。

### 可听

漫画可用对白气泡、旁白框、字效或系统文本表达的内容：

- 角色对白、内心独白、旁白、系统提示。
- 脚步、铃声、撞击、尖叫、机械声、静默后的环境声。

### 可执行

角色、画师、下游提示词能执行的内容：

- 动作任务、表演任务、道具状态、空间运动、关系距离。

禁止把以下内容直接写成画面字段或漫画格提示：

- `他意识到`、`她觉得`、`没人知道`、`真正的规则`。
- `压抑感继续`、`关系破裂`、`命运翻转` 这类抽象结论。
- `规则测试开始`、`第二节课是课堂作业` 这类任务说明。

修复方式：

- 抽象概念 -> 转成可见物或规则显影。
- 心理判断 -> 转成内心独白、表情特写、手部动作或身体停顿。
- 因果解释 -> 分拆成动作、对手反应和下一格代价。
- 比喻结论 -> 转成构图关系、遮挡、留白或道具状态。

## Scene Anchor Policy

漫画 1 号不强制使用影视 slugline，但每组至少要有稳定场景锚点：

- 场所或空间范围。
- 时间、光线或气候状态。
- 本组核心视觉物件。
- 本组角色关系站位。
- 本组最贵的一格或停笔点。

同一地点、同一时间段内的多个 beat 不应伪装成多个场景；除非空间、时间、视角真源或冲突阶段发生实质变化。

## Dialogue Preservation Gradient

| source condition | rule |
| --- | --- |
| 用户要求保真或指定原句 | 对白逐字保留；引号内不混动作 |
| 现实新闻、纪实采访、真实公开发言 | 不伪造为真实原话；可改为旁白转述或虚构承载线 |
| 虚构故事且用户未要求保真 | 可重写对白以服务漫画节奏，但不得改变人物关系核和关键代价 |
| 解说漫兼容 | 旁白可改写为朗读友好的画面句，但必须仍可画 |

## Quality Priority

质量提升顺序：

`保核与事实边界 -> 可画 -> 声画清楚 -> 场景锚点稳定 -> 表演/构图可执行 -> 钩子与页势能`

不得为了“高级感”“电影感”牺牲：

- 现实事实边界。
- 虚构来源的人物关系核、情绪核、卖点核。
- `第N组.md` 作为唯一主真源。

## Good Pattern

```md
对白（苏红叶）：“各位同学，早上好。”
对白画面：苏红叶站在讲台前，教鞭末端的红珠停在桌沿；全班的椅脚同时停止摩擦。

内心独白（林寂）：“她发现了。”
内心独白画面：林寂低着头，指腹压住课本边角，校服后背被冷汗洇深。

音效（走廊外）：“咚……咚。”
音效画面：教室后门的玻璃轻轻震动，门缝里没有人影。
```

## Bad Pattern

- `动作画面：林寂意识到自己拿到了底牌。`
- `对白（某人）：“他沉默了。”`
- `旁白画面：命运的规则开始翻转。`
- `音效（铃声）：“八点四十分铃声。”`
- `视觉焦点：一种无法言说的压抑感。`
