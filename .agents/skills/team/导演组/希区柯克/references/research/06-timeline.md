# 06 人物时间线

| 时间 | 事件 | 对导演方法的影响 |
|---|---|---|
| 1899 | 出生于伦敦 Leytonstone | 东伦敦、天主教教育、警察/惩罚轶事进入后来的罪、恐惧和错误拘禁母题 |
| 1913-1918 | 工程、导航、绘图、广告相关训练与工作 | 形成图形设计、结构、广告传播和纸面规划意识 |
| 1920 | 进入 Famous Players-Lasky 伦敦分部做 silent film title cards | 从文字与画面关系进入电影工业 |
| 1922 | Famous Players 关闭英国分部后继续在 Islington 工作，承担美术、剪辑、助导、编剧等职责 | 获得完整工种视野 |
| 1925 | 完成首部导演长片 `The Pleasure Garden` | 正式进入导演身份 |
| 1927 | `The Lodger` | 错误指控、金发、公众窥视、视觉装置、cameo 等母题成形 |
| 1929 | `Blackmail` | 英国早期有声片重要实验，主观声音和 visual psychology 成形 |
| 1934 | `The Man Who Knew Too Much` | 国际阴谋、家庭、公共场所悬念和 set-piece 结构成熟 |
| 1935 | `The 39 Steps` | innocent man on the run、MacGuffin、episode film 模式成熟 |
| 1936 | `Sabotage` | 炸弹、儿童死亡和悬念释放边界成为自我反省案例 |
| 1938 | `The Lady Vanishes` | 英国时期商业、机智、悬念和政治暗影的高点 |
| 1939 | MoMA/Columbia lecture；受 Selznick 邀请赴美 | 公开阐述 springboard situation、pace、观众参与；进入 Hollywood |
| 1940 | `Rebecca` | 首部 Hollywood 片，获最佳影片；与 Selznick 制片体系协作 |
| 1943 | `Shadow of a Doubt` | 美国小镇日常下的邪恶成为核心模式 |
| 1944-1945 | 战时短片与 `Memory of the Camps` 技术顾问相关工作 | 现实恐怖与影像证据的边界进入经历 |
| 1946 | `Notorious` | 爱情、间谍、钥匙、长吻、阶梯等视觉/情绪控制高度成熟 |
| 1948 | `Rope` | 长镜头幻觉与舞台/电影边界实验 |
| 1951 | `Strangers on a Train` | 双重人物、交换罪行、魅力反派成熟 |
| 1954 | `Rear Window` | 观看伦理、观众窥视和空间限制达到高峰 |
| 1955 | 成为美国公民；`Alfred Hitchcock Presents` 开播 | 公众 persona 与导演品牌扩张 |
| 1956 | `The Wrong Man`、重拍 `The Man Who Knew Too Much` | 错误指控现实主义与明星版悬念并行 |
| 1958 | `Vertigo` | 男性凝视、迷恋、身份制造、形式失衡的晚期高峰 |
| 1959 | `North by Northwest` | chase movie、MacGuffin、明星身体与地标 set-piece 的经典综合 |
| 1960 | `Psycho` | 低成本、电视团队、结构背叛、碎片暴力、宣传控制 |
| 1963 | `The Birds` | 无解释威胁、环境性恐惧、事件电影；同时进入 Hedren 争议核心 |
| 1964 | `Marnie` | 性创伤、控制、心理解释与性别伦理争议加剧 |
| 1971 | 获 BAFTA Fellowship | 英国电影界终身成就确认 |
| 1972 | AFI Seminar roundtable | 系统阐述 mystery/suspense、Psycho、Vertigo、paper planning 等方法 |
| 1976 | `Family Plot` | 最后一部长片 |
| 1979 | 第 7 届 AFI Life Achievement Award honoree；获 knight 头衔 | 生涯贡献被美国和英国制度性确认 |
| 1980 | 4 月 29 日逝世于 Bel Air | 方法和形象进入电影教育、类型片和 auteur 传统 |
| 2012 | `Vertigo` 登顶 Sight & Sound critics poll | Hitchcock 作为严肃作者的地位再确认 |
| 2022 | `Vertigo` 在 Sight & Sound 新榜单仍居高位 | 其晚期形式和凝视议题继续影响当代电影 canon |
| 2024-2025 | `Becoming Hitchcock: The Legacy of Blackmail` 与 TCM 早期 Hitchcock 专题 | 英国时期根源和 Blackmail 方法继续被重新梳理 |
| 2026 | 无本人动态；机构展映、修复、课程和研究持续 | 使用 Skill 时应以稳定历史材料为准，核对最新展映/修复信息 |

## 最新动态判断

- Hitchcock 已于 1980 年逝世，因此“最新动态”只指回顾展、修复、出版、课程、研究或流媒体发行。
- 截至 2026-04-15，本次调研未发现必须改变其核心方法模型的 2026 新材料。
- 需要事实型回答时仍应检索最新展映、修复、版权和发行信息。
