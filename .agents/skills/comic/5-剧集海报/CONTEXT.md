## Repair Playbook

1. 先确认本集事实和角色边界，而不是先润色标题。
2. 若像角色海报，优先补 `representative_scene + foreground/background`，不要只加形容词。
3. 若标题无力，优先从本集冲突、秘密、代价和反转入口重写。
4. 若画面漂亮但不追更，说明钩子没锁住；回 `creative_core`。
5. 若 JSON 结构完整但无法继续生图，优先修 `prompt_package`，不要重写整份合同。
6. 若标题像普通章节名，先判断它缺的是“关系张力、身份错位、危机倒计时、情绪痛感”中的哪一类，而不是盲目加更狠的词。
7. 若标题点击感很强但海报承接不住，优先收窄标题到当前集真实代表性场面，不要让标题跑到下一集或结局去。
8. 先选分发类型再选标题模板：
   - 漫画推文：信息密度 + 搜索词 + 强冲突短句。
   - 解说漫：讲述感 + 结果前置 + 关系冲突。
   - 漫剧/短剧海报：身份、危机、反转、未完成目标。
9. 若用户说“标题位置不对”，先检查是不是把 `center` 当成了单轴；5 号技能默认是双轴居中，不接受只写水平居中。
10. 若用户说“不要脱离项目”，先检查有没有实际读取桥接包和九刀流 JSON；没读就是源层违规，不是审美分歧。
11. 若用户说“风格不一致”，先回 `style_inheritance`，不要直接改 prompt 词面。
12. 若用户说“这个画面不是本集最值钱的”，先回 `highlight_discovery`，重新比较候选高光点。
13. 若用户已经给了标题，先锁 `user_title_text -> hook_title.text`，不要先按模板重写一版再回头覆盖。

## Reusable Heuristics

- 剧集海报最值钱的不是“人物好看”，而是“一张图就把本集最贵的悬念钉住”。
- 如果主体正确但海报仍发空，通常不是角色问题，而是缺少能制造压力的前景物或背景环境。
- `第N集` 是索引，不是卖点；真正卖点必须落在大标题和代表性画面上。
- 海报标题可以更传播，但最好是“从本集事实里抽尖”，而不是凭空制造另一个故事。
- 5 号技能里的“默认居中”一律解释为“水平+垂直双居中”，不是只在画面宽度中线居中。
- 用户显式给标题时，最稳做法不是“参考一下”，而是直接优先采用，再只做最小必要的排版压缩。
- 5 号技能不允许脱离 `projects/comic/<项目名>` 上游真源独立创作；至少要继承桥接包和九刀流 JSON 的角色、场景与风格信息。
- 多集漫画项目中，5 号技能读取上游时最稳规则是：先按 `episode_number` 命中 `第N集-page-group-*.json` 与 `第N集-思考过程摘要.md`，只有 legacy 单文件项目才回退到 `第N集-nine_blade_comic_prompts.json` 或无前缀文件名。
- `prompt_package.positive_prompt` 的门禁应检查它是否足够具体、可直接继续生图，而不是强绑某个英文关键词；中文 prompt 只要结构完整，也应被视为合法下游输入。
- 高光点发现的稳定做法不是“第一眼看上哪张就用哪张”，而是先列候选，再按命题价值和传播性筛掉表面漂亮但不值钱的镜头。
- 好海报的色彩不是装饰，它应该替当前情绪服务：威胁、诱惑、逃亡、崩坏、秘密，各自需要不同光线策略。
- 漫画推文标题最稳的抓手不是“诗意”，而是“关键词可搜 + 冲突可见 + 情绪可点”。
- 解说漫标题最稳的抓手不是“设定全”，而是“先把结论或最痛那一下说出来”，让观众愿意听你往下讲。
- 漫剧/短剧海报标题最稳的抓手不是“完整句”，而是“像弹幕里会被刷屏的一刀”：短、狠、能指向关系或危机。
- 2025 的外部信号显示，短剧爆款标题与题材并没有完全收敛到单一霸总套路，更依赖高情绪爽点、关系错位、IP 系列化和人物共鸣；因此标题库不能只存“总裁/离婚/替嫁”，也要覆盖现实流、学霸女主、系列修仙、长辈重生等新爆点。

### 常见模板库

#### A. 漫画推文标题模板

这些模板优先服务“图文分发 / 搜索流 / 转发流”。

1. `【人群词】+【结果】+原来是【关键冲突】`
   - 例：`真千金翻盘后，最先慌的竟然是她哥`
2. `【时间/集数】+【危机】+【主角动作】`
   - 例：`第7集她被赶出门后，终于不忍了`
3. `【设定词】+【结果前置】+【反转】`
   - 例：`重生太奶奶一出手，全家都不敢装了`
4. `【关系】+【极端动作】+【未说完结果】`
   - 例：`前夫把离婚协议甩过来后，他后悔了`
5. `【身份词】+【被低估】+【亮明底牌】`
   - 例：`所有人都以为她只是侍女，直到她亮出令牌`
6. `【痛点】+【反击】+【结果】`
   - 例：`被全家当佣人使唤后，她直接考了第一`
7. `【数字/阶段】+【目标】+【下一步危机】`
   - 例：`她拿到京市户口这天，更大的麻烦来了`
8. `【题材入口词】+【情绪钩子】+【人群词】`
   - 例：`七零重生后，她不再给任何人当软柿子`

#### B. 解说漫标题模板

这些模板优先服务“口播开场 / 讲述流 / 听感先行”。

1. `她以为【表面事实】，其实【隐藏真相】`
   - 例：`她以为自己只是替身，其实所有人都在骗她`
2. `这一集最狠的，不是【事件】，是【关系动作】`
   - 例：`这一集最狠的，不是离婚，是他亲手把她推出门`
3. `他明明可以【做A】，却偏要【做B】`
   - 例：`他明明可以亮身份，却偏要继续装穷`
4. `所有人都在等【结果】，只有她先看见了【危险】`
   - 例：`所有人都在等她低头，只有她先看见了门后的血`
5. `你以为这是【老套路】，其实它先刀在【新位置】`
   - 例：`你以为这只是豪门虐恋，其实它先刀在母女关系上`
6. `这一集真正的钩子，是【未完成目标】`
   - 例：`这一集真正的钩子，是她终于决定不回头`
7. `从【受辱/误会】到【反击/亮牌】，她只用了【动作】`
   - 例：`从被当众羞辱到反手收网，她只用了一个电话`
8. `最可怕的不是【眼前危机】，而是【隐藏后果】`
   - 例：`最可怕的不是门被推开，而是里面的人早就知道她会来`

### 快速选型法

1. 如果目标是 `漫画推文`：
   - 先塞进 `题材入口词/人群词`
   - 再加 `结果前置` 或 `反转`
   - 最后补一个可搜索关键词
2. 如果目标是 `解说漫`：
   - 先说“最痛/最怪/最危险”的那一下
   - 再补关系或身份错位
   - 尽量让标题读起来像开场第一句
3. 如果目标是 `海报标题`：
   - 保持短
   - 先保危机，再保关系
   - 标题必须能被海报画面承接

### 审核清单

- 这条标题能不能回指本集真实场面？
- 这条标题更像“搜索词”、更像“口播句”，还是更像“海报刀口”？
- 去掉主角名后，标题还有没有冲突？
- 标题是不是已经把结局说完了？
- 如果把这条标题放到封面中央，会不会太长、太散、太像总结？
