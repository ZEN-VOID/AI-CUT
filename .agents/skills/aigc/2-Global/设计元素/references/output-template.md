# 设计元素输出模板说明

本文件解释 `templates/设计元素.template.md` 的写作目标与章节作用。

## Section Intent

1. `项目范围与输入锁定`
   - 说明本轮覆盖哪些剧集 / 阶段，以及主辅输入真源。
2. `世界观与时代判型`
   - 给出时代路由、dominant era、研究/借鉴方法与禁区。
3. `上游 grouped script 证据摘要`
   - 只写真正影响服装和建筑场景的叙事信号、社会秩序、文明压力与阶段变化。
4. `初始化附加约束`
   - 只写来自 `0-Init` 的世界观、目标受众、禁区与长期母题。
5. `服装设计定调`
   - 输出设计主陈述、词汇母句、研究依据、禁借项、阶段变化提醒。
6. `建筑场景设计定调`
   - 输出设计主陈述、词汇母句、研究依据、禁借项、阶段变化提醒。
7. `阶段成长与变化矩阵`
   - 明确不同阶段的设计变化方向和不变量。
8. `时代专属 guardrails`
   - 只写最关键的不可越界项。
9. `下游 handoff 与使用边界`
   - 明确 `4-Design/服装`、`4-Design/场景` 如何继承，以及本文件不直接替代哪些对象级产物。

## Writing Reminders

- 先写“为什么这样定调”，再写“长什么样”。
- 词汇母句要能被下游直接剪切使用。
- 禁区要具体到“什么不能出现”，而不是泛泛而谈。
- 阶段变化矩阵必须能说明“何时收束、何时扩张、何时异化”。
