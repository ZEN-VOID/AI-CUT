# 编剧手册

## Purpose

本文件吸收原 `1-水月` 及其角色/情绪细则，供单技能 `3-Detail` 根包直接调用。
所有条目都只能在 `1-分镜构图` 已先行锁定 `分镜数 / 时间 / 剧本正文 / 主体锚定 / 分镜构图` 之后再进入，绝不反向改骨架。

## 1. 角色表现

### 核心问题

- 谁在承压，谁在主动
- 人物最先哪里露馅
- 这一拍的行为目的是什么
- 若存在对白，谁掌话头、谁吞话、谁截断、谁失口

### 三叶子吸收

#### 内心戏

- 先写角色表面上在维持什么，再写身体哪一处先说真话。
- 微动作优先顺序：
  - 眼神
  - 呼吸
  - 嘴角/下颌
  - 肩颈/背部
  - 手部
- 露馅点必须推动下一拍，不允许只是漂亮细节。

#### 动作戏

- 动作必须具备：
  - 爆发点
  - 攻守关系
  - 阻断点
  - 角色化出手门路
  - 改局后果
- 默认审美标尺不是“热闹”，而是“招路清、攻守明、劲路准、见人”。
- 动作不得写成镜头术语，也不得沦为任何人都能套用的通用动作模板。

#### 对话戏

- 不复述台词内容，专写：
  - 抢话
  - 吞话
  - 截断
  - 改口
  - 气口
  - 断句
  - 压强几何

### 写法要求

- 先判主链，再判辅链，不要三线平均发力。
- `角色表现` 最终应像一个可直接写入 JSON 的字段对象，而不是人物小传。

## 2. 氛围表现

### 核心问题

- 这股场感到底由什么具体条件产生
- 空间如何参与叙事
- 情绪温度如何被景物显影
- 哪个空间结构最值得被记住

### 三叶子吸收

#### 层次

- 先回答：
  - 近处先被感知的是什么
  - 中段承动作的是什么
  - 远处回声是什么
- 前景/中景/背景必须各有叙事职责，不能只是摆件清单。

#### 意境

- 只锁一个主温度，不并列写“冷、悲、空、悬、美”。
- 必须有一个承情物象：
  - 风
  - 雾
  - 水汽
  - 灯色
  - 回声
  - 距离感
- 情与景必须互相沾染，不允许空抒情。

#### 空间诗学

- 先判空间此刻是在：
  - 压人
  - 托人
  - 见证
  - 包围
- 再选一个 `poetic_anchor`：
  - 低梁
  - 门洞
  - 长廊
  - 台阶
  - 立柱
  - 空庭

### 写法要求

- 先有空间支架，再有情绪温度，再有诗性记忆点。
- 一旦空间不参与叙事，就不算成立。
