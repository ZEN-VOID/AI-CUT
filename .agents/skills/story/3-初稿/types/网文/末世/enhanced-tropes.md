# 末世题材增强包

> 补充维度：套路拆解 / 情节模式 / 人物原型 / 反套路策略 / 质量过滤

## 一、核心套路与行规

### 1.1 末世设定变体
| 类型 | 核心威胁 | 叙事焦点 |
|------|----------|----------|
| 丧尸末世 | 感染者/变异体 | 生存+人性+重建 |
| 天灾末世 | 极寒/极热/洪水/陨石 | 资源争夺+秩序重建 |
| 病毒末世 | 生化危机/瘟疫 | 隔离+疫苗+阴谋 |
| 系统末世 | 全球系统化/游戏化 | 规则适应+升级+对抗 |
| 异界入侵 | 异族/怪物入侵 | 战争+联盟+进化 |

### 1.2 末世叙事核心
- **生存是第一法则**：篇幅的30%应服务于生存压力（物资/安全/疾病/信息）
- **人性是终极舞台**：末世真正的威胁不是丧尸/天灾，而是在极端情况下人的选择
- **希望是必要元素**：纯粹的绝望会让读者逃离。在黑暗中埋下少量但真实的希望

### 1.3 常见套路陷阱
- 主角天然知道所有末世知识（全能型主角）
- 道德困境过于模板（"救不救陌生人"）
- 反派全是纯粹恶人（缺乏末世背景下的道德灰度）
- 安全区/基地段千篇一律（永远是争权夺利）

## 二、情节模式与节奏规则

### 2.1 末世篇章结构
```
灾难爆发期(1-2卷)：末日降临→逃生→失去重要的人/物→找到初期的庇护所
动荡求生期(3-5卷)：资源竞争→遭遇其他幸存者→形成小团队→对抗区域威胁
秩序重建期(6-8卷)：建立/加入基地→势力博弈→更大的外部威胁→人性考验
文明复兴期(9+卷)：安全区域扩大→旧世界真相揭露→新文明形态的探索
```

### 2.2 节奏铁律
- 开篇必须有冲击力的末世降临场景（前3章内）
- 安全场景不超过连续2章——舒适区制造麻木
- 每一卷至少有一场牵涉人员生死的大型危机
- 在绝望中每3-5章撒一点希望（新盟友/新发现/新资源/新可能）

### 2.3 爽点设计
| 爽点类型 | 末世实现 | 注意事项 |
|----------|----------|----------|
| 绝境逆转 | 几乎不可能的情况下反杀/逃生 | 代价必须真实 |
| 资源获得 | 发现物资/安全区/强力能力 | 获得过程有危险 |
| 联盟建立 | 收服/联合强力盟友 | 信任建立有过程 |
| 复仇回归 | 被抛弃/背叛后强势归来 | 情感铺垫要充分 |

## 三、反套路策略

### 3.1 反"末世百科全书"
- 主角不是末世上的一切都懂——对未知的恐惧才是好末世的开端
- 信息的逐步获得和错误纠正比全知更有吸引力
- 末世规则的发现过程就是剧情本身

### 3.2 反"道德完美主角"
- 在极端环境中，不完美、有私心但保持底线的主角比圣人更有真实感
- 主角的某些选择可以"错"——现实中没人能在末世做全对的选择
- 让读者看到主角的道德困境和代价，而不是"我全都救到了"

### 3.3 反"丧尸只是靶子"
- 如果是丧尸末世，丧尸不能只是站桩靶子
- 变异、进化、不同类型的威胁让怪物保持压迫感
- 丧尸/怪物群的设计考虑战术层面——数量、速度、特殊能力

## 四、质量过滤规则

### 4.1 末世氛围检查
- [ ] 前3章建立了足够的末世压迫感
- [ ] 资源稀缺性被持续呈现（不只口头说）
- [ ] 安全场景不过度（无连续3章以上完全安全）
- [ ] 人性灰度被展示（不全是纯好/纯坏）

### 4.2 AI腔高发区警示
- 末世描述过度依赖"荒凉/破败/死寂"等套词
- 丧尸/怪物战斗流水账化
- 幸存者心理状态被过度解释而非呈现

### 4.3 失败码
| 失败码 | 症状 | 返工目标 |
|--------|------|----------|
| `FAIL-APOC-ATMO` | 末世氛围不足 | 增加感官描写和资源压力呈现 |
| `FAIL-APOC-MONO` | 反派/幸存者单调模板化 | 增加人性灰度和复合动机 |
| `FAIL-APOC-HOPE` | 连续多章完全绝望 | 埋入微量希望线索 |
