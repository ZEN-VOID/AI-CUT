# 电影学院派知识库

## 1. 定位

这不是一个泛电影资料夹，而是服务于 `.codex/skills/aigc2026/2-导演` 评审工作的电影学院派知识库。

它的核心职责只有两件事：

1. 提供可被直接引用的电影知识内容模块。
2. 让评审意见拥有可回溯的学院依据与通用影视语法依据。

评审规则本身不再放在这里，而是内化到：

- `/Volumes/AIGC/AIGC-ZEN-VOID/.codex/agents/影评人/电影学院派.md`

知识库在这里不承担：

- 输出 schema
- 字段落点矩阵
- 使用协议
- 改写模板
- 评审禁忌

它现在只承担：

- 导演方法知识
- 分镜与镜头知识
- 电影摄影知识
- 原始资料洗涤与索引

## 2. 可信性

- 学院证据可回溯。
- 通用影视语法可解释。

## 3. 适用边界

- 适用：
  - 作为 `电影学院派` Agent 的被引用知识源。
  - 作为 `2-导演` 阶段评审时的知识依据模块。
- 不适用：
  - 作为 Agent 的行为规则仓。
  - 作为评审输出模板仓。

## 4. 目录说明

- `导演手册/`
  - 项目级导演方法、戏剧单元、调度原则、对话场景与执行策略。
- `分镜脚本/`
  - 镜头语法、镜头设计、镜头调度、分镜级镜头技术。
- `电影摄影/`
  - 光影、色彩、镜头、摄影风格与摄影执行知识。

## 5. 使用方式

- 当问题是“为什么这样组织”，优先读 `导演手册/`
- 当问题是“镜头怎么落”，优先读 `分镜脚本/`
- 当问题是“光色镜头如何支撑”，优先读 `电影摄影/`
