# 06-timeline｜川尻善昭：人物时间线与近期动态

## 基本信息

- 姓名：川尻善昭 / Yoshiaki Kawajiri
- 出生：1950 年 11 月 18 日，日本神奈川县横滨市。
- 职业：动画导演、编剧、分镜师、角色设计、动画师。
- 代表作：《妖兽都市》《迷宫物语：走る男》《獣兵衛忍風帖 / Ninja Scroll》《Vampire Hunter D: Bloodlust》《The Animatrix: Program》《Highlander: The Search for Vengeance》。

## 时间线

| 时间 | 事件 | 思维影响 |
|---|---|---|
| 1950 | 出生于横滨 | 城市、港口、战后流行文化与西部片/忍者漫画共同进入早期想象 |
| 1968-1972 | 高中毕业后进入虫プロ，从事动画工作 | 进入手绘动画工业基础训练 |
| 1972 以后 | 参与 Madhouse 创立期工作 | 形成长期 Madhouse 制作网络 |
| 1984 | 《SF新世纪Lensman》导演出道 | 从动画师/分镜进入长片导演 |
| 1987 | 《迷宫物语：走る男》与《妖兽都市》 | 确立高速、黑暗、身体恐怖、成人向类型风格 |
| 1988-1991 | 《魔界都市新宿》《Goku Midnight Eye》《Cyber City Oedo 808》 | 黑色都市、硬汉、超能/赛博动作持续成形 |
| 1993 | 自编自导原创《獣兵衛忍風帖 / Ninja Scroll》 | 将忍者漫画、山田风太郎、骗术戏、西部片孤狼和成人动画压成全球入口作品 |
| 2000 | 《Vampire Hunter D: Bloodlust》 | 哥特荒野、怪物小队、动作 setpiece 与浪漫悲剧结合 |
| 2001 | TV 动画《X》导演 | 长篇系列导演经验，同时影响其参与《Animatrix》的工作负荷 |
| 2002-2003 | 《The Animatrix》：导演《Program》，编剧《Program》《World Record》 | 与 Wachowski 合作，验证个人风格可进入全球科幻 IP |
| 2007 | 《Highlander: The Search for Vengeance》导演 | 再次处理海外 IP 与日式动作动画融合 |
| 2010s | 大量参与分镜/key animation，如《Chihayafuru》《One Punch Man》等 | 从全片导演转向分镜型工匠角色 |
| 2014 | 导演 30 周年特集上映，《Ninja Scroll BURST》相关活动 | 作品被回顾为海外成人 anime 入口之一 |
| 2024 | 《Ninja Scroll》30 周年北美重映，CBR 发表访谈 | 本人回顾旧作、重制可能、现代敏感场景、分镜与体力问题 |
