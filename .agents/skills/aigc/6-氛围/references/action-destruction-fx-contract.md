# Action Destruction FX Contract

本细则用于 `aigc/6-氛围` 中“动作破坏点效果强化”的标准化执行。它只展开 `SKILL.md` 已授权的氛围/现场物理特效层，不改写剧情事实、动作胜负、武器设定、角色能力或 `3-美学` 真源。

## Scope

适用触发：

- 上游表演稿出现冷兵器、徒手格斗、追逐、坍塌、撞击、爆破余波、风浪、船战、室内打斗或其他会改变现场材质状态的动作点。
- 用户要求强化 `动作破坏点`、`破坏感`、`爆点`、`剑风`、`枪风`、`链镰`、`飞剑`、`断链余劲`、`湿泥`、`尘土`、`石粉`、`木屑`、`水汽` 等效果。
- 既有 `氛围画面` 显得生硬、轻飘、只写气氛不写物理后果，或破坏效果越界成法术/现代 CG 光效。

非目标：

- 不创造新的招式、法术、异能、爆炸物、现代武器、机关或额外剧情事件。
- 不输出安全爆破施工方案、设备清单、VFX 参数或图像/视频 prompt。
- 不把所有动作点都强化成爆破点；只处理具有动作强度、节奏爆发或环境承托价值的触发点。

## Source And Style Fit

每个动作破坏点必须同时满足四个来源条件：

| source_layer | requirement |
| --- | --- |
| `action_source` | 原稿已有动作、碰撞、追逐、坍塌、武器挥扫、余劲或环境冲击。 |
| `material_source` | 场景中已有可被影响的石阶、大石、树木、树枝、林地、船板、窗纸、桌案、瓦片、水面、泥地、尘土等材质。 |
| `style_source` | 破坏密度、颗粒度和镜头质感符合 `3-美学` 的画面基调、角色风格、场景风格及项目 `MEMORY.md`。 |
| `boundary_source` | 能解释为物理武器、身体力量、内力外化、风压、水汽、现场碰撞或既有环境余波；不得变成无源法术、激光、能量盾、现代 CG 发光武器秀。 |

## Cold Weapon Action Palette

武侠/冷兵器场景优先按“动作源 -> 受击材质 -> 爆点材料 -> 边界句法”组合：

| action_source | target | effect | boundary_note |
| --- | --- | --- | --- |
| `白刃剑风/刀风` | 石阶、树枝、衣摆、水汽、地面碎石 | 刮出白痕、震开石粉、切断枝梢、劈散雾线、带起湿泥 | 可写刃风、风压、余劲；不写剑光实体化、光刃、法术轨迹。 |
| `枪风/枪尾顿地` | 石阶、青砖、木栏、林地、积水 | 闷响、裂缝、泥水外翻、碎木飞溅、灰尘从缝里顶出 | 强调重、直、沉和短促爆发；不写空气炮或能量冲击波。 |
| `链镰/铁链拖扫` | 树根、石角、门框、地面、枝条 | 刮出火星、树皮炸裂、石角崩白、湿木屑飞散、泥沟翻开 | 链的破坏应有路径和重量，避免像自动追踪法器。 |
| `飞剑/掷剑/抛剑` | 水汽、碎石、木桩、瓦片、布幔 | 空气被切开、碎石二次崩粉、木屑短促散开 | 可写高速冷兵器和风压，不写御剑术、悬空自转、灵光。 |
| `断链余劲/兵器脱手` | 山脊、石面、林地、甲片、碎枝 | 连续浅坑、泥水和石粉爆开、断枝被抽飞、火星短闪 | 余劲要落在现场材质上，结束后回到人物动作和静物余波。 |
| `兵器相撞` | 刀背、枪刃、链扣、护甲、石壁 | 金属火星、震落尘土、回声撞墙、火点迅速熄灭 | 火星短、硬、散；不得形成持续发光边缘。 |

## Construction Pattern

合格的 `氛围画面` 动作破坏句通常包含：

1. `time_anchor`：在某个动作前、同时、之后或余波中。
2. `cause`：剑风、枪风、链镰拖扫、身体撞击、坍塌余波、浪头拍击等源稿已有力量。
3. `target`：明确受影响的现场材质。
4. `material_response`：湿泥、尘土、石粉、木屑、水汽、火星、碎玻璃、灰烬等具体反应。
5. `rhythm`：短促爆点、连续抽击、先闷后爆、余声推远、静物落定等节奏。
6. `boundary`：必要时用“不发光”“只留下白痕/火星/水汽裂缝”等方式把效果收回物理边界。

示例结构：

```text
氛围画面：枪尾落地的同时，石阶先闷响一声，湿泥和白色石粉从裂缝里顶出来，像几个预埋小爆点沿台阶边缘短促炸开。
```

## Hard Prohibitions

- 不写“灵光、仙气、法阵、神力、能量波、激光刃、光轨、护盾、粒子特效、CG 光效”等会把风格推向修仙法术或现代光效秀的表达，除非上游和 `3-美学` 明确允许。
- 不让破坏结果改变剧情胜负、人物受伤程度、场景可通行性或道具状态，除非上游已发生。
- 不把没有硬材质的场景强写成石粉木屑爆破；先确认场景材质。
- 不用同一个“爆开/炸起/卷起”句式连续覆盖整场；同场要有闷响、刮痕、崩角、泥水外翻、余声、静默等变化。
- 不以“港式爆破感”作为正文元说明；正文只能写观众看到/听到/感到的材质反应。

## Report Evidence Requirements

当本细则被触发，执行报告必须记录 `Action Destruction FX Map`，至少包含：

| field | content |
| --- | --- |
| `source_anchor` | 对应原稿动作/环境/音效锚点。 |
| `action_source` | 白刃、枪风、链镰、飞剑、撞击、坍塌、浪头等力量来源。 |
| `target_material` | 被影响的石、木、泥、水、金属、玻璃、布、纸等材质。 |
| `effect_material` | 石粉、湿泥、尘土、木屑、水汽、火星、灰烬等反应材料。 |
| `style_fit` | 对应 `3-美学` 或项目 `MEMORY.md` 的风格继承依据。 |
| `boundary_check` | 如何避免修仙法术、现代 CG 光效、无源爆炸或剧情越权。 |

审查时若发现动作破坏点过弱、无材质、无来源、过度发光、法术化、现代 CG 化或破坏结果改变剧情，触发 `FAIL-ATM-DESTRUCTION-FX`，回到 `N3/N4/N5` 最小修复。
