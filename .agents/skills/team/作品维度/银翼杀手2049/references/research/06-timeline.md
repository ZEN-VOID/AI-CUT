# 06 · 时间线

| 时间 | 事件 | 说明 |
| --- | --- | --- |
| 1968 | Philip K. Dick 出版《仿生人会梦见电子羊吗？》 | empathy / inhuman 问题成为系列思想起点 |
| 1982 | 《银翼杀手》上映 | noir + 赛博未来 + 复制人哲学母体形成 |
| 2011-2014 | 续作开发成形，Hampton Fancher 回归，后由 Michael Green 完成剧本演进 | 续作明确走“延续世界、重写 DNA”路线 |
| 2016 | 完成主要拍摄与前期世界设计 | Deakins / Gassner / Villeneuve 的世界 bible 落地 |
| 2017 | 《银翼杀手2049》上映 | 片长 164 分钟；Ryan Gosling 饰演 K，Harrison Ford 回归 |
| 2018 | 奥斯卡最佳摄影、最佳视觉效果等获奖 | 作品的视听语言被主流奖项确认 |
| 2022 | AFI Silver 等机构继续放映 | 说明其已进入 repertory / contemporary classic 语境 |
