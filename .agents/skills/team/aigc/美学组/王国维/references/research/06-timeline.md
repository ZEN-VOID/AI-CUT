# 06 时间线

| 时间 | 事件 | 对后续美学视角的影响 |
|---|---|---|
| 1877 | 生于浙江海宁 | 传统文化底盘与江南审美经验 |
| 1890s | 读史、治举业、接触上海新学环境 | 从旧学走向现代学问的入口 |
| 1898-1900 | 在上海时务报馆、东文学社学习日文与西学 | 中西交界的认知框架形成 |
| 1901 | 赴日本，原拟修理学，因病归国 | 现实受阻后转向更内在的问题 |
| 1902-1904 | 密集研读哲学，尤其康德、叔本华、尼采 | 早期哲学与美学底盘建立 |
| 1904 | 写《红楼梦评论》《叔本华与尼采》 | 悲剧美学与生命问题意识集中爆发 |
| 1906-1907 | 写《文学小言》《论教育之宗旨》《古雅之在美学上之位置》 | 文学总论、美育、古雅概念形成 |
| 1908-1910 | 《人间词话》陆续刊发 | “境界说”成熟，成为核心美学名片 |
| 1911 | 辛亥后随罗振玉赴日 | 人生与文化立场进入更剧烈张力 |
| 1913 | 《宋元戏曲史》完成 | 从美学判断转入系统戏曲史建构 |
| 1910s | 甲骨、金文、殷周制度、敦煌等研究推进 | 形成更硬的文献-证据方法 |
| 1917 | 《殷周制度论》 | 学术方法论与历史判断能力达到高峰 |
| 1925 | 任清华研究院国学门导师 | 学术人格进入制度传承层 |
| 1927 | 自沉昆明湖 | 留下极重的文化与人格象征 |

## 最新动态（后世研究与纪念）

- 2010 年起，清华持续设立“王国维学术讲座”。
- 2017 年清华举办“独上高楼·王国维诞辰 140 周年纪念展”。
- 2025 年清华人文学院仍发布王国维学术讲座课程预告，显示其学术象征继续在场。
- 截至 2026-04-18，王国维的“最新动态”主要体现为讲座、纪念、全集整理与新论文，而非人物本身的新材料爆发。
