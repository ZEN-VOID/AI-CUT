# 05 决策记录与行动

## 决策 1：把篮球和漫画绑成一个身份

来源：Publishers Weekly 2007  
可信度：高

### 决策

- 没有把“喜欢篮球”和“画漫画”分开处理，而是决定用两者共同构成自己的创作位置。

### 影响

- `SLAM DUNK` 不是“找个题材来画”，而是把自己最真实的身体经验转换成叙事结构。

## 决策 2：用球员视点而不是观众视点画比赛

来源：Publishers Weekly 2007  
可信度：高

### 决策

- 回看《SLAM DUNK》时，井上明确把“从球员出发”当成自己与同类作品的差异点。

### 影响

- 这不仅改变了镜头感，也改变了人物关系写法。
- 比赛不再只是信息输送，而是身体体验与内心压力。

## 决策 3：看到轮椅篮球后，去做现实接触而不是停在感动

来源：Publishers Weekly 2007  
可信度：高

### 决策

- 被电视中的轮椅篮球震动后，他没有停在“这题材很有意义”，而是去采访、接触、结识球员。

### 影响

- `REAL` 的底盘因此不是概念正确，而是现场经验支撑。

## 决策 4：建立 `スラムダンク奨学金`

来源：2017 官方随笔、公开项目脉络  
可信度：高

### 决策

- 井上把对篮球的回馈落成一套面向海外留学与成长路径的制度，而不是一次性公益活动。

### 动机

- 他看到日本篮球长期缺少“通向强国”的桥梁，很多球员知道要出去，却没有路。

### 影响

- 这说明他的体育观非常“路径型”：不满足于热爱、鼓励和口号，而要搭制度桥。

## 决策 5：对《SLAM DUNK》续篇保持克制

来源：官方 FAQ  
可信度：高

### 决策

- 不因为市场期待就轻启续篇，而要等“自然想画”的时机。

### 影响

- 这是他对人物生命与作者诚实度的保护。
- 同时也解释了为什么他愿意用“换视点的电影”而不是“直接画第二部”来回到这个世界。

## 决策 6：接受电影化，但必须由现在的自己重写

来源：官方相关访谈摘要 + 官方随笔 `THE FIRST`  
可信度：中高

### 决策

- 井上一开始对电影化并不积极，后来接受，是因为确认自己有新的视点和新的生命经验可以放进《SLAM DUNK》。

### 影响

- `THE FIRST SLAM DUNK` 的成立逻辑，不是 IP 再开发，而是作者更新后的再进入。

## 决策 7：把漫画推进到空间、展览和纸面之外

来源：`最后的漫画展`、`OTHER HAND`、`pepita`、`円空を旅する`  
可信度：高

### 决策

- 不把自己锁在连载系统里，而让作品进入展览、旅途、建筑、雕塑与现场绘画。

### 影响

- 这让他的图像判断天然带有空间感和对象研究能力。

## 决策 8：继续把 `REAL` 放在当下创作重心

来源：2024 / 2026 官方 topics  
可信度：高

### 决策

- 在 `THE FIRST SLAM DUNK` 大成功后，官方最新持续更新反而集中在 `REAL` 16、17 和联名。

### 影响

- 说明井上当前的作者重心并没有被单一商业成功完全带走。
- `REAL` 仍是理解“现在的井上雄彦”的关键窗口。
