## 附录A：场景情绪声音表（按朝代/环境分类）

> **使用原则**：环境声是表演承托，不只是背景。使用时需回指当前场景的具体情绪压力源。

### A1. 古代/历史场景

| 情绪类型 | 室内（厅/堂/房） | 室外（街/路/野） | 特殊空间 |
|---------|----------------|-----------------|---------|
| **离别** | 门轴轻响、衣料摩擦声、烛火噼啪、漏壶滴答 | 远处马蹄声、脚步渐远、风吹草动、鸦鸣 | 渡口：橹声、水波拍岸；驿站：更鼓声 |
| **紧张** | 更漏声、烛芯爆炭、低频嗡鸣（虫鸣）、纸张翻动 | 远处更鼓、风声忽变、脚步忽远忽近 | 战场：金属碰撞、兵器破空、马蹄纷乱 |
| **孤独** | 空堂回声、烛影摇曳、滴水穿石、风穿窗隙 | 夜风呼啸、旷野虫鸣、无人的街道 | 牢房：铁链轻响、远处隐约人声 |
| **希望** | 檐下燕语、晨鸡初唱、窗纸透光 | 远处孩童嬉笑、牧笛声、春风拂柳 | 寺庙：钟声余韵 |
| **焦躁** | 棋子敲桌、茶水沸腾、衣袖摩擦 | 街市喧嚣、人声嘈杂、车马交错 | 考场：笔尖沙沙、翻卷声 |
| **压迫** | 群体忽然安静、杯底磕桌、金属轻响、呼吸被压低 | 忽然鸦雀无声、远处传来肃穆脚步 | 殿上：玉磬余音、珠帘碰撞 |

| 情绪类型 | 室内（教室/办公室/家） | 街道/公共空间 | 特殊空间 |
|---------|---------------------|--------------|---------|
| **离别** | 门锁咔哒、行李箱轮声、电梯下行提示音 | 站台广播、列车渐远、引擎熄火 | 机场：登机广播、行李传送带 |

| **孤独** | 冰箱低频电流、挂钟滴答、窗框轻响、寂静的电视雪花声 | 雨打车窗、路灯嗡鸣、空旷街道回声 | 医院走廊：仪器的规律性电子音 |
| **希望** | 鸟鸣穿窗、孩童笑声从楼下传来、门铃响起 | 街头艺人弹唱、咖啡店门铃、雨后天晴的清新 | 校园：上课铃、自由活动的嬉闹 |
| **焦躁** | 电话铃持续响、键盘节奏加快、咖啡机运作声 | 施工噪音、堵车喇叭、快餐店叫号 | 办公室：复印机运转、打印机出纸 |
| **压迫** | 群体忽然安静、空调声消失、沉重脚步声靠近 | 人群忽然静默、远处传来整齐脚步 | 审讯室：日光灯电流声、手铐碰撞 |

### A3. 架空/奇幻场景

## 附录B：姿态-情绪对照库

> **使用方法**：先确定情绪类型，再选择对应的具体姿态。同一情绪可选多个变量组合，但需回指上游触发点。

### B1. 紧张/警觉

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **手部** | 手指在杯沿来回摩擦、指腹在桌面轻点、指甲掐进掌心 | 等待消息、面对质问 |
| **肩颈** | 肩膀内收、肩胛骨夹紧、颈侧肌肉绷住 | 听到威胁、不确定安全 |
| **呼吸** | 呼吸变浅变快、胸腔僵硬、吸气只吸一半 | 危险临近、信息冲击 |
| **脚部** | 脚尖在地面轻点、脚跟抬起、脚尖朝外不稳 | 准备逃离、评估距离 |
| **视线** | 视线固定在某点、眨眼频率加快、瞳孔微缩 | 发现异常、盯住威胁源 |

### B2. 压制/强忍

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **手部** | 把手藏进袖中、指节慢慢收紧、掌心攥住袖角 | 压制愤怒、掩饰难过 |
| **面部** | 咬住后槽牙、下颌锁住、嘴角抿紧成线 | 强忍愤怒、不让情绪外泄 |
| **视线** | 视线绕开、目光下沉、眼睛不动 | 不看触发源、避免对视 |
| **身体** | 身体反而静止、动作突然变慢、重心后移 | 愤怒压制、身体防御 |
| **声线** | 声音压低、尾音收住、字与字之间停顿变长 | 强忍不满、压制反驳 |

### B3. 恐惧/惊疑

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **身体** | 身体后撤半步、重心降低、后退贴墙 | 面对威胁、发现危险 |
| **手部** | 手指松开（物件掉落或悬停）、手背挡在身前 | 意外冲击、无法接受 |
| **呼吸** | 呼吸骤停、喉结滚动、吞咽声 | 听到坏消息、发现真相 |
| **面部** | 瞳孔放大、眼睑上抬、嘴唇微张 | 震惊、不可置信 |
| **动作** | 动作变碎、不连贯、停在半空 | 信息过载、无法反应 |

### B4. 愤怒/攻击

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **手部** | 指节发白、拳头攥紧、拍桌/砸物 | 强烈愤怒、压制失败 |
| **身体** | 身体前倾、重心前移、逼近对方 | 攻击态势、威胁 |
| **面部** | 眉心紧皱、鼻翼张开、咬肌收紧、脸色涨红 | 愤怒外显、情绪爆发 |
| **动作** | 动作幅度增大、力度加大、停顿后突然启动 | 愤怒积累、临界爆发 |
| **声线** | 声音变粗变重、音量提高、语速加快 | 情绪宣泄、质问 |

### B5. 难过/不舍

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **视线** | 视线不离开、眼眶湿润但不落泪、目光低垂 | 离别、不舍、无能为力 |
| **手部** | 手指松开又收紧、手背抹过眼眶、指尖抓住对方衣角 | 告别、挽留、忍住悲伤 |
| **身体** | 动作变慢、幅度变小、频率降低、肩膀塌下 | 悲伤打击、失去力气 |
| **呼吸** | 吸气不完整、呼气变长、哽咽声 | 压抑悲伤、情绪临界 |
| **声线** | 话尾音拖长、声音发颤、停顿变多 | 不舍、告别 |

### B6. 害羞/心虚

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **视线** | 视线下落、避开正面、眼睑低垂 | 被戳穿、感到羞愧 |
| **身体** | 身体后缩、后退半步、偏过脸去 | 躲避目光、尴尬 |
| **手部** | 手指抓袖口、玩弄无关物件、整理衣角 | 掩饰心虚、转移注意力 |
| **动作** | 动作先退后进、完成质量低、停顿在关键点 | 想说什么又收回 |
| **声线** | 说话变轻、语速变慢、尾音上扬 | 心虚掩饰、试探 |

### B7. 试探/隐瞒

| 姿态类型 | 具体写法 | 适用场景 |
|---------|---------|---------|
| **视线** | 视线绕开对方、落到对方手或物件上 | 观察对方反应 |
| **动作** | 话说到一半停住、整理无关物、先看对方手再接物 | 保留底牌、观察反应 |
| **停顿** | 停顿在关键信息前、节奏变慢、问句拖长 | 试探对方是否知道 |
| **距离** | 错开距离、身体微后倾、半侧对着对方 | 保持安全、掩饰意图 |

## 附录C：情绪强度梯度指引

> **使用原则**：情绪强度决定表演幅度。上游授权决定允许的最高强度，不得越级。

### C1. 五级强度定义

| 等级 | 强度名 | 表现特征 | 授权门槛 |
|------|--------|---------|---------|
| **1级** | 微痕 | 局部微表情变化，动作幅度极小；呼吸未变，声音未变 | 任何场景均可使用 |
| **2级** | 轻度 | 单一部位明显变化（眼/手/声），其他部位保持；动作完整但克制 | 上游有明确情绪词即可 |
| **3级** | 中度 | 多个部位协同变化（面+手+呼吸），身体姿态明显改变；可能影响声线 | 上游有情绪描述或上下文支撑 |
| **4级** | 强度 | 全身参与，面部+身体+呼吸明显失控；动作可能中断或失控；声线明显变化 | 上游有强烈情绪词或情节高潮 |
| **5级** | 爆发 | 情绪失控外显，身体剧烈反应；可能包含泪水、颤抖、喊叫；动作失控 | 仅在上游明确写出"崩溃/大哭/怒吼/倒下"时使用 |

### C2. 强度选择决策树

```
上游情绪词分析
│
├─ 是否有明确强情绪词？（哭/怒/崩溃/颤抖/大笑/尖叫）
│   └─ 是 → 检查上游是否明确写出这些词
│       ├─ 明确写出 → 最高可授权4级（爆发需额外判断）
│       └─ 未写出 → 最高授权2级
│
├─ 是否有情绪描述？（紧张/害怕/难过/愤怒）
│   └─ 是 → 检查上下文是否支撑
│       ├─ 上下文有充分铺垫 → 最高授权3级
│       └─ 上下文无铺垫 → 最高授权2级
│
└─ 无明确情绪词，只有潜台词/隐含信息
    └─ → 优先使用1-2级，用微表情和身体联动承载
```

### C3. 强度与字段对应

| 强度等级 | 推荐字段 | 示例写法 |
|---------|---------|---------|
| **1级** | `表情特写`、`心理反应` | `眼角微微抽动`、`指腹在杯沿轻压` |
| **2级** | `心理反应`、`角色动作`、`对白画面` | `手指在杯沿来回摩擦`、`声音比前一句轻了半度` |
| **3级** | `角色动作`、`对白画面`、`表情特写` | `肩膀内收、呼吸变浅`、`眼眶发红但没有落泪` |
| **4级** | `角色动作`、`动作画面`、`对白画面`、`群像画面` | `他猛地站起，椅子向后倒去；指节发白，眼眶通红` |
| **5级** | `动作画面`、`群像画面`、`音效画面` | `她忽然崩溃，跪倒在地；哭声穿破寂静，肩膀剧烈起伏` |

### C4. 强度降级规则

以下情况必须降级使用：

| 降级原因 | 降级幅度 | 处理方式 |
|---------|---------|---------|
| 上游无明确情绪词 | 降1-2级 | 用微表情和身体联动替代情绪外显 |
| 场景需要压抑感 | 降1级 | 用压制替代爆发，表面平静但底层暗涌 |
| 多人场景中的配角 | 降1级 | 配角反应应弱于主角，避免喧宾夺主 |
| 规则/系统压迫场景 | 降1级 | 用群体反应和沉默替代个体情绪爆发 |
| 需要留给后续情节空间 | 降1级 | 情绪不完全释放，保留张力到下一场 |

### C5. 强情绪授权判断标准

| 上游原文 | 可授权强度 | 说明 |
|---------|-----------|------|
| `他感到一阵恶心` | 2级 | 直接感受词，无外显描写 → 用微表情/身体联动 |
| `她的眼眶红了` | 3级 | 上游已有可见反应 → 可授权到呼吸/声线变化 |
| `他再也忍不住，怒吼出声` | 4-5级 | 明确写出爆发 → 可授权声音外显和身体失控 |
| `她蹲下身，把脸埋进双手` | 3-4级 | 有具体动作描写 → 强度视上下文判断 |
| `全场陷入沉默` | 1-2级 | 群体情绪，无个体外显 → 用群体反应和空间承托 |
| `他的手指微微发颤` | 1-2级 | 上游只写微颤 → 不追加哭泣、愤怒等高强度动作 |
| `她没有哭，只是静静看着他` | 1级 | 上游明确写"没有哭" → 强制压制到1级 |
