# Scripts

`6-分组/scripts` 只承载机械辅助：

- 读取 Markdown 分组稿。
- 估算每组计入场景标题行、定场镜头、画面构图、构图分区和画面属性后的分镜剧本正文字数，并累计 `分镜N（约X秒）` 的组内显式时长；组底 fenced YAML 本身不计入字数。
- 检查分镜组 ID、场景标题行、定场镜头、画面构图、至少两个有内容的构图分区、第二组起定场镜头的上一组尾帧还原入场锚点、组头不得另列 `角色：` / `场景：` / `道具：` 主体清单、必填标题、全局风格固定前置词、组头第 1 行全局风格整理句不超过 300 字、组底 YAML 字段、精确重复道具名、计入场景标题行/定场镜头/画面构图/构图分区/画面属性后的分镜剧本正文字数、组内显式时长累计、组间连接件结构、连接件场景标题行、连接件三项 north_star 风格行、连接方法、变化过程、主体运动、运镜设计、透视适应和避免元素字段；连接方法只做非空与“不得只填抽象分类名”的机械检查，旧端点字段 `起点尾帧：` / `目标首帧：` 与旧 `连接件提示：` 为 error。
- 输出校验报告；结构错误和单组显式时长累计超过 18 秒均为 error，低于 850 字或低于约 10 秒的组为 warning，必须交给语义 review 判断是否存在短场景例外或是否需要回填。

脚本不得生成分组边界、定场镜头、构图分区、全局风格整理句、组间连接件、角色/场景/道具创作判断、道具同物异名合并结论或 canonical 分组正文。遇到缺失内容或单组超过 18 秒时，应报错并要求 LLM 或人工回到主创环节修复。validator 允许 `定场镜头：` 展开为多行自然段，但只能拦截格式、缺项、场景标题行缺失、定场镜头缺失、画面构图缺失、构图分区不足、组头主体清单残留、第二组起缺少上一组尾帧还原入场锚点或跨场景连接件标题行错误、连接件三项风格行缺失、全局风格整理句超长、连接方法裸分类名、端点字段残留、旧 `连接件提示：`、单组超时、精确重复道具名和明显结构错误，不能替代分组边界、构图分区主体落位准确性、定场镜头主体属性与站位质量、尾帧还原是否具体、当前组风格抽取是否准确、3-4 秒连接件语义、内部方法论选择、主体运动/运镜描述质量、非尾钩判断或道具语义归一判断。
