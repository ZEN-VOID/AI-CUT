# 4-Design 阶段路由矩阵

本文件为 `aigc/4-Design/SKILL.md` 的阶段路由细则载体，用于集中承接 tranche/domain coverage、前置条件、输出落点与 handoff 说明。

## Tranche Matrix

| tranche | 进入条件 | active domains | 父层应核对的输出 | 默认下一入口 |
| --- | --- | --- | --- | --- |
| `1-清单` | 缺对象池，或用户明确要求补清单 | `场景 / 角色 / 道具` | `清单.json / 研究.json / bridge.json` 等 list 真源 | `2-设计/<域>` |
| `2-设计` | list 真源稳定，且命中 active design 域 | `场景 / 角色 / 道具` | 设计 JSON/Markdown + `full_generation_prompt` + 同 stem 单主体图 | `3-面板/<域>` 或 `5-Image` |
| `3-面板` | design 真源、`full_generation_prompt` 与同 stem 图已稳定 | `场景 / 角色 / 道具` | `*-Panel-layout.json`、`_manifest.json`、request sidecar、派生 PNG | `5-Image / 6-Video / review` |

## Domain Coverage Matrix

| domain | 当前状态 | 父层口径 | 风险提示 |
| --- | --- | --- | --- |
| `场景` | `active` | 可进入 `1-清单 / 2-设计 / 3-面板` 对应 leaf | 不得回退到旧路径 |
| `角色` | `active` | 可进入 `1-清单 / 2-设计 / 3-面板` 对应 leaf | 设计与面板都要保持同 stem 图连续性 |
| `道具` | `active` | 可进入 `1-清单 / 2-设计 / 3-面板` 对应 leaf | 不得让 generic prop 漂移抢权 |
| `服装` | `pending-migration` | 只能报告 pending，不得伪装 active runtime | 不能预建 `4-Design/服装/*` active 落盘路径 |

## Runtime Alignment Checklist

### `1-清单`

- 父层只核对 list 真源是否落在 `projects/aigc/<项目名>/4-Design/<域>/1-清单/`
- 不得由父层代写 list 真源

### `2-设计`

- 父层核对设计真源是否落在 `projects/aigc/<项目名>/4-Design/<域>/2-设计/`
- 重点核对：
  - `full_generation_prompt`
  - 同 stem 单主体图
- 若缺任一项，不得把该域写成已可进入 `3-面板`

### `3-面板`

- 父层核对面板真源是否落在 `projects/aigc/<项目名>/4-Design/<域>/3-面板/`
- 重点核对：
  - `*-Panel-layout.json`
  - `_manifest.json`
  - request sidecar / 派生 PNG（若本轮触发生图）
- 不得把派生 PNG 升格为父层 canonical 输出

## Advisory Boundary Checklist

- `roles.supervision` 只能在 leaf 首次落盘前提供 advisory。
- `source_skill_refs` 只作领域提示，不作 post-write reviewer 授权。
- 若本轮没有读取 advisory，closure 中要写 `unused_with_reason`。

## Handoff Matrix

| 当前完成点 | 可进入的下一入口 | 不可越过的前置条件 |
| --- | --- | --- |
| `1-清单/<域>` | `2-设计/<域>` | 该域 list 真源已稳定 |
| `2-设计/<域>` | `3-面板/<域>`、`5-Image` | `full_generation_prompt + 同 stem 单主体图` 已稳定 |
| `3-面板/<域>` | `5-Image / 6-Video / review` | layout JSON 与派生链路可追踪 |

## Stage Report Minimum Notes

父层 closure 至少要写清：

- 本轮 route mode
- 命中的 tranche/domain
- 已执行 leaf 输出
- pending 或 blocked scope
- advisory 是否启用与是否采纳
- 推荐下一入口
