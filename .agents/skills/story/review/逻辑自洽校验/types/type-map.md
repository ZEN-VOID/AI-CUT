# Type Map

## 类型包加载边界

- 每次调用本技能时，必须依据本文件识别并加载同目录 `types/` 中选中的类型包（单选或多选）。
- `types/` 中命中的类型包作为固定上下文加载；`knowledge-base/` 只作为按需检索、切片或向量召回的知识库，不替代类型包。


| signal | route |
| --- | --- |
| 世界规则冲突 | source trace 后判 owner |
| 能力边界漂移 | `1-单章叙事起盘` 或 `1-设定` |
| 例外缺代价 | blocking issue |
| source truth 冲突 | `back_to_source_contract` |
