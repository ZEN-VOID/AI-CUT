# Chapter Rhythm Rules

## Shared Contract

章级节奏的共性真源已回收到：

- `../../../_shared/chapter-rhythm-handoff-contract.md`

本文件只补章级 planning 自身的落盘要求，不再平行维护第二套共性发动机。

## 章级写回重点

- `本章节奏曲线` 必须明确写出 `selected_pack / selected_mode`。
- `七步职责映射` 必须回答本章每一步承担什么推进功能，而不是只写步骤名。
- `规划义务` 必须锁清 `entry_promise / conflict_axis / micro_payoff / exit_hook`。
- `义务段位` 与 `建议写法` 必须分开，避免 drafting 把建议写法误读成不可动的法律。
- Mermaid 图至少能看出起势、转折、升级、高潮、尾钩。

## Chapter-Specific Rule

- 章级是 planning 的最细层，但仍只锁“这章怎么跑”，不直接写正文段落。
