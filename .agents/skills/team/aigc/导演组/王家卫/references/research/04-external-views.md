# 04 他者视角、批评与争议

## 正面外部观察

| 来源 | 观察 | 可提炼点 |
|---|---|---|
| Criterion | 其作品形成“王家卫世界”，时间、欲望、城市孤独和流行音乐高度可识别 | 风格不是单片技巧，而是连续世界观 |
| BFI / Christopher Doyle | 《重庆森林》23天快速拍摄，很多场景临场、街头、受条件限制 | 即兴和限制是方法，不只是混乱 |
| Senses of Cinema | 时间与记忆贯穿作品，个人情绪和香港历史互相纠缠 | 私人爱情常是城市历史的载体 |
| Academy | 《一代宗师》把武术传承看成时间、身体和谱系 | 类型片也被转译成时间伦理 |

## 批评与盲点

1. **制作周期与超支**
   - 多个作品长期拍摄、重写、重剪，带来工业风险。
   - 这种方法可能产出高度个人化作品，也可能伤害合作者和资源可控性。

2. **演员和剧组消耗**
   - 外界长期将其“折磨演员”“没有定稿”的名声浪漫化。
   - Skill 必须把这转化为风险提醒：如果缺少边界、合同和补偿机制，即兴会变成权力滥用。

3. **风格被消费成表面**
   - 广告、MV、短视频常复刻霓虹、慢镜、旧歌，却丢掉时间结构和人物缺席。
   - 这会导致“像王家卫”只剩调色，不剩戏。

4. **女性角色与凝视**
   - 其女性角色常被塑造成谜、影像和欲望对象；作品同时给了她们强烈存在感，也容易被后续模仿者简化为“漂亮又疏离”。

## 2025-2026 《繁花》争议边界

- What's on Weibo 汇总了 2025 年围绕《繁花》的署名、劳动、录音泄露和职场伦理争议。
- 古二/程骏年提出其参与剧本和人物线创作却未获足够署名与报酬的指控。
- 制作方和相关人员有回应和否认，并表示将采取法律行动；公开材料仍处于争议状态。
- 对 Skill 的影响：不能再把“王家卫式折磨和漫长制作”无条件浪漫化。任何使用本 Skill 的导演建议都必须补上制作伦理、署名、劳动边界和协作透明度。

## 可保留的内在张力

- 极高个人作者性 vs 大规模协作的署名和劳动正义。
- 允许现实改写剧本 vs 需要明确合同与责任边界。
- 让演员/城市自然生长 vs 导演权力过强。
- 怀旧美学 vs 对历史活力和现实矛盾的捕捉。
