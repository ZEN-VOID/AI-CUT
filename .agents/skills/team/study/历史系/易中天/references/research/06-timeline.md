# 06-timeline · 易中天时间线

## 基本时间线

| 时间 | 事件 | 可能形成的稳定影响 |
|---|---|---|
| 1947-02-08 | 生于湖南长沙 | 湖湘文化与家庭知识背景构成早期底色 |
| 幼年 | 童年后迁居武汉 | 形成长沙/武汉双重城市经验 |
| 1965-1975 | 赴新疆生产建设兵团 | 长期边地经历强化现实感与韧性 |
| 1975-1978 | 在乌鲁木齐钢铁公司子弟中学任教 | 早期教学训练，强化“讲给人听懂”的能力 |
| 1978 | 武汉大学毕业 | 文学与古代文论训练进入系统阶段 |
| 1981 | 获武汉大学硕士学位并留校任教 | 学术职业正式展开 |
| 后续阶段 | 任厦门大学中文系教授、博导 | 形成跨文学、美学、历史的复合身份 |
| 2005 | 进入央视《百家讲坛》 | 大众传播身份起飞 |
| 2006 | `品三国` 热潮形成 | “公共知识讲述者”人格稳定成形 |
| 2008 | 讲先秦诸子系列，并面对《品三国》错误争议 | 方法与边界被迫公开化 |
| 2013 | 启动 `易中天中华史` 系列 | 从节目名嘴回到长线写作者 |
| 2022-12 / 2023 | 出任厦门大学人文与艺术高等研究院院长并主持系列学术活动 | 角色升级为学术组织者与文化主持者 |
| 2024-06 | 与考古专家共同讲述“我们小时候”系列活动 | 继续参与历史/考古公共传播 |
| 2025-09 至 2025-10 | 围绕《三国的星空第一部》持续在微博发声并参与宣传 | 历史叙事进入影像跨界实验 |
