# Template: 元素替换 / 换背景

Template ID: `photoGPT.元素替换.换背景.v1`

## Input Roles

- 图一：待编辑主图，锁定主体身份、服装、姿态、镜头视角、主体尺度和前景关系。
- 图二：背景参照图，提供场景空间、透视、环境元素、色彩氛围和光线方向。

## Prompt Frame

```text
保留图一中主体人物或主体物体的身份特征、服装造型、姿态动作、镜头视角、主体尺度和前景关系完全不变，仅将原有背景环境替换为图二中展示的场景空间与氛围。

背景替换要求：完整吸收图二的空间结构、透视关系、主要环境元素、色彩氛围、光线方向和空气质感；移除图一原背景中的无关建筑、地面、天空、室内陈设和旧背景残留；主体边缘过渡自然，接地阴影、反射、景深关系和空气透视合理。

主体锁定：面部、发型、服装、肢体姿态、道具持握关系、主体构图位置严禁漂移。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变主体身份、表情、体态、服装和姿势。
- 不保留旧背景残片。
- 不把图二中的人物或主体复制到图一。
- 不产生边缘白边、抠图硬边或光线冲突。
