# Scene Visual Constraint Contract

本文件定义每个场景在进入逐镜设计前必须裁决的场景级视觉约束。这些约束是**内部裁决**，不进入成稿正文，但所有逐镜分镜明细必须在约束框架内展开。

## Core Rule

每个 `visual_unit` 在完成摄影语法裁决（`N6.2-CAMERA-GRAMMAR`）后、进入功能性投影（`N6.4-FUNCTIONAL-PROJECTION`）前，必须形成内部 `scene_visual_constraint`。该约束包含构图布局、构图方式、光源设置、照明类型、色彩体系和摄影技术参数六大分区，覆盖该场景的整体视觉基准。

`scene_visual_constraint` 是内部裁决，不出现在最终正文成稿中。逐镜分镜明细的设计必须在此约束框架内展开，确保同一场景内不同画面句子的镜头设计具有统一的美术基调。

## Scene Visual Constraint Fields

### 构图布局 composition_layout

合理分配和排列场景元素，形成画面的空间骨架。

| field | description | example |
| --- | --- | --- |
| `subject` | 主体位置与占比 | 人物偏右三分线，占画面 40% |
| `companion` | 陪体位置与角色 | 课桌行列作为前景框架压迫主体 |
| `foreground` | 前景元素与遮挡 | 门框/窗帘/桌椅边切割画面左侧 |
| `background` | 背景元素与氛围 | 黑板粉笔字、封死窗帘、冷白墙面 |

裁决时必须回答：主体在哪里、陪体在哪里、前景用什么遮挡或引导、背景用什么承托氛围。

### 构图方式 composition_method

以形状感、线条感、影调感、虚实感、节奏感、纹理质感和气势七个子维度裁决画面的审美骨架。

#### 形状感 shape_sense

具象或抽象的形状感设计，决定画面的几何心理暗示。

| shape | psychology | use when |
| --- | --- | --- |
| 方形 / 矩形 | 人造规划、稳定、坚固；也有局限、封闭之意 | 教室秩序、制度框架、课桌网格 |
| 正三角形（"金字塔"构图） | 稳定、永恒、统治 | 权力人物居中、讲台高处、规则宣判 |
| 倒三角形 | 不稳定、运动、倾斜 | 角色被压迫、空间失衡、规则入侵 |
| 圆形 / 椭圆 | 包围、循环、聚焦 | 群像围困、瞳孔、聚光灯 |
| 对角线菱形 | 动态张力、冲突交汇 | 人物对峙、视线交叉 |
| 不规则碎片 | 混乱、崩塌、失控 | 规则破碎、群体恐慌、空间异常 |

#### 线条感 line_sense

具象或抽象的线条感设计，引导视线方向和情绪节奏。

| line | psychology | use when |
| --- | --- | --- |
| 垂直线 | 高大挺拔、稳定、静默 | 柱子、门框、站姿人物 |
| 水平线 | 平静、开阔、安详 | 地平线、桌面、走廊延伸 |
| 对角线（倾斜线） | 不稳定、动感、运动，暗示空间深度（透视线） | 走廊透视、课桌排列倾斜、角色倾斜身体 |
| 波状线（"S"形） | 动静交替、内在律动、秀美美感 | 走廊曲线、窗帘褶皱、角色行走路径 |
| 放射线 | 聚焦、爆发、冲击 | 灯光扩散、恐惧辐射、注意力中心 |
| 锯齿线 | 紧张、焦虑、危险 | 裂缝、锯齿状阴影、规则边缘 |

#### 影调感 tonal_sense

利用明暗对比增强画面深度，保持亮部质感和暗部层次。

| tonal | use when | execution cue |
| --- | --- | --- |
| 高调 | 明亮、纯净、脆弱 | 过曝教室、白色校服、冷白灯光淹没人物 |
| 低调 | 压迫、神秘、危险 | 深色窗帘吸光、角落暗影、只保留危险轮廓 |
| 中间调 | 自然、日常、观察 | 常态校园光线，不强化明暗倾向 |
| 高反差 | 戏剧、冲突、极端情绪 | 强烈侧光切割脸、窗口高光+室内深暗 |
| 柔和低反差 | 梦境、回忆、温柔 | 柔光包裹、雾气、色块过渡自然 |

#### 虚实感 depth_clarity

合理配置清晰与模糊，突出主题或动态感。

| clarity | use when | execution cue |
| --- | --- | --- |
| 全实 | 空间关系展示、群像、规则布局 | 前中后景信息全部可读 |
| 主实副虚 | 主体突出、信息分层 | 主体清晰，背景化成色块或光斑 |
| 前虚后实 | 发现、揭示、注意力转移 | 前景模糊剪影，后景信息逐渐清晰 |
| 焦点追随 | 动态跟踪、视线引导 | 焦点跟随运动主体，背景流动模糊 |
| 全虚 | 失神、眩晕、意识模糊 | 画面整体失焦，只保留色块和光影轮廓 |

#### 节奏感 visual_rhythm

通过元素的重复、间隔、变化，创造视觉节奏和韵律。设计审美视觉暂留点，引导视线停留。体现秩序感、韵律感、规律性、连续性。

| rhythm | use when | execution cue |
| --- | --- | --- |
| 重复节奏 | 秩序、制度、压迫 | 课桌行列、窗户间隔、日光灯排布 |
| 渐变节奏 | 递进、压迫加深、空间延伸 | 桌椅渐远渐小、光影渐暗渐冷 |
| 交替节奏 | 对比、张弛、对话 | 明暗交替、动静交替、前景后景交替出现信息 |
| 突变节奏 | 惊吓、揭示、入侵 | 秩序被打破的瞬间，颜色或形状突然改变 |
| 自由节奏 | 混乱、失控、边缘 | 元素无规律散布，但保留 1-2 个视觉暂留锚点 |

#### 纹理和质感 texture_and_material

添加特定画面质地效果，增强艺术性和独特性。

| texture | use when | execution cue |
| --- | --- | --- |
| 颗粒感 | 复古、纪实、心理压力 | 胶片颗粒在暗部可见，增加粗糙真实感 |
| 胶片质感 | 时代感、记忆、怀旧 | 色彩偏移、高光溢出、暗部偏绿 |
| 光晕 / 眩光 | 超现实、神圣、危险诱惑 | 窗口光晕、灯管溢光、瞳孔反光扩散 |
| 磨砂 / 雾气 | 梦境、隔阂、不确定性 | 画面带雾感，细节被吞没，只保留大色块 |
| 锐利质感 | 真实、冷酷、制度 | 高锐度让每个细节清晰可读，无处可藏 |

#### 气势 atmosphere_power

通过对时空环境的把握，表现特定的精神风貌和情绪感染力，规定情感表达的倾向和程度。

| atmosphere | use when | execution cue |
| --- | --- | --- |
| 压迫气势 | 制度碾压、权力宣判 | 低角度仰拍+深焦+大面积阴影+硬光 |
| 渺小气势 | 个体被吞没、群像无助 | 大远景+高角度+人物占比极小+环境主导 |
| 紧迫气势 | 追逐、倒计时、危险逼近 | 对角线构图+手持微晃+浅景深+快速运镜 |
| 肃穆气势 | 仪式、宣判、命运 | 对称构图+固定机位+冷色调+静止长镜 |
| 荒诞气势 | 异常、错位、规则荒谬 | 移轴+高饱和+不规则构图+变速运镜 |
| 温柔气势 | 治愈、关系、安全 | 柔光+浅景深+暖色温+慢速横移+自然纹理 |

### 光源设置 lighting_setup

| field | description | example |
| --- | --- | --- |
| `key_light` | 主光源：方向、质感、强度 | 右上方侧窗柔光，照亮主体半张脸 |
| `fill_light` | 辅助光：补光方向与比例 | 左侧反射补光，保持暗部可读轮廓 |
| `back_light` | 逆光：轮廓分离与氛围 | 背后日光灯管勾出肩膀轮廓，与黑窗帘分离 |

### 照明类型 lighting_type

| type | use when | execution cue |
| --- | --- | --- |
| 自然光 | 户外、窗口、时间标记 | 方向和色温随时间段变化 |
| 人工硬光 | 制度、审讯、暴露 | 锐利阴影边缘，高光过曝 |
| 人工柔光 | 日常、对话、观察 | 阴影过渡自然，皮肤质感柔和 |
| 混合光 | 复杂空间、内外对比 | 冷白室内 + 暖色窗外，色温冲突 |
| 实用光 | 写实、场景内光源 | 台灯、手机屏、黑板灯作为画面内可见光源 |
| 造型光 | 风格化、情绪强化 | 轮廓光、背景光、发灯单独塑造主体 |

### 色彩体系 color_system

| field | description | example |
| --- | --- | --- |
| `hue_dominance` | 色相主导 | 冷白+深蓝为底色，红色作为危险点缀 |
| `value_range` | 明度范围 | 高光在 80-90%，暗部保持 15-25% 层次 |
| `saturation_strategy` | 饱和度策略 | 全局低饱和，只有红色道具保留高饱和 |
| `color_temperature` | 色温基调 | 5500K 冷白日光灯为常态，3200K 暖光只用于回忆/安全 |
| `color_psychology` | 色彩心理 | 冷白=制度无温度，深蓝=压抑，红色=危险/欲望，金色=隐藏真相 |

### 摄影技术参数 camera_technical_params

内部裁决的拍摄参数，供下游逐镜设计参考。

| field | description | example |
| --- | --- | --- |
| `camera_model` | 机型 / 传感器风格 | ARRI Alexa Mini / Sony Venice / RED Komodo（影响色彩科学和动态范围） |
| `aperture` | 光圈 | f/1.4 浅景深特写 / f/5.6 中景对话 / f/11 深焦空间 |
| `shutter_speed` | 快门速度 | 1/48s 标准电影运动模糊 / 1/1000s 冻结动作 |
| `iso` | ISO 感光度 | 800 低噪点室内 / 3200 暗光手持 |
| `focal_length` | 焦距 | 24mm 广角空间畸变 / 50mm 标准观察 / 85mm 人像压缩 / 135mm 长焦压缩 |
| `resolution` | 分辨率 | 4K DCI（4096x2160）/ 6K 裁切空间 / 8K 超采样 |

## Internal Constraint Rules

1. `scene_visual_constraint` 是纯内部裁决，不出现在最终正文成稿中。
2. 同一场景内视觉约束不变时只裁决一次，视觉约束发生重大变化时（如从白天到夜晚、从室内到走廊、从平静到恐慌）可重新裁决。
3. 下游 `camera_grammar_plan`、`shot_design_plan`、`functional_projection_plan` 和逐镜分镜明细必须在约束框架内展开，确保同一场景内不同画面句子的镜头设计具有统一的美术基调。
4. 裁决时选取当前场景最关键的 2-3 个构图方式子维度，不要求七个子维度全部覆盖。选取错误比遗漏更危险——错误的形状感或气势会把平淡画面写成夸张宣言。
5. 场景视觉约束的六大分区（构图布局、构图方式、光源设置、照明类型、色彩体系、摄影技术参数）共同构成"这个场景在视觉上长什么样"的内部基准，不输出为参数清单或标签列表。

## Usage Note

场景视觉约束是 5-摄影 阶段的内部裁决框架。正式的"组级画面视觉基调"输出由下游 `6-分组` 阶段负责，后者从组内镜头设计中提炼画面属性并以自然语句写入每个分镜组。两者的区别：
- **场景视觉约束**（5-摄影 内部）：场景级美术基准，约束逐镜设计
- **组级画面属性**（6-分组 输出）：从组内镜头设计提炼的自然语句，供下游消费

## Knowledge Base Binding

- **画面属性**的裁决和落盘以 `knowledge-base/摄影构图/` 作为知识库参考，确保构图布局、构图方式、光源、色彩等审美维度有摄影构图理论支撑。
- **分镜明细**的裁决和落盘以 `knowledge-base/电影镜头/` 作为知识库参考，确保镜头语言、运镜调度、镜头技术等执行维度有电影镜头专业理论支撑。
- **对话戏**的分镜明细落盘应特别参考 `knowledge-base/电影镜头/一流对话场景.md`，该文件包含 100 个对话场景高级拍摄技巧，覆盖力量对比、障碍物、位置交换、深布景、环拍对话、正反打变化等专业技法。当 visual_unit 类型画像命中对白画面、对白身体锚点或关系停顿时，必须从该知识库中选取匹配的对话场景技法指导分镜明细写作。
