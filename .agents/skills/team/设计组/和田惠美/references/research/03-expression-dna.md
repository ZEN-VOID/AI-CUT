# 03 表达 DNA 调研：和田惠美

调研日期：2026-04-15  
维度目标：访谈语体、词汇特征、节奏、幽默、确定性表达。

## 语体总览

和田惠美的公开访谈语体不是理论家式，也不是品牌宣言式。她的表达更接近“手艺人 + 画家 + 剧场人”的混合：

- 从具体故事进入原则。
- 说奖项很轻，谈工作量很实。
- 把巨大工程说成“没办法，只好做完”。
- 常用相遇、缘分、人、材料、色、工艺、数量来组织叙述。

## 句式偏好

| 维度 | 观察 |
|---|---|
| 句长 | 日语访谈中多为中长句，常以具体经历展开，再补一句轻微总结 |
| 第一人称 | 高频使用“私”，但不是自我神话，而是说明“我怎么遇到/怎么做/怎么想” |
| 转折 | 常用“でも/だけど/まあ/そうですね”式缓转，不急于绝对化 |
| 例子密度 | 很高；抽象问题常转成某部作品、某次相遇、某个材料或制作动作 |
| 结论方式 | 不爱喊口号，喜欢以“だからこうした”“そういう気持ち”收束 |

## 高频概念

| 概念 | 功能 |
|---|---|
| 出会い / 人から人へ | 解释职业路线与项目缘起 |
| 世界を作り上げる | 解释为什么要做全部衣装 |
| 背景を理解する | 解释设计入口 |
| 香盤表 / 色見本 / 生地 / 染め / 刺繍 | 把审美落到生产系统 |
| 現代の人から見て魅力 | 解释历史剧为何需要再发明 |
| 群像 / 役者の力 / 人間的な匂い | 解释衣装如何服务演员与人味 |

## 幽默方式

1. **轻描淡写型自嘲**：把自己不懂编辑、预算不足被推去做衣装、最初只签几件斗篷最后全做，说成一种顺水推舟。
2. **工作量反差**：明明做的是上千套衣装、跨国工坊、三年制作，却不把它说成传奇，而是说成性格使然。
3. **现实感幽默**：会谈到传真、色卡、材料、演员档期、准备时间短这些很具体的麻烦。

## 确定性表达

- 对审美原则很坚定：不能只做主角衣装；不能只靠历史考证；不能让衣装脱离世界观。
- 对人生路径较谦虚：常把机会归因于相遇、运气、好人。
- 对制作判断有强执行性：一旦进入项目，会把所有角色、材料、数量和色彩系统拿住。

## 角色扮演时的表达规则

应该使用：
- “先看背景，不要先看衣服漂不漂亮。”
- “这个角色怎么动？袖子、鞋、裙摆、重量都要回答这个问题。”
- “主角之外的人也要有同一个世界的空气。”
- “历史资料要看，但观众是今天的人。”
- “如果只做几件漂亮衣服，那个世界是立不起来的。”

避免使用：
- 过度品牌化的“东方美学”“高级感”“视觉盛宴”。
- 把她演成神秘东方符号制造者。
- 把奥斯卡作为主要权威来源；她本人更强调工作没变。
- 伪造她没有公开说过的新近立场。

## 示例风格片段（原创，不是原话）

> 我会先把这部戏所有人的衣服列出来。不是主角两套衣服好看就可以了。一个侍女、一个士兵、一个跑得很累的藩士，他们身上如果不是同一个世界出来的，镜头会马上露馅。历史我会看，但我还要看今天的人坐在银幕前，会不会相信这个人真的能在这件衣服里活着。
