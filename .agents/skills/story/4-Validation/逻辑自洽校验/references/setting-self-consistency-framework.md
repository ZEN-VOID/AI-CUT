# 逻辑自洽校验 / 设定自洽框架

## Purpose

本文件是 `逻辑自洽校验` 子技能的细则真源，用来把 `.agents/skills/story_backup/templates/worldbuilding/setting-consistency.md` 的跨阶段一致性思路，转译为 `story/4-Validation` 可执行的单维度检查框架。

边界：

- 本文件负责“怎么检查设定与逻辑是否自洽”。
- `SKILL.md` 负责“何时进入、如何路由、怎样落盘”。
- 若发现时间顺序问题，应移交 `时间线`；若发现结构义务没兑现，应移交 `结构兑现`；本文件不抢其他维度的总线。

核心判断句：

- 设定可以大胆，但规则不能失忆。
- 读者可以接受“这个世界有超常规则”，不能接受“规则只在剧情需要时才生效”。

## 1. 世界铁律 / 物理法则自洽

### 核心问题

- 世界已经写明的基本规则，是否被正文偷偷推翻？
- 对象状态、空间锚点、资源消耗是否仍然服从已声明的铁律？
- 同一种规则在不同场景下，是否变成了两套口径？

### 高危失效模式

| fail_shape | 症状 | 默认 metric |
| --- | --- | --- |
| 铁律失忆 | 前文明确“不可能”，本集却无解释地发生 | `world_rule_conflicts` |
| 状态失锚 | 当前地点、伤势、持有物、阵营状态无法在正文站稳 | `state_conflicts` |
| 规则时灵时不灵 | 同一规则在 A 场景严格生效，在 B 场景被静默跳过 | `world_rule_conflicts` |

### 默认修复策略

- 优先补 `source owner`，确认是不是上游真源先写乱了。
- 若是正文问题，回到 `1-单集叙事起盘`，重写触发条件和承接细节。
- 若确实要破例，转到“例外合同”做合法化，而不是把破例当默认常态。

## 2. 力量体系 / 能力边界自洽

### 核心问题

- 角色、道具、系统、阵法、组织资源的能力上限是否稳定？
- 越级、破格、临时爆发是否有明确条件，而不是无限制常态化？
- 使用力量是否伴随稳定代价，而不是只要剧情需要就零成本开挂？

### 高危失效模式

| fail_shape | 症状 | 默认 metric |
| --- | --- | --- |
| 战力漂移 | 同等级强度忽高忽低，没有额外解释 | `capability_conflicts` |
| 越级常态化 | 前面需要神兵/伏击/代价，后面却裸越级 | `capability_conflicts` |
| 代价蒸发 | 规则说要消耗寿命/资源/风险，本集却白嫖 | `exception_cost_gaps` |

### 默认修复策略

- 问清楚：本次成功依赖的是常规能力，还是特殊条件。
- 若是特殊条件，必须把条件和代价写到正文可感知层。
- 若正文无法自证，就上溯 `1-Cards / 2-Planning` 看能力合同是不是本来就写歪。

## 3. 社会生态 / 资源后果自洽

### 核心问题

- 规则带来的经济、组织、人口、秩序后果，是否稳定存在？
- 某个稀缺资源、高危能力、制度约束，是否只有在制造冲突时才出现？
- 世界中的普通人、组织和环境，会不会对超常规则做出合理反应？

### 高危失效模式

| fail_shape | 症状 | 默认 metric |
| --- | --- | --- |
| 资源购买力漂移 | 同一种资源时而稀缺时而廉价，正文无解释 | `social_ecology_conflicts` |
| 制度只在需要时上线 | 原应触发组织/秩序反应的事件，被世界静默吞掉 | `social_ecology_conflicts` |
| 环境后果失踪 | 大规模能力使用后没有任何可见余波 | `contrivance_risk` |

### 默认修复策略

- 把“世界会怎么反应”当成逻辑问题，而不是额外装饰。
- 如果后果被省略，先判断是正文偷懒，还是上游世界观根本没定义约束。
- 对需要长期稳定的社会反应，优先修共享 source，而不是只给当前一集补一段解释。

## 4. 例外合同

当正文要打破既有规则时，不要直接问“能不能爽过这一段”，而要检查以下 4 格：

| slot | 必问问题 | 缺失后果 |
| --- | --- | --- |
| `trigger` | 这次破例由什么条件触发？ | 变成无因果开洞 |
| `cost` | 主角/世界付出了什么代价？ | 变成零成本开挂 |
| `precedent_or_explanation` | 有没有先例、伏笔或当前可核对解释？ | 变成临场补丁 |
| `visibility` | 读者能不能在正文里感知到这件事不是常态？ | 变成规则失忆 |

硬规则：

- 4 格缺一，默认计入 `exception_cost_gaps` 或 `world_rule_conflicts`。
- 若 4 格都存在，但只停在注释层、旁白层而无正文证据，仍然算弱成立。

## 5. Source Trace 矩阵

| 症状 | 优先怀疑层 | 默认回流 |
| --- | --- | --- |

## 6. story2026 维度投影

| 设定自洽母题 | 本技能的结构化落点 |
| --- | --- |
| 世界法则不能随意打破 | `world_rule_conflicts` |
| 力量体系必须前后一致 | `capability_conflicts` |
| 使用力量必有消耗或风险 | `exception_cost_gaps` |
| 社会、经济、组织要对规则做出反应 | `social_ecology_conflicts` |
| 当前状态必须可追踪 | `state_conflicts` |
| 剧情不能靠作者硬推 | `contrivance_risk` |
