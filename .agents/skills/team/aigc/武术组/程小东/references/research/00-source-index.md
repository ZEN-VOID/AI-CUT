# 00 来源索引：程小东

调研时间：2026-04-15

## 来源策略

程小东的公开“方法论文本”很少，核心材料主要来自机构人物资料、电影节/大师班介绍、作品署名、奖项记录、少量访谈和同行/影评对其动作系统的描述。本次蒸馏不把百科、论坛和粉丝整理当作高权重来源；知乎、微信公众号、百度百科不进入证据链。

## 信息缺口

- 缺少系统性自述著作或完整导演手记。
- 公开访谈数量有限，表达 DNA 不能按作家型人物处理，只能从少量访谈、动作设计实践和作品风格归纳。
- 早年训练、出生年份与英文转写在公开资料中存在小幅差异；本 Skill 对此保持谨慎，不以未经核验的单一说法强行定论。
