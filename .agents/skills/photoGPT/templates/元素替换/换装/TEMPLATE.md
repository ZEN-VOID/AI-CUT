# Template: 元素替换 / 换装

Template ID: `photoGPT.元素替换.换装.v1`

## Input Roles

- 图一：角色原图，锁定面部、发型、肤色、体态比例、姿态和构图。
- 图二：服装参照图，提供服装剪裁、面料、色彩、配饰和层次。

## Prompt Frame

```text
保留图一中角色的面部特征、发型发色、肤色、体态比例、姿态动作、镜头视角和构图位置完全不变，仅将角色身上的服装整体替换为图二中展示的服装样式。

服装替换要求：完整还原图二服装的剪裁版型、面料质感、色彩配色、装饰细节、纽扣、刺绣、腰带、配饰和层次结构；服装需自然贴合图一角色的体型，保持衣物褶皱、垂坠、遮挡关系和动作张力的物理合理性。

身份锁定：面部五官、发型、肤色、体态比例、姿态、背景和镜头关系严禁漂移。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变角色脸、发型、身材比例和姿势。
- 不把图二模特的脸或身体复制过来。
- 不改变图一背景，除非用户明确要求。
- 不丢失图二服装关键装饰和层次。
