# Template: 多视图 / 场景

Template ID: `photoGPT.多视图.场景.v1`

## Required Fields

- `scene_id`
- `scene_name`
- `desc`
- 一张或多张场景参考图

## Input Roles

- 主参考图：提供场景核心形象、建筑语法、材质和风格。
- 辅助参考图：只补充局部材质、光影、结构或布局，不覆盖主参考图真源。

## Prompt Frame

```text
### 场景ID: <scene_id>
保留参考图的画面风格和场景核心形象，根据以下要求调整为多视图场景设计页：

设计主体：<desc>
空镜头护栏：画面中不出现任何人物、人形剪影、生物或人影。

布局规格：SCENE_DESIGN_SHEET 模板，16:9 画幅，4K 分辨率（4096x2304），深色中性背景（#1a1a1a），严格 3x3 九宫格，均匀间距与面板比例，干净分割线，无装饰性边框噪声。

布局模块：
P1 氛围全景；P2 氛围中景；P3 氛围特写；
P4 东向视图；P5 西向视图；P6 低角度仰拍；
P7 南向视图；P8 北向视图；P9 高角度俯拍。

一致性约束：所有面板共享相同主光源方向与强度；材质家族、纹理老化状态、标志性锚点元素数量、轮廓、比例和材质跨面板保持稳定。

身份标识：左上角固定身份标识徽章 <scene_id> <scene_name>，位置与字体跨输出保持稳定。
```

## Negative Constraints

- 不出现人物、人影、生物。
- 不把九宫格变成故事板或广告海报。
- 不让同一建筑/空间在不同面板中发生身份漂移。
