# 06 人物时间线

## 关键节点

| 时间 | 事件 | 对导演方法的影响 |
|---|---|---|
| 1947 | 侯孝贤生于广东梅县，幼年随家人迁台，在高雄凤山成长。 | 外省家庭、台湾地方生活、少年经验与语言/身份混杂成为底层记忆。 |
| 1960s | 少年时期大量看电影、读文学，也有街头与叛逆经验。 | 后来形成“电影和文学把人从黑路上带回来”的自我叙事。 |
| 1970s | 进入台湾电影工业，从场记、副导演、编剧等位置工作。 | 作者性从现场工种和商业片工法内长出，而非学院理论先行。 |
| 1983 | 《风柜来的人》成为重要转向。 | 青年经验、地方生活、跳切和新电影现实主义开始成形。 |
| 1984 | 《冬冬的假期》。 | 儿童视角、乡村空间、家庭与时间流动成为方法。 |
| 1985 | 《童年往事》。 | 个人记忆和家庭史明确进入创作核心。 |
| 1986 | 《恋恋风尘》。 | 青年离乡、工作、爱情与时间失落形成生活流叙事。 |
| 1989 | 《悲情城市》获威尼斯金狮。 | 台湾历史创伤、家庭微观叙事、画外暴力进入世界电影史视野。 |
| 1993 | 《戏梦人生》获戛纳评审团奖。 | 口述史、布袋戏、日据经验与个人传记互相嵌合。 |
| 1995 | 《好男好女》。 | 历史、电影制作、当代人物和记忆层次并置。 |
| 1998 | 《海上花》。 | 封闭空间、长镜头、室内调度和文学改编达到高度形式化。 |
| 2001 | 《千禧曼波》入围戛纳竞赛。 | 转向当代青年、夜生活、近距离与霓虹空间。 |
| 2005 | 《最好的时光》。 | 以三段历史时间考察爱情、身体和沟通方式。 |
| 2007 | 《红气球之旅》。 | 跨文化制作、巴黎空间、儿童与日常观察。 |
| 2015 | 《刺客聂隐娘》获戛纳最佳导演奖。 | 用现实主义和限制重写武侠类型，成为晚期代表作。 |
| 2020 | 获第 57 届金马奖终身成就奖。 | 行业承认其对台湾电影和华语电影的长期影响。 |
| 2023-10-25 | 家属公开确认其健康状况；媒体报道其退休并停止《舒兰河上》。 | 最新公开事实边界：不再假定有新导演作品推进。 |

## 最新动态（截至 2026-04-15）

- 可靠公开来源显示，2023 年家属确认侯孝贤被诊断为阿尔茨海默病，并因健康原因退休。
- The News Lens 综合家属声明指出，原筹备影片《舒兰河上》无法继续拍摄。
- 2023 年后未检索到可靠公开证据显示侯孝贤恢复导演新片工作。
