# Inner-Life Execution Playbook

本文件是 `6-心理活动描写` 的本地执行细则。

它只负责把“心理活动描写”映射成可执行重写动作，不再复写父级合同。

## 1. 先锁 POV，再写心理

- 先回答：
  - 这一段是谁在感？
  - 他/她现在最多能意识到哪里？
  - 此刻的心理运动是被什么触发的？
- 若回答不出来，先不要写内心句。

## 2. 抽象心理词的默认落地位

| 抽象词 | 优先落地位 |
| --- | --- |
| 害怕 / 紧张 | 呼吸、手心、视线、肌肉反应 |
| 羞耻 / 心虚 | 回避、嘴硬、动作变形、视线不敢停 |
| 欲望 / 心动 | 身体趋近、注意力被勾住、想说又压回 |
| 愤怒 / 不甘 | 力道、忍住、反击冲动 |
| 怀疑 / 动摇 | 判断翻转、记忆回闪、看法松动 |

## 3. 四种常用重写动作

1. `情绪词 -> 知觉`
   例：不是“她很慌”，而是“她听见门轴一响，后背先凉了一寸”。
2. `总结句 -> 半显性自辩`
   例：不是“他其实已经后悔”，而是“他把那句话在心里压回去，像压一口不该翻出来的旧火”。
3. `概念判断 -> 当下判断`
   例：不是“她终于明白自己爱上了他”，而是“她一听见那名字，先想的不是躲，是他今晚会不会伤着”。
4. `纯心理 -> 心理+动作影子`
   例：让人物的心理变化带出下一拍要靠近、回避、出手或沉默。

## 4. 禁写信号

- 像作者讲主题的总结句
- 一口气把人物想法解释完的整齐长句
- 比人物当前认知更早抵达的终局结论
- 只有“痛、难受、复杂、震惊”等抽象词，没有场面触点
