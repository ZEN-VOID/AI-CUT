# 镜花 创作示例

## 使用说明

- 本文件用短片段示范 cinematic canonical 字段与必要 compatibility projection 应该怎样组织，而不是示范正文扩写。
- 示例只负责校准镜头语言与字段落点，不提供可直接套写的现成 prose。

## 正例

### 正例 1：先有 shot spine，再有摄影

弱写法：

```json
{
  "摄影美学": "幽冷、压迫、孤独"
}
```

强写法：

```json
{
  "slot_id": "1-1-1-s01",
  "分镜ID": "1-1-1-1",
  "beat_refs": ["1-1-1-b01"],
  "时间段": {"开始秒": 1, "结束秒": 5},
  "景别": "半身",
  "镜头框架": "前景压肩",
  "摄影美学": "右侧窗光仅照亮半张脸，屋内冷色压住空间，桌沿留一圈暖灯"
}
```

为什么好：

- 先有分镜组织，再让光色依附在具体 shot 上
- 压迫感来自构图和光位，不是空泛评价

### 正例 2：运镜有动机

弱写法：

```json
{
  "运镜手法": "快速推进，制造紧张感"
}
```

强写法：

```json
{
  "slot_id": "1-1-1-s03",
  "分镜ID": "1-1-1-3",
  "beat_refs": ["1-1-1-b02"],
  "时间段": {"开始秒": 9, "结束秒": 12},
  "运镜手法": "话音压低后缓慢推近半步",
  "镜头速度": "慢",
  "摄影美学": "视线收紧到她捏白的指节"
}
```

为什么好：

- 推近发生在明确情绪节点
- 运镜解释了“为什么此刻需要动”

### 正例 3：转场克制

弱写法：

```json
{
  "转场特效": "炫目闪白进入下一组，制造梦幻感"
}
```

强写法：

```json
{
  "slot_id": "1-1-1-s04",
  "分镜ID": "1-1-1-4",
  "beat_refs": ["1-1-1-b03"],
  "时间段": {"开始秒": 12, "结束秒": 15},
  "转场特效": "由窗纸薄亮自然收白，下一组接室外风声"
}
```

为什么好：

- 先利用已有画面条件，再形成组间尾钩
- 转场收益服务时空翻页，没有额外炫技负担

## 反例

### 反例 1：脱离水月发明镜头叙事

```json
{
  "镜头类型": "突然切到窗外乌鸦飞起，再拉回人物脸上，象征命运降临"
}
```

问题：

- 若 `水月` 没有这个感知信息，这就是额外发明事实
- 象征先于叙事，越权了

### 反例 2：摄影形容词堆积

```json
{
  "摄影美学": "冷冽、精致、电影感十足，色彩高级而克制，质感非常丰富"
}
```

问题：

- 没有任何可执行的光位、色温或材料线索
- 属于评价，不是导演/摄影字段

### 反例 3：特效抢主轴

```json
{
  "转场特效": "每个切换都伴随粒子拖尾和残影，突出人物情绪的撕裂"
}
```

问题：

- 效果压过了角色、动作和构图主轴
- 默认强度远超叙事所需
