# Type Package: story-bound-poster

## Purpose

默认海报设计包。它要求海报从当前剧集事实、角色、冲突、高光点和上游风格中长出来，而不是做通用角色宣传照。

## Fixed Context

- 必须优先读取分组剧本与九刀流 JSON。
- 主体必须来自当前集实际出场角色。
- 代表性画面必须能回指当前集真实事件、情绪或冲突。
- 标题可以传播化，但不得伪造本集不存在的反转。
- prompt 必须包含比例、主体、代表性动作/瞬间、前景/背景、文字系统、氛围配色和风格连续性。

## Review Focus

- 这张海报对应本集哪个钩子？
- 去掉标题后，画面是否仍然能看出冲突？
- 主体是否有上游 presence evidence？
- 风格是否继承了上游 style bible，而不是换作品？
