# Actor Performance Control Contract

## Purpose

本细则把“角色演技”从情绪标签升级为可执行的表演控制系统。它服务 `2-编导` 的演员任务、心理反应、对白画面、角色动作、表情特写、环境声音和受控增强，不授权新增剧情事实、对白、事件结果、人物动机或摄影方案。

核心原则：不要只写“开心、愤怒、难过、害羞、紧张”等情绪结论；必须把情绪拆成上游触发点、人物动机、面部肌肉、身体联动、声音/环境承托和视频微动态限制。否则 AI 视频容易得到模板化表情，而不是有演技的角色。

## Ownership

- 本文件拥有 `actor_performance_control_pass` 的表演层细则：微表情、情绪动机、身体联动、环境声音、非对称瑕疵和微动态限制。
- `performance-and-scene-craft-contract.md` 负责场景状态差、演员任务、潜台词行为和场面调度，本文件只细化“演员如何把情绪演出来”。
- `psychological-reaction-contract.md` 负责 `心理反应` 字段语义和 GETability；本文件为 `心理反应`、`表情特写`、`角色动作`、`对白画面` 提供表演变量。
- `field-routing-and-audio-visual-contract.md` 负责字段纯度；本文件不改变字段边界。
- `visual-aesthetic-contract.md` 负责画面美学；本文件不写机位、景别、镜头运动或分镜编号。

## Trigger Scope

必须执行本 pass 的信号：

- 上游出现强情绪、压抑情绪、情绪切换、沉默反应、未出口对白、强忍、试探、羞惭、紧张、愤怒、离别、恐惧、心虚、关系变化或权力压迫。
- 编导稿中出现“很愤怒、很难过、很害怕、很开心、很紧张、崩溃、压抑、复杂”等情绪标签，但缺少面部、身体、呼吸、声线、道具或空间承托。
- `心理反应` 只能靠字段标题理解，正文没有演员可演、观众可见/可听的反应。
- `对白（角色，状态）` 的第二项或 `对白画面` 只写抽象情绪，没有说话时的节奏、呼吸、停顿、视线、手部或对手反应。
- 用户要求“演技更好”“人物更真实”“表情更精准”“不要模板化表演”“AI 人物不僵硬”。

## Actor Performance System

每个关键情绪 beat 应尽量形成以下五层，不要求每个字段逐项全写，但执行报告证据必须说明取舍。

| layer | question | preferred projection |
| --- | --- | --- |
| `emotion_trigger` | 情绪因上游哪个动作、信息、声音、规则或关系变化发生 | `source_anchor`、`trigger`、`scene_dramatic_map` |
| `inner_motive` | 角色想达成什么，又为什么不能直接说或直接做 | `performance_task_map`、`内心独白（主角）`、`心理反应` |
| `micro_expression` | 情绪出现在眉、眼、嘴、鼻翼、咬肌、下颌、皮肤状态的哪里 | `表情特写`、`对白画面`、`心理反应` |
| `body_linkage` | 情绪如何影响肩、手、呼吸、重心、姿态、道具动作和声线 | `角色动作`、`动作画面`、`对白画面`、`音效画面` |
| `ambient_support` | 环境声、余声、沉默、自然声或空间声音如何放大情绪 | `音效` / `音效画面`、`环境描写`、`群像画面` |

## Five-Part Expression Control Formula

### 1. Muscle Zone Control

把情绪落到面部区域，而不是只写情绪名。

可选变量：

- 眉：眉头内收、眉心竖纹、眉峰轻挑、眉尾压低。
- 眼：半眯、睁大、湿润、失焦、闪躲、盯住某一点、眨眼频率变化。
- 嘴：抿嘴、嘴角上扬又压下、嘴角不对称、牙关咬紧、嘴唇发干。
- 面部肌肉：苹果肌隆起、咬肌收紧、鼻翼轻张、下颌锁住。
- 声线关联：尾音收住、气声变轻、字与字之间停顿变长。

弱写法：

```md
表情特写：她很愤怒。
```

强写法：

```md
表情特写：她的眉心压出一道竖纹，眼睛半眯，视线死死钉在对方脸上；咬肌缓慢收紧，嘴唇抿成一条线。
```

### 2. Three-Layer Emotion Motive

关键情绪不要只写单一情绪。优先判断：

- `surface_emotion`：表层正在展示什么，例如平静、笑、沉默、冷淡。
- `suppressed_emotion`：被压住的真实情绪，例如愤怒、害怕、难过、不甘。
- `hidden_motive`：角色为什么压住它，例如不能暴露底牌、不能让对方担心、不能激怒对手。

示例：

```md
心理反应：阿真听见门外脚步声后没有立刻回头，只把杯沿转了半圈；她嘴角还维持着笑，指腹却在杯壁上停住。
```

这类写法让观众看到“表面平静”和“底层紧张”的冲突。

### 3. Non-Facial Physiological Linkage

真实情绪会影响身体。关键情绪 beat 必须优先寻找脸以外的反应。

| emotion pattern | body variables |
| --- | --- |
| 紧张 | 眨眼变快、肩膀内收、手指摩擦、呼吸变浅、脚尖停住 |
| 强忍难过 | 眼眶发红但不落泪、笑意压回去、吸气不完整、手指抓衣角 |
| 愤怒压制 | 指节发白、下颌绷住、身体反而静止、声线压低、动作变慢 |
| 害羞 | 视线下落、身体后缩、手指抓袖口、说话变轻、停顿变多 |
| 心虚/遮掩 | 整理无关物、先低头再抬眼、话到半句停住、错开距离 |

弱写法：

```md
角色动作：他很紧张。
```

强写法：

```md
角色动作：他把杯子换到另一只手，指腹在杯沿上来回摩擦；说话前吸了一口气，却只吸到一半。
```

### 4. Asymmetry And Imperfection Control

真实表演不应完美对称，也不应像模板表情。关键表演可轻量加入非对称、瑕疵和不完全控制，但不得堆成外貌描写。

允许：

- 一侧咬肌比另一侧更明显。
- 嘴角只抬起一边，又很快压下去。
- 眼角湿润但没有落泪。
- 嘴唇略干、脸色发白、皮肤保留细小疲惫感。
- 右手比左手更用力，袖口被抓出皱褶。

禁止：

- 为追求“真实”新增伤疤、疾病、妆容、外貌设定或与上游不符的身体状态。
- 把瑕疵写成剧情线索、伤势结果或新人物设定。

### 5. Micro-Dynamics And Video Restraint

视频表演最怕表情乱跳、动作过大。关键情绪要写变化顺序和幅度限制。

可用结构：

```text
前段静止/假装平静 -> 触发点进入 -> 局部肌肉或身体变量变化 -> 收住或泄露一点 -> 余波停顿
```

示例：

```md
对白画面：她前两个字说得很稳，第三个字前停了一下；嘴角本来要抬起，随即压回去，手指在袖口里慢慢收紧。
```

约束：

- 不用“大喊、崩溃、泪流满面”替代强情绪，除非上游明确支持。
- 不把“微动态”写成摄影慢动作；只写演员动作的快慢、停顿和幅度。
- 不让角色持续做夸张表情；真实情绪常出现在切换瞬间。

## Environment Sound Support

环境声是表演承托，不只是背景。关键沉默、离别、压迫或紧张场必须考虑是否需要声音层帮助观众 GET 到情绪。

| scene emotion | sound support candidates |
| --- | --- |
| 离别 | 远处车声、脚步声渐远、风吹树叶、门轴轻响 |
| 紧张 | 钟表、低频嗡鸣、短促吸气、衣料摩擦、纸张声消失 |
| 孤独 | 空房间回声、雨声、冰箱/电流低频、窗框轻响 |
| 希望 | 远处孩童笑声、清晨鸟鸣、风声变轻 |
| 焦躁 | 键盘/打字声、电话铃、急促脚步、杂声叠加 |
| 压迫 | 群体忽然安静、杯底磕桌、金属轻响、呼吸被压低 |

使用限制：

- `音效` 写声音本体，`音效画面` 写声音和人物反应/空间承托。
- 环境声不得新增事件、人物、线索或外部因果。
- 能用环境声表达的沉默，不要新增对白解释。

## Projection Targets

表演控制结果必须进入既有字段，不新增 `演技提示词`、`表情控制公式`、`演员调度公式` 等正文第二体系。

| performance material | target fields |
| --- | --- |
| 微表情 | `表情特写`、`对白画面`、`心理反应` |
| 身体联动 | `角色动作`、`动作画面`、`对白画面`、`心理反应` |
| 声线/呼吸 | `对白（角色，状态）` 第二项、`对白画面`、`音效画面` |
| 道具动作 | `角色动作`、`道具特写`、`对白画面` |
| 环境声 | `音效`、`音效画面`、`环境描写` |
| 情绪动机 | 规划证据、`内心独白（主角）`、可见反应；不得写成动作字段主观意图 |
| 微动态限制 | `角色动作`、`对白画面`、`表情特写` 中的停顿、压住、收回、变轻 |

## Review Checklist

- 情绪是否有上游触发点，而不是凭空贴标签。
- 是否至少给出一个微表情变量和一个非面部身体变量；关键情绪场建议两类以上。
- 是否能看出表层情绪、压制情绪或隐藏动机中的至少两层。
- 是否用呼吸、手部、肩颈、重心、声线、道具或空间距离参与表演。
- 是否用环境声或沉默承托关键情绪，而不是只依赖 BGM 或解释性文字。
- 是否避免模板化表情：只写“皱眉、瞪眼、流泪、大笑”但缺少动机和微动态时不合格。
- 是否避免过演：没有上游依据时，不写崩溃、大喊、泪流满面、夸张扭曲。
- 是否避免摄影越权：不写镜头推进、特写、慢镜、快切等摄影方案；只写演员和声画材料。
- 是否全部内嵌到既有字段，没有新增第二套正文体系。
