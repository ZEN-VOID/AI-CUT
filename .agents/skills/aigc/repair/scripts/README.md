# Scripts Boundary

`aigc-repair` 当前不提供专用自动化脚本。本目录用于未来放置只读定位、diff、格式转换、validator 或 report projection 脚本。

允许：

- `rg` / diff / path collection。
- repair report 投影。
- 结构化 JSON/YAML 校验。
- provider evidence 路径检查。

禁止：

- 用脚本生成 canonical creative truth。
- 用脚本替代豆包或 owning stage 的创作性判断。
- 用脚本自动覆盖图像、视频、阶段主稿或项目记忆。
