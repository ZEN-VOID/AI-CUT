# 玄幻剧 Genre Type

本类型卡定义 `玄幻剧` 在 `4-编剧` 中的题材策略。它不是子技能，不拥有输出路径、节点、gate 或最终审查权。

## Identification

| field | value |
| --- | --- |
| `type_id` | `genre-xuanhuan` |
| `axis` | `genre_axis` |
| `aliases` | 玄幻、修炼、仙侠化世界、宗门、秘境、灵力 |
| `primary_signals` | 修炼境界、灵力/灵气、宗门等级、血脉、法宝、秘境、天命、阵法、丹药、神魂、妖兽 |

## Strategy

| strategy_slot | rule |
| --- | --- |
| `dramatic_pressure_bias` | 把冲突落到境界差、宗门秩序、资源争夺、血脉/身份暴露、试炼风险和规则代价。 |
| `rhythm_bias` | 适合“规则压力 -> 异象/证据露出 -> 旁人误判 -> 境界或代价确认 -> 尾钩未解规则”的节奏。 |
| `controlled_supplement_bias` | 可补灵力反应、阵纹变化、法宝状态、身体承压、旁人识别规则的可见证据；不得新增体系规则或能力结果。 |
| `field_material_bias` | 宗门大殿、试炼台、秘境入口、阵纹、灵石、丹炉、法器、血脉纹路、灵压、裂开的地面、群修反应。 |
| `climax_hook_bias` | 高潮优先落到规则被触发、身份/境界证据显现、代价出现、法宝或阵法状态改变。 |

## Prohibited Drift

- 不得新增 source 不支持的境界、法宝功能、血脉身份、宗门规则或世界规则。
- 不得用抽象“威压拉满、天命感”替代可见异象、身体反应和规则后果。
- 不得为了爽点强行越级改变胜负或事件结果。

## Combination Notes

- `正剧 x 玄幻剧`：可用角色对白、动作和场面反应解释规则压力，但必须有 source basis。
- `解说剧 x 玄幻剧`：旁白承接世界规则和关系状态，画面用阵纹、法器、身体承压和群像反应承托。
