# 全局风格词最佳实践

本文件是 `aigc/2-Global` 生成 `project_global.全局风格` 与 `groups[].global.全局风格` 时的专项规则展开。它只细化风格词写法，不拥有入口路由权；若与 `SKILL.md` 冲突，以 `SKILL.md` 为准。

## 默认适用范围

- 默认对象：真人版古装影视项目。
- 默认载体：运行时 `第N集.json.project_global.全局风格`，并默认同步到 `groups[].global.全局风格`。
- 默认用途：作为 AIGC 生图 / 生视频统一视觉风格前缀，而不是剧情摘要、镜头清单或工具参数。
- 若项目源层明确要求非古装、非真人、动画、漫画或强 stylized 方向，应先读取 `types/type-map.md` 形成 `style_type_profile`，再做等价改写。

## 推荐定稿基线

真人版古装影视风格，高清电影级画质与真实摄影语法。强调真实人物皮肤与服饰纹理、自然毛发细节、可感材质反射与空间层次；光影严格遵循场景动机光与物理照明逻辑，保留高光不过曝与暗部细节可读。构图采用电影化纵深与权力关系调度，前中后景关系清晰。镜头语言克制稳定，动作反馈符合真实物理，不使用动画描线、赛璐璐分层或夸张残影。整体统一为写实、细腻、4K HDR、影院级调色与轻胶片颗粒。

## 组织规则

| slot | 必须覆盖 | 避免写法 |
| --- | --- | --- |
| `媒介属性` | 真人版古装影视、高清电影级画质、真实摄影语法 | 把项目写成动画、国漫、插画、游戏 CG |
| `人物与服饰质感` | 真实人物皮肤、服饰纹理、自然毛发、可感材质反射 | 瓷质皮肤、线稿、角色建模、夸张发丝渲染 |
| `光影逻辑` | 场景动机光、物理照明、高光不过曝、暗部细节可读 | 无来源强光、非物理辉光、过曝神光 |
| `空间与构图` | 电影化纵深、权力关系调度、前中后景清晰 | 固定某个景别、固定机位或过细焦段参数 |
| `镜头与动作` | 镜头语言克制稳定、动作反馈符合真实物理 | 大幅残影、粒子拖尾、非物理飘动、动画化速度线 |
| `后期底座` | 写实、细腻、4K HDR、影院级调色、轻胶片颗粒 | 糖果色、赛璐璐分层、工具参数堆叠 |

## 字数与改写

- 默认最终字段建议 180-260 中文字。
- 用户或项目要求更长时可以放宽，但必须仍然是一句可继承的统一风格前缀。
- 可删减顺序：先删重复形容词，再删宽泛审美词，最后才压缩物理光影、真实材质与反动画化禁区。
- 不要为了凑短把禁区删掉；“不使用动画描线、赛璐璐分层或夸张残影”是真人古装写实基线的防污染钩子。

## 交付门

完成 `N3-PROJECT-GLOBAL` 前，必须检查：

1. 字段没有剧情摘要、角色命运判断、具体镜头列表或工具参数。
2. 字段能直接被 `3-Detail / 5-Image / 6-Video` 继承为统一风格前缀。
3. 字段包含真实人物质感、服饰/材质纹理、物理光影、空间层次、克制镜头、真实动作反馈和影院级后期底座。
4. 字段不含动画描线、赛璐璐、夸张残影、粒子拖尾、非物理辉光等污染词。
