# Template: 元素替换 / 换角色

Template ID: `photoGPT.元素替换.换角色.v1`

## Input Roles

- 图一：待编辑原图，提供构图、姿态、服装/场景关系、镜头和光影。
- 图二：角色参照图，提供新角色身份、脸、发型、体态气质和必要服装特征。

## Prompt Frame

```text
保留图一的构图布局、镜头视角、动作关系、场景空间、光影方向和叙事位置，将图一中的目标角色替换为图二中的角色身份。

角色替换要求：完整还原图二角色的面部五官、发型发色、肤色质感、体态比例和整体气质；替换后的角色必须自然融入图一的动作、服装/道具关系、场景透视和光照环境。

保留锁定：图一中的镜头、背景、构图、动作节奏、接触阴影和非目标角色严禁漂移；若用户指定保留图一服装，则服装也必须锁定。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变非目标角色、背景和构图。
- 不混合两位角色造成身份漂移。
- 不新增剧情、道具或场景元素。
