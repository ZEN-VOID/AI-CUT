# 职场婚恋题材模板 (Workplace Marriage)

> **核心卖点**: 职业成长 + 情感博弈 + 都市现实。在职场的硝烟中，谈一场势均力敌的恋爱。

## 1. 核心流派细分

### 办公室恋情流
- **特点**: 同一个公司的上下级/平级恋爱——"白天是同事，晚上是恋人"
- **核心爽点**: 职场暧昧、秘密恋情、公开关系后的反应
- **创作要点**: 职场恋情的"禁忌感"是核心吸引力——被发现的风险

### 职场逆袭流
- **特点**: 草根在职场打拼+遇见爱情
- **核心爽点**: 职场晋升+感情双丰收
- **创作要点**: 职场线和感情线要有机交织——不是"各写各的"

### 商战博弈流
- **特点**: 竞争对手/合作方之间的爱情——"你是我的对手，也是我的心跳"
- **核心爽点**: 智力博弈+情感拉扯——"这局你赢了，但我心甘情愿"
- **创作要点**: 商战要真实——不能是"总裁一句话就收购了"

## 2. 追读心理

| 驱动力 | 执行手段 |
| --- | --- |
| 职场真实感 | 办公室政治、甲方乙方、996——读者看到的是"我的生活" |
| 情感对等 | 两人都有事业和尊严——没有"依附" |

## 3. AI 写作专项指南

### 常见陷阱
1. **职场写成背景**: 办公室就是换个地方谈恋爱 → 职场规则必须推动剧情
2. **霸总职场**: 总裁靠"有钱"解决一切 → 展示真实商业逻辑
