适用于默认项目链路：`projects/aigc/<项目名>/2-编导/第N集.md`。

## Fixed Context

- 保留 `2-编导` 的 frontmatter、`【剧本正文】`、场景标题、字段顺序和对白。
- 运动强化只新增或修订 `运动强化：` 字段，不删除原编导句。
- `2-编导` 已经提供角色动作、表演状态、空间关系和视觉句；本阶段只补运动起点、路径、终点和参照系。

- 报告写 `projects/aigc/<项目名>/3-运动/执行报告.md`。
