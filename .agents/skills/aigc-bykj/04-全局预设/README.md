# 04-全局预设

BYKJ `04-全局预设` 把三种入口收束为统一的通用全局视觉预设。核心产物不是单一风格名，而是：

- 300-500 字中文 `global_style_prompt`，对齐 `.agents/skills/aigc` 的 `north_star.yaml` 全局风格口径。
- `global_style_collection`，包含组别风格集合和设计主体风格集合。
- 图片参考模式包含单/多图权重、逐图证据、可迁移边界和去污染审计。
- 自动解析模式包含 `02` 全本覆盖、组别候选、设计主体候选和证据不足退化标记。
- 通用结构化 JSON、关键词、负面词、场景变体和下游使用边界。

三种入口：

1. 用户指定预设风格。
2. 用户提供参考图并按 12 维视觉风格解析还原。
3. 自动读取 `output/[项目名]/02-剧本处理/` 全本剧本，从故事来源归纳全局视觉风格。

默认输入：

```text
output/[项目名]/02-剧本处理/
```

默认输出：

```text
output/[项目名]/04-全局预设/
├── 全局风格预设.json
├── 视觉风格库.json
├── 全局预设.md
├── 执行报告.json
└── manifest.json
```

本阶段只建立全局风格真源，不改写剧本、不提取资产、不生成分镜或图像任务。
