# 2-摄影美学 模块说明

## 定位

- 本分支负责在已稳定的 shot spine 上补光影、色彩和质感，形成 `cinematography_patch`。
- 它拥有画面执行气候的判断权，但不拥有反向改写分镜骨架的权利。
- 它不是一上来就把 `光影 / 色彩 / 质感` 三叶子平铺展开，而是要先把上游摄影承诺压成当前组可执行的中层策略。

## 使用方法

- 先回看上游摄影底座，明确这组必须继承的风格走廊、导演情绪目标和禁区。
- 再锁 `visual_control_line`，把主体控制、空间剥离、质感方向和观看压力先压清。
- 再收束 `cinematography_strategy_note`，决定这组最该坚持的布光或色彩优先级，以及放弃了哪些备选方向。
- 完成上述中层收束后，再下钻 `光影 / 色彩 / 质感`，汇流顺序固定为 `光影 -> 色彩 -> 质感`。
- `光影` 产出后还要补 `group_lighting_note`，说明组内镜头怎样共享一套光影推进语法，再由总协调压成单一 `cinematography_patch`。
- 任何摄影信息都必须能挂到具体 shot 上，且要显化 `1-水月` 已给出的动作、情绪、空间或关系。

## 失真与修正

- 若一上来就直奔叶子模块，说明跳过了摄影底座回看和视觉控制线收束。
- 若只剩漂亮形容词，说明没有回到具体光位、色温或材料。
- 若每镜都有布光描述但组内没有统一推进，说明缺失 `group_lighting_note`。
- 若为了摄影气质改动了分镜数或插入点，说明越权。
- 若风格很好看但故事收益不清，先保留镜头可读性和上游摄影承诺，再谈审美强化。
