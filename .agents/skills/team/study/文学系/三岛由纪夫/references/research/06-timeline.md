## 年表

| 时间 | 事件 | 对编剧蒸馏的意义 |
|---|---|---|
| 1925-01-14 | 生于东京，本名平冈公威（Hiraoka Kimitake） | 出身、家庭、阶层和祖母影响构成早期“面具/礼法/病弱身体”底色 |
| 1941 | 以《花ざかりの森》出道，并开始使用“三島由紀夫”笔名 | 早期就显示高度修辞与自我造型意识 |
| 1944 | 《花ざかりの森》结集出版；东京帝大法学部入学 | 战时语境与“本该死去却活下来”的经验形成持续阴影 |
| 1948-1949 | 在财政系统工作后离开 | 明确选择“作者人生”而非官僚人生 |
| 1949 | 《仮面の告白》成功 | 面具、欲望、羞耻、自白结构成型 |
| 1950 | 《愛の渇き / Thirst for Love》 | 欲望与毁灭进一步强化 |
| 1951-1953 | 《禁色 / Forbidden Colors》 | 面具、同性欲望、社会表演继续扩大 |
| 1954 | 《潮騒》出版；同年写歌舞伎《鰯売恋曳網》 | 纯爱/神话化身体与戏剧跨媒介能力并行 |
| 1956 | 《金閣寺》与《近代能楽集》 | 美与毁灭、古典结构现代化，两条关键主线同年成熟 |
| 1958 | 结婚（Britannica / 外部研究常提及） | public persona 与家庭体面并存 |
| 1960 | 出演电影《からっ風野郎》；《宴のあと》出版 | 主动进入电影和公共形象经营 |
| 1961 | 《憂国》发表 | 忠诚、身体、死亡和仪式的极端表达成型 |
| 1963 | 第一次诺奖提名 | 国际声望已高度稳定 |
| 1964 | 第二次诺奖提名；构思《豊饒の海》 | 从单书写作转向总工程 |
| 1965 | 第三次诺奖提名；《サド侯爵夫人》；《憂国》电影 | 戏剧与影像进一步把文字兑现为场面 |
| 1967 | 第四次诺奖提名；《春の雪》出版 | 《丰饶之海》启动，时间-轮回结构进入成体系创作 |
| 1968 | 第五次诺奖提名；组建盾会 | 文字、组织、身体、政治表演彻底合流 |
| 1969-05-13 | 与东大全共斗在驹场进行四小时公开辩论 | 三岛式 public conflict 的代表场面 |
| 1969 | 《奔馬》出版 | 理想主义、行动、暴烈历史想象进一步推进 |
| 1970 | 《暁の寺》《天人五衰》阶段；仍考虑写藤原定家历史小说 | 说明晚期不是单线奔向死亡，仍在规划写作工程 |
| 1970-11-25 | 交《天人五衰》定稿，同日于市谷事件后切腹 | 作品闭环与人生结局被设计成同场完成 |
| 1999-07 | 三島由紀夫文学館开馆 | 官方研究与传播进入制度化阶段 |
| 2020 | 死后 50 周年，多篇研究重新讨论《丰饶之海》与其死亡关系 | 当代评价更强调“结构与争议同时成立” |
| 2025 | 生诞 100 周年，多国机构展开纪念项目 | 三岛全球影响继续活跃 |
| 2025-2026 | Japan Society 推出 Yukio Mishima Centennial Series: Emergences | 说明其戏剧与跨媒介遗产在当代仍被持续改编和讨论 |

## 最近 12 个月动态（以 2026-04-16 为准）

### 1. 生诞 100 周年延续到国际文化活动

- Japan Society 官方页面显示，2025 年起启动了 **Yukio Mishima Centennial Series: Emergences**，以世界首演、北美首演和能乐关联项目纪念其百年诞辰。  
- 这说明三岛在今天仍主要以“戏剧、改编、跨媒介灵感源”被持续调用。

### 2. 日本与海外的纪念展持续活跃

- 文学馆与相关新闻均显示 2025 年围绕百年诞辰有专题展与纪念活动。  
- 这意味着三岛不是“学术孤岛作者”，而是仍具有稳定公共传播能量。

### 3. 出版侧继续扩容

- Penguin Random House 作者页截至 2026 仍在持续维护其书目页；近年英文世界有《Voices of the Fallen Heroes》等新一轮整理出版。  
- 对 Skill 的意义是：三岛接受史仍在更新，尤其是短篇与戏剧材料在英语世界继续扩散。

## 时间线结论

三岛最关键的时间线不是“早慧 -> 成名 -> 极端死亡”这么简单，真正的主线更像：

1. 早期病弱与礼法环境  
2. 假面与欲望写作成名  
3. 美、古典、毁灭、戏剧结构成熟  
4. 身体与 public persona 加入工程  
5. 作品总工程与现实总场面在 1970 年重叠闭合

这条线比任何单一标签都更适合转成编剧工具。
