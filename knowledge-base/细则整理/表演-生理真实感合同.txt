## Purpose

本细则定义 `2-编导` performance layer 中的生理真实性约束：情绪变化必须符合人体生理过渡规律，不能瞬间切换，不能无痕迹残留，不能无代价消耗。它不是医学参考手册，而是确保 AI 生成的表演在生理层面让观众信服的硬性规则。

核心原则：演员是肉体，不是情绪开关。愤怒到平静需要时间，大哭到正常需要过渡。任何违反人体生理过渡规律的表演都会让观众出戏——不是因为他们懂生理学，而是因为他们自己就是肉体，知道这些过渡不可能瞬间完成。

## Ownership

- 本文件拥有情绪生理过渡的规则库：六组标准过渡、情绪-生理因果链、生理残留效应、极端情绪的生理代价。
- `psychological-reaction-contract.md` 负责心理反应的可感知化；本文件为心理反应提供生理真实性约束——反应不是凭空出现的，它必须有生理过渡支撑。
- `actor-performance-control-contract.md` 的 Five-Part Expression Control Formula 提供微表情和身体联动的变量；本文件决定这些变量的时序约束——哪些必须先出现、哪些必须延续、哪些之间需要间隔。
- `performance-and-scene-craft-contract.md` 负责场景状态差；本文件确保跨场景的情绪状态变化遵守生理规律。
- `character-arc-performance-contract.md` 负责跨集弧线表演规划；本文件确保弧线变化不违反单次情绪的生理过渡。
- `../../_shared/lived-in-character-behavior-contract.md` 提供角色活人感的小事和下意识反应；本文件确保这些小事在生理层面可信。

## Six Physiological Transition Rules

以下六组规则覆盖最常见的表演情绪切换。每组定义了从前状态到后状态的生理过渡路径、必需的生理表现和最低过渡节拍数。

### 1. Crying → Normal（哭泣 → 平复）

| attribute | specification |
| --- | --- |
| `from_state` | 哭泣中或刚结束哭泣 |
| `to_state` | 表面恢复平静 |
| `physiological_transition` | 眼眶红肿（持续）→ 鼻音残留（逐渐减弱但不消失）→ 手背擦拭眼泪/鼻水 → 深呼吸（胸腔起伏明显）→ 吸鼻子 → 试图控制面部肌肉 → 嘴角微微绷住 |
| `minimum_beat` | 3-5 拍 |
| residuals | 眼眶红肿在场景结束前不消失；鼻音可能持续整个场景；声音比哭之前略微沙哑或不稳定 |
| wrong_example | "大哭结束，下一拍立刻用正常声音说台词" |
| right_example | "大哭结束，先吸了两下鼻子，用手背擦了一下眼睛，深吸一口气才开口说话，但声音还是有点发抖" |

### 2. Anger → Calm（愤怒 → 平静）

| attribute | specification |
| --- | --- |
| `from_state` | 愤怒中或刚爆发完愤怒 |
| `to_state` | 表面恢复冷静 |
| `physiological_transition` | 指节从紧握慢慢松开（先松小指，最后松拇指）→ 下颌线从紧绷逐渐放松 → 呼吸从急促变深长（先快后慢）→ 肩线从耸起回落 → 面部潮红逐渐消退 |
| `minimum_beat` | 2-4 拍 |
| residuals | 声线比平时略微紧绷或发硬；说话节奏偏快或偏重；手指可能仍有微小的握紧-松开循环 |
| wrong_example | "愤怒吼完一句话，下一拍就用温柔语气说话" |
| right_example | "吼完之后先闭了一下嘴，手指慢慢松开，呼了一口长气，再开口时声音比刚才低了两个音调但还是有点硬" |

### 3. Exertion → Still（剧烈运动/体力消耗 → 静止）

| attribute | specification |
| --- | --- |
| `from_state` | 剧烈运动、奔跑、搏斗或体力消耗后 |
| `to_state` | 停下来恢复 |
| `physiological_transition` | 喘息（张嘴呼吸）→ 胸腔明显起伏 → 额头微汗或出现汗珠 → 双手可能撑膝或扶墙 → 呼吸频率从急促逐渐放缓但不能立刻恢复 → 肌肉有延迟性震颤 |
| `minimum_beat` | 2-3 拍 |
| residuals | 汗水在后续场景中持续；说话时偶尔有气口不足；动作幅度比平时略小（肌肉疲劳） |
| wrong_example | "跑完一公里，停下来立刻开始完整对白" |
| right_example | "跑到门口停下来，先弯腰撑着膝盖喘了几秒，额头有细密的汗，站直后说第一个字时声音还带着喘" |

### 4. Fear → Composed（恐惧 → 表面镇定）

| attribute | specification |
| --- | --- |
| `from_state` | 恐惧中或刚经历恐惧 |
| `to_state` | 表面恢复镇定 |
| `physiological_transition` | 手指微颤（控制不住但试图控制）→ 瞳孔未恢复（仍偏大）→ 喉头吞咽（明显的喉结上下）→ 呼吸刻意变慢但仍有不稳定 → 后背或手心有冷汗 |
| `minimum_beat` | 3-5 拍 |
| residuals | 吞咽频率比平时高；说话时偶尔有声音发紧或发虚；与威胁来源保持比平时更大的物理距离；手指有不自觉的小动作（如摸耳朵、捻衣角） |
| wrong_example | "刚从极度恐惧中缓过来，立刻用平稳的声音做出完整判断" |
| right_example | "没有危险了，但手指还在微微发抖，先咽了一下口水，慢慢呼了一口气，再开口时声音比平时轻，第一个字有点发虚" |

### 5. Surprise → Neutral（惊讶 → 中性）

| attribute | specification |
| --- | --- |
| `from_state` | 刚受到惊吓或意外信息冲击 |
| `to_state` | 恢复到中性状态 |
| `physiological_transition` | 眨眼频率突然增加 → 嘴唇微张后慢慢合上 → 头部有微小的后仰或偏移 → 吸一口气 → 表情逐渐归位 |
| `minimum_beat` | 1-2 拍 |
| residuals | 惊讶后的注意力更集中（瞳孔聚焦更快）；如果惊讶是负面的，后续可能转入恐惧或愤怒的过渡路径 |
| wrong_example | "听到消息后大吃一惊，下一拍就完全恢复正常" |
| right_example | "听到消息时眼睛眨了两下，嘴张开又合上，过了两秒才说出第一个字" |

### 6. Temperature → Comfort（温度变化 → 舒适）

| attribute | specification |
| --- | --- |
| `from_state` | 刚从寒冷/炎热环境进入舒适环境 |
| `to_state` | 身体适应舒适温度 |
| `physiological_transition` | 皮肤颜色变化（冷→红，热→褪红）→ 呵气或擦汗 → 肩膀放松或收紧 → 与温度相关的下意识动作（搓手、拉衣领、擦额头） |
| `minimum_beat` | 4-6 拍 |
| residuals | 进入舒适环境后仍有1-2个与温度相关的习惯动作（如冷环境进入后仍在搓手） |
| wrong_example | "从暴风雪中走进室内，下一拍就在正常聊天" |
| right_example | "走进室内先呼了一口白气，搓了两下手，鼻子和耳朵还是红的，过了好一会儿才把外套扣子解开" |

## Emotion-Physiology Causal Chain

情绪不是凭空出现在身体上的，它有明确的生理因果链。表演规划必须尊重这个链路的顺序：

### Primary Causal Chains

| emotion | causal_chain | performance_order |
| --- | --- | --- |
| `anger`（愤怒） | 情绪触发 → 血压上升 → 面部潮红 → 手指收紧（握拳）→ 呼吸变浅变快 → 声线变紧 → 语速变快或音量变大 | 面部先变（1-2秒），手部紧随（2-3秒），呼吸和声音最后（3-5秒） |
| `fear`（恐惧） | 情绪触发 → 肾上腺素分泌 → 瞳孔放大 → 呼吸变浅 → 肌肉紧张 → 手指/肢体微颤 → 声音发紧或发虚 | 眼部先变（瞬时），呼吸紧随（1-2秒），肢体和声音最后（2-4秒） |
| `sadness`（悲伤） | 情绪触发 → 喉头收紧 → 眼眶湿润 → 呼吸不稳（抽泣前兆）→ 嘴角下压 → 肩膀内收 → 声音变低或颤抖 | 喉部先变（1-2秒），眼眶和嘴角紧随（2-3秒），肩膀和声音最后（3-5秒） |
| `joy`（喜悦） | 情绪触发 → 面部肌肉放松 → 眼角出现笑纹 → 嘴角上扬 → 呼吸变轻 → 肩线放松 → 声音变轻变高 | 面部先变（1-2秒），呼吸和姿态紧随（2-3秒） |
| `surprise`（惊讶） | 情绪触发 → 眉毛上扬 → 眼睛睁大 → 嘴张开 → 呼吸暂停 → 身体微后仰 | 眼部和嘴部几乎同时（瞬时），呼吸暂停紧随（0.5-1秒），身体后仰最后（1-2秒） |
| `disgust`（厌恶） | 情绪触发 → 鼻翼皱缩 → 上唇上提 → 喉头反胃感 → 身体后撤或转头 → 呼吸变浅或屏住 | 面部先变（瞬时），喉部和身体紧随（1-2秒） |

### Causal Chain Enforcement

- 表演变量的出现顺序必须符合因果链。愤怒场景中不能先出现语速变快再出现面部潮红——生理上血压上升和面部充血发生在声线变化之前。
- 如果场景需要简化的表演（如远景或快节奏剪辑），可以省略中间环节，但不能颠倒顺序。
- 多种情绪叠加时，每种情绪的因果链独立运行，但在身体上会互相干扰（如愤怒+恐惧=面部潮红和瞳孔放大同时存在）。

## Physiological Residual Effect

前一场景的情绪生理效果必须延续到下一场景，不能在场景切换时被重置：

### Residual Rules

| previous_scene_emotion | required_residuals_in_next_scene | duration |
| --- | --- | --- |
| 大哭 | 眼眶红肿、鼻音、声音沙哑、偶尔吸鼻子 | 至少延续到下一个有时间间隔的场景 |
| 极度愤怒 | 面部残红、呼吸比平时略重、手指偶尔回握 | 2-3个场景逐渐消退 |
| 极度恐惧 | 手指微颤、吞咽频率偏高、与威胁源保持距离 | 持续到角色确认安全之后 |
| 剧烈运动 | 喘息、出汗、肌肉疲劳导致的动作幅度减小 | 根据运动强度，持续2-5个场景 |
| 惊吓 | 注意力过度集中、反应过度（稍有刺激就回头/退缩） | 1-2个场景 |
| 长时间紧张 | 肩颈僵硬、手指有无意识小动作、呼吸偏浅 | 持续到角色真正放松的场景 |

### Scene Transition Application

```yaml
# 正确：上一场景角色大哭，下一场景（30分钟后）
next_scene_residuals:
  眼眶：仍然红肿，但比刚才好一些
  声音：说话时偶尔有鼻音
  行为：比平时更安静，说话更少
  下意识动作：偶尔用手背碰一下眼角

# 错误：上一场景角色大哭，下一场景
next_scene_wrong:
  角色用完全正常的表情和声音与人交谈，没有任何哭过的痕迹
```

## Extreme Emotion Physiological Cost

长时间或高强度的情绪状态会对身体产生代价。这些代价不是可选的表演细节，而是必须体现在后续表演中的生理后果：

### Cost Table

| extreme_emotion | physiological_cost | how_to_manifest |
| --- | --- | --- |
| 长时间哭泣（超过5分钟） | 头痛、脱水、眼睛肿痛、体力下降 | 后续场景中用手按太阳穴、声音比平时哑、动作幅度减小 |
| 极度恐惧（持续超过10分钟） | 虚脱、四肢无力、冷汗、可能呕吐 | 恐惧过后瘫坐在地、站不稳、需要扶墙、脸色苍白 |
| 长时间愤怒（持续高度愤怒） | 声嘶力竭、头痛、手抖、体力消耗 | 吼叫之后声音沙哑到几乎听不见、手握不紧东西、靠墙喘气 |
| 持续高度紧张（超过30分钟） | 肩颈酸痛、头痛、肠胃不适、手心出汗 | 不停揉后颈、按太阳穴、手心湿到拿不住东西 |
| 极度悲伤（含隐忍不发） | 胸闷、呼吸不畅、喉头持续收紧、食欲丧失 | 说话前需要先清嗓子、吃东西时吞咽困难、胸腔起伏比平时大 |

### Cost Enforcement Rules

- 生理代价必须在极端情绪结束后的场景中出现，不能被忽略。
- 代价的严重程度与极端情绪的持续时间和强度成正比。
- 如果角色在极端情绪后需要立即执行高强度动作（如战斗、逃跑），代价会影响动作质量——手握不紧武器、腿发软、视野模糊。
- 角色可以通过意志力部分压制代价的表现，但不能完全消除。压制本身就是一种表演。

## Transition Timing Matrix

表演中情绪切换的最小时间约束：

| from \ to | normal | anger | fear | sadness | joy | surprise | calm |
| --- | --- | --- | --- | --- | --- | --- | --- |
| **normal** | - | 2-3b | 1-2b | 2-3b | 1-2b | 0.5-1b | - |
| **anger** | 3-5b | - | 2-3b | 3-5b | 5-8b | 1-2b | 2-4b |
| **fear** | 3-5b | 2-3b | - | 3-5b | 6-10b | 1-2b | 3-5b |
| **sadness** | 3-5b | 3-5b | 2-3b | - | 5-8b | 1-2b | 3-5b |
| **joy** | 2-3b | 3-5b | 2-3b | 3-5b | - | 1-2b | 1-2b |
| **surprise** | 1-2b | 2-3b | 1-2b | 2-3b | 2-3b | - | 1-2b |

（b = beat，即表演节奏中的最小时间单位）

注意：表中的数字是最低限度。实际表演中，从高强度情绪切换到对立情绪（如极度快乐→极度悲伤，极度愤怒→极度温柔）通常需要更长时间。如果场景要求快速切换，必须有强烈的外部触发事件作为催化。

## Failure Modes

| failure | symptom | fix |
| --- | --- | --- |
| 情绪瞬间切换无过渡 | 角色上一拍在大哭，下一拍就用正常声音说话 | 为每种情绪切换应用 Six Physiological Transition Rules 中的最小beat数 |
| 前场景大哭后场景毫无痕迹 | 场景切换后角色的眼睛、声音、行为完全正常 | 在下一场景中至少保留2-3个生理残留表现 |
| 极端情绪无生理代价 | 角色长时间恐惧后依然精力充沛地行动 | 在后续场景中加入生理代价表现：虚脱、手抖、声嘶等 |
| 生理因果链顺序错乱 | 愤怒场景先出现语速变快再出现面部潮红 | 检查 Emotion-Physiology Causal Chain 中的表演顺序 |
| 高强度情绪到对立情绪瞬间切换 | 极度愤怒下一拍就极度温柔 | 为对立情绪切换添加过渡beat，或添加外部催化事件 |
| 温度变化无身体反应 | 角色从暴风雪走进室内后直接正常活动 | 加入呵气、搓手、皮肤颜色变化等温度适应行为 |
| 隐忍不发的悲伤无代价 | 角色长时间压抑悲伤但身体没有任何体现 | 喉头收紧、胸闷、说话前清嗓子等隐忍的生理表现 |

## Reusable Heuristics

- "演员是肉体，不是情绪开关；愤怒到平静需要时间，大哭到正常需要过渡。"
- "生理残留是场景连续性的锚——前一场景的情绪必须在下一场景留下痕迹，哪怕只是一次吸鼻子。"
- "极端情绪有代价，就像剧烈运动有代价。角色不能在长时间恐惧后依然健步如飞。"
- "因果链的顺序不能乱。生理上先发生的事，表演上必须先出现。"
- "表面镇定和真正平静是两回事。恐惧后的'镇定'手指还在抖，这才是活人。"
- "快速情绪切换不是不可能，但它需要强烈的外部催化。没有催化的快速切换就是表演失真。"
