# Design Style Analysis Contract

`设计风格解析.md` 是 `shot-by-shot` 输出给 `5-设计` 的角色、场景、道具 side context。它只提炼可迁移设计原则，不直接生成正式角色设定、场景设定、道具设定或提示词终稿。

## Required Sections

| section | downstream | requirement |
| --- | --- | --- |
| `## 角色解析` | `5-设计/角色/2-设计` | 身份压力、体态、服装结构、可迁移材质/纹理策略、画面合同边界 |
| `## 场景解析` | `5-设计/场景/2-设计` | 空镜空间秩序、环境压力、可迁移装置关系、无人画面约束 |
| `## 道具解析` | `5-设计/道具/2-设计` | 完整道具主体、功能压力、细节层级、纯色背景 45 度完整近摄约束 |
| `## 视觉转译原则` | all design | 把参考片具体美术表达转译为目标项目自有设定 |
| `## Do Not Import` | all design | 不得复制人物脸、服装纹样、场景构图、道具纹章、地图文字或专属符号 |

## Fixed Image Contracts

- 角色解析服务纯色背景全身服装试装照，无场景环境。
- 场景解析服务空镜、无人、无人物剪影。
- 道具解析服务纯色背景、45 度视角、完整道具近摄、无人物手、无场景环境。
