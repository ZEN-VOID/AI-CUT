# Scripts Boundary

本目录可放置机械辅助脚本，但当前仅保留脚本边界说明。

允许脚本执行：

- 枚举 `projects/aigc/<项目名>/4-分组/第N集.md`。
- 定位分镜组和组底 YAML 块。
- 抽出 YAML `道具` 字段的候选原文。
- 检查 `道具清单.md` 是否为 Markdown table，且列名为 `名称`、`首次登场`、`原文描述（关键词式）`。

禁止脚本执行：

- 自动判断两个候选是否为同一叙事道具。
- 自动过滤背景杂物。
- 自动生成 canonical `名称` 或关键词式描述。
- 从正文、角色表、场景表或外部资料创作新道具项。

上述裁决必须由 LLM 直接完成。
