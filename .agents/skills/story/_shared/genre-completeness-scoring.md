# 题材知识库完整度评分机制

本文件是 `story2026` 的题材知识库完整度评分标准。用于评估 `3-初稿/types/网文/` 下每个题材目录的知识覆盖深度，指导知识库建设优先级。

## 评分维度

每个题材按以下 7 个维度评分，每维度 0-5 分，满分 35 分。

| 维度 | 权重 | 说明 | 评分标准 |
|------|------|------|---------|
| `tropes_and_conventions` | 5 | 题材核心套路与行规 | 0=无；3=列举了常见套路；5=套路+变体+使用场景 |
| `plot_patterns` | 5 | 该题材的情节结构与节奏模式 | 0=无；3=有基本情节模式；5=多种情节模式+节奏规则 |
| `character_archetypes` | 5 | 题材典型人物原型与关系模式 | 0=无；3=列举了原型；5=原型+动机+关系动态 |
| `scene_design` | 5 | 类型化场面的设计与写作要点 | 0=无；3=有场面分类；5=场面功能+双轴路由+写法指导 |
| `anti_trope_strategy` | 5 | 反套路博弈策略 | 0=无；3=提到了避免老套；5=系统反套路策略+替代方案 |
| `quality_filters` | 5 | 题材特化的质量过滤规则 | 0=无；3=有基本质量建议；5=完整质量门禁+失败码 |
| `style_and_voice` | 5 | 题材特化的语言风格与叙事声口 | 0=无；3=有风格描述；5=声口细则+句式指导 |

## 完整度等级

| 总分 | 等级 | 说明 |
|------|------|------|
| 28-35 | **深** | 7维度覆盖完整，可支撑高质量全流程创作 |
| 18-27 | **中** | 关键维度覆盖，需要时可补充 |
| 10-17 | **浅** | 仅有基础描述，多维度缺失 |
| 0-9 | **裸** | 几乎无题材特化知识 |

## 知识文件标准

每个题材的知识文件应遵循以下命名和内容标准：

### 推荐文件结构（中等深度）

```
题材名/
├── 题材名.md             # 总览：核心定义、世界观模板、关键行规
├── plot-patterns.md      # 情节模式：典型结构、节奏规则、变体
├── character-archetypes.md # 人物原型：常见角色类型、动机、关系模板
├── scene-design.md       # 场面设计：类型化场面、双轴路由、写法指导
├── anti-trope.md         # 反套路：常见套路陷阱、反套路策略、替代方案
├── quality-filters.md    # 质量过滤：题材特化的质量门禁、失败码
└── style-voice.md        # 语言风格：声口指导、句式特征、用词边界
```

### 最小可行结构（浅层→中层升级）

```
题材名/
├── 题材名.md             # 已有：核心定义
├── enhanced-tropes.md    # 新增：套路+情节+人物+反套路（合并文件，高效补齐）
├── scene-quality.md      # 新增：场面设计+质量过滤（合并文件，高效补齐）
└── style-voice.md        # 新增：语言风格+叙事声口
```
