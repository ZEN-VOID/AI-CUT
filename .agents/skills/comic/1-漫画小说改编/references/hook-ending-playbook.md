# 章末钩子手册

钩子的本质：打破平衡、留下缺口、触发好奇。

通用公式：

`铺垫（前文已建立） + 打破（台词/动作/画面/信息反转） + 留白（不解释完）`

最重要的负面规则：

- 不把话说完
- 不把事了结
- 不把真相全抖出来

## 1. 六类默认钩子

### 1. 突发反转式

- 用法：在读者以为局面稳定时，突然引入颠覆信息。
- 示例：
  - 他以为一切终于结束，门却被推开，进来的人，是他亲手下葬过的姐姐。

### 2. 致命提问式

- 用法：用一个没有答案的问题收尾。
- 示例：
  - 可她不知道，真正想毁掉她的人，到底是前夫，还是身边一直替她撑伞的那个人？

### 3. 危险逼近式

- 用法：在最紧张的一秒突然停笔。
- 示例：
  - 走廊尽头传来缓慢脚步声，门把一点点压下去，下一秒，门开了。

### 4. 真相留白式

- 用法：只给半截信息，不给最终解释。
- 示例：
  - 她终于看清那张脸，脸色一下白了，连手里的杯子都摔碎了。

### 5. 伏笔回收式

- 用法：把前文随手埋下的线索变成新的阴谋入口。
- 示例：
  - 直到这一刻他才发现，第一次见面时她袖口那道血痕，从来就不是别人的。

### 6. 情感冲击式

- 用法：不用硬悬疑，用一句戳心的话留余韵。
- 示例：
  - 原来她等了那么多年，等来的不是团圆，而是亲手把他送回深渊。

## 2. 可追加参照的钩子类型

### 7. 身份错位式

- 示例：
  - 他摘下面具的那一刻，我才知道，这些年一直追杀我的人，竟然就是我的未婚夫。

### 8. 代价升级式

- 示例：
  - 系统声音再次响起：任务可以取消，但代价，是她的命。

### 9. 时限倒计时式

- 示例：
  - 手机屏幕只剩最后十秒，而门外，已经有人开始输密码。

### 10. 证据失效式

- 示例：
  - 我刚把监控录像交出去，屏幕就突然一黑，那个最关键的人脸，消失了。

### 11. 选择悖论式

- 示例：
  - 只要我点头，所有人都能活，可第一个死的人，会是他。

### 12. 欲望兑现反噬式

- 示例：
  - 她终于得到了想要的一切，可当那枚戒指戴到手上时，她却忽然想起，上辈子她就是死在这只手里。

## 3. 分类型示例

### 言情甜宠 / 虐恋

- 他俯身靠近，温热气息落在耳畔，只说了一句话，就让我浑身僵住。
- 我以为这只是一场逢场作戏，直到他红着眼，说出了那句我从未听过的告白。
- 门被推开，站在外面的人，竟是我这辈子最不想见的那一个。

### 悬疑 / 惊悚

- 我终于找到真相，却发现下一个要死的人，是我。
- 监控里清晰地拍到一张脸，那是早就已经死了的人。
- 身后传来极轻的脚步声，我猛地回头，空无一人。

### 玄幻 / 修仙 / 系统

- 系统提示音再次响起，这次的任务，竟然是让我亲手杀了最信任的人。
- 我抬手接住那道流光，看清里面的东西时，整个人如坠冰窟。
- 他缓缓摘下面具，露出的那张脸，让满殿瞬间死寂。

### 穿越 / 重生 / 复仇

- 我笑着应下，没人知道，上一世就是这句话，将我推进了地狱。
- 她递来的茶杯还带着温热，我却清楚记得，这杯茶里藏着致命剧毒。
- 直到这一刻我才明白，重生一次，我真正要杀的，根本不是仇人。

### 都市 / 热搜 / 现实议题

- 热搜词条刚冲到第一，我的手机就响了，来电备注只有两个字: 撤稿。
- 所有人都以为那段偷拍视频是意外流出，只有我知道，按下发送键的人，是我妈。
- 当城市灯牌亮起的那一刻，她才发现，今晚整座城在等的人，不是明星，而是失踪三年的自己。

## 4. 钩子强度调节

| 强度 | 适用场景 | 写法 |
| --- | --- | --- |
| `balanced` | 甜宠、成长、轻悬疑 | 留一个问题或关系误差即可 |
| `strong` | 复仇、权谋、系统、都市议题 | 需要明确反转或危险逼近 |
| `relentless` | 连载爆款向、强悬疑、极致虐恋 | 几乎每章都在最贵的地方急停 |

## 5. 自检清单

写完章末后，用下面四句检查：

1. 读者会不会立刻想点下一章？
2. 这一章是不是把真相说太透了？
3. 这个钩子和上一章是不是同一种手法？
4. 这个钩子能不能和下章第一场直接咬合？
