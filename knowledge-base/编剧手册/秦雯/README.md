# 秦雯编剧知识库

## 定位

这不是《繁花》剧情摘要，也不是“海派台词风格摘抄本”，而是给 [秦雯 Agent](/Volumes/AIGC/AIGC-FILM/.codex/agents/编剧组/秦雯.md) 提供判断底座的最小编剧手册。

## 目标

- 为年代剧、都市群像、小说改编、上海语境对白和关系戏提供可追溯依据
- 把“繁花感”从腔调、滤镜和旧物装饰，压回关系重排、时代推动、人物分寸、物件承载和对白空隙这些可执行维度
- 在没有私人剧本和内部飞页的前提下，先建立一套可靠、克制、可扩展的基础知识底座

## 当前范围

- 《繁花》改编中的剧作取舍
- 90 年代上海生活与人物命运的关系
- 群像结构与“每个人物自成看点”的写法
- 上海边界感、契约感、职业细节如何进入对白和场景
- 编剧三维写戏与 AI 辅助的边界

## 默认适用任务

- 年代剧大纲修订
- 小说改编方案收束
- 群像关系梳理
- 单场戏重写
- 对白节拍与潜台词校正
- 上海语境人物塑造

## 暂不覆盖

- 逐版剧本考据
- 片场飞页全过程复原
- 白玉兰获奖履历型资料整理
- 粉丝向台词名场面搬运
- 未经资料支撑的幕后八卦

## 使用原则

- 默认先服务编剧动作，不先堆背景资料
- 默认先抽取“为什么这条材料能支撑这次剧作判断”
- 若某条判断无法回溯到现有材料，就降级为经验推断，不写成硬依据
- 导演风格相关判断可借仓内导演手册补强，但不得反向吞掉编剧层的剧作裁决
