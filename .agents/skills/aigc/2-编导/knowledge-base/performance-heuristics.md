# Directing Heuristics

## Practical Heuristics

- 先不要急着“写得像剧本”，先给上游每个信息点找落点；落点稳定后，剧本质感自然会出来。
- 对白周围最容易混入动作，写完后专门扫一遍中文引号，凡是动作都移到 `对白画面` 或 `角色动作`。
- 系统提示适合用 `旁白（系统提示）` 承载文字，用 `系统画面` 做字体、空间、人物反应和规则显影。
- 诡异校园类场景的可拍核心通常在空间压迫、集体静止、规则文字、声音节奏和局部特写之间，不必靠新增剧情制造恐惧。
- 当多个 beat 都发生在同一教室或走廊，场景标题只出现一次，后续用字段推进节奏。
- “他意识到”通常不是动作，也不是合格的 `心理反应`；可转为眼神停顿、手指停住、呼吸变化、道具停点、内心独白或可执行表演提示。
- `心理反应` 以 `references/psychological-reaction-contract.md` 为准，要像演员表演 beat：谁的身体、表情、呼吸、声音、道具或空间关系发生了什么变化；不要写主角抽象想象或作者解释。
- “他不信任/她在试探/他们被压住”不是表演提示；先问角色想达成什么、不能暴露什么，再写停顿、避视、道具动作、身体距离或声线变化。
- 场面调度优先写权力关系：谁站着、谁坐着、谁占据讲台/门口/黑板，谁被群体视线包围，关键道具停在谁手边。
- 沉默不是空白；用纸张声消失、呼吸停住、手指松开、对手不接话、群像迟半拍反应来承托。
- `表演提示` 和 `场面调度` 的规划句不要直接贴到场景末尾；把“想控制风险”拆成低声、停敲、压住关键文书、周围杂声低下去等细节。
- B 路线新增的判断口径：删掉新增细节后，剧情事实、对白、因果、规则和人物决定必须完全不变。
- 能进入 B 路线的新增通常是“质感和承托”，例如环境微动、群体迟疑、手指压纸、道具位置、声音余波；不能是新信息。

## Repair Patterns

| symptom | repair |
| --- | --- |
| 画面字段复述对白 | 改为说话时的表情、对手反应、空间压迫或画面承托 |
| 音效没有画面 | 补声音源头、人物反应或不可见来源的空间处理 |
| 场景标题像摘要 | 改成 `内景/外景 场所 - 日/夜` |
| 独白像旁白 | 判断主体是否在场并承受该经验；若是角色主观经验，用独白或内心独白 |
| 心理反应像内心散文 | 改成主体明确的身体微动作、表情、呼吸、停顿、声线、道具停点、空间距离或对手不接话；纯语言化内容改为独白/内心独白并配画面 |
| 现实灾难插针过长 | 保留为短促画面，不吞掉主场景 |
| 表演提示像心理标签 | 改成目标、阻碍、策略、外显动作；至少写清其中两项 |
| 潜台词靠新增对白解释 | 删除新增对白，改用停顿、视线、道具、身体距离和沉默余波 |
| 场面调度写成镜头方案 | 删除机位、景别、运动和分镜编号，回到人物、空间、道具和视线关系 |
| 场景末尾堆表演/调度总结 | 拆回对应 beat：环境写空间权力，动作写手和视线，对白画面写声音压低和众人反应，道具特写写焦点物 |
| 受控增强变成剧情新增 | 删除新增剧情项，或转为需用户另行授权的候选稿；canonical 只保留有上游锚点的承托项 |

## Atmosphere And Mood Heuristics

- 氛围不靠堆形容词，靠一个精准的感官细节。"杯壁上的指印已经干透"比"房间空荡荡的"更有孤独感。
- 声音消失比声音出现更有力。当一个场景需要压力时，先想"什么声音该停了"。
- 通感的要诀：让不同感官为同一情绪服务。寒冷不只是温度——它可以是月光的颜色、风声的质感、石板的触感。
- 反衬是高级技法：用温暖的灯火写离别，用安静的午后写恐惧，用完整的日常物件写刚刚发生的暴力。观众会自己完成那一步推导。
- 微观放大：情感越强，画面可以越小。巨大的悲伤场不一定需要暴雨和狂风——一杯慢慢变凉的茶、一张被风吹起又落下的纸，往往更有力。
- 留白不是什么都没写，而是写了"事件之后的痕迹"。翻倒的茶杯还在原位、杯中的水被震出一半——观众会想象刚才发生了什么。
- 声景要有层次，不要只写一个声音。远处鼓声→近处呼吸→全部消失→灯芯爆一下——四层声音比一个"安静"有力十倍。
- 时间感是被忽视的意境维度。灰尘的厚度、蜡泪的堆积、杯中茶凉后水面的薄膜——这些都是"时间在画面上"。
- 不要怕"浪费"环境描写。同一个 slugline 内，环境刷新可以让观众感受到时间在流逝、压力在变化。但从室内到门廊、从主厅到角落的环境变化需要真实的空间焦点转移。
- 意境是锦上添花，不是替代戏剧功能。一个没有冲突和人物选择的场景，再美的意境也是空壳。

## Gait And Movement Pattern Heuristics

人物步态是可见的行动语言。写清楚步态能让观众仅从走路方式就读出角色状态、性格和当下情绪，而不需要额外解释。

### Gait Writing Requirements

每个角色在关键场景应有明确的移动方式：

| 步态要素 | 写法要求 | 示例 |
|---------|---------|------|
| 起脚方式 | 脚跟先离地还是脚尖先起，重心在哪只脚 | `他左脚拖在地上，迟迟不肯抬起来` |
| 步幅大小 | 大步还是碎步，步间距是否均匀 | `她一步一步挨过去，步子越来越小` |
| 速度节奏 | 匀速、忽快、忽慢、停顿踉跄 | `他走得像在赶什么，又像不敢走太快` |
| 身体重心 | 前倾、后仰居中、下沉 | `重心压在脚跟上，身子往后仰` |
| 手臂摆动 | 自然摆动、僵硬贴体、一只不动 | `右手插在口袋里，左手垂着不动` |
| 头部位置 | 抬着、低着、偏向一侧 | `头压得很低，只看着自己脚尖` |
| 行走目的 | 去哪里、见谁、为什么停 | `他在门口停住，脚尖抵住门槛` |

### Body Linkage Rules

身体联动让情绪不只是"面部表情"，而是全身协同的反应。

核心原则：一个情绪出现时，身体多个部位应有协同反应，而不是只有脸。

| 情绪类型 | 联动部位 | 典型写法 |
|---------|---------|---------|
| 紧张/恐惧 | 肩→手→呼吸→重心 | `她的肩耸到耳边，手在袖口里攥紧，呼吸发浅，脚步变快又变慢` |
| 愤怒/压抑 | 下颌→手→重心→声线 | `下颌收紧，指节压得发白，重心往下沉，声音却压得更低` |
| 难过/强忍 | 眼→嘴角→手→呼吸 | `眼眶发红，嘴角却还挂着笑，手指在衣角上慢慢揉搓，呼吸停在胸口` |
| 害羞/心虚 | 头→视线→手→身体 | `头低下去，视线绕开，手不知道往哪儿放，身体往后缩了半寸` |
| 期待/兴奋 | 眼→呼吸→重心→动作 | `眼睛发亮，呼吸变快，重心前倾，脚尖在地上轻轻点着` |
| 犹豫/徘徊 | 脚→重心→手→视线 | `脚尖在门槛边磨了两下，重心换来换去，手垂在身侧捏着袖口` |

### Action Chain Breakdown Method

动作链把一个复杂行为拆成可执行的最小单元。每个关键动作写进终稿前，必须能回答：

1. **起点状态**：人物从什么姿态、站/坐/蹲/躺开始
2. **动作向量**：正在向哪里移动、转身、伸手、停住、退开
3. **可达性**：若触碰对象，对象是否在空间可达范围内
4. **互动方式**：如何接触、取用、避开或被阻碍
5. **终点状态**：动作结束后姿态、位置、朝向、身体接触

示例：

```md
# 弱写法：一个模糊动作
角色动作：林寂走上前。

# 强写法：动作链拆解
角色动作：林寂从座位起身，绕过前排同学的椅背，在第三排空位前停住脚步；他的脚尖朝前，但身体还没有完全转过来。
```

### Role-Specific Writing for Multi-Character Scenes

多人场景不能只有主角的戏。写多人场景时，先分配每个角色的权重：

| 角色类型 | 戏份权重 | 写法要求 |
|---------|---------|---------|
| 主角 | 核心驱动 | 完整动作链、心理反应、对白、内心独白 |
| 对手/关键配角 | 次级驱动 | 动作反应、对白、对主角的判断 |
| 群体 | 环境承托 | 群像反应、集体动作、差异化层次 |

多人场景检查清单：
- 每个关键角色在这个 beat 里是否有明确动作或反应？
- 群体的反应是否有 2-3 个差异化层次，而不是全员同频？
- 主角的对手是否在观察/回应主角的行为？
- 动作链是否在多角色间形成因果，而不是并列？

```md
# 弱写法：只有主角
角色动作：林寂握紧拳头。

# 强写法：主角+对手+群体
角色动作：林寂的手慢慢握紧，指节发白。
群像画面：前排有人转过头，又迅速低回去；后排几个人的脚往课桌下缩了半寸。
对白画面：苏红叶的视线在林寂手上停了一拍，嘴角微微一动，又收回去。
```

### Actor Action Chain And Reachability Rule

表演阶段必须把心理、潜台词、沉默和情绪转成连续的人物动作链，而不是并列堆砌动作或借无互动对象制造"电影感"。

每个关键 beat 写入终稿前，内部必须能回答：
- 人物此刻是站、坐、蹲、躺、靠、俯身还是移动中。
- 人物相对他人、门口、桌面、床边、墙面、通道等空间锚点处于什么方位。
- 人物动作向量是什么：靠近、退开、转身、伸手、停住、收回、坐下、起身、保持。
- 如果人物触碰、取用、查看或避开某对象，该对象是否在可达距离和方向上。
- 这个 beat 结束后，人物姿态、位置、朝向、身体接触和注意力落点是什么，能否交给下游摄影和分组。

强情绪 beat 默认只保留最能改变状态差的少数外显动作。连续擦泪、抽泣、避视、摸物、整理衣角、听见环境声等动作，只有在形成清楚起点、过程、结果或角色策略变化时才并列保留；否则优先删掉低信息动作，保留呼吸、重心、手部、肩颈、下颌、眼神、声线、停顿等最直接可演变量。

当动作链与道具承托冲突时，动作链优先。若道具没有互动、关键信息、规则/证据/危险源或必要环境功能，不得把人物影子、反射、轻响、涟漪、桌面边缘等物件反应写成表演节点。若必须保留对象，应让人物真正触碰、移动、查看、避开或受其阻碍，并写清可达空间关系。

## Known Pitfalls

- 把“好莱坞级”理解为更自由的二创，会直接破坏当前阶段的 canonical 职责。
- 把 `动作画面` 当作万能字段，会把心理、规则和背景说明都塞进去，导致下游无法拍。
- 把 `心理反应` 当作万能内心字段，会让工具和观众都只能读字理解；它必须回到表演、声画或道具反应。
- 同一 slugline 重复打印场景标题，会让分组阶段误判为多个场景。
- 把“高段位表演”写成理论词会让演员无事可做；它必须表现为可执行任务和外显行为。
- 把权力关系写成“掌控全场”“关系改变”会失去画面证据；应转为空间站位、视线压力和道具归属。
- “更影视化”不是默认可以加戏；没有 `controlled_enrichment_ledger` 的新增细节很容易滑向自由改编。
- 末尾总结块会让编导稿像审稿批注而不是可拍剧本；终稿应让表演和调度在对应动作时发生。
