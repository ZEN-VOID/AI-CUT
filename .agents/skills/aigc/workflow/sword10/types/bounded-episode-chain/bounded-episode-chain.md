# Bounded Episode Chain

适用于单集或少量明确分集的 `2-编剧 -> 3-美学 -> 4-导演 -> 5-表演 -> 6-氛围 -> 7-分镜 -> 8-摄影 -> 9-光影 -> 10-分组` 连续推进。

## Fixed Context

- 阶段内可以并发，但分集数量小，主窗口应优先给出逐集状态。
- 任一集失败时，默认停止整个链路，等待用户确认 retry 或 repair。
- 输出报告按每集列出 2-10 阶段产物路径；`3-美学` 同时列出父级总览和 6 个风格协议路径。

## Gate Bias

保守处理失败：一集失败通常意味着链路上下文或阶段合同需要先修，不急于推进其他集。
