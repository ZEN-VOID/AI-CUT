# Inactive Init Option Card

This template is inactive for current `$aigc-init`.

Current initialization no longer asks the user to choose automatic or custom team lineup. It needs a project name/path and any initialization information the user wants preserved; team configuration, collaborator/reviewer preferences, reference material summaries, production constraints, and long-term requirements are recorded in `MEMORY.md`. It also creates the neutral project `CONTEXT/README.md`.

If a future workflow reintroduces executable team selection, it must declare a new active output contract and still consume user-specified team preferences from `MEMORY.md` first.
