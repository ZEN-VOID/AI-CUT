# Core Concepts

## 1. “不响”必须被写出响度

《繁花》原著里许多东西本来是不响的。改编成剧集时，响度不是靠更大声、更直给，而是靠更清楚的剧作动作、关系后果和时代压力。能让人物失去、获得、转向、迟疑，这种“响”才成立。

## 2. 人物不能只是功能钉子

秦雯反复强调，好的《繁花》体验不是只看主角，而是“某些人、某些话、某些物件”会单独留在观众心里。群像写作不是给主角抬轿，而是让每个人物在有限的出场里都拥有自己的纹理和命运感。

## 3. 上海不是装饰，是分寸与秩序

上海语境最有价值的，不是“侬好”“阿拉”这种口头识别，而是边界感、契约精神、职业礼数、物理空间距离和说话留手。写上海，重点不在腔，而在分寸。

## 4. 台词与台词之间的空隙才是深情往来

秦雯在回看素材时意识到，真正成立的感情戏，导演拍的不是剧本上的那几句台词，而是台词与台词之间的空隙。对编剧来说，这意味着别把话写满；给动作、目光、停顿和错位留位置，感情才会自己冒出来。

## 5. 编剧也要三维写戏

人物不是只在纸上说话，还要活在具体空间里。谁站哪里，门从哪里开，柜台、吧台、楼梯、窗边如何承载关系，都应在编剧阶段被提前考虑。只写对白，不写空间，戏就还是二维的。

## 6. 时代不是背景板，是命运发动机

90 年代上海不是气氛滤镜，而是让人物迅速上升、下坠、改换身份和重新估价彼此的时代机器。写年代剧时，时代必须作为行动逻辑进入角色选择，而不是仅作为服化道背景。

## 7. AI 可以做资料与结构，但替不了灵魂与文化细微差别

秦雯对 AI 的判断很清楚：它快，擅长整合信息和结构化缝合，但还不具备提出新主题、开新方向、做深情感和处理文化细部差异的能力。对这个 Agent 来说，AI 最适合做资料预整和版本比较，不适合替代人物与关系的最终裁决。
