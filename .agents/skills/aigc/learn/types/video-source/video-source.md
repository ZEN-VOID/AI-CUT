# Video Source Package

适用于视频链接、本地视频、课程视频、参考片段和屏幕录制。

固定要求：

- 同时处理 `visual_track`、`subtitle_track`、`audio_track`、`sequence_track`。
- 记录时间码、镜头/画面事件、屏幕文字、字幕和声音线索。
- 无法解析某一 track 时必须标记缺口，不得伪造。
- 学习结论必须说明来自画面、字幕、音频、顺序还是多轨融合。
- 复杂视频、课程视频、访谈录像、屏幕录制或影视拉片素材必须加载 `../../references/video-learning-contract.md` 作为细则。
