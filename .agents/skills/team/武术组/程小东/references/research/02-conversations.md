# 02 对话与即兴思考：程小东

调研时间：2026-04-15

## 结论摘要

来源类型：以一手/机构活动与访谈材料为主，二手旁证只用于补足语境。

程小东可用的长对话材料有限。2004 年左右的英文访谈、AFA/BAFTA 大师班资料和徐克访谈旁证显示，他的回答重点通常落在现场执行、动作节奏、演员/替身组织、威亚安全、镜头配合和不同导演的美学目标，而不是抽象理论。

## 反复出现的回答模式

### 1. 先问动作的“感”而不是招式名

程小东式动作不是先列拳种和招式，而是先决定这场动作的感：飘、狠、快、妖、浪漫、悲、喜剧、战争、追逐、试探。然后才把身体路线、威亚高度、镜头速度和剪辑节奏放进去。

### 2. 把危险拆成可执行工序

其高强度 wirework 和群战设计必须依赖排练、替身、演员身体能力、威亚点、落点、摄影机位置和剪辑保护。对 AIGC 或分镜方案，这可转为稳定规则：奇观动作必须写清安全与可拍工序。

### 3. 按导演系统变形

同样是动作指导，在徐克系统里偏神怪、速度和武侠浪漫；在张艺谋系统里要服从颜色、仪式、舞蹈和画面秩序；在《投名状》里要更重、更脏、更军事化。程小东的强项不是只会一种风格，而是把自己的飞翔语法接入不同导演美学。

### 4. 打斗要有音乐性

AFA 大师班材料强调 rhythmic pacing。程小东式打戏常常像乐段：起、承、转、爆发、停顿。没有停顿，飞来飞去会变成噪音；没有重拍，动作会失去危险。

## 可运行提炼

| 对话现象 | 转化为 Skill 规则 |
|---|---|
| 先谈动作感 | 输出前先定义动作气质：飘、狠、快、妖、悲、喜或重 |
| 强调现场执行 | 每个高风险动作必须给排练、替身、威亚、落点和镜头护栏 |
| 随导演变形 | 先读取项目美学，再决定动作密度和真实/奇幻比例 |
| 节奏优先 | 每场动作必须写停顿、变奏和最后一击，避免连招噪音 |
