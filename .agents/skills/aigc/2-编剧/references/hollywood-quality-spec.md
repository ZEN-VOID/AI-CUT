# Hollywood Quality Spec

## Quality Position

本规范用于把忠实投影提升到高质量影视剧本形态。它不能覆盖保真规则，不能授权删减、摘要、重排或自由改写上游剧情因果。

## Scene Dramaturgy

每个场景应尽量显性化以下要素：

| quality_axis | requirement |
| --- | --- |
| 场景目的 | 观众在本场景获得什么新信息、压力或人物关系变化 |
| 冲突压力 | 人物面对什么即时危险、规则限制、误解或选择成本 |
| 行动链 | 每个重要角色通过可见行动推动或抵抗局面 |
| 信息差 | 谁知道、谁不知道、谁以为自己知道 |
| 视觉核心 | 本场景最能承载类型感和情绪的画面物件或空间状态 |
| 声音核心 | 最能制造节奏、压力或规则感的声音 |
| 表演任务 | 演员每段对白、沉默、停顿对应的可执行目标 |
| 潜台词策略 | 角色不能直接说出口的信息如何转成行为、停顿、道具和视线 |
| 权力调度 | 人物高低、远近、站坐、门口/讲台/道具归属如何表达关系变化 |
| 场景状态差 | 场景结尾相比开头，在风险、信息、关系或人物处境上发生什么变化 |
| 下游可分组性 | 场景标题、画面字段、声音字段足够稳定，便于 `5-分组` 切分 |

## High-Level Craft Rules

- 场景开头优先用空间状态、人物位置或可见动作入场。
- 重要规则信息尽量绑定道具、系统画面、黑板、纸张、广播、屏幕或可视化显影。
- 角色恐惧不直接写“很害怕”；用呼吸、手、眼神、身体僵硬、回避动作、声线、沉默和空间距离表达。
- 类型片压力要持续可见：校园怪谈可用灯管、窗帘、讲台、走廊、广播、规则文字、集体反应承载。
- 对白画面不复述对白，而是给出演员表演、对手反应、空间压迫和画面承托。
- 表演提示要写成演员可执行任务：目标、阻碍、策略、外显动作，至少命中其中两项。
- 潜台词不写成解释句；用停顿、避视、抢话、反问、手部动作、身体距离和道具转移承托。
- 场面调度服务权力关系和信息差：谁站着、谁坐着、谁占据讲台/门口/黑板/窗边、谁被群体视线压住。
- 高段位表演和场面调度不能在场景尾部总结式列条；规划结果必须拆入对应对白、动作、画面、道具、声音和反应。
- 沉默必须有动作或声音余波；上游没有对白时，不用新增对白替代反应。
- `2-编剧` 不输出 `分镜明细预设` 字段；需要强调的可拍意图必须转化为基础声画字段，不提前替下游导演、表演、摄影或分组写创作方案。
- 转场要服务因果、压力或信息推进，不为炫技而存在。

## Adaptation Quality Without Fact Drift

允许：

- 把小说描述拆成多个可拍字段。
- 把心理压力投影为独白、表演提示、表情特写或可见画面承托。
- 把抽象氛围转成空间、光线、声音、群体行为。
- 为同一事实补足可拍承托，例如“系统公告”可拆成 `旁白（系统提示）` 与 `系统画面`。
- 为上游已有心理和关系变化补足表演任务、潜台词行为、场面调度和沉默余波。

禁止：

- 删除一个剧情事实，只因为它“不够电影化”。
- 把上游角色判断改成另一个因果解释。
- 为提高冲突新增上游没有的对白或事件。
- 把好莱坞节奏理解为快速摘要。

## Quality Gate Checklist

逐集交付前确认：

- 每个场景都有明确空间、日夜和视觉入口。
- 每段声音都有可拍承托。
- 每个核心道具、规则、系统信息有可视化路径。
- 每个重要心理变化至少被投影为独白、可感知心理反应、可执行表演提示或可见行为之一；`心理反应` 必须能被演员表演、观众从声画中接收，不写抽象内心解释。
- 每个关键场景有进入状态、压力源、转折点和退出状态的可拍证据；若只是过渡场，报告中说明它的承接功能。
- 重要 `表演提示` 可由演员执行，不是心理标签或主题判断。
- 重要表演任务与场面调度已内嵌到具体剧本句段，没有在场景或分镜组末尾作为总结块倾倒。
- 权力关系变化有空间、视线、道具或身体距离承托。
- 沉默、迟疑、反应和余波没有被新增对白偷换。
- 没有为了“高级感”牺牲上游信息量。
- 下游可以仅凭场景标题和画面字段判断组起点。
