# 03 表达DNA

## 作品表层语言

- 华丽，但不是空心；总让热闹背后拖着一点旧账和后劲。
- 不是纯抒情，而是把商业、地方生活和感官符号并排放在一起。
- 反复使用“不是 X，而是 Y”的双层修正：不是怀旧，而是转型烈度；不是餐厅，而是地位竞技场。

## 高频表达部件

| 部件 | 作用 |
|---|---|
| 黄河路 / 和平饭店 / 码头 / 苏州河 | 空间等级与情绪坐标 |
| 方言 / 饭局 / 酒 / 服装 / 夜色 | 地方生态与身份外显 |
| 上升 / 翻身 / 机会 / 本钱 / 门面 | 商业与阶层跃迁 |
| 代价 / 失去 / 后劲 / 误解 / 来不及 | 热闹之后的苦味 |

## 风格规则

- 先给判断，再给细部，不写空洞诗句。
- 每次提风格，都要落到一个可以拍、可以写、可以演的物件或动作上。
- 情绪不靠直接告白，常通过饭桌、走位、沉默、服装和空间关系释放。
- 可以浓烈，但不能漂浮；必须有关系账和时代账托底。

## 易误用风险

- 只学霓虹、慢镜、老歌，会失去《繁花》的商业骨架。
- 只学商战结构，会失去《繁花》的地方感和感官层。
