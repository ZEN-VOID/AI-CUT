## 一、氛围不是背景板

氛围要回答三件事：

1. 这个场景此刻让人感觉到什么
2. 这种感觉如何服务人物处境
3. 这种感觉是否推动了章节的情绪方向

若场景描写删掉后，情绪和关系毫无变化，说明它只是背景板，不是氛围。

---

## 二、三层氛围结构

### 1. 外层：物理环境
- 光
- 声
- 温度
- 气味
- 空间开合

### 2. 中层：人物体感
- 冷不冷
- 闷不闷
- 呼吸顺不顺
- 眼睛刺不刺
- 身体有没有下意识动作

### 3. 内层：情绪投射
- 压抑
- 松动
- 期待
- 荒凉
- 欲望
- 告别感

写法顺序一般是：
景物出现 -> 人物体感 -> 情绪渗出

---

## 三、寓情于景的基本方法

### 方法 1：让景物带方向
不是“有什么”，而是“这些景物在朝哪种情绪上推”。

示例：

```
雨不是单纯在下，而是把整条街都压得发灰。
```

### 方法 2：让人物与景物发生接触
风吹、雨打、潮气入骨、灯影晃眼，人物必须感受到。

### 方法 3：让景物成为关系温度计
同一场雨：
- 对重逢是柔软
- 对诀别是发冷
- 对秘密是遮掩

---

## 四、情景交融的四种路径

### 1. 同频
景和情同向共振。

```
他站在暮色里，风一阵一阵灌进空袖子，像这段关系最后一点温度也被吹散了。
```

### 2. 反衬
景很美，情很坏；或景很暖，心很冷。

```
日光明得晃眼，她却觉得胸口像压着一块湿冷的石头。
```

### 3. 递进
景物层层推进，把情绪推到爆点。

### 4. 留白
不直说感情，只让景物和动作把它托出来。

## 五、境界感的实操化理解

这里不做理论考据，直接采用写作可用版：

### 有我之境
景中有强烈的“我”的情绪、欲望、痛感、判断。

适合：
- 爱意
- 悲痛
- 羞耻
- 复仇

写法：
- 允许人物主观地看景、误读景、把景变成自己情绪的扩音器

### 无我之境
情绪不直接喊出来，景物、动作、声响自己成立，读者自然被带进去。

适合：
- 大寂静
- 巨压迫
- 空茫
- 临别

写法：
- 少评判
- 少形容词
- 多留动作和物理细节

结论：
- 抒情强章可偏“有我”
- 高级压氛围、诀别、悬疑章可偏“无我”

## 六、常见氛围类型与抓手

| 类型 | 抓手 |
|------|------|
| 暧昧 | 呼吸、距离、手指、灯光、香气、沉默 |
| 压迫 | 阴影、闷热、狭窄、低声、迟滞 |
| 荒凉 | 空、冷、远、旧、慢 |
| 诡异 | 不对劲的声响、错位光影、熟悉物的异常 |
| 壮阔 | 纵深、层级、天地尺度、人的渺小 |
| 临别 | 风、晚色、未说完的话、停顿、回头/不回头 |

---

## 七、抒情章的控制

抒情不是停剧情。

每一段抒情都要挂一个功能：
- 推进关系
- 固化决心
- 揭示伤口
- 给后续转折蓄压
- 制造章末余味

若一段抒情既不改关系，也不改人物决策，那它大概率只是自我陶醉。

---

## 八、从“有画面”到“有气”

### 低阶
看得到：月亮、风、树、雨。

### 中阶
感得到：冷、湿、响、逼仄、发亮。

### 高阶
留得住：读完后情绪没有散，景物成了这一章的情绪容器。

提升级方法：
- 少堆同类形容词
- 抓一两个核心感官反复回响
- 让景物和人物动作互相印证

---

## 九、氛围修复流程

当场景“有描述但没氛围”时：

1. 先删泛词
- 如：唯美、压抑、浪漫、凄凉

2. 只保留 2-3 个核心感官锚
- 光
- 声
- 温度
- 气味

3. 补人物体感
- 衣角、手背、呼吸、步伐、视线

4. 检查景物是否改变了人物的说话、动作或判断

5. 检查这一段氛围有没有服务本章情绪方向

---

## 十、禁忌

- 禁止一上来就大段空灵辞藻，人物却不在场
- 禁止用“很美/很冷/很压抑”代替具体感官
- 禁止景物和人物情绪完全平行，互不作用
- 禁止每逢抒情都写风、月、雨、夜色如水的套话
- 禁止为了“有境界”而牺牲清晰度
