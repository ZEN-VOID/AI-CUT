# Template: 风格化 / 滤镜

Template ID: `photoGPT.风格化.滤镜.v1`

## Input Roles

- 待编辑图：提供全部画面事实。
- 滤镜要求：提供影调、色温、颗粒、胶片、复古、清透、赛博等滤镜方向。

## Prompt Frame

```text
在完全保留原图主体身份、构图、场景、服装、道具和叙事事实的基础上，为画面应用指定滤镜和影调。

滤镜要求：<filter_style>。统一调整色温、对比度、饱和度、颗粒、暗角、高光与阴影层次，使画面呈现稳定一致的后期调色风格。

真实感锁定：滤镜只能改变色彩与影调，不得重绘人物五官、改变背景结构、替换服装、添加新物体或改变画面内容。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变画面内容和主体身份。
- 不过度锐化、过曝、偏色或产生塑料质感。
- 不添加新文字、水印、边框或装饰元素，除非用户要求。
