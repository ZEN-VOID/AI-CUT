# scripts

本目录只承载 `4-摄影` 的机械辅助脚本，不承担分镜明细、节拍判断、构图、运镜、边界交出、光影或色彩的主创工作。

允许职责：

- 读取 `3-运动/第N集.md`；用户明确跳过 `3-运动` 时回退读取 `2-编导/第N集.md`。
- 统计可能的画面性字段。
- 检查命中字段后是否存在 `分镜明细：`。
- 检查每个分镜明细块中的 `分镜N（约X秒）:` 是否从 1 开始连续编号，并机械确认显式时长存在。
- 统计每集分镜数量分布，并在 2 镜块异常集中时给出机械提示；快节奏平台默认允许高切换密度，该提示只要求 LLM 抽样复核 `beat_map / rhythm_profile / sequence_density_curve / shot_count_decision`，不替代具体节拍判断。
- 检查 frontmatter 中 `stage: 4-摄影`、`source_motion_path` 或 `source_writing_directing_path`、`output_path` 和 `duration_policy` 是否存在。
- 检查 `分镜N` 的秒数是否符合短剧·AIGC 时值提示（<1s 提示闪切确认，>3s 提示必要性确认，>5s 提示强例外确认）。
- 扫描 `分镜明细` 块中是否出现常见抽象阐释词（如象征、隐喻、寓意等），并给出机械提示。
- 检查 `分镜明细：` 块后是否有空块（无 `分镜N` 条目）。
- 输出机械校验报告。
- 作为质量 review 的输入证据之一；脚本通过不等于 `review/review-contract.md` 通过。
- 只有显式传入 `--strict-shot-distribution` 时，2 镜集中才升级为机械错误；默认保持 warning。

禁止职责：

- 用脚本生成分镜明细正文。
- 用规则拼接替代 LLM 判断有效触发点和分镜数量。
- 用脚本按固定比例强行改写 1/2/3/4 镜数量。
- 用脚本判断某个 `分镜2` 该保留、删并或扩展；脚本只负责发现集中风险，具体裁决必须由 LLM 按合同完成。
- 用脚本裁决源句复述扣除测试、功能性影视投影、非复述型分镜、摄影美学或 AI 视频执行稳定性。
- 自动改写原编导稿正文、对白或场景顺序。
