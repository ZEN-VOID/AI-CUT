## Goal

按 F1 full_auto_edit 将文案9、旁白音频和三类视频素材合成为一个 1280x720 硬字幕 final MP4。

## Inputs

- Reference directory: `projects/0622/示例`
- Source material directory: `projects/0622/素材/视频`
- Video manifest: `projects/0622/素材/视频/视频说明.yaml`
- Script text: `projects/0622/素材/文案/文案9.md`
- Voiceover audio: `projects/0622/素材/音频/文案9-音频.mp3`

- EDL/render plan: `projects/0622/结果/文案9/auto-edit/edit/edl.json`

## Routing

- Main workflow: `$F1` full_auto_edit.
- Manifest generation satellite: not triggered; existing `视频说明.yaml` has segment-level fields.

## Fallbacks

- ASR unavailable/not confirmed: use `ffmpeg silencedetect=noise=-35dB:d=0.18` plus LLM semantic chunks; record in dialogue alignment.
- High-density tool/operation footage: use semi-transparent subtitle box and verify frames.
- Reference rhythm: machine ffprobe plus sampled reference frames; no reference footage used in final.

## Acceptance

Final MP4 exists, decodes, has video/audio tracks, SRT/dialogue/visual/style validators pass, and verify frames cover normal subtitles, tool screen, operation demo, AIGC content, and title-card spans.
