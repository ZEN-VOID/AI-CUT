# F1 PRP: 测试项目自动剪辑（男声2）

## Goal

参照 `projects/测试/示例` 的处理节奏，使用 `projects/测试/素材/视频` 目录中的混剪视频、用户文案和 `男声2.mp3` 旁白，生成一个字幕与旁白尽量卡点的硬字幕 final MP4。

## Inputs

- Reference directory: `projects/测试/示例`
- Source video pool: `projects/测试/素材/视频`
- Script text: `projects/测试/素材/文案/直接他妈大结局了.md`
- Voiceover audio: `projects/测试/素材/音频/男声2.mp3`
- Result directory: `projects/测试/结果`

## Outputs

- Final MP4: `projects/测试/结果/直接他妈大结局了_final.mp4`
- Work directory: `projects/测试/结果/auto-edit/edit`
- Master SRT: `projects/测试/结果/auto-edit/edit/master.srt`
- EDL/render plan: `projects/测试/结果/auto-edit/edit/edl.json`
- Execution report: `reports/F1-自动剪辑-测试项目-男声2-20260621.md`

## Routing

- Main workflow: `$F1`
- Downstream skills:
  - `timestamp-extraction`: 参考片场景/黑场、旁白静音边界。
  - `video-editing`: 混剪素材池拼接、画幅统一、导出策略。
  - `wjs-transcribing-audio`: 中文 ASR 优先级判断；本轮无凭证时进入停顿边界 fallback。
  - `wjs-burning-subtitles`: libass 硬字幕、字号边距、抽帧校验。

## Stages

1. Intake and environment check.
   - 检查参考目录、视频池、文案、旁白和结果目录。
   - 若目标 final 已存在，先备份旧版。
2. Reference rhythm analysis.
   - 对 5 个参考样片做 ffprobe。
   - 抽参考帧和检测黑场/场景变化。
   - 参考片只影响节奏、画幅、字幕风格和补位策略，不复用其画面。
3. Subtitle and voiceover alignment.
   - 旁白 `男声2.mp3` 是主时钟，时长约 55.10 秒。
   - 文案是字幕文本真源，不改写口语和脏话。
   - ASR 凭证当前不可用，采用 `silencedetect` 停顿边界 fallback。
   - 由 LLM 先做语义短句分块，再由脚本把分块投影到停顿边界；不得用全片比例分配作为 final。
4. Visual mapping and missing-material fill.
   - 混剪目录 3 个视频作为素材池，按参考片快节奏截取和拼接。
   - 输出横屏 1280x720、30fps、H.264/AAC。
   - 素材总时长足够，不主动使用黑屏；如果渲染中发现缺口，才用黑屏字幕卡补位。
5. Final render.
   - 旁白替换原视频音频。
   - 字幕在最终 filter chain 最后烧录。
   - 不覆盖 `素材/` 和 `示例/` 原文件。
6. Verification and report.
   - 校验 SRT 编号、重叠和最大 cue 时长。
   - 对 final 完整解码、ffprobe、抽取首/中/尾和最长 cue 帧。
   - 写执行报告、project.md 和 Source Sync Check。

## Fallbacks

- ASR unavailable: 当前环境无 `VOLC_ASR_*`、`VOLC_*`、`OPENAI_API_KEY` 凭证变量，使用 `silencedetect n=-35dB d=0.08; min_pause=0.18` 生成语音区间，并记录为降级。
- libass unavailable: 当前环境已有 `subtitles` 滤镜；若渲染失败，按 `wjs-burning-subtitles` 切换可用 ffmpeg。
- Reference rhythm cannot be machine-detected: 以 ffprobe、抽帧和人工观察说明记录节奏依据。
- Source video too short or mismatched: 优先重复使用素材池不同区段；仍不足时用黑屏字幕卡补位并记录。

## Acceptance

- `projects/测试/结果/直接他妈大结局了_final.mp4` 存在且可完整解码。
- final 有视频轨和音频轨，音频主轨来自 `男声2.mp3`。
- final 时长与旁白时长差 ≤ 0.25 秒。
- `master.srt` 通过结构校验，cue 不重叠，最大 cue ≤ 4 秒。
- 字幕硬烧录到 final，抽帧可读、不明显溢出。
- 报告记录输入、路由、ASR 降级、渲染参数、验证结果和 Source Sync Check。
