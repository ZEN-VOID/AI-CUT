# 对手戏 模块说明

## 定位

- 本叶子负责参照共享 [角色表现细则](../../../../references/角色表现.md)，把双人或多人互动压成攻守、距离、视线、让压、回避和位置交换等可执行策略。
- 它不负责说明“关系复杂”，只负责把关系变成演员能执行的互动打法。

## 使用方法

- 优先判断谁逼近、谁回避、当前优势落在谁手里，以及当前空间/站位支持什么打法。
- 关系变化要通过距离、视线、让步、试探、失手或位置交换体现。
- 输出进入 `character_patch` 时，保留最能制造张力的那个互动策略，而不是关系说明。

## 失真与修正

- 若成稿像人物关系说明书，说明没有把关系落成动作。
- 若开始讨论机位、景别或切接，说明越权到镜花层。
- 若不结合当前空间和站位就谈攻守，说明还没真正命中共享 `对手戏模式`。
- 若互动没有压力方向，先把“谁压谁、谁躲谁”补清楚再汇流。
