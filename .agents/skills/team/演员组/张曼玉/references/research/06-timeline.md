# 06 · 人物时间线

| 时间 | 事件 | 表演系统意义 | 来源等级 |
|---|---|---|---|
| 1964 | 出生于香港，后随家人移居英国成长。 | 形成香港/英国、中文/英文、内外身份之间的早期位移。 | Cannes / 香港电影资料馆 |
| 1983 | 参加香港小姐竞选并进入娱乐工业。 | 从模特、电视、商业曝光进入高压镜头训练。 | 香港电影资料馆 |
| 1984-1988 | 进入电影圈，高频出演商业片、动作喜剧和类型片。 | 早期“漂亮女主”位置与工业训练并存。 | 香港电影资料馆 / BFI |
| 1988 | 与王家卫合作《旺角卡门》。 | 开始从表层角色转向更复杂的内在表演。 | Cannes / Guardian |
| 1990 | 《阿飞正传》等王家卫合作继续扩展银幕气质。 | 学会在欲望、等待和失落里少演。 | BFI |
| 1991-1992 | 主演关锦鹏《阮玲玉》，获柏林银熊最佳女演员。 | 形成“演一个被观看的演员”的元表演能力。 | BFI / 奖项资料 |
| 1996 | 主演陈可辛《甜蜜蜜》与 Assayas《Irma Vep》。 | 一边演普通移民爱情，一边把明星身份放进欧洲元电影。 | BFI / Criterion |
| 2000 | 主演王家卫《花样年华》。 | 以服装、空间、停顿和不越界动作建立极致留白表演。 | Cannes / Guardian / Criterion |
| 2002 | 出演张艺谋《英雄》。 | 把武侠姿态、颜色系统与克制情感结合。 | Cannes / BFI |
| 2004 | 主演 Assayas《Clean》，获戛纳最佳女演员；同年客串《2046》。 | 用粗粝角色拆掉优雅滤镜，完成国际奖项高点。 | Cannes |
| 2004 以后 | 剧情长片输出明显降低，低调生活，偶有艺术/音乐/广告公开出现。 | “退出重复”成为其身后形象的一部分。 | GQ / 媒体长文 |
| 2024 | BFI Southbank 举办 Maggie Cheung 作品回顾。 | 其表演被机构化回看为影史参照。 | BFI / AnOther |
| 2025 | 公开加入小红书/Red Note 引发关注。 | 低调多年后的公众出现，不等于明确复出演戏。 | The Straits Times |

## 最新动态（截至 2026-04-15）

- 可核验的最新公开动态主要集中在 2025 年加入小红书/Red Note 后的轻量社交内容与媒体报道。
- 未发现权威来源显示她已正式回归剧情长片主演。
- 2024-2025 年英语影评和机构活动持续回看其作品，尤其围绕《花样年华》《阮玲玉》《Irma Vep》《Clean》。
