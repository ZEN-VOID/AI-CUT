# Natural Shot Detail Writing Contract

本文件定义 `4-摄影` 终稿层面的写作规范：如何把功能 payload 写成自然、流畅、可执行的中文，同时不丢失下游消费所需的影视事实。

## Example Usage Guard

本文件所有示例、正例、反例和表格样句仅用于说明写作判断、质量边界和修复方向，不是输出模板。执行具体任务时，不得复用示例中的人物、场景、道具、动作、句式或镜头组合；必须根据当前 `visual_unit`、上游原文、项目语境和 `shot_design_plan` 重新写成当前画面自己的自然中文分镜。

## Core Rule

分镜画面不是字段填充，不是参数罗列，也不是文学描写。它是能用中文说清楚"摄影机在哪里、怎么看、怎么动、停在哪里、交给谁"的影视事实陈述。

写完分镜后自问：去掉摄影术语后，这句话还能让读者在脑中看见镜头吗？如果只剩下"镜头推进、氛围紧张"，就还没完成。

## 写作原则

### 1. 动态优先于静态

把参数名变成动词和路径：

| ❌ 弱写法 | ✅ 强写法 |
| --- | --- |
| `镜头慢慢推进，氛围更紧张。` | `镜头贴着课桌边缘慢推，路径被桌角切成窄线，最后停在他握紧的手；紧张来自手指发白和脸被空间切开。` |
| `使用俯拍角度展现压迫感。` | `讲台从画面下沿压上来，黑板字悬在学生头顶，老师只露出拿教鞭的手。` |
| `特写规则文字，突出重要信息。` | `粉笔字占满画面左半边，右侧虚着林寂的眼睛；焦点从字尾慢慢拉到他瞳孔里的反光。` |

### 2. 具体优先于抽象

每个光线、构图、运动必须落到可见结果：

| ❌ 抽象词 | ✅ 可见结果 |
| --- | --- |
| `侧光制造层次感。` | `窗户的侧光切过他的侧脸，左眼窝陷进阴影，右颧骨亮起一道白线。` |
| `前景遮挡营造空间感。` | `黑板边缘的竖框从画面左侧伸进来，挡住三分之一的林寂，只露出他正在移向黑板的手。` |
| `光影变化展现心理。` | `日光灯闪了一下，他的瞳孔收缩，影子从墙上一跳，贴回他攥紧的指节。` |

### 3. 动作被镜头包裹，不是镜头跟随动作

| ❌ 跟随式 | ✅ 包裹式 |
| --- | --- |
| `镜头跟随苏红叶走向讲台。` | `镜头从苏红叶背后慢慢绕到他的侧面；每一步讲台都变得更近、更暗，他的影子在黑板字上拉长。` |
| `镜头跟着林寂的手推向黑板。` | `林寂的手从画面右侧伸入，镜头微微前推，粉笔灰在指缝间飞起，黑板字在他手背留下一层白。` |

### 4. 方向有参照，不含混

方向必须相对于画面边界或摄影机，不写"向左/向右"而不说相对于谁：

| ❌ 含混方向 | ✅ 有参照方向 |
| --- | --- |
| `他向左转头。` | `他的视线从黑板字滑向画面左侧三分之一处的苏红叶。` |
| `镜头从右向左横移。` | `镜头从讲台右侧边缘慢慢横移到林寂的背影，左侧课桌行列在画面里逐排消失。` |

### 5. 禁止同构句

同一分镜块内的多条分镜如果句式完全相同（如"从……以……变化到……最终……"重复 3 次），必须换句式或合并。参数不换，内化到画面描述中：

```markdown
❌ [起始秒-结束秒] 镜头从讲台左侧以缓慢速度横移到林寂位置。
❌ [起始秒-结束秒] 镜头从林寂位置以相同速度继续横移到苏红叶位置。
❌ [起始秒-结束秒] 镜头从苏红叶位置以相同速度横移到黑板字。

✅ [起始秒-结束秒] 长焦压扁后排课桌，镜头从讲台左侧慢慢贴近林寂停住的手指。
```

## 禁止写法

以下写法必须在终稿阶段删除或重写：

1. **纯摄影术语拼接**：只写景别词（特写/近景/中景）而没有后续可见结果。
2. **抽象情绪词**：紧张/压抑/恐惧/绝望等词必须转成咬肌/眉心/手指/呼吸等可见动作。
3. **导演阐释**：表达"这个镜头想表达什么"而不是"摄影机在哪里做什么"。
4. **参数清单**：景别+角度+镜头类型+运镜速度的罗列，没有画面落地。
5. **无变化句式**：连续的"镜头推进、镜头推进、镜头推进"或"从A到B、从B到C、从C到D"。

## 情绪词转译规则

| 抽象词 | 可见动作 |
| --- | --- |
| 紧张 | 咬肌收紧、手指发白、呼吸变浅、喉结起伏、肩膀内收、眉心竖纹 |
| 压抑 | 下颌紧绷、眼眶泛红但无泪、坐姿僵直、视线回避 |
| 恐惧 | 瞳孔放大、退后半步、手护胸前、后脑勺贴墙 |
| 震惊 | 嘴微张、身体僵住、视线定在一点、呼吸骤停 |
| 崩溃 | 眼泪涌出但无声、身体瘫软、双臂抱头、声音嘶哑或失声 |
| 压抑愤怒 | 手指握紧到指节发白、咬住后槽牙、声音压低到只有自己能听见 |

## 时值落地

每条 `[起始秒-结束秒]` 必须显式落盘，时间范围必须与内容匹配：

- 0.5-1 秒内部时值：落盘为 `[0-0.5秒]`、`[0-1秒]` 等短范围，用于快切、撞点、音效同步、闪切。
- 1-2 秒内部时值：落盘为连续 `[起始秒-结束秒]` 范围，用于标准切换、反应、交接。
- 2-3 秒内部时值：落盘为连续 `[起始秒-结束秒]` 范围，用于读秒、信息显影、表演过渡。
- 3-4 秒内部时值（tempo=slow_burn/hold）：落盘为连续 `[起始秒-结束秒]` 范围，用于崩溃/强忍/醒悟等情绪场景，由情绪节奏决定，不受普通 evidence 门槛约束。
- 5 秒以上内部时值：只在空间重置、复杂调度、意识流、高点读秒或明确长对白成立时落盘为长范围。

## 成稿检查

写完分镜后按以下顺序检查：

1. **功能性检查**：去掉摄影术语后，主体、动作、路径、落点是否还清楚？
2. **可执行性检查**：下游图像/视频工具能根据这句话生成画面吗？光线有结果吗？方向有参照吗？
3. **情绪转译检查**：有抽象情绪词吗？有咬肌/眉心/手指/呼吸等可见动作吗？
4. **句式检查**：连续分镜有同构句吗？
5. **时值检查**：时间范围与内容匹配吗？

## 典型修复示例

```markdown
❌ [起始秒-结束秒] 中景俯拍，老师走向讲台，氛围紧张。
✅ [起始秒-结束秒] 俯角中景，苏红叶从画面右侧走向讲台；每一步日光灯把他的影子在黑板上压得更低，咬肌从侧脸绷到下颌。

❌ [起始秒-结束秒] 特写粉笔字，镜头推进，突出重要信息。
✅ [起始秒-结束秒] 镜头贴着黑板表面极慢推进，粉笔字从模糊变得锐利；最右侧一行字比其他的都粗，像被什么压过。

❌ [起始秒-结束秒] 镜头跟随林寂的手推向黑板，动作流畅，节奏紧张。
✅ [起始秒-结束秒] 林寂的手从画面右侧伸入，镜头微微前推，粉笔灰在指缝间飞起；他按在黑板字上的力道让指节发白，字迹在他掌根下震颤。
```

## 自然化不等于口语化

分镜画面是影视专业写作，不是新闻报道，也不是小说描写。目标是专业、可执行、自然流畅：

- 用"镜头贴近"而不是"摄像机慢慢靠近"
- 用"停在他握紧的手"而不是"停在手的特写"
- 用"黑板字在画面左半边显影"而不是"我们看到黑板上的字"
