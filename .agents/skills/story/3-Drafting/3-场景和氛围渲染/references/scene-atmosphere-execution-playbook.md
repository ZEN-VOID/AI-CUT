# Scene Atmosphere Execution Playbook

## Positioning

本文件是 `3-场景和氛围渲染` 的本地执行细则。

- 它不替代父级 `SKILL.md`，只负责把 Step 3 的场景重写动作稳定化。
- 本轮沉淀两组可选镜片：
  - `静冷留白镜片`：适合静场、冷场、错身、旧地回返、雪感、物哀和未说出口的关系压力。
  - `身体代价镜片`：适合亲密、禁忌、熟年、婚外、医疗、病房或任何必须补身体真实与社会后效的场面。

## Scene Function First

动正文前，先回答本场四个槽位：

| slot | 要回答的问题 | 允许落到正文的典型形式 |
| --- | --- | --- |
| `scene_function` | 这场戏在剧情里到底承担什么？ | 逼近、阻隔、等待、诱发、审视、暴露、休止 |
| `space_pressure` | 空间怎样限制或推动人？ | 门槛、走廊、车厢、病床、窗、楼梯、拥挤、空旷、冷热 |
| `relationship_temperature` | 人与人此刻是在靠近、躲避还是隔着什么？ | 站位、错身、停顿、看与不看、坐得太近或太远 |
| `environment_effect` | 环境怎样改变行动、判断或情绪？ | 呼吸发紧、手指发冷、衣料摩擦、灯影遮挡、声音逼近 |

若这四项答不出，先不要加氛围句。

## Atmosphere Vector Map

对关键场景，至少锁定以下 5 项中的 3 项：

| vector | 要点 | 常见误用 |
| --- | --- | --- |
| `light` | 冷暖、明暗、照到哪里、遮到哪里 | 只写“昏黄/惨白”，不写它怎样影响人物 |
| `sound` | 空调、雨、鞋跟、远处广播、心跳、喘息、器械声 | 只说“安静/嘈杂”，没有来源 |
| `temperature` | 冷、潮、热、闷、汗、雾气、指尖温差 | 只有抽象情绪，没有身体受感 |
| `material` | 布料、瓷面、铁栏、木门、床单、玻璃、皮肤 | 只有视觉词，没有触感 |
| `distance` | 门内门外、桌两端、床边、走廊尽头、手能否碰到 | 只说关系远近，不写空间距离 |
| `smell` | 药味、潮味、烟味、酒味、雪水味、花香、金属味 | 乱加香气，和场景功能无关 |
| `object_image` | 茶杯、窗、镜、药盒、被角、票据、戒指 | 物象只美，不起作用 |

## Lens Selection Matrix

| 当前场景类型 | 首选镜片 | 重写重点 | 禁止误用 |
| --- | --- | --- | --- |
| 静场、冷场、错身、重逢前后 | `静冷留白镜片` | 主物象、距离、沉默后的后果、冷暖变化 | 连续堆雪、月、风、灯影等装饰意象 |
| 亲密、禁忌、熟年、婚外 | `身体代价镜片` | 身体事实、羞耻余震、社会账单、整理动作 | 只写暧昧气味和漂亮皮肤 |
| 医疗、病房、照护、手术前后 | `身体代价镜片` | 药味、器械、病体反应、职业边界、体面破裂 | 用术语冒充真实，或用医院当猎奇背景 |
| 追逐、威胁、潜入、审问 | `空间阻力镜片` | 阻隔物、脚步声、视线死角、冷汗、呼吸、压迫路径 | 只写“紧张”，不给障碍物和路线 |
| 地方风物、旧城、传统空间 | `静冷留白镜片` 优先，必要时叠加 `身体代价镜片` | 主物象承担记忆裂纹、礼法压力或身份温差 | 把地方文化写成旅游说明 |

## 静冷留白镜片

适用判断：

- 人物之间有距离，但不宜直说。
- 戏要安静，却不能空。
- 需要物象、冷暖、空间留白来承载情绪。

重写动作：

1. 只选一个主物象，不贪多。
2. 让物象和人物关系绑定，而不是独立被欣赏。
3. 删除解释情绪的句子，改成动作、停顿、错身或没完成的触碰。
4. 保留冷暖变化、视线遮挡和距离关系，让“靠近不了”变成场面事实。

通过标准：

- 删掉主物象后，这场戏明显变弱。
- 沉默之后有后果，而不是只剩漂亮空白。

## 身体代价镜片

适用判断：

- 场面里有身体靠近、欲望、羞耻、病体、熟年疲惫、职业边界或社会禁忌。
- 若只写氛围，会把戏写空。

重写动作：

1. 先补身体事实：呼吸、体温、伤口、药味、疲惫、皮肤、衣料、姿势、病历感。
2. 再补关系代价：配偶、体面、职业、子女、名声、时间、后续回避。
3. 亲密之后优先写余震，而不是把场景停在最美的一刻。
4. 若是医疗场景，写职业边界和制度压力，不让医生身份变成氛围外挂。

通过标准：

- 删掉身体细节后，关系推进明显变虚。
- 场景结束后，能感到有人要为此付账。

## Rewrite Priorities

1. 先改环境怎样作用于人，再改景物句式。
2. 先补空间阻力、感官锚点和关系温差，再补修辞。
3. 若场景已经很满，优先删抽象氛围词，不优先加新意象。
4. 若亲密场只剩美感，优先补羞耻、迟疑、整理动作和后效。
5. 若静场过散，优先缩回到一个主物象和一次距离变化。

## Common Failure Patterns

| symptom | direct cause | rewrite move |
| --- | --- | --- |
| 景很美，但戏没往前走 | 场景没有绑定 scene function | 回到 `scene_function / environment_effect` 重写 |
| 只有“冷清/压抑/暧昧”这类词 | 没有具体感官与身体锚点 | 补 `light + sound + temperature` 或 `material + distance` |
| 写了很多地方风物，人物却消失了 | 把环境当明信片 | 只留一个主物象，并让它压到人物选择 |
| 静场写成散文 | 没有距离和沉默后的后果 | 补错身、停顿、未说出口和动作余波 |
| 亲密场只剩香气和皮肤 | 没有身体事实和代价 | 用 `身体代价镜片` 重写，补羞耻、疲惫、关系账单 |
| 医疗场景像电视剧道具间 | 没有职业边界和病体感 | 补器械声、药味、制度压力和照护动作 |

## Regression Guard

- 不为氛围而抢 Step 4 的角色弧光主责。
- 不为抒情而抢 Step 6 的心理剖白主责。
- 不为“更成熟”而默认把所有亲密戏写成欲望场，必须看场景功能。
- 不在同一段里堆两个以上主物象。
- 不把“留白”误写成“什么都不发生”。
- 不把“身体真实”误写成“更露骨”。
