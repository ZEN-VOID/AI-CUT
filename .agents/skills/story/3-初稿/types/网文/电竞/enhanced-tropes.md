# 电竞题材增强包

> 补充维度：套路拆解 / 情节模式 / 人物原型 / 反套路策略 / 质量过滤

## 一、核心套路与行规

### 1.1 电竞叙事双轨
- **游戏内**：战术对抗、个人操作、团队协作、胜负结果
- **游戏外**：训练日常、人际关系、俱乐部运营、舆论压力、商业博弈

两条轨道必须交替推进，不能一头重。游戏外是人物成长的真正舞台，游戏内是人物能力的可视化战场。

### 1.2 关键类型场景
| 场景类型 | 功能 | 写作要点 |
|----------|------|----------|
| 比赛场景 | 展现技术、团队协作、战术博弈 | 有空间感（地图/位置/方向）、有节奏（试探/对攻/转折/决胜）、有解说的同时保持叙事驱动 |
| 训练场景 | 展示付出、策略打磨、团队磨合 | 不是流水账，聚焦关键突破或关键困难 |
| 赛后复盘 | 反思、成长、关系变化 | 胜利后的隐忧、失败后的觉醒 |
| 转会/人员变动 | 关系重构、利益博弈 | 不只是"谁走了谁来了"，而是关系和命运的变化 |
| 舆论/粉丝 | 外部压力、公众形象 | 不只是背景板，影响角色心理和选择 |

### 1.3 常见套路陷阱
- 比赛描写变成"技能名称+连招"的流水账
- 主角队永远不会输（或只会"战略放弃"式输）
- 对手永远是傲慢的反派模板
- 电竞细节写太多而人物成长写太少

## 二、情节模式与节奏规则

### 2.1 赛季节奏模板
```
组建期：组队/磨合/确定风格→首次比赛
上升期：连胜/发现问题/调整/实力提升→关键对决
危机期：核心成员状态下滑/内部矛盾/强敌出现→连败/濒临散队
觉醒期：克服困难/战术突破/关系修复→重返赛场
决胜期：最终Boss/冠军赛→胜利/失败（意义大于结果）
```

### 2.2 比赛节奏规则
- 重要比赛至少2000字以上，有完整的"起承转合"
- 一般比赛可简写（只写关键节点），不必每场都详写
- 赛季中每5-8章一场详细比赛
- 每场比赛的结果必须有意义——改变排名/改变关系/改变角色心态

### 2.3 爽点设计
| 爽点类型 | 电竞实现 |
|----------|----------|
| 技术碾压 | 精彩操作/极限反杀/战术奇袭 |
| 团队默契 | 不需要沟通的完美配合 |
| 逆风翻盘 | 从必输到逆转的大逆袭 |
| 新战术/新英雄 | 秘密武器亮相，打对手措手不及 |
| 复仇之战 | 击败曾经战胜/羞辱过自己的人 |

## 三、反套路策略

### 3.1 比赛不只是"赢"
- 有些比赛的意义是"学到了什么"而不是赢
- 一次有意义的失败胜过十次平淡的胜利
- 让失败塑造人物，而不是为虐而虐

### 3.2 对手不只是"反派"
- 对手有令人尊敬的地方——技术、意志、品格
- 对手打败主角队时读者也能理解为什么
- 宿敌关系可以有温暖的时刻（惺惺相惜/互相激励）

### 3.3 电竞不止是打游戏
- 身体伤病、心理压力、职业生涯短暂——现实的另一面
- 代际交替——老将退役、新人崛起
- 当电竞不是生活的全部——学业/家庭/未来的平衡

## 四、质量过滤规则

### 4.1 比赛描写质量检查
- [ ] 比赛有空间感（地图位置/走位/距离/方向可被读者感知）
- [ ] 比赛有节奏变化（不只"你打我一下我打你一下"）
- [ ] 比赛结果服务于人物弧线（不仅为赢而赢）
- [ ] 游戏术语能被非玩家读者理解（或上下文自解释）

### 4.2 AI腔高发区警示
- 比赛流水账（技能名+连招+血量变化）
- 解说的信息量替代了叙事（"解说员说XXX"）
- 人物训练场景变成"努力"的抽象口号

### 4.3 失败码
| 失败码 | 症状 | 返工目标 |
|--------|------|----------|
| `FAIL-ESP-GAME` | 比赛描写无空间感/无节奏 | 重写比赛段落 |
| `FAIL-ESP-CHAR` | 人物被电竞细节淹没 | 增加游戏外的情感/关系线 |
| `FAIL-ESP-TERM` | 术语过量/不可读 | 减少术语或增加上下文解释 |
