| 时间 | 事件 | 对导演思维的影响 |
|---|---|---|
| 1954 | 出生于台湾屏东县 | 家庭、父亲、教育秩序和台湾文化经验成为早期底层材料 |
| 1975 | 毕业于国立台湾艺术专科学校 | 戏剧与表演训练奠定演员观察能力 |
| 1980 | 伊利诺伊大学取得戏剧 BFA | 进入美国语境，开始跨文化学习 |
| 1984 | NYU Tisch 电影制作 MFA；短片 `Fine Line` 获 NYU 奖项 | 形成纽约电影训练与独立制作基础 |
| 1991 | 长片处女作 `Pushing Hands` | “父亲三部曲”开端，处理移民家庭和文化错位 |
| 1993 | `The Wedding Banquet` 获柏林金熊 | 华语/美国双文化家庭喜剧进入国际视野 |
| 1994 | `Eat Drink Man Woman` | 饭桌、父权、食物与欲望成为家庭秩序的可拍形式 |
| 1995 | `Sense and Sensibility` 获柏林金熊并进入 Oscar 体系 | 证明跨文化情感翻译能力 |
| 1997 | `The Ice Storm` | 从英式礼法转向美国郊区压抑 |
| 2000 | `Crouching Tiger, Hidden Dragon` | 武侠、身体自由、礼法压抑与国际市场结合；获 DGA 奖 |
| 2003 | `Hulk` | 首次大型 Hollywood 技术/超级英雄实验，类型期待错位明显 |
| 2005 | `Brokeback Mountain` | 获威尼斯金狮、DGA 奖、奥斯卡最佳导演；同性爱情与西部神话重写 |
| 2007 | `Lust, Caution` | 再获威尼斯金狮；情色、谍战、表演和政治背叛高度纠缠 |
| 2012 | `Life of Pi` | 第二次获奥斯卡最佳导演；3D 与 CG tiger 成为技术-情感融合样本 |
| 2016 | `Billy Lynn's Long Halftime Walk` | 120fps/3D/4K 实验，评价分裂但显示观看技术野心 |
| 2019 | `Gemini Man` | 继续 HFR 和数字角色实验，市场与评价不稳定 |
| 2024 | 获 Praemium Imperiale 戏剧/电影奖 | 官方确认其跨文化、跨类型与国际艺术地位 |
| 2025-02-08 | 获 DGA Lifetime Achievement Award | 行业层面对其导演生涯和技术创新的总结性认可 |
| 2025-06 | California Film Commission 公布含李安导演项目 `Gold Mountain` 的税收抵免项目 | 最新项目方向回到加州、淘金、华人历史和跨文化史诗 |
| 2026-04-15 | 本 Skill 调研截止 | `Gold Mountain` 后续拍摄、上映和 `Bruce Lee` 传记片状态仍需后续核对 |

## 最新动态

- DGA 于 2025-02-08 第 77 届 DGA Awards 上授予李安 Lifetime Achievement Award。
- California Film Commission 2025-06 公告显示，含李安导演项目 `Gold Mountain` 的项目获得加州 Film & Television Tax Credit Program 支持，并计划主要在洛杉矶以外拍摄。
- 截至 2026-04-15，本 Skill 未把 `Gold Mountain` 视为已完成或已上映作品；涉及开机、演员、剧情细节、发行日期必须重新核对。
