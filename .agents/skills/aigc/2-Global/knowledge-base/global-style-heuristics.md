# Global Style Heuristics

本文件保存 `aigc/2-Global` 在全局风格词生成中的可复用经验，不承载强制入口合同。

## Reusable Heuristics

- 真人古装风格词最容易被动画示例污染；只要出现“线稿、赛璐璐、角色建模、夸张残影、粒子拖尾”，先回到写实摄影基线。
- “真实摄影语法”比“电影感”更稳，因为它会自然牵引物理光影、材质、空间和动作反馈。
- 全局风格不是镜头方案；可写“镜头语言克制稳定”，不要写固定焦段、机位、推拉摇移或景别。
- 材质词可以写高层真实感，例如皮肤纹理、服饰纹理、自然毛发、材质反射；避免写过细内容物，比如某种具体布料或某件道具，除非项目源层硬锁。
- 反动画化禁区应保留在最终字段里，尤其当模板历史中存在国漫、赛璐璐或夸张武侠示例时。

## Repair Playbook

| symptom | likely cause | fix |
| --- | --- | --- |
| 风格词像动画提示词 | 直接继承了旧模板动画示例 | 删除动画化词汇，重写为真人写实摄影、真实材质和物理光影 |
| 风格词像镜头清单 | 把 `3-Detail` 的工作提前写进 `2-Global` | 抽回为高层摄影语法，不写具体镜头操作 |
| 风格词太抽象 | 只写“高级、电影感、统一气质” | 补真实人物、服饰纹理、毛发、材质反射、动态范围和空间层次 |
| 风格词太长 | 同义形容词堆叠 | 保留媒介、真实质感、物理光影、空间镜头、真实动作、后期底座和禁区，删宽泛形容词 |
