# 标准剧 / Output Template

本文件是 `标准剧` 子技能的模板真源。

## `格式合同.md`

```markdown
# 标准剧 格式合同

## 变体定位
- 表演优先
- 旁白从严

## 场景标题规范
- 使用 `### 场景X：<场景信息>`

## 允许字段
- 动作画面：
- 对白（<说话人>）：“……”
- 对白画面：
- 内心独白（<说话人>）：“……”（按需）
- 内心独白画面：
- 旁白（讲述者）：“……”（仅必要时）
- 旁白画面：

## 硬门槛
- 允许零旁白
- 文本字段与画面字段同命题配对
- 动作不进引号正文
```

## `格式样例.md`

```markdown
### 场景一：王府内·夜

动作画面：烛火在屏风边缘跳动，屋内只剩衣料轻擦与急促呼吸。
对白（沈清）：“你现在才来？”
对白画面：沈清站在案前不退，目光先落在来人袖口的血迹上，又抬到对方眼里。
内心独白（沈清）：“他若再迟一步，一切就来不及了。”
内心独白画面：她指尖压住桌沿，指节一寸寸发白，却仍强撑住语气。
```
