# 02 对话与即兴思考

## 对话风格 1：先讲故事，再给结论

- **一手 / 近一手**：`https://m.bjnews.com.cn/detail/160569213715067.html`
  - 新京报深访中，几乎每个回答都先绕进一个小故事，例如小鸟签名的来历、与林怀民/三毛/古龙的往事，然后才顺手给出判断。
- **判断**：
  - 他不喜欢直接抽象论证，更偏好“故事开门，观点落地”。

## 对话风格 2：用个人极端经验当证据

- **一手 / 近一手**：`https://club.commonhealth.com.tw/article/2616`
  - 书摘里反复用“42 天不出门”“凌晨 1 点起床”“10 年又 40 天闭关”证明自己的专注方法。
- **一手 / 近一手**：`https://paper.people.com.cn/hqrw/html/2015-12/06/content_1667677.htm`
  - 《环球人物》里同样把其作息、读佛经往返次数、闭关时间写成量化证据。
- **判断**：
  - 他说服别人时，不靠学术术语，靠“我就是这样做出来的”。

## 对话风格 3：喜欢用反问打断惯性思维

- **一手 / 近一手**：`https://club.commonhealth.com.tw/article/2616`
  - 典型句式是“你为什么不去问，花为何要开？树为何要长？我就是爱画画。”
- **一手 / 近一手**：`https://m.bjnews.com.cn/detail/160569213715067.html`
  - 面对文化产业、电影、美术等问题时，也常先反问“为什么要这样竞争”“为什么要把别人的看法看得那么重要”。
- **判断**：
  - 这类反问不是修辞装饰，而是他的思考入口：先把题目倒过来。

## 对话风格 4：判断极快，而且愿意得罪正确废话

- **一手 / 近一手**：`https://m.bjnews.com.cn/detail/160569213715067.html`
  - 谈中国动画时，他直接说“技术已经够了……但是编剧和导演始终有问题。”
  - 谈题材竞争时，他直接说明自己为何不与日本漫画家拼他们最强的部分。
- **判断**：
  - 他不太会给中庸安慰式答案；更像内容创业者和老派创作者的混合口吻。

## 对话风格 5：在顽皮和极端自律之间来回切换

- **一手 / 近一手**：`https://m.bjnews.com.cn/detail/160569213715067.html`
  - 采访开头部分充满恶作剧式笑谈、插科打诨和顽童气。
- **一手 / 近一手**：`https://paper.people.com.cn/hqrw/html/2015-12/06/content_1667677.htm`
  - 但一进入工作方法与学习，他又变成高度冷硬、自律、强意志的口吻。
- **判断**：
  - “顽皮”不是散漫，而是外层表情；内核依然是强烈目标导向。

## 被追问时的核心模式

1. 把问题缩成最核心的一句话。
2. 用自己的故事或读书经历举一个极端样本。
3. 给出非常明确的判断。
4. 把答案上升为一条可执行方法。

## 对动漫组最有用的对话结论

- 蔡志忠式判断很少停在“画风好不好看”，而会迅速问“你到底在讲什么”。
- 对话里最常见的真正方法不是“灵感”，而是“主题选择 + 闭关补课 + 高密度输出”。
