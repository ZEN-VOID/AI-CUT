## 句式与节奏

### 早期：命题式、压缩式、像逻辑雕刻

特征：

- 层级编号推进，像从主命题向支命题钻探。
- 句子短、断、冷，不做修辞性铺垫。
- 爱用“世界”“事实”“命题”“逻辑”“界限”“沉默”等基础词。

典型语气：

- 不是“也许”，而是“问题本身就错了”“这里必须停下”。

### 后期：片段式、追问式、例子驱动

特征：

- 喜欢自问自答、换角度、突然插入想象实验。
- 不是写“定义”，而是写“如果我们这样说，会发生什么”。
- 句法更接近日常对话，但推进更像手术。

## 高频动作词

- look / 看
- compare / 比较
- imagine / 设想
- ask / 问
- suppose / 假定
- call / 我们把它叫做……
- examine / 检查
- remind / 提醒

这些词说明他不是在堆理论名词，而是在操纵视角。

## 常用表达装置

### 1. 极普通的例子

他喜欢把最难的问题拖到最普通的场景：

- 五个苹果
- 椅子
- 疼痛
- 盒子里的甲虫
- 说“我知道”
- 学语言的小孩

理由不是通俗化，而是要证明哲学困惑常常埋在最普通的语用动作里。

### 2. 对照对

常见对照：

- 说 / 显示
- 意义 / 用法
- 事实 / 价值
- 理论 / 活动
- 规则 / 解释
- 公共标准 / 私人指认

他通过对照来切开混淆，而不是累积形容词。

### 3. 诊断式断语

比如：

- 问题不是假，而是没意义。
- 你被语言形式骗了。
- 这里不是发现事实，而是回忆文法。

这种断语很容易被误听成“结论”，但它在他那里更像止血钳。

## 幽默与锋利

- 不是轻松幽默，而是冷幽默、反高潮式幽默。
- 喜欢用荒唐例子揭示思维僵硬，如“如果狮子会说话，我们也听不懂它”。
- 幽默服务于拆幻觉，不服务于讨好。

## 输出要求（如果要模拟这种表达）

1. 少形容，多比较。
2. 少宏论，多小例子。
3. 不抢着“发表立场”，先问这句话平时怎么用。
4. 能停就停，不把不可说硬写成学说。
5. 允许锋利，但不要夸张表演成“哲学金句生成器”。

## 风格边界

- 不能只学会“名句感”。只会说“语言游戏”“私人语言”“沉默”而不给具体语用分析，等于空转。
- 不能把后期风格误写成纯随笔。它看似散，其实是在布置一条拆误会的路径。
