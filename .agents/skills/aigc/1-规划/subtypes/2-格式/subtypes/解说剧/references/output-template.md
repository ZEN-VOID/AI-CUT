# 解说剧 / Output Template

本文件是 `解说剧` 子技能的模板真源。

## `第N集.md`

```markdown
# 第N集

## 来源画像
- 故事主源：
- 预设保留：
- 明细扩写：
- 锁定轴：
- 当前集继承锚点：
- 场景编号规则：仅按连续时空编号；镜号与锚点单独保留

### 场景1：地点·时间
- 镜号范围：
- 锚点继承：

旁白（讲述者）：“……”

旁白画面：

镜头语言预设：

动作画面：
```

硬规则：

- 结果稿必须可直接阅读，不得写成合同说明或样例说明。
- 非对白文本优先旁白化。
- 旁白主体统一为 `讲述者`。
- 动作不进引号正文。
- 若命中混合源/分镜源且锁定了 `scene_boundary`，同一连续时空跨组续写时允许使用 `场景X（续）`，不得把镜号逐条升格为新场景号。
- 若上游分镜脚本明确写出运镜/镜头提示，必须优先保留为 `镜头语言预设：...`，并紧跟相关 `*画面` 条目；未明确写出时，不得脑补新增。
- 对分镜源输入，优先规范化整理已有 `*画面` 与场景结构，不以“补齐缺口”为默认目标。
- 场景标题若可从分镜脚本已有结构稳定提取，必须优先复用该结构。
