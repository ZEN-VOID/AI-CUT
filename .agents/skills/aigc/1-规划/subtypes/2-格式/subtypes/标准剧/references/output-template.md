# 标准剧 / Output Template

本文件是 `标准剧` 子技能的模板真源。

## `第N集.md`

```markdown
# 第N集

## 来源画像
- 故事主源：
- 预设保留：
- 明细扩写：
- 锁定轴：
- 当前集继承锚点：（仅在上游显式提供时写出）
- 场景编号规则：按当前来源粒度写明

### 场景1：地点·时间
- 镜号范围：（仅在上游显式提供或锁轴要求保留时写出）
- 锚点继承：（仅在上游显式提供或锁轴要求保留时写出）

动作画面：

镜头语言预设：（仅在上游显式提供时写出）

对白（角色）：“……”

对白画面：
```

硬规则：

- 结果稿必须可直接阅读，不得写成合同说明或样例说明。
- 默认只附加字段标题，不改写原文正文。
- 允许零旁白。
- 文本字段与画面字段应同命题配对。
- 动作不进引号正文。
- 若命中混合源/分镜源且锁定了 `scene_boundary`，同一连续时空跨组续写时允许使用 `场景X（续）`，不得把镜号逐条升格为新场景号。
- `当前集继承锚点 / 镜号范围 / 锚点继承` 都是条件字段；只有上游显式提供，或当前锁轴明确要求保留时，才允许写入。
- 若上游分镜脚本明确写出运镜/镜头提示，必须优先保留为 `镜头语言预设：...`，并紧跟相关 `*画面` 条目；未明确写出时，该字段不得出现。
- 对分镜源输入，优先规范化整理已有 `*画面` 与场景结构，不以“补齐缺口”为默认目标。
- 场景标题若可从分镜脚本已有结构稳定提取，必须优先复用该结构。
