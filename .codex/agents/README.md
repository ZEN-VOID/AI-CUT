# Agents

Store specialized agent role documents here.

Recommended pattern:

- One file per role
- Short purpose statement at the top
- Clear scope, inputs, outputs, and guardrails
- Reference shared rules instead of duplicating them
- Group closely related roles into a domain folder when they belong to one governance surface

Examples:

- `planner.md`
- `reviewer.md`
- `researcher.md`
- `release-manager.md`
- `harness治理/中书省.md`
- `harness治理/门下省.md`
- `harness治理/尚书省.md`

Use the template in `../templates/agent/agent-template.md` when creating a new role.
