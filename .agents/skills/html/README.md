# HTML Skills

Codex-adapted HTML design skill group.

## Installed Packages

- `claude-design/`: adapter for `jiji262/claude-design-skill`.
- `html-anything/`: adapter for `nexu-io/html-anything` / Open Design template skills.

The upstream materials are preserved under each package's `references/upstream/` directory. Local `SKILL.md` files are the Codex runtime contracts.
