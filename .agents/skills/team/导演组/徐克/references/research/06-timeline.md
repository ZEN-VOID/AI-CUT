# 06 人物时间线：徐克

调研时间：2026-04-15

## 关键时间线

| 时间 | 事件 | 对导演方法的影响 |
|---|---|---|
| 1950 | 出生于越南，后移居香港 | 跨地域经验进入香港身份与文化混杂的创作底色 |
| 1960s-1970s | 在美国学习电影，后毕业于 University of Texas at Austin 相关电影训练 | 获得西方电影教育与技术视野 |
| 1977 | 回到香港，进入电视制作 | 把电影训练接入香港本地快速工业 |
| 1979 | 导演《蝶变》 | 以类型片进入香港新浪潮语境 |
| 1980 | 《地狱无门》《第一类型危险》 | 黑色、暴力、荒诞与社会压力显露 |
| 1983 | 《新蜀山剑侠》 | 以特效和神怪重塑武侠空间 |
| 1984 | 与施南生创办 Film Workshop | 作者冲动获得生产系统；进入导演/制片复合身份 |
| 1986 | 《刀马旦》；参与《英雄本色》生产链 | 性别、戏曲、政治、动作混合；同时推动英雄片浪潮 |
| 1987 | 《倩女幽魂》作为监制/制片体系关键作品 | 神怪、爱情、动作、音乐和明星系统融合 |
| 1990 | 《笑傲江湖》相关项目 | 金庸文本现代化改编进入核心经验 |
| 1991 | 《黄飞鸿》 | 重启宗师武侠，把家国、现代性和动作美学叠合 |
| 1993 | 《青蛇》 | 以欲望和性别角度改写神话 |
| 1995 | 《刀》 | 将武侠拆到粗粝、残酷、身体化 |
| 2000 | 《顺流逆流》 | 城市动作、速度、空间实验 |
| 2010 | 《狄仁杰之通天帝国》 | 探案、历史、奇幻和大特效工业重组 |
| 2011 | 《龙门飞甲》 | 华语 3D 武侠重要节点 |
| 2014 | 《智取威虎山》 | 传统样板/战争/动作大片的类型再造 |
| 2021-2022 | 《长津湖》《长津湖之水门桥》相关联合导演/监制工作 | 进入超大规模主旋律战争工业 |
| 2024 | M+ 推出 “Tsui Hark, the Free-Spirited Trailblazer” 放映调查项目 | 其香港电影遗产继续被机构化整理 |
| 2025-01-29 | 《射雕英雄传：侠之大者》中国大陆春节档上映 | 年逾七十仍重返金庸武侠和大规模类型奇观 |
| 2025-03 | *Legends of the Condor Heroes: The Gallants* 在法国等市场有限上映并获媒体评价 | 国际影评继续关注其奇观与浪漫并置能力 |

## 最近 12-18 个月动态

- 2024 年，M+ Cinema 秋季节目策划 “Tsui Hark, the Free-Spirited Trailblazer”，把其作品作为香港电影重要作者/工业现象重新展映。
- 2025 年 1 月，《射雕英雄传：侠之大者》定档春节档并上映。公开报道显示其预售在春节档前期居前，影片主打“侠之大者，为国为民”的金庸武侠精神。
- 2025 年 3 月，Le Monde 对 *Legends of the Condor Heroes: The Gallants* 发表评论，认为徐克仍能在宏大手段和亲密细节之间调度，并强调影片围绕忠诚、选择和和平价值展开。

## 时间线提示

徐克的创作阶段不宜简单切成“早期好/晚期坏”。更准确的结构是：

1. 早期：新浪潮锋利类型实验。
2. 黄金期：Film Workshop 型工业组织与类型重启。
3. 风格极端期：神话、武侠、身体和欲望的极端实验。
4. 技术大片期：3D/CG/历史奇观与大陆合拍工业。
5. 晚近：经典 IP 与大型工业系统内的再武侠化。
