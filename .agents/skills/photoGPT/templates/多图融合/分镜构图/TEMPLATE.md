# Template: 多图融合 / 分镜构图

Template ID: `photoGPT.多图融合.分镜构图.v1`

## Input Roles

- 主体/角色图：提供角色或主体身份。
- 场景图：提供空间和环境。
- 分镜/构图参考图：提供机位、景别、运动方向、画面重心和视线引导。

## Prompt Frame

```text
基于多张参考图生成一个统一的分镜构图画面：主体身份来自主体/角色图，场景空间来自场景图，机位、景别、画面重心、视线引导和动作方向来自分镜/构图参考图。

构图要求：严格遵守参考构图的镜头高度、景别、主体位置、前中后景层次、运动方向和视觉重心；主体与场景透视匹配，光线和阴影统一。

叙事锁定：不新增用户未声明的剧情事件、角色关系或道具；只把指定主体放入指定构图与场景。

补充要求：<user_desc>
```

## Negative Constraints

- 不把分镜参考图中的人物身份复制到输出。
- 不改变主角身份、服装和关键姿态，除非用户要求。
- 不生成多格漫画或九宫格，除非用户明确要求。
