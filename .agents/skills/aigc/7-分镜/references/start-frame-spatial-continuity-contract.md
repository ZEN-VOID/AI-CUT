# Start Frame Spatial Continuity Contract

本细则约束 `7-分镜` 如何从单个画面点解析 `前景 / 中景 / 后景 / 主体站位`，以及同一画面点内多条分镜之间如何保持这些空间层次的连续性。它展开 `N5-COMPOSITION-DESIGN` 与 `N7-CONTINUITY`，不改变 `SKILL.md` 的输出格式。

## Core Rule

每条分镜必须落成：

```text
分镜N（N-N秒）：景别，景深，构图形式，前景，中景，后景，主体站位。
```

`N-N秒` 只表示输出格式；`N` 必须落在 0.5 秒网格，可为整数或 `.5` 小数。

同一画面点内的多条分镜必须共享同一空间基底。允许景别、焦点、构图形式变化，但前景、中景、后景和主体站位的变化必须能从上一条分镜读出承接或明确调度原因。

## Single Point Spatial Parse

进入 `N5` 时，先为当前画面点建立内部 `spatial_field_map`：

| field | parse_question | output_requirement |
| --- | --- | --- |
| `foreground` | 哪个物体、身体局部、遮挡、门框、树根、兵器、雾线或光影最靠近镜头？ | 写成可见物或可见遮挡，不写抽象情绪 |
| `midground` | 当前观看任务的主要人物、动作、危险或信息落在哪里？ | 通常承载主体或核心动作，不复述整句剧情 |
| `background` | 后方提供什么空间方向、危险来源、退路、目标、证据或情绪压力？ | 写出可见背景与它的叙事作用 |
| `subject_position` | 主体站、坐、跪、伏、贴墙、压边、居中、偏左、偏右，和谁/什么保持关系？ | 必须包含主体空间位置或姿态，不只写主体姓名 |

解析顺序：

1. 先问当前画面点的观看任务：观众此刻要看见什么变化。
2. 再锁定主体、危险源或信息源。
3. 再找可见空间锚点：门框、墙根、树根、石阶、浅沟、桌案、窗、雾线、光斑、道具。
4. 最后填入 `前景 / 中景 / 后景 / 主体站位`，不得直接复制原画面字段的半句。

## Within Point Continuity

同一画面点有多条分镜时，必须建立内部 `within_point_spatial_continuity_map`：

| continuity_field | requirement |
| --- | --- |
| `base_space` | 当前画面点的固定空间基底，例如“左侧树根护送线 / 右侧浅沟危险线” |
| `shot_1_spatial_state` | 第一条分镜的前景、中景、后景、主体站位 |
| `shot_n_delta` | 后续分镜相对上一条改变了什么：景别推进、焦点转移、遮挡变前景、主体前移、危险显形、后景转为目标方向 |
| `carried_anchor` | 至少保留一个连续锚点：主体姿态、道具、方向线、遮挡、危险源、目标方向或声音/光色落点 |
| `illegal_jump_check` | 不得让主体凭空换边、危险源凭空换方向、前景遮挡凭空消失、后景目标凭空改写 |

允许变化：

| change_type | legal_when |
| --- | --- |
| 景别变化 | 仍能追踪同一主体、同一动作或同一道具 |
| 景深变化 | 焦点变化服务注意力转移，关键危险或关系没有被错误虚化 |
| 前景变化 | 新前景来自上一镜可见元素、主体移动后形成遮挡，或画面点内动作造成 |
| 中景变化 | 主体或核心动作确实移动，且主体站位说明变化 |
| 后景变化 | 后景被揭示、被遮挡、被焦点弱化，或成为下一条分镜的目标方向 |
| 主体站位变化 | 有动作、推拉、转身、跌倒、护送、逼近、后退等可见原因 |

不合法变化：

- 分镜1 主体在画面左侧，分镜2 无解释变成右侧。
- 分镜1 后景目标是门外，分镜2 无解释变成窗内。
- 分镜1 前景遮挡是树根，分镜2 同一画面点内树根消失且无构图原因。
- 每条分镜只换几个词，无法读出主体站位和空间层次变化。

## Output Examples

合格：

```text
分镜1（0-1.5秒）：中景，深景深，框架式构图，织坊木架占前景两侧，令狐冲与剑尖血絮压在中景偏右，后景湿布和井状暗处收紧空间，令狐冲站在右侧护住出入口。
分镜2（1.5-3秒）：近景，浅景深，前景遮挡构图，剑尖血絮贴近前景边缘，令狐冲眼神压在中景右侧，后景木架与湿布虚成封闭暗线，令狐冲仍守右侧出入口。
```

不合格：

```text
分镜1（0-1.5秒）：中景，深景深，框架式构图，木架，令狐冲，湿布，令狐冲站着。
分镜2（1.5-3秒）：近景，浅景深，中心构图，剑尖，眼神，背景，令狐冲看着。
```
