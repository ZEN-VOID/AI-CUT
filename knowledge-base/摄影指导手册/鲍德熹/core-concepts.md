# Core Concepts

## 1. 摄影先服务戏，再服务传奇

基于 ASC 对《卧虎藏龙》的拍摄复盘，可以推断鲍德熹最重要的判断不是“把武侠拍得更华丽”，而是“别让华丽把人物和戏挤出去”。因此：

- 先问这场戏的情绪压力由谁承担
- 再问镜头、对比度和色路怎么把这份压力说出来
- 最后才问奇观和风格是否值得被放大

## 2. 低反差不等于平

ASC 资料显示，他会为了戏剧浓度主动收掉高反差、硬光和高饱和。这里的低反差不是“灰”，而是：

- 保留面部体积
- 保护细节和层次
- 让人物情绪先被感到，而不是先被色彩轰到

## 3. 色彩应该沿剧情走，而不是整片锁死

可复用原则：

- 色彩路线应和叙事情绪同步推进
- 不同段落可以有不同主色压力
- 色彩变化要有戏剧动机，不能只因为“这一段想更好看”

## 4. 人脸和兵器要被区别对待

从现有材料可以归纳出两套常见任务：

- 对脸：柔和、体积、呼吸、层次
- 对钢与玉：反射、锐点、危险、诱惑

这意味着：

- 主体人物不该被一刀切成广告柔焦
- 关键兵器和物件也不该被平均照亮
- 光线必须区分“谁该被托住，谁该被点亮”

## 5. 动作成立，先靠地理，再靠速度

对武侠和动作场面，优先级应是：

1. 观众知道人物在哪里
2. 观众知道危险从哪里逼来
3. 观众知道镜头为何靠近
4. 然后才允许速度和奇观放大

如果前 3 点没成立，再快只会更乱。

## 6. 近身而不粗暴

ASC 材料里最有价值的一点，是镜头可以进入动作，但要保持一种“优雅的近身感”。这对 AIGC 摄影规划的启发是：

- 靠近动作不等于手忙脚乱
- 运动镜头应像在与演员共舞，而不是追着演员乱跑
- 即使场面激烈，也要保留节律和呼吸

## 7. 后期与数字修整只能做秩序维护

现有材料表明，后期调色和数字修整在鲍德熹方法里是维持摄影意图一致性的工具，不是第二导演。因此：

- 不能靠调色挽救前期没有想清的摄影逻辑
- 数字处理优先修整风格连续性和技术痕迹
- 若前期的戏剧承光力不成立，后期只会把问题放大

## 8. AIGC 场景下的最小迁移规则

当把这些原则迁移到 AIGC 工作流时，默认遵守：

- 先写 `visual_thesis`，再写镜头和光色参数
- 先给 `contrast_strategy` 与 `palette_route`，再给滤镜或质感词
- 先校正 `geography_readability`，再讨论快慢、飘逸或史诗感
- 先明确 `beauty_softness` 的适用对象，再做统一人脸处理
