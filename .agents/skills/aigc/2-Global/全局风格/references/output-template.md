# 全局风格输出模板说明

canonical 输出固定为：

- `projects/aigc/<项目名>/2-Global/全局风格/全局风格设计.md`

## 必备章节

1. `项目范围与输入锁定`
2. `上游 grouped script 证据摘要`
3. `初始化附加约束`
4. `大师级灵感映射`
5. `全剧视觉命题`
6. `风格词汇造句`
7. `稳定继承法则`
8. `下游使用边界`
9. `通用全局风格提示词前缀（150字内）`

## 章节写法要求

- `项目范围与输入锁定`
  - 明确主输入来自 `1-Planning/3-分组`
  - 明确附加上下文来自 `0-Init`
- `上游 grouped script 证据摘要`
  - 必须体现跨集重复信号，而不是只写剧情梗概
- `大师级灵感映射`
  - 每个锚点写“借用机制 / 适配原因 / 拒绝照搬”
- `风格词汇造句`
  - 至少给出 2 句可复用风格母句
- `稳定继承法则`
  - 说明后续阶段应继承什么、不要误读什么
- `下游使用边界`
  - 说明本文件不直接替代 `类型元素`、`设计元素` 或 shot-level 设计
- `通用全局风格提示词前缀（150字内）`
  - 必须放在文档最后
  - 必须是全局可复用前缀，不得写项目专名、剧情事件或单集细节
  - 总字数不得超过 `150` 字

## 模板使用规则

- 写回前必须读取 `templates/全局风格设计.template.md`
- 可以在不破坏章节顺序的前提下补充小节
- 不得删除上述必备章节
- `通用全局风格提示词前缀（150字内）` 必须保持为最后一节
