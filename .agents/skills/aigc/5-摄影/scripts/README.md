# scripts

本目录只承载 `5-摄影` 的机械辅助脚本，不承担分镜明细、节拍判断、构图、运镜、边界交出、光影或色彩的主创工作。

允许职责：

- 读取 `4-表演/第N集.md`。
- 统计可能的画面性字段。
- 检查命中字段后是否存在 `分镜明细：`。
- 检查每个分镜明细块中的 `分镜N（约X秒）:` 是否从 1 开始连续编号，并机械确认显式时长存在。
- 统计每集分镜数量分布，并在 2 镜块异常集中时给出机械提示；该提示只触发 LLM 复判，不替代节拍判断。
- 检查 frontmatter 中 `stage: 5-摄影`、`source_performance_path`、`output_path` 和 `duration_policy` 是否存在。
- 检查 `分镜N` 的秒数是否符合短剧·AIGC 时值提示（<1s 提示闪切确认，>3s 提示必要性确认，>5s 提示强例外确认）。
- 扫描 `分镜明细` 块中是否出现常见抽象阐释词（如象征、隐喻、寓意等），并给出机械提示。
- 检查 `分镜明细：` 块后是否有空块（无 `分镜N` 条目）。
- 输出机械校验报告。
- 作为质量 review 的输入证据之一；脚本通过不等于 `review/review-contract.md` 通过。

禁止职责：

- 用脚本生成分镜明细正文。
- 用规则拼接替代 LLM 判断节拍点。
- 用脚本按固定比例强行改写 1/2/3/4 镜数量。
- 用脚本裁决源句复述扣除测试、功能性影视投影、非复述型分镜、摄影美学或 AI 视频执行稳定性。
- 自动改写原编导稿正文、对白或场景顺序。
