# Output Template

| Output Contract field | Template alignment |
| --- | --- |
| Required output | 时间线 `dimension_packet` 与 `dimension_report_ref`。 |
| Output format | Markdown 维度报告 + 结构化 packet。 |
| Output path | `projects/story/<项目名>/review/第V卷/时间线.md`。 |
| Naming convention | 文件名按父层 registry `report_filename`。 |
| Completion gate | 时间锚、时序证据与 `spoiler_risk` 可被父层聚合。 |
