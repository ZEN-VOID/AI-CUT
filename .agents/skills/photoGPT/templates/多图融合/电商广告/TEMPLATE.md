# Template: 多图融合 / 电商广告

Template ID: `photoGPT.多图融合.电商广告.v1`

## Input Roles

- 商品图：锁定商品身份、轮廓、材质、颜色和 logo/包装事实。
- 场景图：提供空间、台面、环境光和广告氛围。
- 风格/构图参考图：只提供版式、色彩、光影或镜头语言。

## Prompt Frame

```text
将输入图中的商品主体融合到统一的电商广告视觉中。商品身份必须来自商品图，场景空间和氛围来自场景图，版式与光影可参考风格/构图参考图。

商品锁定：保留商品外形比例、品牌标识、包装结构、材质反射、颜色层级和关键文字，不得重绘成其他商品。

广告画面要求：构图清晰、商品为第一视觉中心，背景服务商品卖点；光线、阴影、反射、景深和接触关系自然统一；画面具有高端商业摄影质感，适合电商主图/详情页/广告横幅。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变商品身份、logo、包装结构和关键文字。
- 不复制参考图中的无关人物、水印、杂物或竞品。
- 不让背景抢走商品主体焦点。
