本文件承载 `aigc 3-明细 / 2-角色表现 / 动作戏` 的路由策略、VSM 与局部回退规则。

## Local Route Note

- 本技能是父级裁定后的唯一 leaf 入口，不在本层再分叉。
- 进入前提以主 `SKILL.md` 的 `When to Use / When Not to Use` 与父级路由结果为准。
- 若当前问题跨 sibling、输入前提不齐或边界不清，先回退父技能重新判路。

## 港式武指可回放标准

1. 攻防必须清楚，不允许“谁打中谁”含混不清。
2. 借势必须成立，不允许人物像无重量纸片般乱飞。
3. 环境和道具必须参与动作，不允许人物在真空里互殴。
4. 每段动作都要写出“前一招逼出的态势，如何成为下一招入口”。
5. 默认基线是写实武侠；若题材命中 `仙侠 / 异能 / 超能 / 机甲近战`，允许超物理强化，但仍必须保留 `起点 -> 触发 -> 迁移 -> 落点 -> 余势` 合法链。

## 物理锚点合同

| 锚点 | 最低要求 | 常见空泛写法 |
| --- | --- | --- |
| 切入线 | 交代正切、斜切、下钻、横扫、贴身压入或绕后切入 | 只写“冲上去” |
| 接触点 | 写清腕骨、肋下、下颌、膝弯、桌沿、门框等具体落点 | 只写“打中对方” |
| 受力传递 | 说明力从哪发、经哪一节传到哪一处 | 只写“猛地一击” |
| 失衡与补偿 | 写命中或落空后重心怎么歪、怎么补 | 只写“被打退几步” |
| 脱离与再逼近 | 写双方如何分开、谁先抢回线位 | 只写“再次缠斗” |
| 借景借物 | 让墙、栏杆、桌面、楼梯、车门等进入结构 | 只写“场面很乱” |
| 伤势与余势 | 写喘、麻、滑、兵器下沉、再起困难等后果 | 只写“受伤倒地” |

## 兵器子类型速查

| subtype | 优势距离 | 主任务 | 更稳的交换链 | 易写虚点 |
| --- | --- | --- | --- | --- |
| `WT-01` 短刀/匕首 | 贴身到半步 | 抢线、突刺、割腕、逼退 | 贴身切入 -> 短促发力 -> 立刻抽线 | 写成长兵器式大开大合 |
| `WT-02` 长刀/单刀 | 半步到一步半 | 断线压迫、横斩驱离、砍肩断步 | 压距离 -> 放刀路 -> 收刀防贴身 | 只剩“刀光霍霍” |
| `WT-03` 长棍/短棍 | 半步到两步 | 点腕、压喉、撬膝、封线 | 棍端试线 -> 压线/别步 -> 抽回再点 | 把棍写成刀 |
| `WT-04` 长枪/矛 | 一步半到三步 | 占中线、拒贴身、逼绕线 | 中线试探 -> 拍偏/回中 -> 再刺或扫退 | 写成连续乱舞 |
| `WT-05` 双匕/双短兵 | 贴身到半步 | 一手挂线、一手进刺 | 挂线 -> 错拍 -> 进刺 -> 抽回换手 | 写成双手乱刺 |
| `WT-06` 软兵器/链鞭 | 一步到多步 | 缠械、抽脸、封退路 | 预摆蓄势 -> 改圈入线 -> 回收再甩 | 只写“鞭影翻飞” |
| `WT-07` 徒手擒拿/夺械 | 贴身 | 抢关节、断根、控人 | 抢接触点 -> 贴入重心 -> 转轴发力 -> 锁控 | 只写“一个擒拿制住” |
| `WT-08` 泛化保守分支 | 未知 | 保守写优势距离和失衡链 | 先距离 -> 再接触点 -> 再余势 | 在证据不足时硬判具体套路 |

## 反空泛动作门禁

- 下列句式若出现，后面必须立刻跟细节链，否则判空泛：
  - “两人迅速缠斗在一起”
  - “几招过后，他被击飞出去”
  - “双方打得难分难解”
  - “她一个漂亮转身把对方解决”
- 单次补写不求招数最多，但至少要有一个完整交换回合。
- 动作如果只剩炫技点而没有冲突目标，优先回到 `目标 -> 阻力 -> 结果` 主链重判。
