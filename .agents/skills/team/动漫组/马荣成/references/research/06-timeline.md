# 06 人物时间线

| 时间 | 事件 | 证据/来源 | 对方法论的影响 |
|---|---|---|---|
| 1961 | 生于香港 | HKCCE；香港舞蹈团节目册 | 本土港漫与香港城市文化语境 |
| 1976 左右 | 14 岁进入漫画行业，15 岁发表首作 | HKCCE | 早期进入商业连载和师徒/助理生产体系 |
| 1970s | 《徙置区风云》《五兄弟》等早期作品 | 香港商报 | 从当时流行港漫画风摸索起步 |
| 1981 | 加盟黄玉郎旗下，开始《中华英雄》相关创作 | 香港商报 | 获得试验画技空间 |
| 1982-12-28 | 《中华英雄》第一期单行本推出 | 香港商报 | 写实画功、电影分镜、武打动作冲击行业 |
| 1980/1982 | 《中华英雄》时间存在来源差异：节目册写 1980 launched，HKCCE 写 1982 started composing | 香港舞蹈团节目册；HKCCE | Skill 中必须避免把单一年份写死为无争议事实 |
| 1989 | 创办天下出版/Jonesky Limited，编创《天下画集》并启动《风云》体系 | HKCCE；香港商报；节目册 | 作者从画师转为出版与 IP 主导者 |
| 1989 前后 | 举办个人漫画展，展示《两极》等 | 香港商报；节目册 | 主张港漫可成为艺术品 |
| 1990s-2000s | 多部作品与《风云》持续出版、外地发行、跨媒介改编 | 节目册；HKCCE | 形成跨地域、跨媒介 IP 影响 |
| 2002 | 与张艺谋电影《英雄》相关推出电影漫画并办原画展 | 香港舞蹈团节目册 | 电影与漫画之间的视觉互译 |
| 2012 | 香港漫画星光大道；马荣成与多位漫画家入选/留手印，步惊云成为公共角色 | Bangumi；香港政府新闻网 2016 回顾 | 《风云》进入香港公共文化地标 |
| 2013 | “一个英雄的诞生”马荣成漫画作品展；台湾“80/90 港台起风云”展 | 香港舞蹈团节目册 | 代表作回顾与跨地展示 |
| 2015 | 《风云》完结，并举办结局篇原画展 | 香港舞蹈团节目册；香港商报 | 长篇 IP 阶段性收束 |
| 2017 | 参与《风云 5D 音乐剧》；香港邮政发行《风云》特别邮票 | 香港舞蹈团节目册；香港邮政 | 跨舞台、邮票与城市文化资产化 |
| 2023 | 香港商报画技专访，透露正筹备《风云》正式结局篇 | 香港商报 | 代表作终局继续被修订 |
| 2025 | 《风云》户外 3D 光影舞蹈表演，马荣成为漫画原著及艺术指导 | 香港舞蹈团节目册 | 从漫画原作进入大型视觉艺术指导 |
| 2026-01 | 第五届港漫动力报道，马荣成为评审委员会主席 | 文汇报 | 晚近角色：基本功与创新并重的产业评审 |

## 最新动态（截至 2026-04-16）

- 2026 年 1 月，文汇报报道第五届“港漫动力”中，马荣成为评审委员会主席，评审标准强调画功、风格、故事架构、演绎能力和创意。
- 2025 年香港舞蹈团《风云》户外 3D 光影舞蹈表演将《风云》继续转译为现场视觉表演，马荣成为漫画原著及艺术指导。
- 2023 年香港商报专访称其正筹备《风云》正式结局篇。
