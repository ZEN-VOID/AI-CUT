# 06 人物时间线：程小东

调研时间：2026-04-15

## 关键时间线

来源类型：以一手/机构时间线、片目和放映资料为主，二手材料仅作交叉核对。

| 时间 | 事件 | 对武术指导方法的影响 |
|---|---|---|
| 1950s | 出生于香港电影/戏曲相关家庭；父亲程刚为电影人 | 早期接触电影工业和传统身段系统 |
| 童年-少年 | 接受戏曲、武术和武行训练 | 建立身段、节奏、翻腾、兵器和现场纪律 |
| 1970s | 进入影视圈，参与电视与电影武术工作 | 形成快速拆动作、排练和现场执行能力 |
| 1979 | 电视武侠《天龙诀》等署名武术指导资料可见 | 早期武侠电视工业经验 |
| 1983 | 《生死决》成为早期代表性导演/动作作品 | 决斗、剑术、奇幻感和动作诗性显露 |
| 1987 | 《倩女幽魂》上映，程小东导演，徐克监制 | 神怪、爱情、喜剧、武侠、音乐与威亚浪漫融合 |
| 1990 | 《笑傲江湖》相关工作 | 金庸武侠现代化动作系统进入高峰 |
| 1992 | 《笑傲江湖之东方不败》 | 飘忽、性别流动、速度和不可捉摸的武侠身体成为标志 |
| 1993 | 《新龙门客栈》《东方三侠》等时代动作语境延续 | 群像、客栈空间、女性动作与类型混合扩展 |
| 2001 | 《少林足球》动作指导 | 武术动作进入喜剧、漫画化和特效物理 |
| 2002 | 《英雄》动作指导 | 色彩寓言、剑术、威亚、舞蹈和国际大片美学合成 |
| 2004 | 《十面埋伏》动作指导 | 竹林、舞蹈、追逐、试探和情感背叛结合 |
| 2006 | 《满城尽带黄金甲》动作指导 | 宫廷阵列、盔甲、群体调度和仪式动作 |
| 2007 | 《投名状》动作指导 | 更重、更脏、更军事化的动作物理 |
| 2011 | 《白蛇传说》导演 | 神怪、爱情、CG 与威亚传统再结合 |
| 2019 | 《诛仙 I》导演 | 仙侠、CG、偶像工业与传统威亚动作继续交接 |
| 2025 | 《倩女幽魂》4K 修复版在 AFI Silver 等机构展映信息出现 | 1980 年代作品继续作为华语奇幻武侠经典被重新放映 |

## 最近动态

- 截至 2026-04-15，未发现程小东 2025-2026 年新的已上映导演长片或大型动作指导作品。
- 2025 年，AFI Silver 页面显示《倩女幽魂》4K 修复版在 2025-11-21 开映，说明其经典作品仍在国际机构放映系统中被重新呈现。
- 其晚近公共讨论主要以大师班、回顾放映、经典作品修复和华语武侠动作遗产为主。

## 时间线提示

程小东的职业阶段不宜简单切成“早期香港/后来内地大片”。更准确的是：

1. 武行与电视武侠训练期。
2. 徐克/香港新武侠合成期。
3. 金庸与神怪浪漫高峰期。
4. 张艺谋国际大片转译期。
5. 仙侠/CG 与传统威亚交接期。
