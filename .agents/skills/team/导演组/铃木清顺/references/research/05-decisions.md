# 05 决策记录与行动

## 关键决策链

| 时间 | 决策/事件 | 可提炼规则 |
|---|---|---|
| 1954 | 转入日活 | 接受制片厂工业命题，而非从独立作者姿态起步 |
| 1956-1967 | 高频拍摄日活类型片 | 在流水线中训练快速执行和低成本风格化 |
| 1963-1966 | 风格越来越强烈 | 在商业框架内逐步提高形式偏航程度 |
| 1967 | 拍摄 `Branded to Kill` | 将杀手片推到叙事断裂和感官符号中心 |
| 1968 | 被日活解雇 | 形式越界产生真实职业后果 |
| 1968-1970s | 诉讼与长期难以拍片 | 作者性与工业制度冲突形成创作断层 |
| 1980 | `Zigeunerweisen` 复出 | 从片厂类型转向梦境、记忆和仪式 |
| 2001 | `Pistol Opera` | 晚期回访杀手神话并进一步抽象化 |
| 2022 | `Branded to Kill` 4K 修复获 Venice Classics 最佳修复片 | 曾经的工业问题片被经典化和修复化 |

## 决策启示

1. **不要先逃离类型**
   - 清顺式反叛建立在类型任务之内。
   - 对创作者的实用启示：先保留入口，反叛才有对象。

2. **越界有成本**
   - `Branded to Kill` 的后果说明“不可理解”不是纯美学胜利，也会带来职业断层。
   - 商业项目必须写出可读性底线。

3. **复出不是回到原点**
   - `Zigeunerweisen` 之后的清顺不再只是日活动作片导演，而是把流亡经验转化为梦幻仪式。
   - 适用于创作者转型、被行业排斥后重建风格的问题。

4. **低成本可以变成方法**
   - 抽象布景、强色彩、物件符号和剪辑断裂是预算限制下的高杠杆设计。

## 矛盾点

- 清顺需要制片厂类型片给他材料，但他的形式推进最终超出制片厂容忍。
- 他的作品被后世经典化，但其方法并不天然适合所有商业交付。
- 他常被说成“反叙事”，但最有效的清顺式作品仍有清晰的类型入口。
