# 解说剧 / Chain Of Thought

本文件承载 `解说剧` 子技能的局部思维链细则。

## Pass Blueprint

### Pass 1: Entry Check

- 确认父级已裁决到 `解说剧`
- 若没有明确解说信号，应回退父级重判

### Pass 2: Narration Leadership

- 先确认哪些信息属于非对白解释通道
- 旁白是主通道，但不等于吞掉对白

### Pass 3: Dialogue Fidelity

- 对白必须保真保留
- 旁白负责解释、连接、补充非对白信息

### Pass 4: Voice Unification

- 旁白主体统一为 `讲述者`
- 内心独白默认关闭，避免叙述口径膨胀

### Pass 5: Downstream Readiness

- 样例必须让下游一眼看出“旁白主导 + 对白保真”的分工
- 若读起来像普通标准剧或纯讲解稿，视为失败
