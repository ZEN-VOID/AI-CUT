# 06 · 时间线

主要来源：Session Press（https://www.sessionpress.com/books/solace）、Zolima CityMag（https://zolimacitymag.com/two-weeks-quarantine-with-photographer-wing-shya/）、Initium（https://campaign.theinitium.com/20201128-photo-wingshya/index.html）、SCoP（https://artguide.artforum.com/uploads/guide.004/id16598/press_release.pdf）、香港旅发局（https://www.discoverhongkong.com/content/dam/dhk/intl/corporate/newsroom/press-release/hktb/2024/06-Movie-Tourism-Local-Premiere-E.pdf）、SCMP（https://www.scmp.com/magazines/style/people/celebrities/article/3346439/their-words-photographer-wing-shya-recalls-20-years-style-highs-hong-kong）。

| 时间 | 事件 | 影响 |
|---|---|---|
| 1964 | 出生于香港 | 后续所有视觉实践都围绕香港流行文化、电影和城市气质展开 |
| 1980s 末 | 对平面设计产生兴趣，赴加拿大 Emily Carr 学习 | 接触更自由的艺术教育，形成“自己找路”的创作方式 |
| 1991 前后 | 回到香港，进入广告、电台、唱片工业 | 从 fine art 转向 commercial/pop，开始高密度图像生产 |
| 1990s 中 | 为歌手设计和拍摄唱片封套 | 建立人物、平面设计、流行文化的混合能力 |
| 1996 | 参与王家卫 Takeo Kikuchi 商业短片相关拍摄 | 进入王家卫系统的前奏 |
| 1997 | 赴阿根廷担任《春光乍泄》剧照摄影和平面设计 | 形成“流动中捕捉情绪”的关键转折 |
| 2000 | 参与《花样年华》剧照和平面视觉；与张国荣拍摄写真《庆》相关素材 | 明星肖像、电影剧照和香港流行文化记忆绑定 |
| 2004 | 继续参与王家卫相关作品，包括《2046》 | 稳定成为电影世界延展者，而非普通片场记录者 |
| 2006 | 东京森美术馆个展《Distraction/Attraction》 | 进入国际艺术展览系统 |
| 2010 | 与陈国辉共同执导《全城热恋》 | 从摄影/平面转为导演 |
| 2011 | 继续共同执导《全球热恋》 | 商业爱情电影路径延续 |
| 2017-2018 | 上海摄影艺术中心大型回顾展《夏永康：越轨》 | 25 年跨媒介创作被系统回顾 |
| 2018 | 《Sweet Sorrow》在 Art Central 展出 | 用大型个人项目回应香港青年和社交媒体时代 |
| 2020 | Blue Lotus《Happy Together》回顾展；Kyotographie《One Light, Different Reflections》 | 疫情期间继续以展览方式回看旧作和个人方法 |
| 2021 | 北京《MOODS 情绪》展 | “情绪”作为其作品关键词被进一步强化 |
| 2024 | Session Press 出版《Solace》；执导《Hong Kong Classics Reimagined》中王家卫致敬短片《Illicit》 | 以摄影集和短片继续整理香港电影记忆 |
| 2026 | SCMP Style 采访回顾香港 20 年 style 变化 | 仍活跃于香港时尚/艺术公共讨论 |

## 最新动态

- 2024 年《Solace》出版，是夏永康首本美国 monograph。
- 2024 年香港旅发局《Hong Kong Classics Reimagined》收录他执导的《Illicit》。
- 2026 年 3 月 SCMP Style 发布他关于香港 style 变迁的口述采访。
