# 03-智能分集

BYKJ `03-智能分集` 把 `02-剧本处理` 的处理后剧本拆成下游可消费的集级文件、分集总表、执行报告和 manifest。

默认输入：

```text
output/[项目名]/02-剧本处理/
```

默认输出：

```text
output/[项目名]/03-智能分集/
├── 分集总表.md
├── 执行报告.json
├── manifest.json
└── episodes/
    ├── 第1集.md
    └── 第N集.md
```

本阶段吸收 `aigc/1-分集` 的分集能力，但不继承其默认小说源路径和 `projects/aigc/<项目名>/1-分集/` 输出路径。BYKJ `03` 的默认正文真源是上游 `02-剧本处理` 输出。
