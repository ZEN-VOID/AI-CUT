# Performance And Scene Craft Contract

## Purpose

本合同把忠实剧本化投影升级为导演和演员可执行的高质量场景设计。它不授权删减事实、改写对白、重排事件或新增剧情因果；它只规定如何把上游小说中的心理、关系、压力和隐含信息转成可见、可听、可表演的戏剧动作。

## Scene Turn Pass

每个场景必须尽量形成可追踪的状态差。若上游只提供连续叙述，也要从原文中提取已有变化，而不是发明新转折。

| axis | question | output evidence |
| --- | --- | --- |
| `entry_state` | 场景开始时，人物、空间、规则或信息处于什么状态 | 场景开头的空间、人物位置、道具状态或群体反应 |
| `pressure_source` | 压力来自谁、什么规则、什么误解、什么时间限制或什么危险 | 可见阻碍、声音提示、系统文字、对手动作或群体视线 |
| `active_want` | 关键角色此刻想达成什么 | `表演提示` 中的目标和行为策略 |
| `obstacle` | 角色为什么不能直接达成目标或直接说出口 | 空间距离、对手注视、规则限制、被听见风险、身份风险 |
| `turning_point` | 上游中哪一个信息、动作、声音或反应改变了局面 | 被强化的画面、声音、道具、沉默或群像反应 |
| `exit_state` | 场景结束时，关系、风险、信息或人物处境和开头有什么不同 | 最后一组动作、表情、空间关系或道具状态 |

`scene_turn_pass` 的输出是 `scene_dramatic_map`，供 `N5-DRAFT` 写作时消费。不得新增 `场景转折说明` 之类解释性字段；转折必须落入既有画面、声音、表演、道具或群像字段。

## Integrated Performance Projection Rule

`scene_dramatic_map`、`performance_task_map`、`blocking_power_map` 是编导规划证据，不是终稿可直接倾倒的段落。终稿不得在场景末尾、分镜组末尾或段落末尾用总结式 `表演提示：...`、`场面调度：...` 汇总角色目标、权力关系和空间布局。

必须把规划结果拆进对应剧本句段：

- 角色目标、阻碍、策略 -> 拆入该角色的 `对白画面`、`角色动作`、`表情特写`、可感知 `心理反应` 或 `内心独白画面`；`心理反应` 必须像演员表演 beat，不能写心理结论。
- 进入 `角色动作` / `动作画面` 的文本必须只保留镜头可实拍的客观动作、神态、语气、生理反应和空间运动；规划层可以判断目标和阻碍，但终稿动作句不得写“试图、想要、打算、意图”等主观预判或心理意图词。
- 主角视角下对他人行为的判断 -> 拆入 `内心独白（主角）` 与 `内心独白画面`，或拆成主角观察到的客观证据和主角反应；不得写成全知视角的客观总结。
- 直接情绪感受 -> 拆入微表情、肢体动作、生理反应、声线变化或主角内心独白，不直接写“感到恶心/难受/愤怒”。
- 空间权力关系 -> 拆入场景开头的静态空间/道具 `环境描写`、人物入场的 `角色动作`、对抗中的 `对白画面`、群体反应或道具位置；人物动作、说话引出和关系判断不得借 `环境描写` 承载。
- 道具归属与视线焦点 -> 拆入 `道具特写`、`对白画面`、`群像画面` 或具体动作。
- 沉默、压制、命令感 -> 拆入声音变弱、动作停顿、众人反应、呼吸和手部细节。
- 动作速度和力度 -> 拆入 `角色动作` / `动作画面`，用快慢、停顿、迟疑、骤发、收住、压住、松开等可表演词体现状态差；不得写成摄影慢镜、快切或节奏说明。
- 转场和未出口信息 -> 优先从道具状态、声音桥、群体反应、空间阻隔、动作中断、环境刷新、人物选择性沉默中选取；视线转移只在上游已有明确观看对象、空间目标或权力关系需要时使用，不作为默认解法。

错误：

```md
表演提示：主位人物想把外部风险转化成可控制的执行命令；他用低声压住满室杂声，外显为指节停敲、目光不离桌上文书。

场面调度：主位人物位于长案尽头，众人沿案两侧低伏；文书和器具停在他手前，所有视线都被迫汇向同一点。
```

正确：

```md
环境描写：议事长案横在室内中央，两侧席位低伏成两列，案首只留出主位人物面前一片空位；文书压在器具旁，纸角被边缘抵住。

角色动作：主位人物一下下敲着案面，忽然停住，指腹压住文书边缘，视线没有离开纸上的名单。

对白（丰臣秀吉，压低声音，字字停顿）：“<上游原句>”
对白画面：他没有抬高声量，长案两侧的杂声却一层层低下去；众人的视线顺着器具和文书收束到他手前。
```

错误：

```md
环境描写：令狐冲咬了一口番薯，甜得烫嘴，他凑到她耳边小声说：任盈盈别过头去，没说话，只是指尖悄悄攥住了他的手腕。
```

正确：

```md
环境描写：扁舟贴着潮雾缓缓起伏，船板边缘积着薄薄盐霜。
角色动作：令狐冲咬下一口番薯，被热气烫得偏过头，仍慢慢往任盈盈身边挪近半寸。
对白画面：任盈盈别过脸，没有接话；她的指尖在袖下收紧，扣住令狐冲的手腕。
```

若必须保留 `表演提示` 或 `场面调度` 字段，只能作为紧贴具体 beat 的局部执行提示，不得出现在场景/分镜组末尾，也不得写成“角色想要 X，因此用 Y，外显为 Z”的规划摘要。默认优先内嵌到正式画面、动作、道具、声音和反应字段；一旦进入 `角色动作` / `动作画面`，必须删除主观意图词，只留下演员能做、镜头能拍的外部行为。

## Subtext To Behavior Rule

小说中的“不信任、试探、心虚、压迫、想保护、想隐瞒、想摊牌”等潜台词，不得直接写成结论。必须转为带目的的行为。

### Actor Performance Control Anchor

当潜台词、心理变化或沉默反应涉及强情绪、压抑情绪、情绪切换、未出口对白、害羞、紧张、愤怒、离别、心虚或恐惧时，必须同时消费 `actor-performance-control-contract.md`。本文件负责“角色要达成什么、被什么阻碍、采取什么行为策略”；`actor-performance-control-contract.md` 负责把该策略细化为微表情、面部肌肉分区、非面部生理联动、环境声音、非对称瑕疵和微动态限制。

终稿不新增 `演技提示词` 或 `表情控制公式` 字段。表演控制只能拆入：

- `表情特写`：眉、眼、嘴、咬肌、鼻翼、下颌等可见变化。
- `角色动作` / `动作画面`：肩、手、重心、呼吸、姿态、道具动作和空间运动。
- `对白画面`：说话前后的停顿、声线变化、手部、对手反应和身体距离。
- `心理反应`：有主体、有触发点、有 GETability 的表演反应。
- `音效` / `音效画面`：环境声、沉默余波、呼吸、衣料、纸张、脚步和空间声音。

如果只得到“他很愤怒 / 她很难过 / 角色很紧张”这类标签，视为 `actor_performance_control_pass` 未完成，必须回到表演规划层重写。

### Transition And Subtext Variety Rule

“看向远方 / 避开对方 / 顺着视线望去”可以是有效的转场和潜台词动作，但它只是行为库中的一种。若连续两个 beat 都用视线承担未出口信息，或一个场景多次用远方地标承接下一场压力，必须换用其他可拍变量。

优先从以下变量中选择，且每次选择要回指当前场景的真实压力：

| variable | use when | example projection |
| --- | --- | --- |
| `prop_state` | 信息压在物件上，角色不能直接说破 | `对白画面：她把锦缎折角压平，指尖停在海浪纹断开的针脚上。` |
| `sound_bridge` | 下一场压力可由声音先进入 | `音效画面：港外锚链声从雾里拖过来，石阶上的笑声慢了半拍。` |
| `group_reaction` | 角色不愿让旁人担心 | `群像画面：孩子们仍围着酒坛笑，最前排的人还没察觉两人的沉默。` |
| `body_interruption` | 角色把话咽回去或临时改口 | `角色动作：他喉结动了一下，把葫芦换到另一只手，没把后半句说出来。` |
| `spatial_blocking` | 风险来自距离、门槛、队列、船舷或城墙 | `场面调度：欢呼的人群隔在两人和码头之间，空出的石阶像一道窄窄的缝。` |
| `environment_refresh` | 同一 slugline 内背景压力改变 | `环境描写：海雾压低后，城墙旗影从雾缝里断断续续露出。` |
| `unfinished_action` | 人物选择不行动本身就是戏 | `角色动作：他本要上前，脚尖却停在湿石边，只把袖口收回去。` |

禁止把转场固定写成：

- `角色A没有看角色B，而是看向远方。`
- `角色B顺着角色A的视线望去。`
- `远处某地标在风里飘着。`

上述写法只有在“观看”本身是上游信息、空间侦察、威胁发现或权力对抗时才可使用；否则优先改为声音、道具、群像、空间或动作中断。

| source pattern | weak projection | strong projection |
| --- | --- | --- |
| `他不信任她` | `表情特写：他露出不信任。` | `表演提示：他接过纸条前停半拍，先看她的手，再看纸条背面。` |
| `她在试探他` | `对白画面：她试探地看着他。` | `对白画面：她把问题说到一半停住，视线落在他握紧的手指上，等他先补完后半句。` |
| `他不想暴露自己知道规则` | `心理反应：他隐藏秘密。` | `心理反应：他没有立刻追问，先低头整理课本，把真正的问题压到最后两个字才抬眼。` |
| `全班被压住` | `群像画面：大家都很害怕。` | `群像画面：前排学生的笔尖悬在纸面上，后排有人把椅子往桌下缩了半寸。` |
| `关系发生微妙变化` | `环境描写：两人的关系变了。` | `场面调度：陈大雷把纸条推过桌线，林寂没有收进书包，只用指尖按住边角。` |
| `主角判断她在试探` | `对白画面：她是在试探他。` | `内心独白（林寂）：“她在等我先补后半句。”` + `内心独白画面：她的问题停在半截，视线落到林寂握紧的手指上。` |
| `角色感到愤怒但不能发作` | `角色动作：他试图压住怒火。` | `角色动作：他把纸角抚平两次，第二次指节压得发白；抬眼时下颌绷住。` |

## Actor Task Rule

`表演提示` 在规划层必须写成演员可执行的任务，而不是角色心理标签；在终稿层应优先拆入对应剧本句段，而不是作为场景末尾总结块。

推荐结构仅用于规划层，不得原样作为终稿动作句：

```text
表演提示：<角色>想<即时目标>，但<阻碍/风险>；他/她通过<具体行为策略>完成，外显为<眼神/呼吸/停顿/手部/身体距离/声线/道具动作>。
```

允许写：

- 目标：试探、拖延、安抚、遮掩、施压、求证、转移注意、争取同盟。
- 阻碍：不能暴露信息、不能激怒对方、不能被旁人听见、规则限制、身份风险、时间压力。
- 策略：反问、沉默、避视、抢先递道具、后退半步、把问题压到最后、用玩笑卸力。
- 外显：呼吸变浅、手指压纸、眼神绕开、声线压低、停顿半拍、身体前倾、脚尖退回桌下。

禁止写：

- `表演提示：他很害怕。`
- `表演提示：她感觉自己赢了。`
- `表演提示：他似乎拿到底牌。`
- `表演提示：这是命运的转折。`
- `角色动作：他试图让自己冷静下来。`
- `角色动作：她想要装作没听懂。`

## Blocking And Power Rule

场面调度必须服务人物关系、权力变化和信息差。`2-编导` 可以在规划层使用 `场面调度`，但终稿应优先把它拆入环境、动作、对白画面、道具和群像反应；不得提前写摄影机位、景别、镜头运动或分镜方案。

优先使用以下可拍变量：

- 高低：谁站着，谁坐着，谁被迫抬头或低头。
- 远近：谁靠近，谁后退，谁保持安全距离。
- 门槛：谁占据门口、讲台、窗边、黑板、走廊口。
- 视线：谁看谁，谁避开，谁被群体视线压住。
- 道具权力：纸条、课本、教鞭、黑板字、手机、系统提示在谁手里或谁面前。
- 空间隔离：课桌、讲台、门框、窗帘、阴影、空座位如何分隔人物。

示例：

```md
场面调度：苏红叶站在讲台内侧，教鞭横在讲桌边缘；林寂坐在第三排靠窗，课桌和前排同学的肩膀把他半遮住。
```

## Silence And Reaction Rule

高质量编导稿必须给沉默和反应留下可执行承托。上游没有新对白时，不得为了补戏新增对白；优先使用反应、停顿、声音空缺和动作余波。

| situation | preferred projection |
| --- | --- |
| 角色听到威胁 | 呼吸停顿、手指松开、视线落到出口、肩颈僵住 |
| 规则信息出现 | 群体停止动作、纸张翻动声消失、黑板字反光映在眼睛里 |
| 对白后有潜台词 | 对手不接话、道具停在两人中间、目光先避后回 |
| 高点兑现后 | 声音短暂抽空、群像迟半拍反应、角色身体状态改变 |

沉默不是空白，必须写成可见/可听的状态变化。

## Novel-To-Performance Rewrite Matrix

| novel source | primary field | supporting field |
| --- | --- | --- |
| 主观恐惧、羞耻、心虚 | `表演提示` / 可感知 `心理反应` | `表情特写`、呼吸、手部、身体距离 |
| 推理、意识到、记起 | `内心独白` / 可感知 `心理反应` | `道具特写`、系统文字、角色视线停点、停顿或吞咽 |
| 主角视角判断他人 | `内心独白（主角）` / `心理反应` | 对方当前可见动作、语气、停顿和主角反应 |
| 直接情绪感受 | `心理反应` / `表情特写` / `角色动作` / `内心独白` | 微表情、肢体动作、生理反应、声线变化 |
| 关系变化、信任变化 | `场面调度` / `表演提示` | 道具转移、距离变化、视线变化 |
| 压迫感、危险气质 | `环境描写` / `群像画面` | 光线、门窗、停顿、声音空缺 |
| 规则、身份、数值、公告 | `旁白（系统提示）` / `规则显影` | `系统画面`、黑板、纸张、屏幕、反应 |
| 爽点、高光、认知翻转 | `道具特写` / `群像画面` / `表演提示` | `音效画面`、状态差、余波 |

## Quality Gate

交付前必须检查：

- 每个关键场景至少有一个可回指上游的 `turning_point` 或明确说明该场只是承接/过渡。
- 重要心理变化没有只停留在“他意识到/她觉得/大家害怕”的结论句。
- 主角视角下对他人行为的判断没有写成客观第三方概括；必要时已转入主角内心独白。
- `角色动作` / `动作画面` 没有“试图、想要、打算、意图”等主观预判词，只写可实拍客观动作、神态、语气和生理反应。
- 主观情绪感受没有直接写“感到恶心/难受/愤怒”，已转成微表情、肢体动作、生理反应或主角内心独白。
- `心理反应` 没有写成抽象想象、内心散文或因果解释；每条都能被演员表演、观众从画面或声音中接收。
- `表演提示` 至少包含目标、阻碍、策略或外显动作中的两项。
- 规划层的表演任务和场面调度已内嵌到对应剧本句段，没有在场景末尾或分镜组末尾总结式列出。
- 权力关系变化通过空间、视线、道具或身体距离表现，而不是只写关系结论。
- 转场、潜台词和未出口对白没有连续依赖同一种视线动作；若出现看向远方，能说明观看对象为何是当前 beat 的真实压力或信息来源。
- 沉默和反应被写成可见/可听状态，未用新增对白替代。
- 没有出现摄影越权字段，如 `分镜明细预设`、机位、景别或分镜编号。
