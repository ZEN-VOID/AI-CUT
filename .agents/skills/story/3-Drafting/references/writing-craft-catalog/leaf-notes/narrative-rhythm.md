## 一、节奏感到底是什么

节奏感不是“写得快”，而是读者在阅读时能感到：

- 什么时候该快
- 什么时候该慢
- 为什么现在停一下
- 为什么下一句必须追下去

节奏失衡常见症状：
- 全章都一个速度
- 段内解释太满
- 情绪刚起就被说明打断
- 关键处不加速，过渡处却写很重

---

## 二、五个节奏单位

### 1. 句子节奏
- 短句：打击、判断、失控、惊讶、爆点
- 长句：观察、铺垫、迟疑、压抑、拉长痛感

### 2. 段内节奏
一段内部最好有一个明确重音：
- 动作落点
- 情绪落点
- 信息落点
- 关系落点

### 3. 段落节奏
段落与段落之间要有快慢差：
- 短段提速
- 中段承接
- 长段蓄压

### 4. 场景节奏
场景内部的推进顺序：
- 进入
- 对位
- 变化
- 落点

### 5. 章节节奏
整章的呼吸：
- 开场抓手
- 中段推进
- 后段变奏
- 结尾钩子

## 三、段内叙事节奏模板

### 模板 A：动作 -> 反应 -> 判断

```
门开了。
她下意识抬头，手指顿在半空。
不是他。
```

适合：
- 突发
- 误判
- 小反转

### 模板 B：观察 -> 细节 -> 情绪渗出

```
窗台上还放着那只旧茶杯，杯沿有一道细小裂痕。
她记得那是去年冬天摔的。
胸口忽然就空了一下。
```

适合：
- 抒情
- 回忆
- 关系余味

### 模板 C：对白 -> 动作锚 -> 关系位移

```
"你可以不说。"

他把伞往她那边偏了偏，却没再看她。

"可你今晚要是走了，以后就别再回来找我。"
```

适合：
- 对手戏
- 暧昧攻防
- 临界关系

## 四、快慢切换原则

### 什么时候该快
- 动作已经明确
- 情绪已到爆点
- 决策必须立刻落地
- 战斗、追逐、争吵进入正面交锋

### 什么时候该慢
- 关键信息刚出现，需要读者消化
- 情绪刚拐弯，需要身体感受
- 场景初入，需要建立空间和气息
- 决策前，需要让代价浮起来

原则：
- 快不是乱跳
- 慢不是注水

---

## 五、信息密度控制

一段里尽量不要同时承载：
- 世界观解释
- 人物情绪
- 关系变化
- 行动推进
- 伏笔回收

若一段塞了 4-5 件事，节奏通常会坏。

优先做法：
- 一段一主功能
- 次功能只做轻触

---

## 六、段落长短的情绪作用

| 段落形态 | 常见效果 |
|----------|----------|
| 单句短段 | 重击、停顿、强调 |
| 两三句短段 | 快速推进、持续压迫 |
| 中段 | 稳定承接、正常推进 |
| 长段 | 蓄压、说明、沉浸，但风险是拖 |

修复口诀：
- 爆点前短
- 爆点时更短
- 爆点后给半口气

---

## 七、常见失衡类型

### 1. 说明压死情绪
角色刚有感觉，作者立刻解释原因、背景、意义。

修复：
- 先保留身体反应和动作
- 解释延后一拍

### 2. 全程短句导致单调
看似很快，实际像一直敲木鱼。

修复：
- 在关键处插一段稍长的观察或心理

### 3. 全程长段导致发闷
信息都在，但读者滑不动。

修复：
- 拆段
- 给对话
- 给动作锚
- 给段尾落点

### 4. 爆点不给加速
高潮来了，句子和段落还在慢慢解释。

修复：
- 删解释
- 保动作
- 保结果
- 保代价

---

## 八、节奏与题材的适配

### 战斗章
- 快段多
- 动作优先
- 信息只给最必要的

### 暧昧章
- 快慢交替
- 停顿很重要
- 空白比解释重要

### 悬疑章
- 关键不是一直快，而是“收”和“放”
- 该藏的时候藏，该炸的时候炸

### 过渡章
- 不能全平
- 至少要有 1-2 个小脉冲

---

## 九、节奏修复流程

1. 标出本段主功能
2. 找出本段重音句
3. 检查是否有无效解释压在重音前
4. 调整句长
5. 调整段长
6. 确认这一段结束后，读者得到的是推进、余味还是压力

若三者都没有：
- 这段大概率该删或重写

---

## 十、禁忌

- 禁止所有段落都差不多长
- 禁止每个情绪节点都立即解释透
- 禁止在爆点段塞进大段设定说明
- 禁止把“断句很多”误当成“节奏很好”
- 禁止一章从头到尾没有节拍变化
