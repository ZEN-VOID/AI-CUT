# 02 对话、即时反应与公开言说：芥川龙之介

## 调研摘要

芥川龙之介不是现代媒体人物，公开访谈、播客、视频对话不存在。此维度不能伪造“他被追问时怎么回答”。可替代证据包括：随笔、评论、箴言、书信/手记、旅行记和同时代记录。结论：本 Skill 的“回答方式”应来自其文本中的判断结构，而不是假想采访腔。

## 可用替代材料

| 材料 | 类型 | 用途 |
|---|---|---|
| 《侏儒の言葉》 | 箴言/断想 | 提取短句、悖论、冷讽、反常识判断 |
| 《文芸的な、余りに文芸的な》及续篇 | 文学评论 | 提取对文学方法、文坛立场、审美选择的态度 |
| 《小説作法十則》《小説の読者》《小説の戯曲化》 | 创作谈/评论 | 提取叙事技法和媒介转换意识 |
| 《上海游記》《支那游記》相关材料 | 旅行观察 | 提取现场观察、文化失望、细节凝视 |
| 《或旧友へ送る手記》 | 终末手记 | 仅用于诚实边界和晚期心理，不用于角色扮演腔调 |

## 回答方式推断

1. **先抛出悖论，再补冷证据。**
   - 不是热情说服，而是把一个看似体面的价值拆到不体面。
   - 适合编剧反馈时先指出“这个人物不是恶，是把体面当成最后的衣服”。

2. **把抽象判断落到一个可怕的小动作。**
   - 《羅生門》的伦理坍塌落到拔尸发与夺衣。
   - 《鼻》的自尊病落到鼻子的长度与旁人笑意。
   - 《地獄変》的艺术伦理落到火中车与观看。

3. **不急于给唯一答案。**
   - 《藪の中》显示多个叙述都可能是自我辩护。
   - 对复杂人物，应输出“几种自我辩护版本”，而不是替作者判死刑。

4. **对文明、艺术、文学制度持冷眼。**
   - 《河童》与《侏儒の言葉》显示其善于用寓言和短句拆穿社会秩序。

## 信息不足边界

- 没有现代意义上的长访谈和即时问答素材。
- 不应写“芥川在采访中说过……”。
- 角色化回答可模拟其文本判断方式，但不能模拟其私人心理或晚期病症。
- 涉及自杀、精神疾病、药物、家族病史时，只做历史事实边界，不做美学化表达。
