# Field Routing And Audio-Visual Contract

## Audio-Visual Pairing

声画一体性是强制门禁。文本与声音条目必须就近配对同命题画面字段：

| 声音字段 | 必须配对的画面字段 |
| --- | --- |
| `对白（角色，语态）` | `对白画面` |
| `独白（角色）` | `独白画面` |
| `内心独白（角色）` | `内心独白画面` |
| `旁白（主体）` | `旁白画面` |
| `音效（来源）` | `音效画面` |

字段纯度：

- `对白/独白/内心独白/旁白/音效` 只写可听见、可说出或可被声音设计呈现的文本/声音本体。
- `对白` 字段标题固定为 `对白（真实角色名，语态/状态短语）`，例如 `对白（阿真，眼里带笑）`；`原文角色`、`角色名`、`某人` 只是规则占位，不得作为角色名进入终稿。
- 第二项用于展示上游已有或可从上下文判断的说话状态，可以是声音、呼吸、节奏、情绪、姿态或关系压力，例如 `压着笑意`、`声音发颤`、`故作轻松`、`眼里带笑`、`酒意未散`、`冷静得近乎无声`。不要求一词，不要求以“地”结尾；不得借该项改变对白含义、人物态度或新增心理解释。
- 对应 `*画面` 只写可见画面、表演、空间或承托。
- 画面字段不得重复解释声音文本。
- 声音字段不得塞入动作描写。
- `*画面` 字段不得写抽象概念、心理结论、主题解释、比喻结论或任务说明；必须落到可见的身体、空间、道具、文字、光线、运动或表情。
- `角色动作` / `动作画面` 只写镜头可实拍的客观动作、神态、语气、生理反应和空间运动；不得写“试图、想要、打算、意图、准备借此、为了”等主观预判、心理意图或目的说明。
- 主角视角下对他人行为的主观解读不得写成客观画面结论；应转入 `内心独白（主角）`、主角的 `心理反应`，或拆成主角能看见/听见的证据。
- 直接主观情绪词不得作为画面结论，例如“感到恶心、难受、愤怒、害怕、崩溃”；应转成微表情、肢体动作、生理反应、声线变化或主角内心独白。
- 抽象概括、无法实拍的总结句，尤其是体现重复、熟悉、往日常态的句子，不得进入画面字段；只保留当前场景当下可拍的核心动作、状态和情节。
- `心理反应` 不是内心解释字段。它只能写观众可通过画面、声音或演员表演接收到的反应：身体微动作、表情变化、呼吸、停顿、声线、吞咽、手部处理、道具停点、空间距离、对手不接话或短促可听独白；不得写“他意识到/她觉得/内心崩塌/灵魂震动/因为所以”等离开文字无法感知的抽象心理。
- `表演提示` 在规划层必须可执行：写演员可做的身体任务、目光任务、呼吸、停顿、动作力度和对手关系；终稿应优先拆入具体 `对白画面`、`角色动作`、`表情特写`、`群像画面` 或 `心理反应`，不得在场景末尾总结式列出。
- `场面调度` 在规划层只写人物与人物、人物与空间、人物与道具之间的可见关系，例如站坐、高低、远近、遮挡、门口/讲台/窗边占位、道具归属与视线方向；终稿应优先拆入环境、动作、道具、对白画面和群像字段，不得写摄影机位、景别、镜头运动或分镜编号。
- 视线动作不得成为潜台词、转场和未出口对白的默认承托。若同一场景已用过“避开视线 / 看向远方 / 顺着视线望去”，后续同类 beat 应优先改用手部、道具、声音、群像、身体距离、动作中断、空间阻隔或环境刷新。
- 任何字段都不得输出任务说明、占位说明或规则复述，例如“本场按上游原文顺序承接...”“说话者的视线...”“不新增事件结果”“引号内不加入动作”。这些只能作为内部执行约束，不能进入 canonical 剧本正文。

## Environment Field Purity

`环境描写` 只写场景本身的可见写景材料：地点、空间结构、自然景物、自然条件、空气介质、光照状态、承载面、围护面、开口边界、静置物件和整体氛围。它服务美术、置景、情绪烘托和场景进入感，不承载剧情推进、人物动作、对白引出、心理解释或规则说明。

`环境描写` 不限定一场只出现一次。场景开篇的环境描写用于建立地点、空间、光线和氛围；同一 slugline 内，若剧情推进导致可见环境焦点变化，也可以再次出现环境描写，作为“环境刷新”。

允许重复出现的触发：

- 空间层次变化：从室内移动到门廊、窗边、院中、甲板外沿、楼梯口、屏风后、角落或另一处可见背景。
- 视野背景变化：人物仍在同一场景标题下，但画面背景从主厅转到墙角、案边、窗外、门口、船舷、廊下、阴影处。
- 光线/空气变化：日光被云遮住、火光压低、雾气加重、尘土被卷起、雨声贴近、烟气散开。
- 材质/承载面变化：地面、桌面、墙面、甲板、水面、门槛、石阶等成为新的情绪或信息承托面。
- 情绪背景变化：同一空间的自然景物、静物、空位、阴影、余声或沉默随剧情压力显出新的氛围层。

使用限制：

- 不因每个动作或每句对白机械重复环境描写；只有可见环境焦点真的变化，或需要承托新的情绪/空间压力时才刷新。
- “环境刷新”不得替代 `角色动作`、`对白画面`、`群像画面`、`道具特写` 或 `场面调度`。
- 若角色从室内走到室外但仍属于同一连续空间和同一 slugline，可在同一场景标题下追加新的 `环境描写`；若地点或日夜已构成真实场景变化，应按 slugline 规则另开场景。

允许写：

- 自然景物与自然条件：飘雪、落叶、朝露、风沙、雨丝、雾、烟、尘、潮气、热浪、冷气、风。
- 光照状态：日光、月光、烛火、灯管、火光、屏幕光、阴影边界。
- 承载面与围护面：地面、甲板、桌面、墙面、门窗、舱门、城墙、窗棂。
- 静置物件和痕迹：家具、光源、痕迹、纸张、衣料、器具在空间中的状态。
- 群体造成的环境状态，例如“甲胄声在长案两侧压低”，但具体人物动作仍应拆入 `角色动作` 或 `群像画面`。

## Atmospheric Environment Enrichment

优秀的 `环境描写` 不是寥寥几笔的地点点缀。它可以用景境本身烘托叙事心境：飘雪、落叶、朝露、风沙、雨丝、雾气、日影、火光、尘土、潮气等自然景物，都可以在符合情境时增强孤独、压迫、松弛、危险、离别、迟疑、肃杀或温柔。

允许增强的前提：

- 与场景地点、季节、时段、类型气质和上游情绪压力相容。
- 只改变表现层，不改变剧情事实、人物行动条件、事件结果或因果。
- 自然景物只做氛围和心境承托，不成为新线索、新障碍、新规则、新危险或新事件。
- 若上游没有明确写到该自然景物，但场景语境允许，可作为 `controlled_enrichment` 的环境氛围项写入，并在报告中记录 `source_signal / added_detail / risk_check`。

禁止增强：

- 为了戏剧化强行改天气、改季节、改地理条件。
- 让新增自然景物造成角色受伤、迷路、失明、误判、延误或发现线索。
- 用“他心里很冷”“她的孤独铺满天地”这类心理解释替代可见景物。

合格写法：

```md
环境描写：院中薄雪落在石阶边缘，廊下灯火被风压得发颤，未化的雪水沿门槛慢慢聚成一道冷亮细线。
```

这类写法可以承托肃杀、迟疑或离别，但不新增事件结果。

禁止写入 `环境描写`：

- 人物连续动作：咬、凑近、握住、别过头、说话、收袖、拔刀、奔跑、回望、灌酒。
- 对白引出句：`他小声说：`、`她低声对他说了一句话：`、`沉声吐出日语：`。
- 剧情结果、背景概要、任务说明或固定承接规则。
- 关系结论或心理判断，例如“两人关系变了”“众人被震慑”“无人知晓”。

若一句同时包含写景和人物动作，必须拆分：

```md
环境描写：海雾贴着船舷，湿木板上浮着一层淡淡盐霜。
角色动作：令狐冲咬下一口番薯，被热气烫得偏过头，仍往任盈盈身边挪近半寸。
对白（令狐冲，声音压低）：“<上游原句>”
对白画面：任盈盈别过脸，没有接话；她的指尖在袖下收紧，扣住令狐冲的手腕。
```

同一 slugline 内环境刷新示例：

```md
### 场景3：内景 旧宅正厅 - 夜

环境描写：正厅梁木低沉，桌上油灯把墙面照出大片暗黄，窗纸外的雨声被门缝压成细线。
角色动作：阿真把信纸压在灯下，指尖停在落款处。
对白（阿真，声音很轻）：“<上游原句>”
对白画面：对面的人没有立刻接话，视线落到信纸边缘的水痕上。
环境描写：门廊外的雨水沿石阶往下淌，廊柱背后空出一片黑，风把半扇门推得慢慢晃动。
角色动作：阿真转到门边，手掌贴住门框，回头看向屋内。
```

这里第二条 `环境描写` 不是新事件，也不是人物动作；它刷新了同一场景内从正厅到门廊背景的空间和氛围。

## Action Field Dynamics

`角色动作` / `动作画面` 不只回答“人物做了什么”，还要在上游已有或戏剧需要明显时回答“动作以什么速度、节奏和力度发生”。速度感只强化可拍执行，不改变事件结果。

优先写入的动作维度：

- 身体部位：手、眼、肩、背、脚步、呼吸、重心。
- 位移与空间：靠近、后退、转身、扑倒、掠过、贴墙、越过门槛。
- 姿态变化：站起、伏低、僵住、偏头、俯身、半跪。
- 速度节奏：骤然、猛地、飞快、迟半拍、慢慢、缓缓、停半拍、一下下。
- 力度质感：压住、攥紧、松开、砸下、轻碰、拖拽、收住。

使用规则：

- 动作字段必须是客观可拍文本，只写身体行为、神态、语气、呼吸、停顿、重心、空间位移、道具操作和生理反应。
- 动作字段禁止主观意图词：`试图`、`想要`、`打算`、`意图`、`准备借此`、`为了让/为了掩饰`、`想借此`。若这些词承载主角内心，应改入 `内心独白（主角）`；若只服务表演，应改成可见动作。
- 上游写出明确速度或力度时必须保留，例如“鞭影一闪”应转成快速动作，“缓缓收进袖中”应转成慢动作。
- 上游没有速度信号时，只在表演、悬念、危险或状态差需要时补轻量节奏词；不得为了“更燃”把普通动作改成高速动作。
- 快慢必须服务人物状态或场景压力：快动作常用于袭击、逃离、抢夺、反应失控；慢动作常用于迟疑、压迫、隐藏、仪式感或决定落下。
- `动作画面` 不写摄影术语，例如慢镜头、快切、推镜；只写人物动作本身的速度与节奏。

不合格：

```md
角色动作：林寂试图压住怒火，想要装作没听见。
```

合格：

```md
角色动作：林寂把纸角抚平两次，第二次指节压得发白；他抬眼时下颌绷住，没有接话。
```

## Prop Close-Up Field Purity

`道具特写` 让物件承担信息、压力、规则、关系或状态变化。它不限定具体物件种类，而是按物件在场景中的功能进入字段。

优先写入的道具维度：

- 信息载体：纸面、屏幕、牌匾、衣料、卷轴、刻痕、标签、地图、名单、印记。
- 规则显影物：黑板、系统界面、公告、符号、红线、编号、封条、仪式物。
- 权力与归属：谁持有、谁压住、谁推开、谁不敢碰、物件停在谁面前。
- 状态变化：打开、合拢、折起、破损、染色、烧焦、沾湿、断裂、移位、消失。
- 线索痕迹：血迹、灰尘、划痕、针脚、缺口、指印、水痕、反光、气味来源的可见痕迹。

使用规则：

- `道具特写` 写物件的可见状态和信息，不写角色对物件的主观解释；主观判断拆入 `心理反应`、`内心独白` 或 `对白画面`。
- 物件被角色拿、推、收、抢、按住时，可写 `角色动作`；当重点是物件信息、状态变化或归属压力时，写 `道具特写`。
- 道具本身不应自动新增功能、规则或线索含义；只能显影上游已有信息或承托已存在的关系压力。

## Psychological Reaction Field Anchor

`心理反应` 的详细字段语义、类型化转译、演员表演口径、证据结构和 review checklist 以 `references/psychological-reaction-contract.md` 为准。本文件只保留字段路由入口：

- `心理反应` 必须写明确主体的可感知反应，不写无主体心理结论。
- 只能靠文字解释才能成立的内容，应改入 `独白（角色）` / `内心独白（角色）` 并配对对应画面；主角视角下对他人行为的判断优先进入 `内心独白（主角）`。
- 已能拆入 `对白画面`、`角色动作`、`表情特写`、`道具特写`、`群像画面` 或 `场面调度` 的内容，优先拆入具体字段。
- 任何 `心理反应` 都不得新增对白、幻象、回忆、事件、因果、答案或人物动机。

## Required Subject Marking

- `对白（林寂，呼吸发紧）：“这什么情况？”`
- `独白（林寂）：“国运？副本？SSS级？”`
- `内心独白（林寂）：“她发现了。”`
- `旁白（系统提示）：“天道系统·全球公告。”`
- `音效（高跟鞋）：“嗒、嗒、嗒。”`

所有对白、内心独白、旁白都必须显式带主体。对白还必须显式带语态或角色状态，格式为 `对白（角色名，语态/状态短语）`。音效必须显式带来源；来源不可见时可写 `音效（不可见来源）`。

## Sound Literal Rule

声音字段必须写声音本体，而不是事件说明、时间说明或叙述概括。

| 错误 | 原因 | 修正 |
| --- | --- | --- |
| `音效（铃声）：“八点四十分铃声。”` | 这是时间说明，不是声音 | `音效（铃声）：“叮铃铃——”`，时间写入 `音效画面` 或上下文 |
| `音效（走廊外）：“一声短促的闷响。”` | 这是描述性句子，不是可听声音本体 | `音效（走廊外）：“呃。”` 或 `音效（走廊深处）：“咚……咚。”` |
| `音效（苹果）：“尖叫声。”` | 声音类别，不是声音表现 | `音效（苹果）：“咿——！”` |
| `对白（某人）：“他沉默了。”` | 叙述被误作对白 | 若上游不是直接引号对白，改为 `动作画面` 或 `表演提示` |
| `对白（原文角色）：“别闹。”` | 把规则占位误当角色名 | `对白（阿真，眼里带笑）：“别闹。”` |
| `对白（阿真）：“别闹。”` | 缺少说话状态展示口 | `对白（阿真，眼里带笑）：“别闹。”` |

对白字段只承接上游明确引号对白，或用户明确授权新增的对白。不得把非引号叙述、概括性说明、作者判断转为对白。若上游原文已有引号，例如 `“普通反应。”`，则按对白冻结保留。

| 地点、空间结构、自然条件、空气介质、光照状态、承载面、围护面、开口边界、静置物件，以及同一 slugline 内空间/背景/光线/空气/材质焦点变化 | `环境描写` | 只写场景本身的写景材料，服务氛围与后续美术；可以在场景开篇和剧情推进后的环境刷新处重复出现；不混入人物动作、对白引出、心理解释、剧情结果或任务说明 |
| 高跟鞋、铃声、苹果尖叫、机械提示、走廊异响 | `音效` + `音效画面` | `音效` 写声音来源、质感、节奏和空间方位；`音效画面` 写声音触发时的可见源头、人物反应、空间承托或不可见来源的画面处理 |
| 信息载体、规则显影物、关键物件、线索痕迹、归属关系、状态变化 | `道具特写` / `规则显影` | 让物件承担可见信息、规则压力、关系归属或状态变化；不把物件解释成角色心理结论 |
| 天道系统、真理之眼、好感度、死亡通报 | `系统画面` / `规则显影` / `旁白（系统提示）` | 文字信息可由旁白承载，画面字段负责可视化呈现 |
| 群体反应、全班凝固、集体低头、哭声扩散 | `群像画面` | 用可见行为表达群体心理 |
| 角色主观判断、恐惧、推理、记忆回闪 | `角色动作` / `表情特写` / `道具特写` / `心理反应` / `独白` / `内心独白` | 优先转成可见身体、表情、道具停点、空间距离或可听独白；主角判断他人行为时优先用 `内心独白（主角）`；`心理反应` 必须写演员可表演、观众可接收的反应，不写抽象想法；不要塞进 `动作画面` |
| 主角视角下对他人行为的解读、判断、怀疑 | `内心独白（主角）` / `内心独白画面` / `心理反应` | 把“她在试探他/他早有准备/对方在撒谎”等改成主角内心独白，并配当前可见证据；不得写成客观第三方概括 |
| 直接情绪感受：恶心、难受、愤怒、害怕、崩溃 | `心理反应` / `表情特写` / `角色动作` / `内心独白` | 转成微表情、肢体动作、生理反应、声线变化或主角内心独白；不得直接写“感到...” |
| 往日常态、重复熟悉、过往背景、物品来历 | 删除 / `内心独白（主角）` / 当下可拍物证 | 非当前主线必要信息删除；必要信息只转成当前场景可拍证据或主角内心，不新增回忆镜头或来历解释 |
| 角色目标、阻碍、试探、隐瞒、潜台词 | `对白画面` / `角色动作` / `表情特写` / `心理反应` | 转成带目的的行为策略：停顿、避视、道具动作、身体距离、声线变化、对手不接话，不写心理标签；`表演提示` 只作规划证据或紧贴 beat 的局部提示 |
| 权力关系、信任变化、压迫位置、群体注视 | `角色动作` / `道具特写` / `群像画面` / `对白画面` / 紧贴 beat 的 `场面调度` | 用站坐、高低、远近、门口/讲台/黑板占位、道具归属和视线压力表达关系变化；只有纯空间或静置道具状态可进入 `环境描写`，`场面调度` 不作为场景末尾总结字段 |
| 对白后的沉默、迟疑、反应和余波 | `表演提示` / `表情特写` / `群像画面` / `音效画面` | 不新增对白；用呼吸、停顿、纸张声消失、手部动作、对手不接话等承托潜台词 |
| 转场、未出口对白、下一场压力浮现 | `音效画面` / `道具特写` / `群像画面` / `角色动作` / `环境描写` / `转场` | 从声音桥、物件状态、群体欢呼变弱、动作中断、空间阻隔或环境刷新中选择；不要默认写角色看向远方或另一人顺着视线望去 |
| 角色服装、色彩构成、危险气质、非动作造型信息 | `角色造型` / `表情特写` | 服务造型、美术和表演，不作为动作句处理 |
| 国运灾难、新闻碎片、现实世界同步后果 | `现实灾难画面` / `系统画面` | 作为短促压力插针，不挤占主叙事 |

## Concrete Field Rewrite Examples

| 不合格字段 | 问题 | 合格投影 |
| --- | --- | --- |
| `道具特写：字迹确实是他的...他从来不这样写字。` | 后半句是主观判断 | `道具特写：扉页写着“林寂”；“寂”字最后一撇异常拉长。` + `心理反应：林寂盯住那一撇。` |
| `道具特写：这件东西证明他已经掌控全局。` | 解释性结论，不是道具状态 | `道具特写：名单被压在他掌心下，只露出最后一行红字；对面的人没有伸手去拿。` |
| `道具特写：线索说明凶手就在屋里。` | 推理结论越过物件可见信息 | `道具特写：门闩内侧留着半枚新鲜指印，木屑还卡在锁孔边缘。` + `心理反应：他盯住锁孔，没有立刻开口。` |
| `心理反应：林寂意识到苏红叶已经掌控全场。` | 抽象认知和权力结论，观众离开文字无法感知 | `心理反应：林寂的目光从教鞭移到第一排同学僵直的肩背，手指慢慢离开桌沿，没有再碰那张纸。` |
| `心理反应：她心里像被撕开一道口子。` | 文学比喻，不是演员可执行反应 | `心理反应：她吸了一口气没有吸满，手背贴住杯壁，杯中水面轻轻晃了一下。` |
| `系统画面：规则测试开始。` | 任务说明，不是画面 | `系统画面：黑板中央浮出“课堂纪律条例”标题前的冷白光点。` |
| `环境描写：第二节课是课堂作业。` | 课程说明，不是环境 | 删除，或写 `动作画面：苏红叶把白纸发到每张课桌。` |
| `系统画面：没有字数限制。` | 规则文本被误放进系统画面 | `旁白（系统提示）：“没有字数限制。”` + `旁白画面：白纸题目下方留着大片空白。` |
| `对白画面：陈大雷语气压低，却有一种正面摊牌的爽快。` | 感受性判断 | `对白画面：陈大雷身体前倾，手掌摊开压在桌面，眼睛直视林寂。` |
| `角色动作：林寂试图压住怒火，打算装作没听见。` | 主观意图和情绪结论混入动作 | `角色动作：林寂把纸角抚平两次，第二次指节压得发白；他抬眼时下颌绷住，没有接话。` |
| `对白画面：她是在试探林寂。` | 主角视角判断被写成客观概括 | `内心独白（林寂）：“她在等我先补后半句。”` + `内心独白画面：她的问题停在半截，视线落到林寂握紧的手指上。` |
| `心理反应：林寂感到恶心。` | 直接书写主观感受 | `心理反应：林寂喉头轻轻一滚，嘴角压住，手背抵到唇边又放下。` |
| `环境描写：这种压抑的课每天都重复，所有人早已熟悉。` | 无法实拍的往日常态总结 | 删除，或 `群像画面：前排学生没有抬头，笔尖停在同一行空格上。` |
| `表演提示：他似乎拿到一张别人没有的底牌...` | 小说式抽象 | `表演提示：林寂不抬头，指腹压住课本边角，眼神保持余光警戒。` |
| `表演提示：林寂不信任陈大雷。` | 心理标签，演员不可直接执行 | `表演提示：林寂接过纸条前停半拍，先看陈大雷的手，再看纸条背面。` |
| `环境描写：苏红叶掌控全场。` | 权力关系结论 | `场面调度：苏红叶站在讲台内侧，教鞭横在讲桌边缘；第一排学生坐得笔直，没人越过桌线。` |
| `对白画面：她在试探他。` | 潜台词结论 | `对白画面：她把问题说到一半停住，视线落在他握紧的手指上，等他先补完后半句。` |
| `对白画面：她没有看他，而是看向远方；他顺着她的视线望去。` | 视线转场模板化，两个角色重复同一动作 | `对白画面：她把臂弯里的布料收紧，指尖停在折口；身后的欢呼声一点点散开。` + `角色动作：他喉结动了一下，把涌到嘴边的话咽回去，只低头抹掉葫芦口的酒渍。` |
| `环境描写：令狐冲咬了一口番薯，甜得烫嘴，他凑到她耳边小声说：任盈盈别过头去，没说话，只是指尖悄悄攥住了令狐冲的手腕。` | 人物动作、对白引出和对手反应混入环境 | `角色动作：令狐冲咬下一口番薯，被热气烫得偏过头，仍往任盈盈身边挪近半寸。` + `对白（令狐冲，声音压低）：“<上游原句>”` + `对白画面：任盈盈别过脸，没有接话；她的指尖在袖下收紧，扣住令狐冲的手腕。` |
| `对白画面：说话者的视线、手部动作、身体距离和对手反应就近承托对白；引号内不加入动作。` | 规则说明泄露到正文 | 删除，并改写为具体可见承托，例如 `对白画面：说话者的指腹压住文书边缘，桌案两侧的杂声一层层低下去。` |
| `动作画面：雪千寻攻击忍者。` | 动作结果过于概括，丢失速度和力度 | `动作画面：雪千寻腕口一抖，鞭影骤然掠过甲板；忍者还未转身便扑倒在月光里。` |
| `角色动作：他把关键物件收进袖中。` | 丢失上游“缓缓”的决定感 | `角色动作：他沉默良久，缓缓将关键物件折入袖中，指尖在袖口停了半拍。` |

## Bad Patterns

- `动作画面：林寂意识到自己拿到了底牌。`
- `角色动作：林寂试图压住怒火，想要装作没听见。`
- `场面调度：两人的权力关系发生变化。`
- 场景末尾集中列出 `表演提示：角色想要...外显为...` 与 `场面调度：人物位于...所有视线...`
- `系统画面：规则测试开始。`
- `环境描写：永夜中学的第一节课，在压抑的气氛中继续。`
- `环境描写：本场按上游原文顺序承接地点、人物、证据、伤势、环境要素、道具和群体反应，不新增事件结果。`
- `环境描写：令狐冲咬了一口番薯，甜得烫嘴，他凑到她耳边小声说：任盈盈别过头去，没说话，只是指尖悄悄攥住了他的手腕。`
- `心理反应：因为没有人敢在课堂时间缺席。`
- `心理反应：林寂意识到自己拿到了底牌。`
- `心理反应：她的内心像坠入无边黑暗。`
- `心理反应：林寂感到恶心。`
- `对白画面：她是在试探林寂。`
- `环境描写：这种场面他们早已见过无数次。`
- `道具特写：这把刀来自他多年前的旧事。`
- `对白（苏红叶，微笑保持不动）：“欢迎来到这里。”她微笑着看向全班。`
- `对白画面：苏红叶说欢迎来到永夜私立中学。`
- `对白画面：说话者的视线、手部动作、身体距离和对手反应就近承托对白；引号内不加入动作。`
- `音效：大家害怕得不敢说话。`
- `音效（铃声）：“八点四十分铃声。”`
- `场景2：苏红叶入场震慑全班`

## Good Patterns

```md
音效（高跟鞋）：“嗒、嗒、嗒。”
音效画面：教室前门外，红色细高跟鞋的影子压过门缝；三十多名学生同时停住动作，看向门口。

场面调度：苏红叶站在讲台内侧，教鞭横在讲桌边缘；林寂坐在第三排靠窗，前排同学的肩膀把他半遮住。

对白（苏红叶，微笑保持不动）：“各位同学，早上好。”
对白画面：苏红叶站在讲台前，教鞭末端的红珠停在桌沿，嘴角保持完美到不自然的微笑。

内心独白（林寂）：“她发现了。”
内心独白画面：林寂低着头，后背的校服布料被冷汗洇深；苏红叶的视线从他身上移开。

心理反应：林寂盯住黑板上最后一行规则，喉结轻轻动了一下，压在课本边缘的手指慢慢收紧。
```
