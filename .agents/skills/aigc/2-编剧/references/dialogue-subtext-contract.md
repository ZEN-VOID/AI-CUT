# Dialogue Subtext Contract

## Purpose

在对白标签层增加潜台词行为标注：这句台词在执行什么戏剧动作。对白不只是"谁说什么"，更是"这句话在做什么"——它在试探、隐瞒、施压还是引诱？本合同为编剧阶段提供统一的戏剧动作标注框架，让下游导演、表演和摄影知道每句台词的真正功能，而不只是它的字面含义和语气状态。

## Ownership

- **产出者**：2-编剧 在剧本对白形成阶段为关键对白标注戏剧动作
- **消费者**：
  - 3-导演 消费戏剧动作决定对白场景的压力设计和镜头引导
  - 4-表演 消费戏剧动作决定"怎么演这句话"——语气只是表面，动作才是内核
  - 5-摄影 消费戏剧动作决定对白场景中注意力的分配方向
- 本合同不改变对白本身，只在对白标签层增加行为标注层。

---

## 九类戏剧动作

每句关键对白都至少属于以下一类戏剧动作。一句对白可以同时执行多类动作，但必须有一个主动作。

### 1. probing — 试探

| 维度 | 内容 |
| --- | --- |
| 定义 | 用问题、沉默或看似无关的话题去探测对方的底线、态度或信息 |
| 表面形态 | 可以是关心、闲聊、好奇、质疑 |
| 实际目的 | 获取对方不想主动给出的信息或态度 |
| 典型对白 | "最近还好吗？"（实际想知道：你知道那件事了吗） |
| 表演要求 | 语气轻松但注意力集中在对方反应上——关键不在自己说什么，在对方怎么接 |
| 画面承托 | 试探方的眼神跟随对方反应、手指无意识地轻敲、身体微微前倾 |

### 2. deflecting — 转移回避

| 维度 | 内容 |
| --- | --- |
| 定义 | 用另一句话、另一个话题或另一个对象把当前的压力点从自己身上移开 |
| 表面形态 | 可以是幽默、反问、转移话题、指向第三方 |
| 实际目的 | 不想面对当前问题，但不想暴露自己在回避 |
| 典型对白 | "你说这个干嘛，先吃饭吧。"（实际动作：我不想回答这个问题） |
| 表演要求 | 表面自然但有微小的节奏变化——语速突然快半拍、笑容出现的时机微妙不对 |
| 画面承托 | 回避方的视线移开、手开始做无关的事、身体重心后移或转身取物 |

### 3. concealing — 隐瞒伪装

| 维度 | 内容 |
| --- | --- |
| 定义 | 刻意让对方相信一个不是事实的版本，或隐藏自己的真实状态 |
| 表面形态 | 可以是平静、无所谓、开心、愤怒——任何与真实相反的状态 |
| 实际目的 | 保护自己、保护对方、维持局面或等待时机 |
| 典型对白 | "我没事。"（实际状态：崩溃在即） |
| 表演要求 | 演两层——表面的平静和底下的裂缝。裂缝通过微表情、呼吸、声音控制泄露 |
| 画面承托 | 伪装方的手在桌面下攥紧、笑容维持但眼神没有到达、呼吸频率与语调不匹配 |

### 4. pressuring — 施压威胁

| 维度 | 内容 |
| --- | --- |
| 定义 | 用语言、沉默、存在感或信息的暗示向对方施加压力，迫使对方做出反应或让步 |
| 表面形态 | 可以是直白威胁、暗示、反问、沉默不语、刻意放慢节奏 |
| 实际目的 | 迫使对方暴露、让步、做出选择或暴露弱点 |
| 典型对白 | "你确定要这样？"（表面是确认，实际是施压） |
| 表演要求 | 控制感——不是在发怒，是在控制节奏。关键在于"让对方等" |
| 画面承托 | 施压方占据空间优势、目光不移、动作缓慢从容；被压方缩小、后退、不敢对视 |

### 5. seducing — 引诱拉拢

| 维度 | 内容 |
| --- | --- |
| 定义 | 用魅力、利益、认同感或情感联结引导对方走向自己想要的方向 |
| 表面形态 | 可以是赞美、共情、分享秘密、展示脆弱、给予好处 |
| 实际目的 | 让对方放下防备、建立信任、产生依赖或做出利于自己的选择 |
| 典型对白 | "你和他们不一样。"（表面是认同，实际是拉拢） |
| 表演要求 | 真诚感——引诱越真诚越有效。不能让对方感到在被操纵，要让对方感到被看见 |
| 画面承托 | 引诱方身体靠近、目光柔和但不移开、声音降低变柔；对方身体放松、防备卸下 |

### 6. confessing — 坦白泄露

| 维度 | 内容 |
| --- | --- |
| 定义 | 有意或无意地暴露自己的真实想法、秘密、脆弱或弱点 |
| 表面形态 | 可以是主动坦白、说漏嘴、情绪崩溃时的口不择言、醉后的真话 |
| 实际目的 | 主动坦白=想获得原谅/理解/释放；被动泄露=防线被击穿 |
| 典型对白 | "其实那天……我没有走。"（主动坦白）/ "你以为我想这样吗！"（被动泄露） |
| 表演要求 | 区分主动和被动——主动坦白是"决定开口"的过程，被动泄露是"没忍住"的瞬间 |
| 画面承托 | 坦白方的眼神从回避到直视的转变、声音从控制到失控的过程；对方的震惊/沉默/微表情变化 |

### 7. testing — 考验设陷阱

| 维度 | 内容 |
| --- | --- |
| 定义 | 已经知道答案或已有预判，用对白测试对方是否诚实、忠诚或会犯什么错 |
| 表面形态 | 可以是提问、给选择、故意留出说谎的空间、给对方一个"安全"的错误选项 |
| 实际目的 | 验证怀疑、确认判断、收集证据或给予最后机会 |
| 典型对白 | "你昨天几点走的？"（实际已知道答案，测试对方是否说谎） |
| 表演要求 | 考验方表面平静但注意力高度集中——关键在"等待回答的那几秒"。被考验方不知道自己在被考验 |
| 画面承托 | 考验方给出问题后不追加解释、不催促，只是等；手指微微转动杯子、视线偶尔移开又回来 |

### 8. bonding — 亲近共鸣

| 维度 | 内容 |
| --- | --- |
| 定义 | 通过共享经历、共同弱点、默契暗示或情感暴露来建立或加深情感连接 |
| 表面形态 | 可以是自嘲、提及共同记忆、在关键时刻站到对方一边、无言的陪伴 |
| 实际目的 | 建立信任、修复关系、确认同盟、给予归属感 |
| 典型对白 | "我懂。"（表面只有两个字，实际动作是"我站在你这边"） |
| 表演要求 | 不需要大段对白。bonding 的力量往往在少说——一个眼神、一个点头、一个不说话的陪伴 |
| 画面承托 | 双方距离缩短、身体朝向一致、共享同一个画面空间；沉默不再尴尬而变成舒适 |

### 9. asserting — 宣示边界

| 维度 | 内容 |
| --- | --- |
| 定义 | 明确告诉对方"到这里为止"，划定自己的底线、立场或不可侵犯的区域 |
| 表面形态 | 可以是直接声明、沉默拒绝、动作阻隔、改变语调、起身离开 |
| 实际目的 | 终止对方的试探/施压/越界、保护自我、重新定义关系规则 |
| 典型对白 | "别再问了。"（表面是拒绝，实际是宣示"这是我的边界"） |
| 表演要求 | 力量感不在音量，在确定性——不是在生气，是在声明一个不会改变的事实 |
| 画面承托 | 宣示方身体从松弛变紧实、目光稳定不闪躲、空间上占据或守住自己的位置 |

## 对白标签增强方案

### 现有格式

```
对白（角色，语气/状态）："台词内容"
```

### 增强格式

在现有对白标签下方可选增加潜台词行为标注：

```
对白（角色，语气/状态）："台词内容"
表演提示：戏剧动作——具体语境描述
```

### 标注示例

```markdown
对白（苏红叶，微笑保持不动）："林寂同学，最近还好吗？"
表演提示：试探——她已经知道林寂在调查第三层规则，这句话是看他会不会露出破绽。

对白（林寂，声音平静）："挺好的，谢谢老师关心。"
表演提示：隐瞒伪装——他正在调查的事绝不能让苏红叶知道，平静是装出来的。

对白（陈大雷，语气随意）："你说这个干嘛，先吃饭吧。"
表演提示：转移回避——他不想让林寂继续追问昨晚的事，但不想暴露自己在回避。

对白（苏红叶，声音压低）："你确定要这样？"
表演提示：施压威胁——不是真的在问，是在告诉他"我看到了你做的事"。
```

### 标注规则

- 不是每句对白都需要标注。过场对白、日常寒暄、纯信息传递不需要戏剧动作。
- 关键场景（对质、试探、揭示、冲突）的每一句对白都应该有标注。
- 标注描述应具体到场景语境，不能只写"试探"两个字——必须写清楚"试探什么"和"在什么压力下"。
- 不用标注的情感表达类对白（"我爱你"、"对不起"）如果在关键场景中，仍然需要标注——因为纯情感对白也可能在执行隐瞒、施压或考验等动作。

## 潜台词行为与画面承托的对应关系

| 戏剧动作 | 对白画面承托重点 | 心理反应承托重点 | 道具/空间承托 |
| --- | --- | --- | --- |
| `试探` | 试探方的眼神追着对方反应 | 对方是否上钩的微表情变化 | 试探方的手指无意识动作 |
| `转移回避` | 回避方的视线和身体转向 | 被回避方的失落或困惑微表情 | 回避方抓取无关道具填补空白 |
| `隐瞒伪装` | 伪装方的表面状态与微裂缝 | 裂缝泄露的瞬间反应（声音、呼吸） | 手在桌面下攥紧、杯子被无意识转动 |
| `施压威胁` | 施压方的空间控制和目光 | 被压方的缩小、后退、吞咽 | 施压方缓慢靠近、占据关键位置 |
| `引诱拉拢` | 引诱方的靠近和目光柔和 | 对方防备卸下的身体变化 | 物品被分享、空间距离缩短 |
| `坦白泄露` | 坦白方眼神从回避到直视 | 坦白瞬间的声线变化、呼吸断裂 | 手从桌上收回、身体微微前倾 |
| `考验设陷阱` | 考验方等待回答的静止 | 被考验方回答时的微犹豫或过度自然 | 杯子停在唇边、纸张被翻到关键页 |
| `亲近共鸣` | 双方朝向一致、距离缩短 | 沉默变得舒适而非尴尬 | 共享食物、并排而坐、共同看向同一方向 |
| `宣示边界` | 宣示方身体紧实、目光稳定 | 对方被挡回后的退缩或愤怒 | 手平放在桌上阻止对方靠近、起身离开 |

## 多层次潜台词

每一句有戏剧动作的对白都有至少三个层次：

| 层次 | 说明 | 示例（对白："你最近挺忙的吧？"） |
| --- | --- | --- |
| 表层 — 说的 | 台词字面含义 | 表面是关心对方近况 |
| 行为层 — 做的 | 这句话在执行什么戏剧动作 | 试探——确认对方是否在秘密行动 |
| 内心层 — 想的 | 说话者真正的情感状态 | 不安和恐惧——害怕对方已经知道了真相 |

### 三层标注示例

```markdown
对白（苏红叶，语气平和）："林寂同学，最近挺忙的吧？"

 表层：关心学生近况
 行为层：试探——确认林寂是否在暗中调查第三层规则
 内心层：戒备——她已经收到消息有人在查，需要确认是不是林寂

表演提示：试探——表面是教师对学生的关心，实际在确认林寂是否对她构成威胁。
```

### 三层使用规则

- 表层和行为层是必填的——导演和演员都需要知道"台词表面说什么"和"实际在做什么"。
- 内心层是选填的——只在关键对白中使用。内心层写太多会变成心理解释，限制表演空间。
- 内心层不得作为对白画面或角色动作的内容——它是编剧内部的参照，不直接进入视觉层。

## Failure Modes

| symptom | root_cause | fix |
| --- | --- | --- |
| 对白只有语气没有戏剧动作 | 只标注了"声音压低""故作轻松"等状态，没有回答"这句话在做什么" | 每句关键对白必须回答：这句话在对对方执行什么戏剧动作 |
| 潜台词太明显，失去张力 | 戏剧动作直接写在对白画面上，变成"她在试探他" | 戏剧动作只在表演提示中标注，对白画面只写可见行为 |
| 潜台词标注变成心理解释 | 标注写成"她内心很痛苦但不想让他看到" | 标注只写"隐瞒伪装——表面说没事，实际在撑"，不写内心描写 |
| 所有对白都是同一种戏剧动作 | 没有区分不同角色在不同压力下的不同行为策略 | 检查同一场景内是否有3种以上不同戏剧动作 |
| 戏剧动作与角色状态矛盾 | 角色刚刚经历重大打击，下一句对白却是"引诱拉拢" | 戏剧动作必须与角色当前情绪状态和关系压力兼容 |
| 对白潜台词太复杂，演员无法执行 | 一句对白标注了三种以上戏剧动作 | 每句对白只标一个主戏剧动作，最多一个辅助动作 |

## Reusable Heuristics

- 对白不只是谁说什么，更是这句话在执行什么戏剧动作。语气是表面，动作是内核。"对不起"可以是坦白、可以是试探、可以是施压、也可以是引诱——决定它是什么的不是语调，是场景压力和关系状态。
- 最好的对白是"说一件事，做另一件事"。当台词的字面意思和戏剧动作一致时，对白是透明的；当两者不一致时，对白才产生潜台词。
- 好的演员不需要心理解释，需要戏剧动作。"她内心很痛苦但假装没事"是心理解释，演员不知道怎么演。"隐瞒伪装——表面平静但声音控制过度到有点不自然"是可执行的。
- 潜台词的力量在于"观众能感觉到但说不清"。如果潜台词太明显，它就不再是潜台词，而是台词。好的潜台词让观众事后回想才恍然大悟。
- 一场好的对手戏通常像一场没有裁判的棋局——两个人都在用对白执行不同的戏剧动作，而观众看到的是两个不同的游戏在同时进行。
- 九类戏剧动作不是角色性格标签，而是"此刻的压力策略"。温柔的角色在被逼到墙角时也会施压，强硬的角色在面对心爱之人时也会试探。压力决定动作，不是性格。
- 沉默也是一种戏剧动作。角色选择不说话、不回答、不接话，往往比说话更有力。当角色沉默时，编剧必须标注这个沉默在执行什么动作——是回避？是宣示边界？还是考验？
