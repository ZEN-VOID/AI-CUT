# scripts

`shot-by-shot` 的脚本只允许做机械辅助：

- 检查主报告和项目 `CONTEXT/` 解析文件是否存在。
- 检查 Markdown 是否包含必需标题。
- 统计逐镜表行数、临摹原则数量、禁止照搬项数量。
- 提示缺失字段或阶段解析缺口。
- 检查 `画面基调解析.md`、`角色风格解析.md`、`场景风格解析.md`、`道具风格解析.md`、`摄影风格解析.md`、`分镜风格解析.md` 以及可选 stage 解析的标题与落点。
- 检查 `分镜脚本.md` 是否包含参照 Numbers 示例的 19 列表头。

脚本不得替代 LLM 对视频内容、镜头边界、摄影美学、导演调度、临摹原则或阶段桥接策略的主创判断。
