# Scripts

Scripts in `$fine-tuning` are mechanical helpers only.

Allowed:

- validate source-layer package structure
- check required stage scheme markers
- check unresolved placeholders
- check report template sections
- check JSON prompt schema

Forbidden:

- generate, rewrite, polish, insert, judge, or repair creative tuning content
- produce screenplay, style protocol, subject design, storyboard, camera movement, grouping text, image prompt, or LibTV node prompt
- replace LLM-first comparison or owner boundary judgment

## Available Scripts

| script | purpose |
| --- | --- |
| `scripts/validate_fine_tuning_skill.py` | Mechanical package and marker validation for this skill. |
