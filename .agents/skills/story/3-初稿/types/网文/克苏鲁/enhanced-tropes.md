# 克苏鲁题材增强包

> 补充维度：套路拆解 / 情节模式 / 人物原型 / 反套路策略 / 质量过滤

## 一、核心套路与行规

### 1.1 克苏鲁叙事的独特法则
- **不可名状 > 明确描述**：不写怪物长什么样，写看到它的人什么反应
- **认知污染 > 物理伤害**：真正的威胁不是肉体毁灭，而是精神/认知的崩塌
- **无力感 > 力量对抗**：人类无法战胜古神，只能延迟/换取/承受
- **碎片 > 全貌**：真相永远碎片化呈现，读者拼图的乐趣大于被告诉的满足

### 1.2 恐怖来源分级
| 层级 | 来源 | 写法 |
|------|------|------|
| 表层 | 异象/怪物/变异 | 可感知但不可理解 |
| 中层 | 记忆篡改/时间错乱/因果扭曲 | 熟悉事物的陌生化 |
| 深层 | SAN值崩塌/认知框架碎裂 | "我"不再可信 |
| 终极 | 唤醒/降临/不可逆 | 人类尺度的彻底失效 |

### 1.3 常见套路陷阱
- 把克苏鲁写成"触手怪大战"（丢失了不可名状的核心）
- SAN值变成血条（精神崩塌不是血量变化）
- 怪物图鉴式展开（每个怪物都详细描述）
- 主角总能保持清醒（失去理智才是克苏鲁）

## 二、情节模式与节奏规则

### 2.1 调查者叙事弧
```
正常 → 发现异常 → 调查深入 → 第一次超出理解 → 强制复盘 → 
更深的恐怖 → 认知框架第一道裂痕 → 绝望的选择 → 
接受/拒绝真相 → 被改变（不可逆）
```

### 2.2 节奏铁律
- 恐怖不能一直高潮——有张有弛，平静中的不安比尖叫更持久
- 信息释放渐进化——每一步揭示都带来更大的疑问
- 人物精神状态是节奏的晴雨表——注意力的涣散、记忆的缺失、时间的错乱

## 三、反套路策略

### 3.1 反"主角免疫"
- 主角会san值归零——不是变疯然后就好，而是永久的改变
- 接触过不可名状之物后，主角再也不能回到正常社会
- 胜利的定义改变——不是击败，而是"没有被完全吞噬"

### 3.2 反"解释一切"
- 留下不可解释的东西。读者不需要理解一切，需要的是"感受到无法理解"
- 空的房间比装满怪物的房间更恐怖
- 暗示 > 展示 > 解释

### 3.3 反"理性优先"
- 理性是最先崩塌的——当因果不再成立，逻辑本身成为恐怖来源
- 记忆不可信——前一章的"事实"在后一章被发现是假的/扭曲的
- 时间的非线性——不是倒叙/插叙，而是时间本身的碎裂

## 四、质量过滤规则

### 4.1 不可名状检查
- [ ] 至少40%的恐怖场景不直接描写威胁的具体样貌
- [ ] 精神/认知崩塌有真实的心理过程（不是"他疯了"）
- [ ] 世界观的不完整性被保持（不是所有规则都讲清楚）

### 4.2 失败码
| 失败码 | 症状 | 返工目标 |
|--------|------|----------|
| `FAIL-CTC-NAMED` | 不可名状之物被详细描写 | 移除直接外貌描写，保留反应描写 |
| `FAIL-CTC-SAN` | SAN值写成血条 | 重写为认知崩塌的心理过程 |
| `FAIL-CTC-EXPLAIN` | 过度解释世界观 | 增加留白和不可解释要素 |
