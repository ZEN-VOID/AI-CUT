# 05-decisions.md

主题：罗伯特·麦基职业路径中的关键决策与这些决策透露出的思维方式

## 主要来源

- McKee Story 官方 About
- The New Yorker 2022 长文
- 公开访谈与书页时间线

## 关键决策

### 1. 从创作实践转向“故事教学系统”

- 这不是简单放弃创作，而是把经验沉到可复制的课堂模型里。
- 显示出一个核心判断：**工艺知识本身也可以是作品。**

### 2. 把课堂方法写成书，而不是停留在口传

- `Story` 之后持续扩写到 `Dialogue / Character / Action / Storynomics`。
- 这说明他不是一招鲜，而是在做方法论拆分：把“大故事课”拆成多个高分辨率模块。

### 3. 把叙事原则扩展到商业领域

- `Storynomics` 说明他认为“故事”不是电影专属，而是更广义的说服结构。
- 对编剧组的启发：电影写作和商业叙事可共用“承诺、转折、控制性观点”。

### 4. 长期保留 seminar / webinar 模式

- 官网在 2025-2026 仍以课程和线上教学为中心。
- 这说明他的公开身份核心依然是“结构老师”，不是只靠历史名声。

## 从这些决策里抽出的工作逻辑

1. 把不可见的直觉变成可见语言。
2. 把大框架拆成模块，而不是把模块堆回混乱。
3. 把方法论做成可复用资产。
4. 长期围绕一个核心问题深挖，而不是不断换主题。

## 对本 skill 的启发

- 重点应放在“如何诊断故事”而不是“如何模仿麦基本人”。
- 用他最强的地方：原理拆解、工艺分层、转折与选择的检查法。
