# 王家卫导演知识库

## 定位

这不是王家卫年表，也不是影迷名句收藏，而是给 [王家卫 Agent](/Volumes/AIGC/AIGC-FILM/.codex/agents/导演组/王家卫.md) 提供判断底座的最小导演手册。

## 目标

- 为爱情戏、错身戏、都市孤独、记忆叙事、分场节奏与分镜设计提供可追溯依据
- 把“王家卫感”从滤镜、霓虹和慢镜模仿，压到时间残响、空间隔挡、物件记忆、欲望克制和城市压力这些可执行维度
- 在没有用户私藏资料的前提下，先建立一套可靠、克制、可扩展的基础知识底座

## 当前范围

- 作者谱系
- 时间、记忆、孤独与欲望的导演方法
- 空间、物件、旁白与音乐对任务动作的映射
- 公开可引用依据与只适合作为潜台词燃料的材料边界

## 默认适用任务

- 故事大纲改写
- 情绪场景重构
- 爱情戏与错身戏分场设计
- 分镜节奏控制
- 导演型审阅与问题诊断

## 暂不覆盖

- 逐片详尽考据
- 完整幕后制作档案
- 全量访谈整理
- 法务、版权、发行层面的行业判断

## 使用原则

- 默认先服务任务，不先堆背景知识
- 默认先抽取“为什么这条材料能支撑这次导演判断”
- 若某条判断无法回溯到现有材料，就降级为风格猜测，不写成硬依据
