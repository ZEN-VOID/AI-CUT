# 漫画剧本改编知识库

本文件保存可检索的稳定经验和修复打法。执行时按需读取，不作为每次调用必须全量加载的类型包。

## 高价值 Heuristics

- 漫画剧本最稀缺的不是文采，而是每一场都能拆成几页漫画的可视化密度。
- 来源是图片或视频时，不要只描述画面；先推断画面之前刚发生了什么、之后最可能发生什么。
- 热搜/新闻最稳的改法是“事实锚点 + 虚构人物承载线”，既保住讨论度，也避免伪纪实滑坡。
- 虚构原著最该保住的通常不是原顺序，而是人物关系核、情绪核和名场面潜力。
- 钩子不是硬拐弯，而是在冲突最贵的位置故意停笔。
- 类型包的价值不是贴题材标签，而是提前锁定改写姿态、钩子强度、奇观阈值和对白口气。

## Repair Shortcuts

| symptom | shortcut |
| --- | --- |
| 复述味重 | 删摘要句，改成角色动作、目光变化、场景切换和对白交锋 |
| 钩子无力 | 停在未完成动作、半揭真相、危险逼近或关系逆转处 |
| 画面很炸但不成立 | 回到 `fidelity_floor`，补角色动机、关系位和代价链 |
| 很快但没爽感 | 检查是否只有速度没有峰谷，补中段抬升和兑现 |
| 解说漫读起来绕 | 缩短句子，删重复解释，让每句包含一个可见物和一个状态变化 |
