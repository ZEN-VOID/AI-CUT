# Cinematic Technique Library

本文件是可动态扩展的摄影技法库。执行时按当前画面句子的戏剧任务选择，不要求每个分镜机械覆盖所有栏目，但每个 `分镜N` 默认应显式包含“景别/景深、镜头视角、镜头类型、运镜速度”四类摄影执行参数；只有当参数已由上下文完全自明时才可省略。具体表述必须遵守 `dynamic-lens-language-contract.md`，写成动态变化而不是静态标签列表。

## 摄影执行参数

### 景别

| shot_size | use when | execution cue |
| --- | --- | --- |
| 大远景 / 远景 | 建立封闭空间、群体处境、人物被制度吞没 | 人物小于环境，课桌、黑板、门窗成为压迫主体 |
| 全景 | 交代人物站位、动作路径、权力距离 | 保留脚下空间、讲台高度、座位行列和门框关系 |
| 中全景 | 人物动作与环境压力同时重要 | 看见上半身动作和周围桌椅/门/黑板的束缚 |
| 中景 | 对话、课堂秩序、人物被点名 | 保留手、肩、课桌边缘和脸部反应 |
| 近景 | 压迫、隐瞒、警觉、呼吸变化 | 让脸和肩占主导，背景仍保留危险色块 |
| 特写 | 眼神、嘴唇、手指、道具、规则文字 | 信息必须可读，避免只为了漂亮而特写 |
| 大特写 | 瞳孔反光、指腹泛白、字迹异常、血色液体 | 只用于关键揭示或情绪峰值，保持稀缺 |

### 景深

| depth | use when | execution cue |
| --- | --- | --- |
| 深景深 | 空间关系、群像压迫、前中后景同时有威胁 | 黑板、学生、门口或窗帘同时可读 |
| 中景深 | 人物与近旁道具/同桌关系并重 | 主体清楚，背景仍有可辨危险轮廓 |
| 浅景深 | 主体孤立、心理压迫、信息暂时遮蔽 | 背景化成冷白/黑/红色块，焦点锁住眼或手 |
| 焦点拉移 | 同镜头完成信息转交 | 从人物眼神拉到规则字、道具、门缝或反应对象 |
| 失焦再合焦 | 眩晕、醒来、规则显影、感知异常 | 失焦不能太久，合焦点必须落在新信息上 |

### 镜头视角

| angle | use when | execution cue |
| --- | --- | --- |
| 平视 | 保持观察冷静、人物间暂时平等 | 适合林寂克制分析和低声谈判 |
| 低角度 | 老师、规则、黑板、道具压迫角色 | 让讲台、教鞭、黑板字向画面下方压来 |
| 高角度 | 学生无助、群体被编排 | 课桌像棋盘，人物像被固定在格子里 |
| 俯拍 | 制度、规则、命运操控感 | 用于全班、座位、规则布局，不滥用到普通对话 |
| 仰拍 | 入场、宣判、危险权力增强 | 苏红叶、黑板公告、门口轮廓可用 |
| 过肩视角 | 角色被凝视或观察他人 | 前景肩背形成遮挡，后景目标带危险距离 |
| 主观视角 | 真理之眼、眩晕、发现隐藏文字 | 需配合光晕、焦点、眨眼或视野边缘变化 |
| 画外窥视视角 | 监控感、被观察、走廊未知威胁 | 门缝、窗帘边、课桌缝隙作为遮挡框 |

### 镜头类型

| lens_type | use when | execution cue |
| --- | --- | --- |
| 广角镜头 | 空间畸变、近距离压迫、教室变形 | 近前景夸张，后景被拉远，适合恐惧空间 |
| 标准镜头 | 保持现实感、对话、观察 | 不抢戏，让表演和规则压迫自己发生 |
| 长焦镜头 | 压缩空间、制造无路可退 | 课桌距离被压扁，讲台像贴到学生脸上 |
| 微距镜头 | 字迹、红珠、苹果咬痕、指腹汗水 | 材质、液体、纸纹和微小异常必须清楚 |
| 移轴 / 倾斜焦平面 | 规则世界错位、空间不可信 | 少量用于异常感，不作为常规漂亮效果 |
| 手持镜头 | 群体慌乱、呼吸不稳、危险靠近 | 微晃而非乱晃，信息必须可读 |
| 稳定器镜头 | 冷酷跟随、秩序化运动 | 适合苏红叶移动、规则宣判、走廊逼近 |
| 固定机位 | 权力碾压、课堂制度、不可反抗 | 镜头不动，让人物在框内被规则处理 |
| 组合运镜 | 信息、情绪和空间需要连续转交 | 两到三个运动动作串联，例如“慢推接焦点拉移”“横移接急停” |

### 运镜速度

| speed | use when | execution cue |
| --- | --- | --- |
| 静止 | 宣判、僵持、被迫服从 | 镜头不救角色，压力由表演和声音累积 |
| 极慢 | 危险无声逼近、意识慢慢发现 | 推拉摇移速度低于观众警觉阈值 |
| 慢速 | 观察、悬疑、信息揭示 | 让观众有时间读文字、看反应、感受空间 |
| 中速 | 普通动作承接、视线转交 | 不炫耀运动，清楚把注意力送到目标 |
| 快速 | 突发、惊吓、动作冲击 | 只用于强节拍，之后给反应镜头落地 |
| 急停 | 危险确认、人物被点名、规则锁定 | 运动突然停止在眼睛、手、字或门口 |
| 变速 | 感知异常、规则显影、心理断裂 | 加速/减速必须对应可见信息变化 |

## Dynamic Parameter Composition

执行时不要把参数写成清单，而要组合成运动句：

| static parameter list | dynamic expression |
| --- | --- |
| `近景、浅景深、长焦、慢推` | 从中景的课桌边缘慢慢压成近景，长焦把后排空间挤扁，景深逐步收窄到林寂的眼睛 |
| `特写、微距、焦点拉移` | 先让苹果红色表皮占满画面，再把焦点从光滑反光拉到咬痕里渗出的红液 |
| `俯拍、深景深、慢速横移` | 从棋盘式课桌俯拍开始慢速横移，让每一排学生像被规则逐格扫描 |

## 经典电影构图

| technique | use when | execution cue |
| --- | --- | --- |
| 对称压迫构图 | 校园秩序、讲台权力、黑板规则压住学生 | 中轴线、桌椅行列、门框或黑板作为几何牢笼 |
| 前景遮挡 | 角色被规则、课桌、门框、他人身体压住 | 用黑色窗帘、课桌边、门框切割画面 |
| 低机位权力镜头 | 老师、规则、系统公告占上风 | 让人物或文字向下压迫观众视线 |
| 高机位俯视 | 群体无助、棋盘式命运 | 课桌排列像格子，人物缩在固定座位中 |
| 负空间孤立 | 某人被点名、被凝视、被规则锁定 | 让空白黑板、封死窗帘或冷白墙面吞掉人物 |
| 深焦层次 | 同时呈现前景反应、中景动作、后景威胁 | 前中后三层都放入可读信息，避免扁平舞台感 |

## 高超运镜手法

| technique | use when | execution cue |
| --- | --- | --- |
| 隐形推轨 | 危险或信息慢慢逼近 | 推进速度低于观众意识阈值，结尾落在眼睛、手、文字 |
| 视线牵引摇镜 | 注意力从角色视线转向目标物 | 先给眼神，再沿视线摇到门、黑板、道具 |
| 焦点拉移 | 同一镜头内完成主体切换 | 前景人物虚化，后景规则或危险变清晰 |
| 轴线压缩 | 角色退无可退、空间变窄 | 长焦压扁课桌距离，让教室像被折叠 |
| 甩镜/急摇 | 惊吓、骤停、突发入场 | 只用于强节拍，不覆盖需要冷静压迫的段落 |
| 手持微晃 | 人群恐惧、失控边缘 | 控制晃动幅度，保留信息可读性 |
| 静止长镜 | 权力碾压、规则宣判 | 镜头不动，让演员和声音造成压力 |

## 高能转场设计

| transition | use when | execution cue |
| --- | --- | --- |
| 声音先行转场 | 下一画面的声音先于画面入侵 | 高跟鞋、铃声、教鞭、日光灯嗡鸣先进入当前画面 |
| 形态匹配剪辑 | 两个画面有相似轮廓、色块或运动方向 | 白光到日光灯、红珠到红苹果、黑板波纹到瞳孔反光 |
| 动作匹配剪辑 | 一个动作的方向或速度承接下一个画面 | 推门、翻页、抬眼、教鞭点地成为切点 |
| 光变转场 | 过曝、熄灯、显影、冷光闪动 | 光线吞没或露出信息，转入新画面 |
| 文字显影转场 | 规则、系统字、板书成为画面接口 | 字体浮现、波纹扩散、金色小字擦过画面 |
| 反应切转场 | 从事件切到被事件击中的脸或手 | 恐惧不直接展示完，切到指尖、瞳孔、肩膀 |

## 光影美学

| lighting | use when | execution cue |
| --- | --- | --- |
| 冷白日光灯 | 校园制度感、非人秩序、医疗式冷酷 | 顶光压脸，眼窝发暗，皮肤失温 |
| 黑窗帘吸光 | 封闭、无外部世界、逃生路径被抹除 | 让窗帘成为吞光背景和负空间 |
| 红色危险点光 | 苏红叶、红珠、红苹果、血色液体 | 红色只在危险或诱饵上出现，保持稀缺 |
| 金色异质微光 | 真理之眼、隐藏规则、主角特殊信息 | 金光贴近眼底或文字边缘，不污染全局色调 |
| 半明半暗轮廓 | 人物身份不稳定、威胁不确定 | 日光灯边缘切脸，一半服从秩序，一半藏入阴影 |

## 色彩美学

| palette | use when | execution cue |
| --- | --- | --- |
| 冷白 + 深蓝 | 学生校服、日光灯、制度空间 | 作为常态底色，压低生命感 |
| 黑 + 白 + 红 | 苏红叶和规则惩罚的视觉母题 | 黑白建立秩序，红色成为警告和欲望 |
| 灰绿旧纸色 | 课本、规则、陈旧校园物件 | 让道具带腐败感，不显现代明亮 |
| 金色小字 | 隐藏信息、主角能力 | 与冷白系统字区分，强调“第二层真相” |

## Extension Rule

- 新增技法必须写明 `use when` 和 `execution cue`，避免只添加审美名词。
- 新增摄影参数必须能落到 `分镜N` 的可执行表述中，例如“中近景、浅景深、过肩视角、长焦压缩、极慢推轨”。
- 若某技法改变输出验收或执行流程，必须同步回改 `SKILL.md` 的 `Reference Loading Guide` 或 `Execution Contract`。
