# Directing Heuristics

## Scene Staging And Power

- 场面调度优先写权力关系：谁站着、谁坐着、谁占据讲台/门口/黑板，谁被群体视线包围，关键道具停在谁手边。
- 沉默不是空白；用纸张声消失、呼吸停住、手指松开、对手不接话、群像迟半拍反应来承托。
- `表演提示` 和 `场面调度` 的规划句不要直接贴到场景末尾；把"想控制风险"拆成低声、停敲、压住关键文书、周围杂声低下去等细节。
- 把高段位表演写成理论词会让演员无事可做；它必须表现为可执行任务和外显行为。
- 把权力关系写成"掌控全场""关系改变"会失去画面证据；应转为空间站位、视线压力和道具归属。
- 诡异校园类场景的可拍核心通常在空间压迫、集体静止、规则文字、声音节奏和局部特写之间，不必靠新增剧情制造恐惧。
- 末尾总结块会让编导稿像审稿批注而不是可拍剧本；终稿应让表演和调度在对应动作时发生。

## 转场技法

### 猪笼城寨式连续转场

一种以**主角行进/行动为主线**，通过**感知触发物**串联多个场景和人物的电影感快节奏转场技法。适用于需要在有限篇幅内建立群像和世界观的开场或群戏场景（村落、市集、军营、客栈、战场等）。

**核心公式：**

```text
主角动作/对话 → 感知触发物（声音/视觉/气味/触觉）→ 焦点切换到新人物/新细节 → 互动 → 下一个触发物
```

**操作规则：**

| 规则 | 说明 |
| --- | --- |
| 感知触发物必须具体 | 每次转场必须有一个可被摄影机/麦克风捕捉的触发物：水花、杖声、酒气、布帘、喊声、刀响、靴底声、弩弦声、竹叶落等。禁止用"忽然注意到""目光所及"等抽象过渡。 |
| 镜头跟着注意力走 | 读者/观众的注意力 = 主角的注意力。主角听到声音→转头→看见新人物，读者自然跟着转。不要用全知视角硬切。 |
| 人物通过动作和对话出场 | 不用旁白介绍身份。新人物出场时必须有一个可拍的动作（扛鱼篓、拨灯芯、掀布帘、拔刀）或一句可听的对白，身份和性格在互动中自然带出。 |
| 细节在互动中带出 | 伤痕、习惯、规矩、权力关系等信息通过动作碰撞和对话摩擦自然暴露，不做独立段落解说。 |
| 转场密度随节奏变化 | 开场建立群像时密度高（3-5个触发物/页），战斗/紧张时密度极高（每个动作都可能是触发物），喘息/对话时密度降低。 |
| 主线不断 | 主角的行进/行动是贯穿线。即使焦点短暂切到其他人，主角必须始终在读者感知范围内（脚步声、呼吸、影子、回话）。 |

**触发物分类：**

| 类型 | 示例 | 适用场景 |
| --- | --- | --- |
| 声音 | 杖声、靴声、弩弦声、刀出鞘声、喊声、鸦叫、竹哨 | 隐蔽接近、危险逼近、新人物入场 |
| 视觉 | 水花、影子、灯火变色、刀光、蓑衣角、泥印 | 空间转移、威胁显影、线索发现 |
| 气味 | 酒气、桐油味、香灰、血味、苦涩草药味 | 场景氛围、身份线索、毒药/迷香 |
| 触觉 | 脚底湿滑、肩头被拍、石子溅起泥水、露滴打湿衣料 | 身体入场、地形参与、环境压迫 |

**反模式：**

| 错误做法 | 正确做法 |
| --- | --- |
| "令狐冲注意到了远处的番伞" | "伞在雾里旋了半圈，伞面上的雨水被甩出去，甩成一圈细碎的水珠帘。伞后面是一双手。" |
| "他忽然闻到了一股酒味" | "酒舍里飘出一股红薯酒的酸甜气。令狐冲鼻尖一动，酒葫芦在腰间晃了晃。" |
| "陈阿叔走了过来，他是村里的渔民" | "陈阿叔扛着满满一篓海鱼自码头方向走来，竹篓压得他左肩微微塌陷，那道从肩头延伸到锁骨的陈年鞭痕在潮气里泛着暗红。" |
| 平铺直叙介绍场景切换 | 用感知触发物自然连接：石子溅水→脚印→人→杖声→酒气→布帘 |

**成功案例：**

《笑傲江湖4》第1章开场：令狐冲修机轴 → 水花（稚子跑过）→ 脚印（石板路延伸）→ 陈阿叔（扛鱼篓/鞭痕）→ 杖声（王哑伯拨灯芯）→ 酒气（酒舍飘出）→ 阿真（喊声/草纸）→ 布帘（阿袖探身）→ 女眷簇拥 → 海面桅灯。一个开场在一页半内建立了主角日常、十余个有名有姓的村民、村庄空间布局和萨摩威胁，零旁白介绍。

## Atmosphere And Mood Heuristics

- 氛围不靠堆形容词，靠一个精准的感官细节。"杯壁上的指印已经干透"比"房间空荡荡的"更有孤独感。
- 声音消失比声音出现更有力。当一个场景需要压力时，先想"什么声音该停了"。
- 通感的要诀：让不同感官为同一情绪服务。寒冷不只是温度——它可以是月光的颜色、风声的质感、石板的触感。
- 反衬是高级技法：用温暖的灯火写离别，用安静的午后写恐惧，用完整的日常物件写刚刚发生的暴力。观众会自己完成那一步推导。
- 微观放大：情感越强，画面可以越小。巨大的悲伤场不一定需要暴雨和狂风——一杯慢慢变凉的茶、一张被风吹起又落下的纸，往往更有力。
- 留白不是什么都没写，而是写了"事件之后的痕迹"。翻倒的茶杯还在原位、杯中的水被震出一半——观众会想象刚才发生了什么。
- 声景要有层次，不要只写一个声音。远处鼓声→近处呼吸→全部消失→灯芯爆一下——四层声音比一个"安静"有力十倍。
- 时间感是被忽视的意境维度。灰尘的厚度、蜡泪的堆积、杯中茶凉后水面的薄膜——这些都是"时间在画面上"。
- 不要怕"浪费"环境描写。同一个 slugline 内，环境刷新可以让观众感受到时间在流逝、压力在变化。但从室内到门廊、从主厅到角落的环境变化需要真实的空间焦点转移。
- 意境是锦上添花，不是替代戏剧功能。一个没有冲突和人物选择的场景，再美的意境也是空壳。
