# 北京电影学院 Agent 知识包

## 定位

这不是一套新的电影教材，而是 `北京电影学院` Agent 的知识调度包。

它的职责只有两件事：

1. 把现有十份学院派资料编排成可调用的证据层。
2. 让 Agent 的导演判断、镜头诊断与摄影建议都有明确回溯路径。

## 适用边界

- 适用：
  - 导演阶段审校
  - 导演说明与分镜返工
  - 镜头调度、轴线、景别、光色与对话场景诊断
- 不适用：
  - 代替 Agent 合同本体
  - 代替上游业务 schema
  - 作为“泛电影学百科”漫灌引用

## 最小包

- `source-map.md`
  - 说明每一份资料支持什么判断动作。
- `usage-protocol.md`
  - 说明什么时候必须先读知识库、什么时候只能把知识当潜台词支撑。
