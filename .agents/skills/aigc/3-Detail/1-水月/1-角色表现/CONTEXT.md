## Repair Playbook

1. 先确认当前 root 与命中 scope 是否唯一，避免在旧快照上做表演判断。
2. 再判断这一拍的主问题到底是露馅、出手，还是攻守试探；不要先写字段。
3. 只选一条主链和最多一条辅链，先提证据，再压回 schema 必填双槽；有明确对白时再补 `对话戏`。
4. 若 `对话戏` 命中且收益明显来自门槛、障碍、逼近、孤立、混乱或私密贴近，先写演员/空间压强几何，再决定是否回交 `运镜手法` 放大。
5. 若 `动作戏` 已成立但需要精确位移或走位，立即回交 `运动表现`，不要继续往 `角色表现` 里塞路径。
6. sidecar 写回前，最后检查一次 target path 是否只命中 `角色表现`，且 patch 中没有偷入镜头术语。

## Reusable Heuristics

- `角色表现` 最稳的写法不是“把三条叶子都写一遍”，而是“先判主问题，再把最有戏的一条主链和一条辅链压成必填双槽；有明确对白时再补 `对话戏`”。
- `动作戏` 回答“这一拍身体或外显动作怎样成立”；`对话戏` 回答“这一拍谁先发、谁截断、谁吞回、哪句怎么带压”；`内心戏` 回答“哪股意图被压在动作底下”。
- 对话戏最容易失真成“对白内容摘要”；一旦开始概括台词含义，就要立刻改回演员可执行的发法、气口和潜台词落点。
- 吸收运镜知识的正确方式，不是把 `推近 / 越轴 / 长焦 / 环拍` 抄进 `对话戏`，而是先把它们逆译成权力差、门槛、障碍、逼近、孤立、混乱或私密贴近这些对白压强几何。
- 当一场对白“值得被拍得更紧”时，`角色表现` 只负责写出为什么紧：谁堵门、谁逼近、谁被排除、谁说到一半失口；“如何拍紧”继续回交 `运镜手法`。
- 只要一句话还能直接替换成“他很难过/她很紧张/他们关系复杂”，通常就还没落成可演证据。
- `角色表现` 不再承担识别锚点；服装/身份识别应优先回读 `出场角色及穿搭` 或交给组级/下游消费层。
- 如果一个判断需要位置、动线或景别才能成立，那通常已经越过 `角色表现` 的字段边界。
