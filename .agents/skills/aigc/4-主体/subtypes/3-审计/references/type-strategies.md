## 路由与回退卡

- 判定顺序：`证据链 -> 失败维度 -> 修订建议 -> 下一入口`
- 冲突解消规则：证据不足优先于偏差判断，先补证据再定责
- unknown 默认路由：保留待核结论，不直接出终局 PASS
- 失败重试上限：2
- 停止条件：关键输入链缺失，无法形成成立的失败维度
