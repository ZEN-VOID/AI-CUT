# 高武题材模板 (High Martial Arts)

> **核心卖点**: 武道等级 + 战力跃迁 + 现代博弈。在现代社会框架中，用武道力量改写命运。

---

## 创意约束推荐（可选）
- 推荐约束包：Pack M15
- 通用叠加：Pack U03

## 1. 核心流派细分

### 全球高武流
- **特点**: 全球武道觉醒，各国角力
- **核心爽点**: 国战武道、世界排名、为国争光
- **代表作**: 《全球高武》——现代高武的标杆
- **创作要点**: 武道不只是个人实力——是国家战略资源

### 武道宗师流
- **特点**: 传统武术在现代都市中的传承与复兴
- **核心爽点**: 师门传承、武德精神、以武止戈
- **创作要点**: 现代人对传统武术的"偏见"是天然冲突

### 格斗竞技流
- **特点**: 格斗赛事为核心舞台
- **核心爽点**: 擂台对战、排名晋升、冠军之战

### 武道修真流
- **特点**: 传统修真与现代武道融合——气血=灵气
- **核心爽点**: 修仙体系和现代社会的碰撞

---

## 2. 战力体系设计

### 等级体系
```
武者 → 武师 → 大武师 → 宗师 → 大宗师 → 王者 → 武圣 → 武神
```

### 能力维度
| 维度 | 说明 | 训练方式 |
|------|------|----------|
| 气血 | 身体根本 | 药物/功法/战斗 |
| 战技 | 战斗技巧 | 实战/传承/领悟 |
| 精神力 | 感知/意志 | 观想/磨砺/顿悟 |
| 武道意志 | 核心信念 | 经历/顿悟 |

---

## 3. 追读心理

| 驱动力 | 执行手段 |
| --- | --- |
| 战力成长 | 每卷至少一次战力突破——且有"展示"场景 |
| 世界排名 | 排名系统可视化——读者可以追踪主角在世界上的位置 |
| 武道对决 | 宿敌线的建立和收束——每卷至少一场"不能输"的战斗 |

## 4. AI 写作专项指南

### 常见陷阱
1. **战力崩坏**: 50章就到顶了后面没法写 → 提前规划等级体系，每阶差距呈几何级递增
2. **战斗流水账**: "一拳→对方挡住→主角发力→对方倒下" → 战斗要写战术决策和身体代价
