# 悬疑灵异题材模板 (Supernatural Suspense)

> **核心卖点**: 灵异规则 + 心理恐惧 + 真相揭秘。在不可见的世界里，用理性对抗未知。

---

## 创意约束推荐（可选）
- 推荐约束包：Pack M06
- 通用叠加：Pack U04

## 1. 核心流派细分

### 灵异探险流
- **特点**: 主角主动或被动探索灵异地点——古墓、凶宅、废弃医院
- **核心爽点**: 灵异发现、危险逃脱、真相揭露
- **创作要点**: 灵异地点的"历史"是最好的恐怖素材——每个鬼都有故事

### 都市传说流
- **特点**: 以都市传说为蓝本——裂口女、如月车站、红衣小女孩
- **核心爽点**: 传说验证、规则破解、传说背后真相
- **创作要点**: 利用读者已有的"都市传说记忆库"——不需要铺垫就能让人毛骨悚然

### 驱魔除灵流
- **特点**: 专业驱魔人/道士除灵——职业选手降维打击
- **核心爽点**: 专业碾压、法器对决、道术斗法
- **创作要点**: 驱魔要有"工作感"——不是热血，是"这单很难搞"

### 民俗恐怖流
- **特点**: 以中国传统民俗恐怖为核心——纸人、棺材、冥婚、扎纸
- **核心爽点**: 文化恐怖共鸣——"我奶奶也说过这个"
- **创作要点**: 民俗恐怖的优势——不需要解释，中国人天生就害怕这些

---

## 2. 恐怖氛围营造

### 恐惧金字塔
```
第一层——感官恐惧: 突然的声音、诡异的画面
第二层——认知恐惧: 习以为常的东西变得不对劲
第三层——存在恐惧: 自己的存在/记忆/身份被质疑
```

### 信息缺失原则
- **不要写"鬼是什么样子"** → 写影子、写声音、写温度变化
- **不要写"为什么会这样"** → 留白比解释更恐怖
- **不要写"它很恐怖"** → 写主角的生理反应——心跳、冷汗、腿软

---

## 3. 追读心理

| 驱动力 | 执行手段 |
| --- | --- |
| 灵异好奇 | 渐次揭示灵异的规则和来源——每章一个"新发现" |
| 安全恐惧 | 在恐怖中加入"安全区"（白天/特定地点）——让读者喘口气 |
| 解谜欲望 | 每个灵异事件背后要有一个"可理解的真相" |

## 4. AI 写作专项指南

### 常见陷阱
1. **形容词喊恐怖**: "恐怖""阴森""诡异"堆砌 → 写具体感官扭曲
2. **鬼怪过早显形**: 第一章就写清楚鬼长什么样 → 前50%只写影响不写本体
