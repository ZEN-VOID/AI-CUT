# scripts

本目录只承载机械辅助脚本，不替代 LLM 对剧本化投影、导演判断、表演工艺、画面化语言和编导质量的主创判断。

允许脚本做：

- 检查 frontmatter 和 `【剧本正文】` 是否存在。
- 检查场景标题是否符合 slugline 基本格式。
- 检查对白、独白、旁白、音效是否就近配对对应画面字段。
- 检查每个场景是否至少有一条正式画面字段。
- 检查 `场面调度` 是否混入机位、景别、镜头运动、分镜编号等摄影越权词。
- 检查场景是否以总结式 `表演提示` 或 `场面调度` 结尾；这类规划结果应内嵌到对应 beat。
- 检查内部任务说明、模板占位句或规则复述是否泄露到剧本正文。
- 检查 `环境描写` 中明显混入的人物动作或对白引出词。
- 检查 `心理反应`、`表演提示` 和画面字段中明显抽象、不可见、不可听、不可演的心理解释词；具体语义以本阶段字段分流、导演层和表演层合同为准。
- 检查 `角色动作` / `动作画面` 中明显的主观意图词，例如”试图、想要、打算、意图”。
- 检查 `心理反应` 中直接书写的主观情绪感受词，例如”感到恶心/难受/愤怒/害怕/崩溃”。
- 检查 `内心独白（角色）` 中第三人称自指问题，例如”他心想/她暗想/他知道/她觉得”等应转为第一人称的表述。
- 检查画面字段中的往日常态总结词，例如”早已熟悉/早已习惯/每天都/一直以来/无数次”。
- 检查场景末尾和倒数第二字段是否为总结式 `表演提示` 或 `场面调度`。
- 检查直接主观情绪词、主角视角判断被客观化、往日常态总结和无关前史/物品来历等高频风险词。
- 标记基础结构风险；`controlled_enrichment_ledger` 的上游锚点和剧情越权风险仍需 LLM/人工 review。
- 输出可复查的机械校验报告。

禁止脚本做：

- 自动生成剧本正文。
- 自动改写对白、独白或剧情事实。
- 通过模板拼接替代 LLM 编导创作。
- 自动生成演员表演任务、潜台词行为或场景状态差。
- 自动判断下游受控增强是否拥有充分上游锚点。
- 将机械校验通过误标为剧情事实完整承接通过。
