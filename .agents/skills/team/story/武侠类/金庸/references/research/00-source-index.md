# 金庸调研来源索引

更新时间：2026-04-18

## A. 一手 / 准一手材料

1. 金庸，`《谈批评武侠小说的标准》`，1957-10-05 原刊于《新晚报》。
   - 当前使用形态：公开评论与书评中的原文转引。
   - 关键点：主题思想、人物刻画、故事与结构、环境刻画；并明确希望武侠小说“有资格被称为文学”。
   - 可见转引：
     - 齐鲁晚报书坊周刊 2025-03-07 书评转引：<https://epaper.qlwb.com.cn/mobile/qlwb/content/20250307/ArticelA09002GN.htm>
     - 搜索摘要补充：<https://sjb.qlwb.com.cn/qlwb/PDF/20250307/A09.pdf>
   - 可信度：中（原文可确认存在，但当前使用的是转引）

2. 金庸公开访谈节录，关于写作作息与“理性/浪漫”。
   - 可见转引：Bilibili 公开视频页摘要
   - URL：<https://www.bilibili.com/video/BV1qW4y1S7Sw/>
   - 关键点：白天写小说、晚上写社评；理性与浪漫并存，且本人偏爱浪漫。
   - 可信度：中

3. 金庸 1972 年《明报》社评 `《作读者的耳目和喉舌》` 相关引文。
   - 当前使用形态：经济观察网文章引述原社评
   - URL：<https://www.eeo.com.cn/2013/0620/245508.shtml>
   - 关键点：办报是企业，但对中国事务有报道与表意见的责任。
   - 可信度：中

4. 金庸接受《南方周末》采访的公开节录。
   - 当前使用形态：搜狐娱乐转载页
   - URL：<https://yule.sohu.com/64/06/article211690664.shtml>
   - 关键点：文学审美不必等同道德说教；阅读小说包含读者共同形象化人物的过程。
   - 可信度：中
