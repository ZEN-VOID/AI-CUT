# 类型片灵感矩阵

本文件用于把“从各类类型片获取灵感和标志性元素”的动作压成稳定的 inspiration 抽取规则。

硬规则：

1. 只借运作逻辑、标志性元素和情绪组织方式，不复刻具体镜头、人物配置或 IP 设定。
2. 先判断当前项目主类型 corridor，再从适配族群里选 inspiration，不反过来让 inspiration 改写项目主型。
3. 每次抽取都要同时记录：
   - 借什么
   - 为什么适配
   - 不能借什么
   - 要如何翻译成 `全集设计.md` 或 `分组设计.md` 的字段

## Extraction Checklist

每次使用本矩阵时，至少要回答：

1. 当前项目真正的主类型 corridor 是什么。
2. 需要从哪两到三类类型族群借力。
3. 借的是“运作逻辑”还是“表层样式”。
4. 这些 inspiration 如何翻译成项目级总则与组级差异。
5. 哪些错误类型是当前项目最容易误滑进去的。
