# 3-Detail Examples

## 正例

### `角色表现.动作戏`

- 好写法：
  - `她把话咽回去，手还停在门把上，肩线僵着不肯转身。`
- 原因：
  - 可见
  - 可演
  - 给后续镜头运动留下空间

### `角色表现.动作戏`

- 好写法：
  - `他先退到窗边，再沿桌角绕向门口，把两人的距离从对峙拉成斜向错位。`

- 原因：
  - 动作方向清楚
  - 可直接被运镜承接
  - 不需要额外再拆单独的运动字段

### `分镜构图.构图形式`

- 好写法：
  - `以门框做内框，把她压在右侧窄幅里，左侧留给空走廊和他迟到的影子。`

### `摄影表现.光影`

- 好写法：
  - `走廊冷白顶灯把她脸上的犹豫照得发硬，门缝里漏出的暖光只勾住手指边缘。`

## 反例

### 空泛

- 差写法：
  - `情绪成立，关系压力更强，电影感更足。`
- 问题：
  - 没动作
  - 没空间
  - 没物件
  - 没法直接消费

### 越权

- 差写法：
  - `角色表现.动作戏：镜头快速推进到她脸上。`
- 问题：
  - 把表演字段写成了运镜字段

### 失焦

- 差写法：
  - `人物在这里情绪复杂，空间也很有感觉。`
- 问题：
  - 既没有镜级正文承接
  - 也没有主体锚定
  - 下游无法消费
