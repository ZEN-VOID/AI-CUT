# 05-decisions｜川尻善昭：重大创作决策与行动逻辑

## 调研结论

川尻善昭的重要决策体现出三条线：抓住时代窗口把个人趣味压到极限；在他人 IP 中寻找个人化入口；根据制作体力与生产方式调整角色，从导演转向分镜/脚本等更可独立完成的工作。

## 决策 1：《Ninja Scroll》不直接改山田风太郎，而是做全球可读的原创忍者娱乐

- 背景：川尻长期想把山田风太郎式忍法帖做成动画，但原作强依赖日本历史事实。
- 决策：保留忍者、骗术、异能、成人暴力和时代动作快感，创造观众更能共情的主角与更全球化的故事入口。
- 证据：CBR 2024 中他谈到山田风太郎历史事实对不了解背景者可能难懂，因此希望做出能打动年轻动画爱好者和全球观众的作品。
- 逻辑：不是“忠实改编”，而是把题材机制抽出来，重新组织成动画类型片。

## 决策 2：在《Animatrix》中拒绝只执行他人脚本，改写个人化短片

- 背景：Wachowski 发送脚本时，川尻正做 TV 动画《X》，时间紧张，且最初觉得脚本“不适合自己”。
- 决策：在对方允许下自己写脚本，形成《Program》和《World Record》。
- 证据：CBR 2024 中他描述准备拒绝后，对方说想写就写，他写了两个脚本且都被喜欢。
- 逻辑：当外部 IP 不贴合自己时，寻找能表达个人性的入口，而不是勉强执行。

## 决策 3：把《World Record》交给小池健，而非自己全部导演

- 背景：他无法在《X》之外同时导演两部短片。
- 决策：自己导演《Program》，小池健导演《World Record》。
- 证据：CBR 2024 与 Matrix 访谈均提到这一分工；Matrix 访谈中他说小池版比他自己可能做出的版本更好。
- 逻辑：强蓝图作者也要识别合作者的生成力；放手不是损失，而可能让作品更强。

## 决策 4：近年以分镜为主，承认导演工作需要更多体力和团队沟通

- 背景：近年作品记录更多出现在 storyboard / key animation / 分镜协力，而非原创长片导演。
- 决策：更多承接分镜、脚本等可独立完成的工作。
- 证据：CBR 2024 中他说明分镜和脚本几乎可独自完成，而导演要同各环节团队沟通，年龄与体力让导演变困难。
- 逻辑：创作角色选择受体力、年龄、生产结构约束；不要把“再导演”当作无成本意愿问题。

## 决策 5：对《Ninja Scroll》重制倾向保留故事、更新技术与伦理处理

- 背景：30 周年语境下被问是否重制。
- 决策：故事和角色不必改；若重制，重心在 3D 动画等新表达，同时敏感场景今天难以原样保留。
- 证据：CBR 2024。
- 逻辑：经典 IP 的真源是结构与角色，不是每个历史时代场景的原样复制。

## 决策 6：在《Vampire Hunter D: Bloodlust》中把菊地秀行素材处理成哥特动作爱情

- 背景：菊地秀行小说拥有黑暗幻想和吸血鬼猎人世界。
- 决策：影片突出荒野哥特、怪物遭遇、交通工具、猎人与被追者关系，以及 Leila / D 情感锚点。
- 证据：Den of Geek 对其视觉地点、装甲车袭击、最终剑战和 D / Leila 关系的评价；川尻本人 Matrix 访谈称自己很满意该片。
- 逻辑：黑暗幻想不能只靠设定，必须有动作 setpiece 和情感尾音。

## 言行一致/不一致观察

- 一致：他反复说视觉先行，作品也明显由强图像和分镜动作驱动。
- 一致：他强调大众娱乐入口，《Ninja Scroll》《Animatrix》都把复杂题材变成清晰类型动作。
- 张力：他擅长强控制分镜蓝图，却能承认小池健改写自己的想象更好。
- 张力：他珍视旧作完整性，也承认现代敏感场景需要处理。
