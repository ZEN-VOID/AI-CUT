# Cinematic Technique Library

本文件只保留静态摄影构图与画面组织技法，用于在 `4-摄影` 内部裁决单个时间段的静态画面骨架：景别、景深、镜头视角、镜头类型、构图方式、光影层次、色彩组织和画面质地。运镜、速度、方向参照、焦点拉移、景别切换、连招、动态高点处理和边界交出锚点已迁移至 `camera-movement-emotion-contract.md`。

## Example Usage Guard

本技法库中的表格、适用场景和示例句只提供静态构图候选语言与判断维度，不是必须输出的摄影参数清单。执行具体任务时，不得机械堆满景别、焦段、视角、光影或构图词；必须只选择当前节拍需要的最小充分静态画面要素，并内化为自然分镜文字。

## 静态摄影执行参数

### 景别

| shot_size | use when | execution cue |
| --- | --- | --- |
| 大远景 / 远景 | 建立封闭空间、群体处境、人物被制度吞没 | 人物小于环境，课桌、黑板、门窗成为压迫主体 |
| 全景 | 交代人物站位、动作路径、权力距离 | 保留脚下空间、讲台高度、座位行列和门框关系 |
| 中全景 | 人物动作与环境压力同时重要 | 看见上半身动作和周围桌椅/门/黑板的束缚 |
| 中景 | 对话、课堂秩序、人物被点名 | 保留手、肩、课桌边缘和脸部反应 |
| 近景 | 压迫、隐瞒、警觉、呼吸变化 | 让脸和肩占主导，背景仍保留危险色块 |
| 特写 | 眼神、嘴唇、手指、道具、规则文字 | 信息必须可读，避免只为了漂亮而特写 |
| 大特写 | 瞳孔反光、指腹泛白、字迹异常、液体边缘 | 只用于关键揭示或情绪峰值，保持稀缺 |

### 景深

| depth | use when | execution cue |
| --- | --- | --- |
| 深景深 | 空间关系、群像压迫、前中后景同时有威胁 | 黑板、学生、门口或窗帘同时可读 |
| 中景深 | 人物与近旁道具/同桌关系并重 | 主体清楚，背景仍有可辨危险轮廓 |
| 浅景深 | 主体孤立、心理压迫、信息暂时遮蔽 | 背景化成受控色块或光斑，焦点锁住眼或手 |
| 全实 | 空间关系展示、群像、规则布局 | 前中后景信息全部可读 |
| 主实副虚 | 主体突出、信息分层 | 主体清晰，背景降级为情绪层次 |
| 前虚后实 | 发现、揭示、注意力分层 | 前景模糊剪影，后景信息保持清楚 |
| 全虚 | 失神、眩晕、意识模糊 | 画面整体失焦，只保留色块和光影轮廓 |

### 镜头视角

| angle | use when | execution cue |
| --- | --- | --- |
| 平视 | 保持观察冷静、人物间暂时平等 | 适合克制分析、低声谈判和中性观察 |
| 低角度 | 老师、规则、黑板、道具压迫角色 | 让讲台、教鞭、黑板字向画面下方压来 |
| 高角度 | 学生无助、群体被编排 | 课桌像棋盘，人物像被固定在格子里 |
| 俯拍 | 制度、规则、命运操控感 | 用于全班、座位、规则布局，不滥用到普通对话 |
| 仰拍 | 入场、宣判、危险权力增强 | 门口轮廓、黑板公告或权力人物可用 |
| 过肩视角 | 角色被凝视或观察他人 | 前景肩背形成遮挡，后景目标带危险距离 |
| 主观视角 | 感知异常、发现隐藏信息 | 需配合边缘遮挡、焦点状态或视野限制 |
| 画外窥视视角 | 监控感、被观察、走廊未知威胁 | 门缝、窗帘边、课桌缝隙作为遮挡框 |

### 镜头类型

| lens_type | use when | execution cue |
| --- | --- | --- |
| 广角镜头 | 空间畸变、近距离压迫、教室变形 | 近前景夸张，后景被拉远，适合恐惧空间 |
| 标准镜头 | 保持现实感、对话、观察 | 不抢戏，让表演和规则压迫自己发生 |
| 长焦镜头 | 压缩空间、制造无路可退 | 课桌距离被压扁，讲台像贴到学生脸上 |
| 微距镜头 | 字迹、咬痕、指腹汗水、微小异常 | 材质、液体、纸纹和异常边缘必须清楚 |
| 移轴 / 倾斜焦平面 | 规则世界错位、空间不可信 | 少量用于异常感，不作为常规漂亮效果 |
| 固定机位 | 权力碾压、课堂制度、不可反抗 | 镜头不动，让人物在框内被规则处理 |

## 构图方式

构图方式从七个子维度裁决画面的审美骨架。执行时按当前画面的戏剧任务选取最关键的 2-3 个子维度，不机械覆盖全部。完整的组级画面属性合同参见 `scene-visual-constraint-contract.md`（原 `group-visual-profile-contract.md`，2026-05-14 重命名）。

### 形状感

| shape | psychology | use when |
| --- | --- | --- |
| 方形 / 矩形 | 人造规划、稳定、坚固；也有局限、封闭之意 | 教室秩序、制度框架、课桌网格 |
| 正三角形（"金字塔"构图） | 稳定、永恒、统治 | 权力人物居中、讲台高处、规则宣判 |
| 倒三角形 | 不稳定、倾斜、压力倒置 | 角色被压迫、空间失衡、规则入侵 |
| 圆形 / 椭圆 | 包围、循环、聚焦 | 群像围困、瞳孔、聚光灯 |
| 对角线菱形 | 张力、冲突交汇 | 人物对峙、视线交叉 |
| 不规则碎片 | 混乱、崩塌、失控 | 规则破碎、群体恐慌、空间异常 |

### 线条感

| line | psychology | use when |
| --- | --- | --- |
| 垂直线 | 高大挺拔、稳定、静默 | 柱子、门框、站姿人物 |
| 水平线 | 平静、开阔、安详 | 地平线、桌面、走廊延伸 |
| 对角线（倾斜线） | 不稳定、紧张，暗示空间深度 | 走廊透视、课桌排列倾斜、角色倾斜身体 |
| 波状线（"S"形） | 动静交替、内在律动、秀美美感 | 走廊曲线、窗帘褶皱、角色行走路径 |
| 放射线 | 聚焦、爆发、冲击 | 灯光扩散、恐惧辐射、注意力中心 |
| 锯齿线 | 紧张、焦虑、危险 | 裂缝、锯齿状阴影、规则边缘 |

### 影调感

利用明暗对比增强画面深度，保持亮部质感和暗部层次。

| tonal | use when | execution cue |
| --- | --- | --- |
| 高调 | 明亮、纯净、脆弱 | 过曝教室、白色校服、冷白灯光淹没人物 |
| 低调 | 压迫、神秘、危险 | 深色窗帘吸光、角落暗影、只保留危险轮廓 |
| 中间调 | 自然、日常、观察 | 常态校园光线，不强化明暗倾向 |
| 高反差 | 戏剧、冲突、极端情绪 | 强烈侧光切割脸、窗口高光+室内深暗 |
| 柔和低反差 | 梦境、回忆、温柔 | 柔光包裹、雾气、色块过渡自然 |

### 节奏感

通过元素的重复、间隔、变化，创造视觉节奏和韵律。这里的节奏只指静态画面内元素排布，不包含镜头运动或剪辑节奏。

| rhythm | use when | execution cue |
| --- | --- | --- |
| 重复节奏 | 秩序、制度、压迫 | 课桌行列、窗户间隔、日光灯排布 |
| 渐变节奏 | 递进、压迫加深、空间延伸 | 桌椅渐远渐小、光影渐暗 |
| 交替节奏 | 对比、张弛、对话 | 明暗交替、前景后景交替出现信息 |
| 突变节奏 | 惊吓、揭示、入侵 | 秩序被打破的瞬间，形状或亮度突然改变 |
| 自由节奏 | 混乱、失控、边缘 | 元素无规律散布，但保留 1-2 个视觉暂留锚点 |

### 纹理和质感

添加特定画面质地效果，增强艺术性和独特性。

| texture | use when | execution cue |
| --- | --- | --- |
| 颗粒感 | 复古、纪实、心理压力 | 胶片颗粒在暗部可见，增加粗糙真实感 |
| 胶片质感 | 时代感、记忆、怀旧 | 色彩偏移、高光溢出、暗部保留层次 |
| 光晕 / 眩光 | 超现实、神圣、危险诱惑 | 窗口光晕、灯管溢光、瞳孔反光扩散 |
| 磨砂 / 雾气 | 梦境、隔阂、不确定性 | 画面带雾感，细节被吞没，只保留大色块 |
| 锐利质感 | 真实、冷酷、制度 | 高锐度让每个细节清晰可读，无处可藏 |

### 气势

通过对时空环境的静态组织，表现特定精神风貌和情绪感染力。

| atmosphere | use when | execution cue |
| --- | --- | --- |
| 压迫气势 | 制度碾压、权力宣判 | 低角度仰拍、深焦、大面积阴影、硬光 |
| 渺小气势 | 个体被吞没、群像无助 | 大远景、高角度、人物占比极小、环境主导 |
| 紧迫气势 | 倒计时、危险逼近 | 对角线构图、浅景深、压缩空间、边缘遮挡 |
| 肃穆气势 | 仪式、宣判、命运 | 对称构图、固定机位、严整线条、低饱和影调 |
| 荒诞气势 | 异常、错位、规则荒谬 | 移轴、不规则构图、反常尺度关系 |
| 温柔气势 | 治愈、关系、安全 | 柔光、浅景深、自然纹理、留出呼吸空间 |

## 经典电影构图

| technique | use when | execution cue |
| --- | --- | --- |
| 对称压迫构图 | 校园秩序、讲台权力、黑板规则压住学生 | 中轴线、桌椅行列、门框或黑板作为几何牢笼 |
| 前景遮挡 | 角色被规则、课桌、门框、他人身体压住 | 用窗帘、课桌边、门框切割画面 |
| 低机位权力镜头 | 老师、规则、系统公告占上风 | 让人物或文字向下压迫观众视线 |
| 高机位俯视 | 群体无助、棋盘式命运 | 课桌排列像格子，人物缩在固定座位中 |
| 负空间孤立 | 某人被点名、被凝视、被规则锁定 | 让空白黑板、封死窗帘或冷白墙面吞掉人物 |
| 深焦层次 | 同时呈现前景反应、中景动作、后景威胁 | 前中后三层都放入可读信息，避免扁平舞台感 |

## 光影与色彩组织

本节只记录静态画面中的光影和色彩组织结果；光源叙事功能详见 `light-as-narrative-contract.md`，场景整体色彩约束详见 `scene-visual-constraint-contract.md`。

| visual_layer | use when | execution cue |
| --- | --- | --- |
| 冷白日光灯 | 校园制度感、非人秩序、医疗式冷酷 | 顶光压脸，眼窝发暗，皮肤失温 |
| 黑窗帘吸光 | 封闭、无外部世界、逃生路径被抹除 | 让窗帘成为吞光背景和负空间 |
| 稀缺危险点色 | 危险、诱饵、惩罚或异常信息 | 只在危险或诱饵上出现，保持稀缺 |
| 异质微光 | 隐藏规则、特殊信息、第二层真相 | 贴近眼底或文字边缘，不污染全局色调 |
| 半明半暗轮廓 | 人物身份不稳定、威胁不确定 | 光线边缘切脸，一半服从秩序，一半藏入阴影 |
| 常态底色 | 制度空间、日常压迫、低生命感 | 作为常态底色，压低活跃度 |
| 秩序对比 | 规则、惩罚、视觉母题 | 高对比建立秩序，危险点成为警告和欲望 |
| 陈旧纸色 | 课本、规则、旧校园物件 | 让道具带腐败感，不显现代明亮 |

## 静态技术参数

内部裁决的静态拍摄参数，供 `shot_design_plan` 使用。成稿中只显式写出影响当前画面最关键的技术选择，不逐项列出全部参数。动态范围、运动模糊、手持临场感和快门运动控制归入 `camera-movement-emotion-contract.md`。

| parameter | options | use when |
| --- | --- | --- |
| 机型 / 传感器风格 | ARRI Alexa Mini、Sony Venice、RED Komodo、DSLR/Mirrorless | 影响色彩科学、动态范围、噪点特征和制作质地 |
| 光圈 | f/1.4-f/2、f/2.8-f/4、f/5.6-f/8、f/11-f/16 | 控制景深和进光量 |
| ISO 感光度 | 100-400、800、1600-3200、6400+ | 控制噪点和暗光表现力 |
| 焦距 | 16-24mm、35mm、50mm、85mm、135-200mm | 控制空间透视和主体占比 |
| 分辨率 | 2K、4K DCI、6K、8K | 控制细节精度和裁切空间 |

## Extension Rule

- 新增静态构图技法必须写明 `use when` 和 `execution cue`，避免只添加审美名词。
- 新增条目不得包含运镜、速度、景别切换、动态连招、焦点拉移、方向参照或跨镜交出设计；这些内容必须写入 `camera-movement-emotion-contract.md`。
- 若某技法改变输出验收或执行流程，必须同步回改 `SKILL.md` 的 `Reference Loading Guide` 或 `Execution Contract`。
