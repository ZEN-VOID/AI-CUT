# 06 作品时间线

| 时间 | 事件 | 作品意义 |
|---|---|---|
| 2013 | 金宇澄《繁花》出版 | 奠定上海百科式、细节繁密、多时间线的文学母体 |
| 2015 | 《繁花》获茅盾文学奖；王家卫取得改编权 | 作品从文学热议转入长线影像开发 |
| 2017 | 剧组研究工作展开 | 年代考据、空间重建、服装与饮食档案系统逐步成形 |
| 2019 | 拍摄启动 | 作品进入实体表演、空间和方言生态建构阶段 |
| 2020-2022 | 疫情与封控中断拍摄，剪辑持续推进 | 停摆被转化为重新组织长篇叙事的时间 |
| 2023-12-27 | 剧集播出 | 《繁花》成为中国大众文化事件 |
| 2024-01 | 普通话版与沪语版同步热议 | 作品的地方语言策略被大规模验证 |
| 2025-11-24 | Criterion Channel 北美首播 | 作品进入更明确的国际作者剧集通道 |
| 2026-02-26 | 英国 MUBI 首发 | 国际流通继续扩大 |
| 2026-03-05 | 上海官方发布扩大国际发行消息 | 《繁花》完成从本地现象到国际样本的再升级 |
