# 黑暗题材模板 (Dark Themes)

> **核心卖点**: 复仇执念 + 人性深渊 + 反英雄弧光。在黑暗中行走的人，可能成为光——也可能彻底沦为暗。

---

## 创意约束推荐（可选）
- 推荐约束包：Pack M25
- 通用叠加：Pack U04

## 1. 核心流派细分

### 复仇流
- **特点**: 主角为复仇而生——复仇是唯一的信念
- **核心爽点**: 深谋远虑的复仇布局、复仇揭开更大阴谋
- **核心问题**: 复仇完成后，主角还剩什么？这是他的人物弧光

### 黑化反英雄流
- **特点**: 主角并非传统英雄——手段灰色，目标可能是善的
- **核心爽点**: 以恶制恶、"比坏人更坏"才能对付坏人
- **核心问题**: 屠龙者会不会变成新的恶龙？

### 生存流
- **特点**: 残酷世界的生存挣扎——没有对错，只有活着
- **核心爽点**: 绝境求生、以最小代价换取最大收益
- **核心问题**: 为了活命做的事，还能回头吗？

---

## 2. 世界观设计

### 黑暗程度阶梯
| 等级 | 描述 | 读者的希望感 |
|------|------|-------------|
| 灰色 | 世界有暗面，但光明也在 | 有——主角的黑暗是为了守护光明 |
| 黑色 | 世界基本没救了 | 微弱——主角是最后的烛火 |
| 深渊 | 世界本身就是恶的 | 几乎没有——唯一的希望是"改变规则本身" |

---

## 3. 追读心理

| 驱动力 | 执行手段 |
| --- | --- |
| 复仇期待 | 复仇不是一次性的——分阶段揭示仇人的全貌 |
| 底线拉扯 | 每卷至少一次"选择是否越界"——主角每次选择定义了他的弧光 |
| 正义焦虑 | 读者想知道"这么做对吗"——不给答案，但给后果 |

## 4. AI 写作专项指南

### 常见陷阱
1. **黑暗无目的**: 为黑而黑 → 每条黑暗线都要有"为什么"——不展示黑暗本身，展示黑暗的根源
2. **主角沦为反派**: 主角做的事和反派没有区别 → 保留"不可跨越的底线"——这是主角和反派的区别
