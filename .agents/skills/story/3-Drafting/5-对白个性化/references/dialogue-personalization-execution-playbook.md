# Dialogue Personalization Execution Playbook

本细则服务 `3-Drafting / 5-对白个性化`，用于把 Step 4 已显影的人物行动逻辑，稳定翻译成“像这个人此刻会说的话”。

## Positioning

- Step 4 负责“这个人会怎么做”。
- Step 5 负责“这个人会怎么开口、怎么绕、怎么压、怎么躲、怎么解释自己”。
- Step 6 负责“这个人心里怎么感到、怎么忍住、怎么自辩”。
- 因此 Step 5 先改话语策略与声口，再决定句长、停顿和措辞；不要反过来只做表层润句。

## Dialogue Function Routing Matrix

| `dialogue_function` | 当前目标 | 优先改写点 | 最常见失败 |
| --- | --- | --- | --- |
| `confession_or_defense` | 承认、辩解、求轻判、遮羞 | 羞耻动作、自辩漏洞、说不全的地方 | 只剩诉苦或大道理 |
| `probing_or_pressure` | 试探、逼问、压位、迫使表态 | 追问路径、称呼变化、利害关系、话语留白 | 双方把动机解释得太完整 |
| `persuasion_or_deterrence` | 劝阻、诱导、拖延、换策略 | “现在动会怎样 / 晚一步会怎样” 的现场算局 | 写成作者替角色讲道理 |
| `confrontation_or_break` | 翻脸、对峙、拆穿、划清界线 | 抢话、打断、回避、针锋相对的节奏 | 只是轮流发言，没有压迫感 |
| `warmth_or_repair` | 缓和、示弱、靠近、试探和解 | 退半步的措辞、称呼松动、没说满的善意 | 过甜、过顺、失去人物防御 |
| `concealment_or_misdirection` | 隐瞒、移话题、转移风险 | 避答、答非所问、局部真话、故意说漏半句 | 角色像讲题人，藏不住信息 |
| `testimony_conflict` | 多人各执一词、抢占解释权 | 各自维护的体面、利益、恐惧、物证锚点 | 变成机械复盘案情 |

## Persona Binding Slots

对本场主要说话者，先补足以下槽位，再改正文：

| slot | 要回答的问题 | 正文中的典型落点 |
| --- | --- | --- |
| `sentence_length` | 他通常是短句压人，还是长句铺网？ | 断句、补句、是否一句说满 |
| `address_pattern` | 他怎么称呼人，何时改称呼？ | 直呼其名、敬称、绰号、故意不叫 |
| `info_density` | 他是一次说透，还是只抖一半？ | 留白、绕路、故意漏关键信息 |
| `attack_defense_mode` | 他更擅长顶、拖、躲、试、哄还是算？ | 抢话、拖字、赔笑、冷问、反问 |
| `avoidance_path` | 他最不肯直说的是什么？ | 换话题、拿别人说事、突然沉默 |
| `shame_trigger` | 什么会让他瞬间想遮掩体面？ | 赔笑、嘴硬、先发制人、装轻松 |
| `self_justification` | 他会怎样把自己修剪成还能接受的样子？ | “我不是…只是…”、“你也知道…” |

## Literary Lens Routing

只有当当前对白已经命中相应结构症状时，才按需借用外部文学镜片；镜片服务结构，不服务模仿。

| 触发症状 | 首选镜片 | 可借用的结构逻辑 | 严禁误用 |
| --- | --- | --- | --- |
| 失败者赔笑、自白越写越平、羞耻不落地 | `太宰治` | 羞耻动作、自辩露馅、轻佻外壳下的伤口、生活账压到说话方式上 | 直接模仿私小说口吻、把角色写成文学人物附体 |
| 多人对质只是“各有版本”、冲突像会议纪要 | `芥川龙之介` | 自我辩护证词、体面与利益绑定、物件/动作当证据、真相缺口保留 | 只做阴郁氛围，不做证词结构 |

## Boundary Guard

1. 文学镜片只增强“为什么这样说”，不直接替换人物卡。
2. 若镜片引入的措辞让角色失去当前题材、时代、阶层或项目语感，立即回退到人物卡事实。
3. 若问题本质是行为失真，回 Step 4；若问题本质是内心评论腔，回 Step 6；不要让 Step 5 吞并其他工序。
4. 若一句台词主要功能是替作者解释结构、主题或设定，默认删减或转写为利害关系。

## Rewrite Priority

1. 先改说话目标，再改攻防方式，再改句长和词面。
2. 先让每个人开口时都在做事，再追求“好听”。
3. 对强冲突对白，优先保留压力和缺口，不要急着补齐逻辑闭环。
4. 对关系回暖对白，优先保留人物原本的防御，不要突然全说开。
5. 若句子很聪明但不像现场说出来的，先拆完整度，再保留锋利处。

## Common Failure Patterns

| symptom | direct cause | rewrite move |
| --- | --- | --- |
| 台词自然，但谁说都行 | 只做口语化，没有做人设绑定 | 回到 `Persona Binding Slots` 补说话画像 |
| 台词有信息，但没有动作感 | 把对白当说明书 | 给每句补话语目标和攻防方向 |
| 自白很重，却不刺人 | 羞耻和自辩没有露馅 | 借太宰镜片补赔笑、遮掩、生活账 |
| 对质很激烈，却像轮流复盘 | 证词没有各自的体面/利益 | 借芥川镜片重绑自我辩护与物证锚点 |
| 文学镜片一上就像模仿名家 | 把镜片当文风滤镜 | 只保留结构逻辑，改回角色自己的时代与身份语言 |
