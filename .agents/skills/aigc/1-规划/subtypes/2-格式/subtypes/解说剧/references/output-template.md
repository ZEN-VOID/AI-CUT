# 解说剧 / Output Template

本文件是 `解说剧` 子技能的模板真源。

## `格式合同.md`

```markdown
# 解说剧 格式合同

## 变体定位
- 旁白主导
- 对白保真

## 场景标题规范
- 使用 `### 场景X：<场景信息>`

## 允许字段
- 动作画面：
- 对白（<说话人>）：“……”
- 对白画面：
- 旁白（讲述者）：“……”
- 旁白画面：
- 内心独白（<说话人>）：“……”（仅用户显式要求时）
- 内心独白画面：

## 硬门槛
- 非对白文本优先旁白化
- 旁白主体统一为 `讲述者`
- 动作不进引号正文
```

## `格式样例.md`

```markdown
### 场景一：码头·夜

旁白（讲述者）：“风先到了，消息却比船更慢。”
旁白画面：夜雾贴着江面缓慢推来，栈桥尽头的灯火被潮气揉成一团模糊金影。
动作画面：老旧木船撞上岸桩，绳索在粗糙木柱上猛地绷紧，又发出一声迟来的呻吟。
对白（周临）：“人都到齐了吗？”
对白画面：周临踩上潮湿木板，先扫过桥头那几张紧绷的脸，才把声音压低。
```
