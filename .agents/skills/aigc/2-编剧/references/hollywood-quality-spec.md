# Hollywood Quality Spec

> **契约地位**：本规范用于把忠实投影提升到高质量影视剧本形态。它不能覆盖保真规则，不能授权删减、摘要、重排或自由改写上游剧情因果。

---

## 1. 场景戏剧学（Scene Dramaturgy）

> **来源**：Hollywood Screenwriting Fundamentals

每个场景应尽量显性化以下戏剧要素：

| quality_axis | requirement | execution_field |
|---|---|---|
| **场景目的** | 观众在本场景获得什么新信息、压力或人物关系变化 | 场景入口状态 |
| **冲突压力** | 人物面对什么即时危险、规则限制、误解或选择成本 | 压力源画面 |
| **行动链** | 每个重要角色通过可见行动推动或抵抗局面 | 角色动作、群像画面 |
| **信息差** | 谁知道、谁不知道、谁以为自己知道 | 内心独白、场面调度 |
| **视觉核心** | 本场景最能承载类型感和情绪的画面物件或空间状态 | 环境描写、道具特写 |
| **声音核心** | 最能制造节奏、压力或规则感的声音 | 音效、音效画面 |
| **表演任务** | 演员每段对白、沉默、停顿对应的可执行目标 | 表演提示 |
| **潜台词策略** | 角色不能直接说出口的信息如何转成行为、停顿、道具和视线 | 对白画面 |
| **权力调度** | 人物高低、远近、站坐、门口/讲台/道具归属如何表达关系变化 | 场面调度 |
| **场景状态差** | 场景结尾相比开头，在风险、信息、关系或人物处境上发生什么变化 | 场景出口状态 |
| **下游可分组性** | 场景标题、画面字段、声音字段足够稳定，便于 `5-分组` 切分 | slugline 稳定性 |

## 2. 高级工艺规则（High-Level Craft Rules）

### 2.1 画面入场原则

| 规则 | 说明 | 示例 |
|------|------|------|
| **空间优先** | 场景开头优先用空间状态、人物位置或可见动作入场 | 用"教室窗外的光线"而不是"林寂感到紧张"开场 |
| **道具即信息** | 重要规则信息尽量绑定道具、系统画面、黑板、纸张、广播、屏幕或可视化显影 | 黑板中央浮出冷白光点 |
| **身体即情绪** | 角色恐惧不直接写"很害怕"；用呼吸、手、眼神、身体僵硬、回避动作、声线、沉默和空间距离表达 | 后背僵住、喉结滚动 |
| **类型片氛围持续** | 校园怪谈用灯管、窗帘、讲台、走廊、广播、规则文字、集体反应承载持续压力 | 灯管闪烁、窗帘飘动 |
| **对白画面不复述** | 对白画面给出演员表演、对手反应、空间压迫和画面承托，不复述对白内容 | 视线、身体距离、道具停点 |
| **表演任务可执行** | 表演提示写成演员可执行任务：目标、阻碍、策略、外显动作，至少命中两项 | "林寂：把纸条藏起来" + "阻碍：苏红叶的目光" + "策略：压低声音" |
| **潜台词不说出** | 潜台词不写成解释句；用停顿、避视、抢话、反问、手部动作、身体距离和道具转移承托 | 问句后停半拍等对方先开口 |
| **权力调度显性化** | 场面调度服务权力关系和信息差：谁站着、谁坐着、谁占据讲台/门口/黑板/窗边、谁被群体视线压住 | 讲台高出一个台阶 |
| **高段位表演不列条** | 高段位表演和场面调度不能在场景尾部总结式列条；规划结果必须拆入对应对白、动作、画面、道具、声音和反应 | - |
| **沉默有动作余波** | 沉默必须有动作或声音余波；上游没有对白时，不用新增对白替代反应 | 纸张声消失、呼吸声变清晰 |
| **转场服务因果** | 转场要服务因果、压力或信息推进，不为炫技而存在 | 声音桥、物件状态串联 |
| **不替下游写方案** | `2-编剧` 不输出 `分镜明细预设` 字段；需要强调的可拍意图必须转化为基础声画字段 | - |

### 2.2 禁止越权清单

| 禁止越权类型 | 原因 | 正确做法 |
|-------------|------|----------|
| **机位描述** | 属于导演 `3-导演` 创作范围 | 用场面调度、空间关系表达 |
| **景别描述** | 属于导演 `3-导演` 创作范围 | 用画面字段描述可见内容 |
| **镜头运动** | 属于导演 `3-导演` 创作范围 | 用动作节奏、表演节奏表达 |
| **分镜编号** | 属于分镜师或导演 `3-导演` 范围 | 用场景标题和转场衔接 |
| **剪辑节奏** | 属于剪辑师或导演 `3-导演` 范围 | 用声音桥、动作中断、转场处理 |
| **调色方案** | 属于调色师或导演 `3-导演` 范围 | 用光线、色温、氛围描述 |

## 3. 三幕结构适配（Three-Act Structure）

> **来源**： Syd Field, Robert McKee

### 3.1 剧集层面的幕次功能

| 幕次 | 集数位置 | 戏剧功能 | 编剧任务 |
|------|----------|----------|----------|
| **第一幕** | 前 25% | 建置：建立世界、人物、规则、日常 | 场景压缩、信息释放节奏控制 |
| **第二幕前半** | 25%-50% | 上升：冲突加剧、阻碍增加、选择成本上升 | 压力叠加、潜台词深化 |
| **第二幕后半** | 50%-75% | 高潮准备：危机逼近、中点转折 | 道具/规则显影加速 |
| **第三幕** | 后 25% | 解决：高潮对决、代价清算、新平衡 | 场景收缩、情绪出口 |

### 3.2 场景层面的幕次功能

| 场景类型 | 幕次位置 | 质量要求 |
|----------|----------|----------|
| **建置场景** | 第一幕 | 快速建立空间、人物、关系基线；为后续变化提供对比锚点 |
| **压力场景** | 第二幕 | 持续叠加可见压力；每个 beat 必须有升级证据 |
| **转折场景** | 中点/第三幕入口 | 场景出口状态必须有质变；道具/关系/信息必须有一个被不可逆改变 |
| **释放场景** | 幕次结尾 | 压力释放必须有物理/声音/动作承托；不得用旁白总结 |

## 4. 类型片节奏标准（Genre-Specific Pacing）

> **来源**： Genre Theory, Blake Snyder's Save the Cat

### 4.1 类型化场景节奏

| 类型 | 节奏特征 | 场景时长参考 | 画面压力要求 |
|------|----------|--------------|-------------|
| **校园规则怪谈** | 均匀压迫、间歇爆发 | 3-5 分钟/场景 | 灯管/广播/规则文字持续承压 |
| **战斗/动作** | 短促有力、即时反应 | 1-2 分钟/场景 | 速度/力度/空间位移必须精确 |
| **悬疑/推理** | 信息释放控制、停顿制造张力 | 4-6 分钟/场景 | 道具/视线/身体距离承载信息差 |
| **情感/亲密** | 节奏放缓、微表情放大 | 3-5 分钟/场景 | 呼吸/停顿/手部动作承载情绪 |
| **恐怖/惊悚** | 节奏撕裂、声音断点 | 1-3 分钟/场景 | 黑暗/声响/空间封闭承载恐惧 |
| **喜剧** | 节奏明快、反应放大 | 1-2 分钟/场景 | 身体喜剧、道具反转承托笑点 |

### 4.2 节奏控制手段

| 手段 | 应用场景 | 执行字段 |
|------|----------|----------|
| **动作停顿** | 制造张力、等待危机落地 | 角色动作、表演提示 |
| **声音消失** | 强化寂静、压迫感升起 | 音效、音效画面 |
| **重复打断** | 阻止角色完成行动、加剧焦虑 | 对白画面、群像画面 |
| **信息延迟** | 让观众先于角色知道危险 | 旁白、系统画面 + 角色动作错位 |
| **节奏加速** | 危机逼近、行动紧迫 | 短句、动作连击、环境刷新 |

## 5. 场景类型化质量标准（Scene-Type Quality Standards）

### 5.1 建制场景（Establishing Scene）

| 质量维度 | 达标要求 | 常见问题 |
|----------|----------|----------|
| **空间建立** | 明确内/外、场所、日夜、围合边界 | 空间模糊、无法支撑后续调度 |
| **人物基线** | 角色入场有可见动作/位置/道具 | 人物突然出现无铺垫 |
| **规则显影** | 规则/系统信息通过道具/文字/声音出现 | 规则由旁白独白直接告知 |
| **氛围锚定** | 类型感通过光线/声音/空间质感建立 | 氛围依赖旁白描述而非画面 |
| **下游可接** | 场景状态可被 `5-分组` 清晰切分 | 场景结尾无稳定画面状态 |

### 5.2 对话场景（Dialogue Scene）

| 质量维度 | 达标要求 | 常见问题 |
|----------|----------|----------|
| **对白冻结** | 对白逐字保留，不增删改 | 对白被润色或语序调整 |
| **潜台词可见** | 不能说出口的信息通过停顿/视线/道具表达 | 潜台词被写成解释句 |
| **权力动态** | 人物高低、远近、谁先说/谁不接话 | 权力关系由旁白告知 |
| **沉默余波** | 沉默有呼吸/动作/声音消失承托 | 沉默无任何物理证据 |
| **对手反应** | 每段对白后有可见反应 beat | 只写说话者，写完即过 |

### 5.3 压力场景（Pressure Scene）

| 质量维度 | 达标要求 | 常见问题 |
|----------|----------|----------|
| **压力可见** | 规则/威胁/时间压力通过道具/系统/空间承托 | 压力由旁白/内心独白告知 |
| **升级链条** | 每个 beat 有压力升级证据 | 所有 beat 同等级别 |
| **选择成本** | 角色面临的选择有可见代价 | 选择由旁白解释其重要性 |
| **抵抗可见** | 角色通过动作/对白抵抗压力 | 抵抗被省略或跳过 |
| **身体化** | 压力通过呼吸/身体僵硬/回避动作表达 | 心理反应写成抽象内视 |

### 5.4 转折场景（Turning Point Scene）

| 质量维度 | 达标要求 | 常见问题 |
|----------|----------|----------|
| **质变证据** | 场景结尾道具/关系/信息有不可逆改变 | 转折被旁白总结 |
| **物理承托** | 转折通过道具状态/位置变化/系统显影 | 转折只在内心发生 |
| **下行/上行** | 转折方向明确：下行（危机加深）或上行（突破出现） | 转折模糊、两可 |
| **下游锚定** | 转折后场景状态可被下游承接 | 转折后无稳定新状态 |

### 5.5 释放场景（Release Scene）

| 质量维度 | 达标要求 | 常见问题 |
|----------|----------|----------|
| **物理释放** | 压力通过可见动作/声音/空间释放 | 释放由旁白总结 |
| **声音承托** | 寂静消失、新声音出现、环境声变化 | 声音无变化 |
| **身体化** | 释放通过叹气、松手、后退、闭眼表达 | 释放只在心里 |
| **不过度** | 释放有节制，不变成情绪狂欢 | 过度渲染、演员提示过长 |

## 6. 视觉叙事原则（Visual Storytelling Principles）

### 6.1 画面核心（Image Core）

每个重要场景必须有一个**画面核心**——最能承载类型感和情绪的视觉物件或空间状态。

| 类型 | 画面核心候选 |
|------|-------------|
| 校园怪谈 | 黑板规则、课桌编号、窗帘、灯管、走廊尽头 |
| 战斗动作 | 武器轨迹、尘土飞扬、衣摆、水花、血迹扩散 |
| 悬疑推理 | 文字痕迹、指纹、破碎物件、阴影切割 |
| 情感亲密 | 手部接触、目光交汇、距离缩短、道具交换 |
| 恐怖惊悚 | 黑暗边界、闪烁光源、镜像、镜面反射 |

### 6.2 构图原则

| 原则 | 说明 | 违反示例 |
|------|------|----------|
| **可见承托** | 每个抽象信息必须有物理承托 | "规则压力很大"无画面证据 |
| **类型感物质化** | 类型情绪通过可见物承载 | "恐怖氛围"无视觉承托 |
| **层次分离** | 前景/中景/背景有意识分离 | 所有元素挤在中景 |
| **视线引导** | 观众视线被引导到信息焦点 | 视觉焦点模糊 |
| **动静对比** | 动态角色与静态环境形成对比 | 环境与角色同等动态 |

### 6.3 光线叙事

| 光线状态 | 情绪/类型信号 | 执行要求 |
|----------|-------------|----------|
| **顶部直射** | 审讯、压迫、权力 | 光线从上方投下阴影 |
| **侧光** | 悬疑、不确定性、两面性 | 半边脸在阴影中 |
| **底部反光** | 恐怖、威胁、不祥 | 脸下半部被照亮 |
| **柔光** | 亲密、回忆、温暖 | 光线散射、边缘模糊 |
| **硬边光** | 对抗、冲突、紧张 | 光影边界清晰 |
| **闪烁忽明忽暗** | 危险、不稳定、设备失灵 | 灯管或火光闪烁 |

## 7. 声音设计标准（Sound Design Standards）

### 7.1 声音核心（Sound Core）

每个重要场景必须有一个**声音核心**——最能制造节奏、压力或规则感的声音。

| 类型 | 声音核心候选 |
|------|-------------|
| 校园规则怪谈 | 铃声、广播、机械提示、脚步声、翻纸声 |
| 战斗动作 | 金属碰撞、尘土落下、呼吸急促、脚步、断裂声 |
| 悬疑推理 | 钟表滴答、纸张摩擦、静电、耳鸣、心跳 |
| 情感亲密 | 呼吸声、沉默延长、衣料摩擦、环境白噪 |
| 恐怖惊悚 | 寂静断点、低频嗡鸣、远处声响、脚步回声 |

### 7.2 声音设计原则

| 原则 | 说明 | 执行要求 |
|------|------|----------|
| **声音纯度** | 音效字段只写声音本体，不写时间/事件说明 | "叮铃铃——"不是"八点四十分铃声" |
| **声音消失** | 寂静和声音消失是有效声音设计 | 虫声断掉、呼吸声变清晰 |
| **声音来源** | 音效必须标注来源（可见或不可见） | 音效（高跟鞋）："嗒、嗒、嗒" |
| **声音桥接** | 转场用声音桥连接两个场景 | 前场景尾声音与后场景声音衔接 |
| **声音与画面配对** | 声音字段必须与对应画面字段就近配对 | 音效 + 音效画面 |

## 8. 保真与质量的平衡（Fact Fidelity vs. Quality Enhancement）

> **来源**：`script-adaptation-contract.md` §Fidelity Rules

### 8.1 允许的质量增强

| 增强类型 | 具体形式 | 限制 |
|----------|----------|------|
| **画面密度提升** | 把一句小说描述拆成多个可拍字段 | 不改变事实、因果、顺序 |
| **心理投影** | 把心理压力投影为独白、表演提示、表情特写或可见画面承托 | 不得新增心理结论 |
| **氛围物质化** | 把抽象氛围转成空间、光线、声音、群体行为 | 不得新增威胁或危险 |
| **系统信息可视化** | "系统公告"可拆成旁白（系统提示）与系统画面 | 旁白内容必须来自上游 |
| **关系外化** | 为上游已有的关系变化补足表演任务、潜台词行为、场面调度 | 关系结论必须来自上游已有信息 |
| **沉默余波** | 为对白后的沉默、迟疑、反应补足动作或声音余波 | 不得新增对白替代反应 |

### 8.2 禁止的质量增强

| 禁止类型 | 原因 | 正确做法 |
|----------|------|----------|
| 删除"不够电影化"的剧情事实 | 违反 FR-1 事实保真 | 保留并找到可拍承托 |
| 把角色判断改成另一个因果解释 | 违反 FR-1 事实保真 | 保留原判断，用可见证据承托 |
| 为提高冲突新增上游没有的对白或事件 | 违反 FR-1 事实保真、FR-3 对话冻结 | 找到已有冲突的可拍承托 |
| 把好莱坞节奏理解为快速摘要 | 违反 FR-2 信息量保真 | 保持信息量，改进可拍性 |
| 旁白成为默认信息堆放处 | 违反 FR-2 信息量保真 | 优先用画面承托，旁白作为补充 |

## 9. 质量门清单（Quality Gate Checklist）

> **来源**：本规范是 `2-编剧` 的质量达标标准，用于逐集交付前的自检。

### 9.1 基础门（每个场景必须通过）

| # | 检查项 | 达标标准 |
|---|--------|----------|
| 1 | 场景有明确空间、日夜和视觉入口 | slugline 格式正确 |
| 2 | 每段声音都有可拍承托 | 音效 + 音效画面就近配对 |
| 3 | 每个核心道具、规则、系统信息有可视化路径 | 有 `道具特写` / `系统画面` / `规则显影` |
| 4 | 每个重要心理变化有投影证据 | 独白 / 心理反应 / 表演提示 / 可见行为之一 |
| 5 | 对白逐字保真 | 引号内无增删改 |

### 9.2 进阶门（每个关键场景必须通过）
