# CHANGELOG

## 2026-04-28

- 更新 north_star 投影合同：每组第 1 行全局风格必须在原词基础上追加固定词 `不生成字幕，不生成BGM，要生成物理互动音效与环境音。`，并同步模板、review gate、workflow、入口元数据与 validator 检查。
- 新增空间一致性合同：空间锚点作为内部参照系使用，不作为分镜组组头字段输出；逐 `分镜N:` 补入 `站位和位移：` 辅助行，并同步模板、review gate、workflow、入口元数据与 validator 检查。
- 收紧 `站位和位移：` 归属层级：由镜头语言容器级改为逐 `分镜N:` 明细级，validator 按分镜明细检查非空辅助行。
- 收敛空间锚点输出：`空间锚点` 改为 `站位和位移：` 的内部参照系，不再作为分镜组组头字段或提示词独立段落输出；validator 改为禁止输出 `空间锚点：`。
- 收紧 `站位和位移：` 主语规则：必须使用明确角色名或上游已命名的稳定群体称谓，禁止 `画面主体`、`主体`、`人物`、`角色`、`主角` 和代词等模糊指代，并加入 validator 检查。
