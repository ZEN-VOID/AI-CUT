# 06 人物时间线

## 关键节点

| 时间 | 事件 | 对导演方法的影响 |
|---|---|---|
| 1970 | 生于山西汾阳。 | 家乡、县城、边缘地方和返回母题成为长期起点。 |
| 1992 | 报考北京电影学院时开始深入理解第六代语境。 | 独立电影与现实观察意识形成。 |
| 1995 | 成立青年实验电影小组。 | 在学院训练之外开始自我组织影像实践。 |
| 1997 | 毕业于北京电影学院；`小武` 作为首部长片在国际影展获关注。 | “普通人 + 家乡 + 转型社会”方法定型。 |
| 2000 | `站台` 入围威尼斯主竞赛。 | 个人成长与时代精神史缠在一起。 |
| 2002 | `任逍遥` 入围戛纳；更深入进入数字影像实践。 | 青年停滞感、流行音乐和数字机动性进入方法。 |
| 2003 | 成立 Xstream Pictures。 | 从作者延展到青年导演平台建设。 |
| 2004 | `世界` 问世。 | 现实中的超现实、景观复制和全球化问题进入核心。 |
| 2006 | `三峡好人` 获威尼斯金狮。 | 地景变迁、拆迁与时间崩塌成为世界级代表方法。 |
| 2008 | `二十四城记`。 | 纪实/虚构混写方法成熟。 |
| 2010 | 公开发表关于第六代、市场和创作独立性的核心文章。 | 独立性与公共空间协商被系统表达。 |
| 2013 | `天注定` 获戛纳最佳编剧。 | 类型暴力成为扩大现实冲击面的工具。 |
| 2017 | 发起创办平遥国际电影展。 | 从单个导演走向生态建设者。 |
| 2018 | `江湖儿女` 入围戛纳主竞赛。 | 爱情、江湖和时代跨度重新卷合。 |
| 2022 | 《Jia Zhangke on Jia Zhangke》出版。 | 对其创作方法有了更系统的一手对谈文献。 |
| 2024-10-08 | NYFF62 举办贾樟柯 Amos Vogel Lecture。 | 《风流一代》被视作其二十多年影像实践的集中回望。 |
| 2026-02-06 | 亚洲电影大奖学院公布 2026-03-15 贾樟柯大师班。 | 最新公开活动继续强调普通人如何成为时代记忆。 |

## 最新动态（截至 2026-04-15）

- 2024-10-08，Film at Lincoln Center 为贾樟柯举行 Amos Vogel Lecture，配合《风流一代》在 NYFF62 主单元放映。
- 2025-01-25，`Le Monde` 对谈中，贾樟柯系统解释《风流一代》的非线性结构、疫情转折与 AI 语境。
- 2026-02-06，亚洲电影大奖学院公布 2026-03-15 的大师班活动，标题直接概括其方法论。
