# 05-资产提取

BYKJ `05-资产提取` 从 `02-剧本处理` 的格式化剧本全本中提取上游资产主体，并路由到三个子技能完成提取后的设计规格 JSON：

```text
05-资产提取/
├── 角色提取/
├── 场景提取/
└── 道具提取/
```

默认输入：

```text
output/[项目名]/02-剧本处理/
```

默认输出：

```text
output/[项目名]/05-资产提取/
```

父级只做路由与汇总；具体提取和设计规则以三个子技能的 `SKILL.md + CONTEXT.md` 为准。canonical 输出为 JSON，包括 `资产清单.json`、各子技能 JSON、JSON 报告和 `manifest.json`。
