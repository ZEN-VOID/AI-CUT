# 02 长对话与即兴回答

## TIME 2004 访谈

- Cannes 版本并非最终完成版；王家卫明确说当时声音、音乐、CG 都还没完全到位，后续又调整了场景顺序与结构。
- 他把结构变化归因为两点：一是赶 Cannes 截止；二是影片本身的结构一直在变，只要抽掉一场，很多场都必须重排。
- 《2046》最早与《花样年华》是两个项目，后来在拍酒店房间时被记忆触发，才接上同一个数字和同一个情感世界。
- 他强调未来段落不是写实科幻，而是 1966 视角下的未来想象。
- 五年制作周期不是浪漫传说，而是“极难维持焦点和信念”的现实组织问题。
- 张子怡的表演不是凭空即兴，而是通过大量时代作业、服装体感和动作训练进入角色。

## Moving Image 对谈

- 王家卫反复澄清：所谓“没有剧本”常是误读，更准确的是“没有不能改的终稿”。
- 他讲《花样年华》与《2046》时，强调旧材料、旧人物和新写作会互相污染，电影会在拍摄与剪辑中继续长。
- 对演员的使用不是让他们背概念，而是让角色适配演员的面貌、速度、沉默和时代气味。

## 可转化为 Skill 的判断规则

1. 《2046》式结构允许变，但变动必须围绕同一记忆核，而不是随便拼贴。
2. 如果你要借未来、梦境或虚构文本，目的应是绕开直说，把旧伤移到更安全的壳层里。
3. 角色的时代质感要靠服装、动作、体感和礼貌方式入戏，不靠标签台词。
4. 若项目本身跨年长拍，必须把版本、结构调整和贡献记录写清楚；《2046》的完成史本身就是方法提醒。
