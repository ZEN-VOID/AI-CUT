# Shared Prompt Principles

本文件只记录 `C.主体参照` 对共享 `图生视频` 句法原则的本地承接方式。

## Shared Source

- 跨视频叶子的共享原则以 `.agents/skills/aigc/6-Video/_shared/image-to-video-prompt-principles.md` 为主。
- 本文件只保存主体参照方向的局部 specialization，不改写父级共享原则。

## Subject Reference Local Rules

- 视频 prompt 应先让主要主体可识别，再让镜头运动、空间与氛围服务主体连续性。
- 当存在角色和服装资产时，主体描述优先用“角色身份 + 当前服装 + 可见动作”组织，不把服装拆成孤立装饰词。
- 当存在道具资产时，道具应绑定到角色动作或场景用途，不单独堆成资产清单。
- 当存在场景资产时，场景作为环境锚点进入组级设计块和镜级空间句，不替代镜头事实。
- 不得为了适配参照图而改写上游剧情、主体身份或动作事实。
