# Type Map

## 类型包加载边界

- 每次调用本技能时，必须依据本文件识别并加载同目录 `types/` 中选中的类型包（单选或多选）。
- `types/` 中命中的类型包作为固定上下文加载；`knowledge-base/` 只作为按需检索、切片或向量召回的知识库，不替代类型包。


| signal | route |
| --- | --- |
| planning 缺去向 | `back_to_source_contract / 2-卷章` |
| 正文缺汇聚动作 | `back_to_drafting_nodes` |
| 支流有明确保留开放 | 非 blocking，记录后续追踪 |
