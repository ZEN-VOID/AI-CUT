# 角色表现 模块说明

## 定位

- 本分支负责把人物压力、行为目的和关系攻守汇成 `character_patch`。
- 它拥有“人物怎么显露自己”的判断权，不拥有空间调度和镜头组织的判断权。

## 使用方法

- 当分镜组里有人物行动、对话或关系张力时启用本分支。
- 先问清“谁在承压、谁在主动、关系怎么变”，再分别下钻 `内心戏 / 动作戏 / 对手戏`。
- 汇流时优先保留角色个性、行为目的和关系攻守，不要把三条叶子都写成并列摘要。

## 失真与修正

- 若文本只剩“紧张、愤怒、悲伤”之类情绪标签，说明 `内心戏` 没有落成可见信号。
- 若人物写得很满但空间开始混乱，立刻把方位和动作连续性交还 `运动表现`。
- 若关系判断变成解释说明，删掉结论句，改回逼近、回避、失手或对视等可演动作。
