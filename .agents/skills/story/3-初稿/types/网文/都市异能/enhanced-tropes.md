# 都市异能题材增强包

> 补充维度：套路拆解 / 情节模式 / 人物原型 / 反套路策略 / 质量过滤

## 一、核心套路与行规

### 1.1 都市异能的核心张力
- **日常与超凡的碰撞**：异能者生活在普通人的世界，隐藏身份/能力/秘密是持续的张力来源
- **规则的灰色地带**：异能社会的规则与正常社会的法律/道德/秩序冲突
- **双重生活**：白天的普通人身份与夜晚/暗面的异能者身份

### 1.2 异能体系设计
| 设计维度 | 关键问题 | 写作要点 |
|----------|----------|----------|
| 来源 | 异能从哪来？（变异/传承/实验/觉醒/契约） | 来源可以成为核心悬念 |
| 分类 | 异能如何分类？（元素/精神/肉体/规则/领域） | 分类有逻辑且内洽 |
| 限制 | 每种异能的代价/限制是什么？ | 无限制的能力=无趣 |
| 社会 | 异能者如何组织？（自由/组织/官方/地下） | 组织有规则、历史和政治 |
| 隐藏 | 如何向普通人隐藏异能世界？ | 隐藏机制本身就是故事 |

### 1.3 常见套路陷阱
- 主角能力最强/最特殊→其他异能者沦为陪衬
- "扮猪吃虎"过度使用——每次都被看不起然后打脸
- 异能世界与普通人世界割裂——应该交织而非平行
- 能力体系越写越大——前期铺垫的规则后期随意破坏

## 二、情节模式与节奏规则

### 2.1 都市异能三幕结构
```
第一幕(觉醒)：发现异能→初次使用→遭遇第一个其他异能者→了解世界的另一面
第二幕(卷入)：被卷入异能事件→加入/对抗组织→保护普通人/对抗反派→能力成长
第三幕(面对)：更大的阴谋浮出→终极对抗→身份/命运的选择
```

### 2.2 节奏铁律
- 每卷至少一次在公众场合的异能事件（暴露风险）
- 能力成长有渐进（不是跳级——一级一级来）
- 日常与异能场景的切换比例≈4:6（太多日常失去张力，太多异能失去日常的锚）

### 2.3 爽点设计
| 爽点类型 | 都市异能实现 |
|----------|-------------|
| 能力觉醒/突破 | 关键战斗中突破极限 |
| 身份揭露 | 隐藏的能力被发现（友军/敌人/爱人） |
| 以弱胜强 | 利用规则/环境/智慧战胜强敌 |
| 守护 | 用能力保护重要的人/地方 |
| 揭秘 | 发现隐藏的真相/组织/历史 |

## 三、反套路策略

### 3.1 反"龙傲天"
- 最强的能力不是战斗力——信息收集/治愈/防护/预知可能比单纯的攻击更有叙事价值
- 能力的代价比能力本身更有趣——使用能力失去记忆/寿命/情感/人性
- 顶尖异能者不一定是最能打的——战略/关系/信息控制力才是真正实力

### 3.2 反"日常只是背景"
- 普通人的身份不是装饰——家庭/工作/朋友是真实的情感锚点
- 异能生活中的普通时刻——异能者也关心房租、食物、和在意的人的关系
- 双重身份的痛苦——不能在普通人面前暴露、对亲近的人说谎

### 3.3 反"异能=暴力"
- 异能的应用远超打架——治愈医生、寻人侦探、天气预报员、造假鉴定师
- 异能社会的日常——异能者有集市/法庭/学校/婚恋/传承
- 社会褶皱——异能者阶层分化（注册vs野生、世家vs散人、强能力vs弱能力）

## 四、质量过滤规则

### 4.1 能力体系自洽检查
- [ ] 所有异能展示的力量水平前后一致
- [ ] 能力突破有铺垫和代价
- [ ] 不同能力之间有自然的克制/配合关系
- [ ] 没有"剧情需要就解锁新能力"

### 4.2 日常/异能平衡检查
- [ ] 普通人身份有实质性剧情（不只是背景设定）
- [ ] 没有连续3章以上完全无视普通人世界
- [ ] 普通人角色的存在影响主角的选择

### 4.3 失败码
| 失败码 | 症状 | 返工目标 |
|--------|------|----------|
| `FAIL-URB-GOD` | 主角能力过强/唯一特殊 | 增加限制/代价/更强的对手 |
| `FAIL-URB-SPLIT` | 异能世界与日常割裂 | 增加交织场景和双重身份的冲突 |
| `FAIL-URB-SCALE` | 能力体系前后矛盾/失控 | 回源到能力体系设计 |
