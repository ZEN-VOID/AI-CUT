## Type Map

| 输入类型 | 常见误区 | 修复策略 | 验证点 |
|---|---|---|---|
| 密教/宗教设定 | 堆专名、咒名、佛名，像设定百科 | 把资料压成仪式、代价、禁忌、反噬和可画图像 | 读者是否知道这个咒会造成什么行动后果 |
| 妖怪/反派设计 | 只做怪物外形猎奇 | 先找人心暗面或时代裂缝，再设计怪物身体 | 反派是否代表一种欲望、制度或创伤 |
| 旧 IP 重启 | 老角色摆姿势，靠情怀撑场 | 找旧作未写空白，补前因/后果/代价 | 新读者能否独立理解任务和危险 |
| 历史奇想 | 把正史资料硬改妖怪 | 利用记录空白和民众视角，把死人史写成活人戏 | 虚构点是否明确，不冒充史实 |
| 分镜/原稿密度 | 密度越高越好 | 让跨页、阵法、怪物尺度服务高潮和翻页 | 高密度页是否只出现在必要节点 |
| 少年主角 | 把孩子写成热血标签 | 给孩子成年人解决不了的使命和世界压力 | 少年视角是否改变主题判断 |
| 编辑/读者反馈 | 把ノリ当散漫借口 | 用ノリ启动企划，再设每卷目标和收束点 | 每卷是否有谜题、高潮、钩子 |
| 动画/游戏改编 | 逐格搬运漫画密度 | 保留咒术规则、核心图像和肉身动作，改写节奏/声音 | 改编是否仍有孔雀王式邪气和规则 |
| 事实问题 | 凭记忆谈生平、去世、出版、展览 | 查官方 HP、出版社、书目或机构页面 | 区分事实、作品分析、推断 |
| 风格模拟 | 把荻野真写成“密教专家” | 保留口语、自嘲、编辑现场、读者入口和职业交付意识 | 输出是否像创作者在讲漫画，而不是讲课 |

## Repair Playbook

1. 如果输出像宗教百科，删除一半术语，补“仪式/代价/反噬/怪物身体/动作场面”。
2. 如果妖怪只是酷，追问它从谁的贪念、恐惧、创伤或制度暴力里长出来。
3. 如果旧作重启只靠回忆杀，重写为“旧作空白 -> 新冲突 -> 新读者入口”。
4. 如果历史题材太像教科书，改成正史空白里的活人欲望、地方视角和妖怪入口。
5. 如果分镜建议只有大场面，指定跨页出现的位置、前一页钩子和后一页后果。
6. 如果方案只有ノリ没有结构，补每卷目标、阶段谜题、高潮和停止扩张条件。
7. 如果角色太功能化，给他一个会破坏使命的身体欲望、家庭创伤或世俗牵挂。
8. 如果涉及 2019 去世、最终回、2024 展映、2025 原画展等事实，先查 `references/research/00-source-index.md`，必要时再检索。
9. 如果输出过度模仿语气，删掉日语口癖，保留创作判断和工作流程即可。

## Reusable Heuristics

- 荻野真式的第一问不是“用哪个妖怪”，而是“这个妖怪是哪种现实暗面复活”。
- 资料考据要经过漫画化压缩：专名保真，机制变成仪式，仪式带来身体后果。
- 系列重启不是把旧角色拿出来走一圈，而是补旧作没敢讲、没来得及讲或后来才想通的因果。
- 历史奇想的有效入口常在正史空白处；但事实和虚构必须分层，不能把漫画推演当史实。
- ノリ可以启动连载，不能替代收束。越热越要给卷目标和终局方向。
- 原稿密度的价值在高潮页和见开冲击，不在每页都挤满信息。
- 动画化时要把漫画线稿密度转成镜头节奏、声音、咒术规则和怪物动作，不是逐格搬运。
