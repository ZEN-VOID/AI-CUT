# 寺山修司时间线

调研截止：2026-04-15

## 说明

此页只记录可核对的时间线事实，优先采用寺山修司记念馆、青森县立美术馆、Harvard Film Archive、Japan Society、MoMA 等官方或机构来源。对生平关键节点、主要作品、海外活动、死后纪念与近年遗产动态做了交叉核对。

## 年表

| 时间 | 事件 | 说明 |
|---|---|---|
| 1935-12-10 | 出生 | 寺山八郎、寺山はつ之长男，生于青森县弘前市紺屋町。户籍上记为 1936-01-10。 |
| 1945 | 战后迁居三泽 | 青森大空袭后被迫迁出，寄居三泽站前伯父经营的寺山食堂二楼。 |
| 1948 | 入读中学 | 入学三泽古间木中学校，开始显露写作与编辑能力。 |
| 1949 | 回到青森 | 被青森市母方叔父夫妇收养，生活重心转向青森市。 |
| 1951 | 青森高校 | 入学青森高校，参加新闻部、文艺部，进入更系统的创作训练。 |
| 1954 | 早稻田大学 / 短歌崭露头角 | 入学早稻田大学教育学部国语国文学科；以「チェホフ祭」获得《短歌研究》第二回新人奖，确立短歌声名。 |
| 1955 | 住院与早期戏剧 | 因肾病长期住院；写出处女剧作《忘れた領分》，并在早稻田上演。 |
| 1957 | 第一部作品集 | 作品集《われに五月を》刊行。 |
| 1958 | 第一歌集 / 广播剧起步 | 《空には本》刊行；在谷川俊太郎建议下开始写广播剧，《ジオノ》播出。 |
| 1959 | 广播剧获奖 | 广播剧《中村一郎》获得民放会长奖。 |
| 1960 | 戏剧与电影脚本 | 《血は立ったまま眠っている》在剧团四季上演；为篠田正浩《乾いた湖》写剧本。 |
| 1961 | 成为职业脚本家 | 在广播与电视领域迅速走红，形成跨媒介工作方式。 |
| 1962 | 结婚 | 与松竹女演员九条映子结婚。 |
| 1963 | 《家出のすすめ》 | 以《現代の青春論》名义整理《家出のすすめ》，转向更强烈的青年论述。 |
| 1964 | 放送诗剧获奖 / 《あゝ、荒野》连载开始 | NHK《山姥》获意大利奖大奖，CBC《大礼服》获艺术祭奖励奖；同年 3 月起在《現代の眼》连载《あゝ、荒野》。 |
| 1966 | 《あゝ、荒野》单行本 | 《あゝ、荒野》结集出版，成为其唯一长篇小说。 |
| 1967 | 天井棧敷成立 | 与横尾忠则、东由多加、九条映子等成立演剧实验室「天井棧敷」。 |
| 1968 | 继续扩张 | 开始以马主身份参与赛马世界，持续将兴趣扩展到戏剧外部。 |
| 1969 | 天井棧敷馆与地下小剧场 | 涩谷天井棧敷馆及地下小剧场落成；词作《時には母のない子のように》大热。 |
| 1970 | 力石徹葬礼 / 公众事件化 | 以「喪主」身份主持《あしたのジョー》力石徹葬礼，成为高度戏剧化的公共事件。 |
| 1971 | 电影导演成型 | 《書を捨てよ町へ出よう》之后，电影创作与剧场工作并行；《Throw Away Your Books, Rally in the Streets》后来成为国际代表作。 |
| 1972 | 海外大型舞台 | 在慕尼黑奥运会纪念艺术节上演野外剧《走れメロス》。 |
| 1974 | 《田園に死す》 | 长篇电影第二作《田園に死す》脚本、导演完成，并获艺术祭奖励新人奖。 |
| 1975 | 市街剧与欧洲活动 | 《ノック》在杉并区街头演出时遭警方介入；《疫病流行記》在荷兰首演，之后展开欧洲巡演。 |
| 1976 | 天井棧敷移转 / 伊朗演出 | 天井棧敷馆移至元麻布；应邀参加伊朗设拉子、波斯波利斯艺术节，上演《阿呆船》。 |
| 1977 | 国际影展与舞台并进 | 《ボクサー》上映；《中国の不思議な役人》上演，继续强化戏剧与电影双轨。 |
| 1979 | 肝病入院 | 因肝硬变住院，并继续进行公共工作与创作。 |
| 1980 | 社会评价定着 | 在《シティロード》读者票选中连续两年获演剧部门第一。 |
| 1981 | 病情恶化与《路地》取材 | 肝硬变恶化再度住院，仍持续进行书写与取材。 |
| 1982 | 《さらば箱舟》与最后海外公演 | 在冲绳拍摄《さらば箱舟》；10 月于巴黎国立夏约宫剧场上演《奴婢訓》，这是其最后一次海外公演。 |
| 1982-1983 | 与谷川俊太郎交换 Video Letter | 进入生命最后阶段，仍保持与同代创作者的对话。 |
| 1983-05-04 | 去世 | 以《墓場まで何マイル？》为绝笔，因肝硬变、腹膜炎并发败血症去世，享年 47 岁。 |

## 死后纪念与遗产动态

| 时间 | 动态 | 说明 |
|---|---|---|
| 1984 | 《Farewell to the Ark》上映 | 《さらば箱舟》于 1984 年公开上映，作为其去世后的代表作继续流通。 |
| 1997-07 | 三泽市寺山修司记念馆开馆 | 由遗族捐赠遗物保存公开而建，成为其故乡三泽的核心纪念设施。 |
| 2017-11 | Harvard Film Archive 回顾展 | HFA 举办 “Shuji Terayama, Emperor of the Underground” 回顾展，按电影、短片与 Video Letter 体系重审其跨媒介影响。 |
| 2023 | 没后 40 年官方纪念年 | 记念馆/TERAYAMA WORLD 推出没后 40 年认定事业，包含演出、展览、放映、出版等多条线。 |
| 2025 | 生诞 90 年官方纪念年 | 官方认定事业明确将 2025 年作为生诞 90 年节点，集中推出演剧、展览、出版、电影节、音乐会等活动。 |
| 2025-01-15 至 2025-01-18 | Japan Society 北美首演 | 《Shuji Terayama’s Duke Bluebeard’s Castle》在纽约 Japan Society 上演，作为北美首演的当代重构，显示其剧作仍在英语语境中持续被再生产。 |
| 2025-04-01 至 2026-03-21 | 弘前市立郷土文学馆企画展 | 第49回企画展《生誕90年 寺山修司－放たれた歌》持续展出，聚焦其第一歌集《空には本》和现代短歌影响。 |
| 2025-07-18 至 2026-05-31 | 三泽市记念馆《あゝ、荒野》原画展 | 以生诞 90 年为名，围绕《あゝ、荒野》原作与新连载漫画原稿展开两期展览；2025-08-12 还有夜间开馆与特设灯光活动。 |
| 2026-01-08 至 2026-01-25 | 岡山「ジャパン・アヴァンギャルド」展 | 官方认定事业中的联动展，把天井棧敷与 1960-70 年代前卫剧团的视觉史重新展示给当代观众。 |

## 信息缺口

1. 早期作品的首次发表日并非每一项都能在同一官方年表中精确到日，需要在专题作品目录里再补一轮核对。
2. 广播剧、电视剧、杂志连载、舞台改稿之间存在交叉版本，现阶段只记录了官方年表中最稳定的节点。
3. 海外演出与海外放映的完整巡回路径仍可继续细化，尤其是 1969-1977 年间各地城市、主办方与具体剧目对应关系。
4. 死后纪念部分优先记录了可验证的机构级活动；若要做更细的“遗产史”，还需要继续补齐出版社复刊、大学讲座、馆藏入藏与国际影展的年际链条。
