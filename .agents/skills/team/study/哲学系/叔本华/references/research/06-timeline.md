# 06 时间线

## 关键年表

| 时间 | 事件 | 对思维的影响 |
|---|---|---|
| 1788-02-22 | 生于但泽（Danzig） | 商人家庭、国际化早年经历与后来的世界性视角有关 |
| 1793 | 因普鲁士接管但泽，全家迁至汉堡 | 早期就把政治变迁与私人命运联系起来 |
| 1803-1804 | 随父母游历欧洲 | 直接见到贫困与阶层差异，悲观底色提前形成 |
| 1805 | 父亲去世，通常被视为疑似自杀 | 人生脚本断裂，进一步转向内省与哲学 |
| 1807-1809 | 在 Gotha / Weimar 补足学术准备 | 从商业线转向学术线 |
| 1809 | 入哥廷根大学学医 | 接触自然科学，为后来自然哲学与动机解释提供素材 |
| 1811-1813 | 转入哲学，赴柏林听费希特、施莱尔马赫 | 形成对当时学院哲学的反感 |
| 1813 | 完成《充足理由律的四重根》 | 奠定分析世界的“找根据”框架 |
| 1813-1814 | 与歌德来往；经 Friedrich Majer 接触印度思想 | Plato + Kant + Upanishads 的三重谱系开始成形 |
| 1816 | 完成《论视觉与色彩》 | 显示其愿意在已知理论上做对抗性修正 |
| 1818/1819 | 出版《作为意志和表象的世界》初版 | 叔本华系统首次成形，但几乎未获重视 |
| 1820 | 获准在柏林大学授课，并与黑格尔撞档 | 公开挑战学院正统，结果惨败 |
| 1821-1826 | 与 Caroline Luise Marguet 诉讼 | 现实人格中的粗暴与哲学中的同情形成硬冲突 |
| 1831 | 因霍乱离开柏林，定居法兰克福 | 退出学院主场，转入长期独立写作期 |
| 1836 | 出版《论自然中的意志》 | 试图让自然科学为意志论背书 |
| 1839 | 《论意志自由》获挪威皇家科学院奖 | 首次获得公开学术认可 |
| 1840/1841 | 《论道德基础》参评丹麦学院未获奖，后与前文合刊 | 进一步强化其反义务论伦理立场 |
| 1844 | 《作为意志和表象的世界》第二版大扩充 | 系统更完整，但仍未立刻爆红 |
| 1851 | 《Parerga and Paralipomena》出版 | 进入更广泛公共读者视野 |
| 1853 | John Oxenford 评论带来英国与德语世界关注 | 叔本华迟来的成名节点 |
| 1859 | 主著第三版 | 继续修补而非改宗 |
| 1860-09-21 | 卒于法兰克福 | 生前晚成名，死后影响剧烈扩散 |

## 后世回响（精确到版本而非“最新动态”）

- 2010：Cambridge 出版 `The World as Will and Representation` 第一卷新译本。
- 2018：Cambridge 出版第二卷新译本。
- 2020：SEP 参考书目已收录 `The Oxford Handbook of Schopenhauer` 等较新的综合研究。
- 2026-02-18：Britannica 条目仍以“悲观主义哲学家”与“意志形而上学”作为主入口。

说明：

- 叔本华为历史人物，不存在“最近 12 个月本人动态”。
- 因而时间线的“近期”应理解为后世译本、研究与影响延续，而非本人活动。
