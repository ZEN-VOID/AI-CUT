# 06 人物时间线

| 时间 | 节点 | 思维影响 |
|---|---|---|
| 1967 | 出生于加拿大魁北克 | 魁北克、法语文化、宽阔地景与北方自然感成为后续表达底色 |
| 1990s 初 | 在加拿大国家电影局相关环境起步 | 形成短片、纪录与公共电影传统中的工艺训练 |
| 1994 | `REW FFWD` | 早期已经关注陌生空间中的观看与迷失 |
| 1998 | `Un 32 août sur Terre` 入选 Cannes Un Certain Regard | 第一部长片即带梦境、存在危机和地景感 |
| 2000 | `Maelström` | 超现实叙事和罪责主题延续 |
| 2008 | `Next Floor` 获 Cannes Critics' Week First Prize | 短片形式中形成荒诞、秩序崩塌和视觉寓言 |
| 2009 | `Polytechnique` | 处理真实暴力与创伤，奠定克制、伦理和伤痛边界 |
| 2010 | `Incendies` | 家族秘密、战争创伤、身份反转；获得奥斯卡外语片提名 |
| 2013 | `Prisoners` | 进入英语工业，继续处理暴力、父权、道德困境 |
| 2013 | `Enemy` | 身份分裂、城市压迫、梦魇结构 |
| 2015 | `Sicario` 入选 Cannes Competition | 毒品战争、边境、国家暴力和道德浑浊 |
| 2016 | `Arrival` | 科幻转向语言、悲伤、时间和沟通伦理；获奥斯卡最佳导演提名 |
| 2017 | `Blade Runner 2049` | 经典科幻续作、身份、记忆、巨型未来地景 |
| 2018 | Cannes 主竞赛评委 | 从参赛导演进入国际电影制度核心观察位置 |
| 2021 | `Dune: Part One` | 长期梦想项目落地；大格式、自然光、世界建造与神话政治 |
| 2024 | `Dune: Part Two` | 视觉语言扩展、IMAX 全片思考、Fremen 视角、Paul 神话危险 |
| 2025-06 | Amazon MGM 宣布其执导下一部 James Bond | 从 `Dune` 转向 Bond，进入另一项传统压力极高的 IP |
| 2026-04-14 | AP 报道 WB 在 CinemaCon 展示 `Dune: Part Three` 前七分钟 | 第三部公开定位为更强烈、更情感化的 thriller；Dune 收束进入宣传前线 |

## 最新动态（截至 2026-04-15）

- `Dune: Part Three` 已进入公开物料阶段；AP 报道 CinemaCon 展示前七分钟，Villeneuve 将其定位为 thriller，且更 intense / emotional。
- Legendary 页面列出 `Dune: Part Three` 原始发行日期为 2026-12-18，发行方 Warner Bros.。
- Amazon MGM 已宣布 Villeneuve 执导下一部 James Bond；官方尚未在已核对来源中给出最终片名、演员和上映日期。
