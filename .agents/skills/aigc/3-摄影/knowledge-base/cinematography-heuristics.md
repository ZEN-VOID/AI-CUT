# Cinematography Heuristics

本文件保存 `3-摄影` 的可复用摄影经验。

## Project-Derived Heuristics: 诡校-测试版

- 封闭教室是主视觉牢笼：课桌行列、黑窗帘、讲台、黑板、前门构成固定权力几何。
- 苏红叶的视觉母题是黑、白、红：白衬衫、黑裙、红高跟鞋、红珠和暗红瞳孔应形成稀缺而刺眼的危险信号。
- 林寂的视觉母题是冷白秩序中的金色异质信息：真理之眼、隐藏规则、小字显影要区别于系统冷白字。
- 日光灯不是普通照明，而是审讯灯、制度灯和病理灯；它应该压平皮肤、制造眼底白线和无处可逃的顶光。
- 黑板是“规则显影屏”，不是背景板；文字出现时应给它可读性、波纹质感和角色瞳孔反光。
- 高跟鞋、教鞭、铃声可以作为声音先行转场的主材料，但不能滥用到每个段落。

## Shot Craft Heuristics

- 先决定观众应该看哪里，再决定镜头怎么炫；炫技必须藏在注意力转交里。
- 对话画面的摄影重点通常不是“谁在说”，而是“这句话落到谁身上、谁被规则压住”。
- 规则文字显影应至少考虑三层：文字本身可读、角色眼中反光、旁人是否不可见。
- 群像不等于平铺人群；要选一个传播路径，如一个人闭嘴、旁边人跟着闭嘴、全班静止。
- 道具特写要让材质和危险可见：红苹果的光滑、咬痕、红色液体、滚动轨迹都比“诡异”二字更有效。
- 表演提示的镜头语言应落在身体证据：肩膀、喉结、眼神停顿、手指、呼吸、汗、校服背部暗色。
- `分镜N` 最稳的执行骨架是：景别与景深先定观看距离，镜头视角定权力关系，镜头类型定空间质感，运镜速度定心理时间。
- `分镜N` 不应停在执行骨架，要继续写出变化：从什么起，经过什么运动，速度如何变，最后落到什么信息。
- `分镜N` 也不能只看自己；写之前先在内部回看临近至少前 3 个画面单位，确认当前镜头从哪里接入、如何延续空间轴线、最后把注意力交给哪里，但输出不要机械展示这段回顾。
- `分镜N` 还要有呼吸：低信息句收敛，关键揭示发散，强表演留停顿，惊吓和规则入侵才允许断裂式爆发。

## Repair Heuristics

- 如果镜头语言读起来像影评，改成现场执行指令。
- 如果镜头语言读起来像图像提示词，补上时间性、切换点和镜头运动。
- 如果镜头语言读起来像导演口号，补构图锚点、运动轨迹、光源或色彩动机。
- 如果镜头语言读起来像只有“推近、特写、压迫”，补景深、视角、镜头类型和运镜速度，让下游知道怎么拍。
- 如果镜头语言读起来像静态参数表，改成“从 A 到 B”的动态句，并让上一注意力点自然滑向下一注意力点。
- 如果镜头语言读起来像每条都在重开一部电影，先在内部补连续性判断；输出只在需要解释跳变时短写承接入口。
- 如果镜头语言读起来像每条都同等铺张，给每条补 `rhythm_profile`，让过场收敛、重点发散。
- 如果每个画面句子都是三分镜，说明节拍判断失效；回到 beat trigger matrix 重判。
