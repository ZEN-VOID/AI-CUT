# 05 - 决策记录与行动逻辑

调研对象：梁朝伟（Tony Leung Chiu-wai）  
调研时间：2026-04-15

## 关键决策线索

1. **从电视训练进入电影长期主义**
   - 早期 TVB 训练和电视曝光建立了职业基本功，但他没有停留在电视明星路径。
   - 行动逻辑：把曝光转化为更高难度角色，而不是只维持人气。

2. **反复与强作者导演合作**
   - 王家卫、侯孝贤、李安、吴宇森、刘伟强/麦兆辉、张艺谋等合作路线显示，他愿意进入不同的导演系统。
   - 行动逻辑：演员不是固定技法输出者，而是进入导演语言。

3. **接受高风险心理角色**
   - 《花样年华》《无间道》《色，戒》《一代宗师》都需要长时间压抑、身份撕裂或高心理暴露。
   - 行动逻辑：不靠外部爆发争取注意，而靠内部压力推动观众靠近。

4. **进入国际商业片但保持低噪表演**
   - 《尚气与十环传奇》让他进入全球主流商业片，但角色文武仍保留伤、执念和父亲身份的复杂性。
   - 行动逻辑：类型片可以大，人物内核不能扁。

5. **晚近继续接受跨国作者项目**
   - 2025 年后公开资料显示其参与伊尔蒂科·茵叶蒂《Silent Friend》等国际项目。
   - 行动逻辑：晚期不是只消费荣誉，而是继续进入新语言、新工业和新表演环境。

## 决策启发式候选

| 决策规则 | 案例证据 | 可转成 Skill 规则 |
|---|---|---|
| 有导演语言，才有表演大小 | 王家卫/侯孝贤/李安路径差异 | 先读镜头，再定情绪强度 |
| 类型片也不能丢掉秘密 | 《无间道》《尚气》《金手指》 | 商业角色先找不能说的东西 |
| 奖项不是方法 | 戛纳/威尼斯/HKFA 荣誉很多，但表演仍靠细节 | 不用荣誉替代动作分析 |
| 高风险戏先做边界 | 《色，戒》等心理与身体暴露 | 亲密/暴力/羞辱场景先写安全和排练 |
