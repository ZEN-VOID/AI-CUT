# 金宇澄小说创作知识库

## 定位

这不是金宇澄年表，也不是《繁花》剧情提要，而是给 [金宇澄 Agent](/Volumes/AIGC/AIGC-FILM/.codex/agents/小说组/金宇澄.md) 提供判断底座的最小小说创作手册。

## 目标

- 为“《繁花》原著向”的小说、场景、对白和群像重写提供可追溯依据
- 把“繁花感”从电视剧式繁华表象，压回闲谈推进、改良沪语、块状结构、城市细部和不响留白这些可执行维度
- 在没有全文、批注本和作者私人手稿的前提下，先建立一套克制、可靠、可扩展的基础知识底座

## 当前范围

- 金宇澄的小说方法与《繁花》原著的叙事倾向
- 改良沪语与可读性边界
- 不写透心理、依赖上下文与对话成立人物的策略
- 城市细部、器物、地名、称呼对场景温度的作用
- 原著小说与电视剧改编之间的边界说明

## 默认适用任务

- 小说大纲或章节的原著向校正
- 场景白描与生活纹理补写
- 对白去说明化
- 群像关系梳理
- 原著改编前的气质防漂移审阅

## 暂不覆盖

- 《繁花》逐章逐段考据
- 金宇澄全部作品的完整文献整理
- 电视剧版《繁花》的视听美学方法
- 生平、版本学、版权与出版史的全量档案

## 使用原则

- 默认先服务创作任务，不先堆背景知识
- 默认先抽取“这条材料如何支撑本次小说判断”
- 若某条判断无法回溯到现有材料，就降级为风格猜测，不写成硬依据
- 默认以原著小说为准，电视剧《繁花》只作为边界提醒，不作为本知识库的正向真源
