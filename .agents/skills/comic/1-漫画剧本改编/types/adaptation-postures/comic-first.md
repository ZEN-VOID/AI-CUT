# Comic-First 改编姿态

## Match Signals

- 虚构故事、网文、剧情摘要、用户要求更漫画、更刺激、更抓人、更适合九刀流。

## Fixed Context

- 优先保住 `人物关系核 / 情绪核 / 卖点核 / 关键代价`。
- 允许换序、并戏、删过桥解释、前置名场面、放大关系失衡、提高视觉奇观密度。
- 当解释完备度与页势能冲突时，优先让读者停住、翻页、被炸到，再把解释拆回后续场景。
- 漫画化增强不等于失控胡改；角色动机、关系位和代价链不能断。

## Review Gate

- 前段有抓停点，中段有抬升点，组末有急停点。
- 重排后仍能追溯到原素材的核心卖点。
