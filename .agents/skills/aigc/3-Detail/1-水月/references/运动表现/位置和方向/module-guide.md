# 位置和方向 模块说明

## 定位

- 本叶子负责把人物站位、朝向、移动路径和距离变化说清楚。
- 它不负责搭建完整空间世界观，只负责主动作所需的最小方位真相。

## 具体创作方法

- 每次优先回答三件事：人物此刻相对什么锚点、动作朝哪个方向展开、人与目标的距离如何变化。
- 锚点要选读者最容易抓住的主参考系，例如门口、桌边、窗前、走廊尽头、对手身前，而不是临时再造一套坐标系统。
- 写法上优先用“靠近 / 退开 / 侧身让出 / 贴近 / 回身望向 / 横切过去”这类可视化动词，而不是抽象描述空间关系。
- 若一个动作里同时包含位移和转向，先写位移主向量，再补转向；不要反过来把朝向写成主句、位移写成附带信息。
- 当多个人同时存在时，只锁与主动作最相关的那一组相对位置，其余人物不必一并编组说明。

## 思维·执行节点

1. `主参考系选择`
   选一个最稳的空间锚点，确保全句围绕同一参考系展开。
2. `方向向量判断`
   判断人物是逼近、退开、横移、转身还是离场，并选一个最清楚的动词表达。
3. `距离变化落句`
   交代与目标对象的距离是缩短、维持还是拉开，这决定张力如何被读懂。
4. `多人物去噪`
   若出现多个人或多物件，只保留主动作真正依赖的空间关系，删去旁枝。

## 节点延展

- 对峙戏里，位置和方向的重点往往不是“走了几步”，而是“逼近半步”“让开一道身位”“把视线或出口卡住”。
- 追逐或阻拦场景里，可以适度增加路径感，但仍应围绕一个主参考系展开，避免空间漂移。
- 延展的目标是让读者脑中能立即立体化，而不是证明作者已经画出平面图。

## 失真与修正

- 若空间开始像地图导览，说明方位词过量。
- 若人物移动了但朝向不清，补一个明确转向或接近/离开的表达。
- 若生成第二套空间坐标，删掉次级参照，只留主参考系。
- 若句子里出现多个“左/右/前/后”却仍然难懂，说明缺的是锚点，不是更多方向词。
