# Director Style Index

本文件是本地导演/作品资料索引，不是执行规则。执行规则以 `../SKILL.md` 为准。

## Usage

- 若用户指名导演，先在本文件或同目录资料中查找是否已有风格包。
- 若没有本地资料，执行者可使用模型已有知识和网络搜索资料建立 `director_style_profile`。
- 网络资料必须在执行报告中记录来源名称、链接、检索日期和使用边界。

## Local Profiles

当前暂无内置导演风格包。

建议后续以以下格式人工加入：

```markdown
### 导演名

- representative_works:
- usable_directing_dimensions:
- forbidden_misreadings:
- source_notes:
```
