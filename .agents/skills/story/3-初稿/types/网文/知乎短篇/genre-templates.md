# 知乎短篇题材模板 (Zhihu Short Stories)

> **核心卖点**: 开篇高概念 + 反转密集 + 社交传播力。在8000-20000字内完成一个"值得转发"的故事。

---

## 1. 知乎短篇铁律

### 开篇300字
- 必须在300字内抛出核心冲突/高概念
- 第一人称优先——"我"的代入感最强
- 示例开篇: "我老公失踪三年了。今天，我在商场看见了他——怀里抱着另一个女人。"

### 反转密度
- 全文至少3次大反转
- 反转节奏: 300字/1000字/5000字/全文结尾
- 每次反转必须有前文伏笔

### 全文篇幅
- 标准: 8000-20000字
- 一章到底不分章
- 节奏: 前30%建冲突、中40%升级、后30%收束

---

## 2. 高概念开头模板

| 类型 | 示例 |
|------|------|
| 身份反转 | "我不是她的替身——她才是我的替身。" |
| 时间错位 | "我明天就死了。但今天，我收到了三天后的自己的短信。" |
| 规则悖论 | "这个游戏的规则是：不要相信任何规则。" |
| 认知颠覆 | "我妈妈不是我妈——我昨天做了DNA检测。" |

---

## 3. 追读心理

| 驱动力 | 执行手段 |
| --- | --- |
| 开篇钩子 | 300字内让读者"不得不看完" |
| 反转密集 | 每1000-2000字一个"小反转"，每5000字一个"大反转" |
| 社交货币 | 结尾必须让读者"想转发给朋友看" |

## 4. AI 写作专项指南

### 常见陷阱
1. **开篇平**: 第一章在介绍背景 → 知乎短篇没有"第一章"——第一句话就是钩子
2. **反转硬塞**: 最后塞个反转 → 每次反转前至少1处伏笔
