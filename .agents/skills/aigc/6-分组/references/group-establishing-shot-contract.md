# Group Establishing Shot Contract

本文件定义每个分镜组开头的 `定场镜头：` 输出标准。该字段用于让每个分镜组在独立生成时重新锁定场景身份、环境身份、全部角色站位、姿态、空间方位和朝向关系。

## Core Rule

每个分镜组标题后必须先写当前场景标题行，再写一段 `定场镜头：`。即便连续多个分镜组仍在同一场景内，也必须重新写定场镜头，因为后续视频生成可能按组独立消费，不能依赖上一组的空间记忆。

`定场镜头：` 必须来自本组正文、上游 `5-摄影` 分镜和可确认的场景事实，不得新增人物、剧情事实、道具互动或未被正文支持的空间动作。

`定场镜头：` 必须服从 `../../_shared/scene-shot-identity-contract.md` 的身份先行规则：开头先回答“这是什么场景”，明确年代/社会语境/空间功能、环境材质、光线和声音基底，再写角色站位和动作承接；不能写成“某角色正在做什么”的动作摘要。

`定场镜头：` 也必须服从 `../../_shared/action-first-continuity-contract.md` 的人物动作链优先规则：在场景身份成立后，先承接人物姿态、位置、朝向、身体接触、未完成动作和注意力落点，再写环境锚点；道具只作定位、互动、关键信息、规则、证据、危险源或必要交代，不作为无互动氛围焦点。

`定场镜头：` 同时承担面向生成的主体信息整理职责。角色、群体、场景锚点和关键道具不再在分镜组开头另行输出 `角色：` / `场景：` / `道具：` 这类结构化清单，而是自然融合进定场描述和构图分区本身：每个会影响画面生成的主体都应带有必要的形容属性、状态属性、站位属性、朝向或注意力落点。组底 YAML 仍保留短名单统计，但它只服务校验和索引，不替代定场镜头的主体传达。

`定场镜头：` 必须内置一套构图分区主体账本。该账本不是普通格式化排版，而是 `6-分组` 的专项裁决能力：从 `5-摄影` 的画面、角色动作、镜头设计、分镜明细、对白主体和道具特写中提取主体与空间证据，再把相关主体准确布置到画面构图中的 `左侧：`、`中间：`、`右侧：`、`前景：`、`中景：`、`背景：` 六个固定分区。六个分区必须按该顺序完整输出，且每个分区都必须写出具体可见主体、环境锚点、遮挡、光影材质或空场压力；分区描述必须说明主体的形容属性、状态属性、站位属性、朝向/视线、与其他主体或道具的距离关系；不得只有名称列表，也不得写成“主体保持当前组正文中的站位”“关键动作锚点由本组正文中的……”或“固定环境锚点保持可读”这类泛化占位句。

从每集第二个分镜组开始，定场镜头必须在保留原有场景身份、环境身份、角色站位、姿态、空间方位和朝向关系的基础上，增量添加上一组尾帧还原入场。尾帧还原入场不是替代定场镜头，也不是固定套话；它是定场镜头第一层连续性账本，用上一组最后一个原始分镜或连接件抵达后的尾帧，复现本组开端的入场画面。

- 同场景相邻组：继承上一组末尾角色位置、姿态、朝向、视线或未完成动作，除非本组正文明确给出变化。
- 跨场景相邻组：以组间连接件抵达后的新场景首态作为开端，同时保证不与上一组尾帧和连接件过程冲突。
- 输出文字不再要求、也不鼓励使用 `承接上一组结束状态`、`承接上组末尾` 这类固定句式；必须具体写出 `上一组末帧`、`上一组尾帧`、`上一组最后一帧` 或等价明确锚点，并还原其中的可见画面状态。
- 尾帧还原入场必须复现上一组尾帧中的主体位置、姿态、朝向、视线、身体接触、道具关系、光线、烟雾、水纹、碎片、衣袍、发丝或其他仍在延续的可见状态；随后用本组定场镜头的场景/环境/站位关系重新锁定全局空间。
- 若需要制造跨组连续分镜序列感，只能在定场镜头和画面属性中说明本组从同一尾帧状态改用不同景别、机位高度、焦点、构图或观看距离进入；不得改写 `5-摄影` 原正文或原有 `分镜明细`。
- 不能把上一组已经结束的动作、位置或朝向突然重置为无证据的新状态；需要变化时必须能回指本组正文或组间连接件。

## Required Content

定场镜头使用自然中文描述和构图分区账本，必须覆盖：

- 场景身份：年代/时代、社会语境、空间功能和场景内子区域，例如教学空间前排、公共走廊门口、家庭餐桌左侧。
- 环境身份：空间固定锚点、材质新旧、氛围、光影、色彩和不新增事件的声音底色；氛围/光影/色彩只描述当前场面已经成立的视觉结果，不新增事件。
- 本组画面中全部可见或需要生成的角色；画外声音只在会影响画面调度时标注为画外。
- 主体属性：对角色、群体、关键道具和场景锚点写出必要的外观、身份、情绪压力、状态变化或风险属性，例如“苍老疲惫的长老”“前倾压迫的派系领袖”“被火光照亮的战争地图”，不得退化为只有名称的清单。
- 每个角色的姿态：站立、坐着、蹲下、躺着、靠着、俯身、跪坐等。
- 角色之间的空间关系：前后、左右、内外、近远、半步、一桌之隔、门内门外等。
- 角色朝向或视线方向：面向谁、背对谁、侧身朝向哪里、看向哪个空间锚点。
- 画面构图：用 `画面构图：` 概述本组起始画面的主体布局逻辑，例如环桌构图、门内外对峙、纵深压迫、前景遮挡、中景人物对峙、背景环境压迫等。
- 构图分区：必须按固定顺序完整输出 `左侧：`、`中间：`、`右侧：`、`前景：`、`中景：`、`背景：` 六个标签。每个分区都要落到本组事实：有角色、群体或关键道具时写具体主体属性、状态、姿态、朝向、视线和关系；没有独立人物时也必须写当前画面中该区域真实可见的环境锚点、遮挡、光影、材质、空场压力或边界关系。不得写“无内容可省略”，不得用“本组正文中的主体/关键动作/固定环境锚点”替代具体主体信息。
- 从第二组开始，上一组尾帧如何作为本组入场画面，例如“上一组末帧中，林寂仍坐在第二排靠窗座位，脸从黑板方向转向苏红叶；本组从同一姿态改用更近的过肩景别进入，同时保留苏红叶在课桌右前方半步处的站位关系”。
- 关键场景锚点：门、窗、讲台、床边、餐桌、车门等；只用于定位，不借机添加无互动道具镜头。
- 本组动作链牵引：若上一组末尾留下站坐、靠扶、牵手、转身、伸手、低头、看向某处等持续状态，本组定场必须明确该状态如何延续或在正文支持下变化。

## Evidence-To-Layout Workflow

生成 `定场镜头：` 时必须先做内部取证，再落盘构图分区：

1. 从 `5-摄影` 本组正文收集主体证据：对白主体、角色动作主体、表演提示主体、明确可见群体、关键道具、场景锚点、镜头设计中的景别/机位/前景/中景/背景/视线/运动方向。
2. 合并同一主体：同一角色、同一群体、同一关键道具不得因镜头景别、状态或持有人变化拆成多个主体；状态差异写在主体描述里。
3. 建立空间轴：先判断左右轴、前后纵深轴、内外/高低/主次轴哪些在本组起始画面中真实成立；不可为了填标签凭空创造方位。
4. 分配构图分区：把每个生成主体放进最稳定的构图位置；同一主体可同时说明“右侧中景”“前景左下”等复合位置，但必须保持空间关系清楚。落盘时仍按 `左侧：`、`中间：`、`右侧：`、`前景：`、`中景：`、`背景：` 六字段完整输出，不能因为某一区域没有人物就省略字段。
5. 写入形容属性和站位属性：每个重要主体至少包含身份/状态/姿态/朝向/视线/与其他主体或关键道具关系中的若干项，避免只有名称。
6. 复核连续性：第二组起先检查上一组尾帧和连接件抵达首态，构图分区不得让主体左右反转、前后跳变、坐站突变或视线无证据改变。

## Placement

每个分镜组开头固定顺序：

```text
## x-y-z

场景N：<场景标题>

定场镜头：<场景身份、环境身份、氛围、光影、色彩、声音底色；第二组起还要先复现上一组尾帧状态>
画面构图：<本组起始画面的构图逻辑和空间轴>
左侧：<左侧具体角色/群体/道具/锚点；若无人物，写真实可见的左侧环境边界、光影或空场压力>
中间：<中间或主位具体主体；写身份、状态、姿态、朝向、视线和关系>
右侧：<右侧具体主体或真实环境锚点；写与中间/左侧主体的距离、对峙或遮挡关系>
前景：<前景具体主体、遮挡、道具焦点、空间入口或可见材质；必须回指本组画面>
中景：<中景具体主体和动作链主关系；写站位、身体状态、朝向或注意力落点>
背景：<背景群体、固定锚点、光影色彩压迫或环境状态；必须是本组成立的具体画面>

画面属性：<从组内镜头设计提炼的自然语句>

视频生成的画面风格...
```

`定场镜头：` 位于场景标题行之后、`画面属性：` 和 north_star 风格行之前。`画面构图：` 与构图分区标签属于 `定场镜头：` 的内部账本，也位于 `画面属性：` 之前。不得输出独立的 `角色：` / `场景：` / `道具：` 清单。

## Count Boundary

`定场镜头：` 计入 YAML `字数统计`，并进入 1680 字目标上限 / 1980 字硬上限的辅助风险口径；但不计入 `时长估算`，也不改变组内 `分镜N（约X秒）:` 的累计结果。

## Do Not

- 不把定场镜头整体写成摄影机运动、景别或镜头语言；但第二组起允许用一句说明“同一尾帧状态改用更近/更低/更侧/更长焦的入场观看方式”，用于建立跨组连续分镜序列感。完整摄影语言仍归属 `画面属性`、分镜正文或连接件。
- 不写成单纯人物动作摘要；开头不能只回答“谁在做什么”，必须先让场景、环境和角色身份成立。
- 不在分镜组开头追加结构化主体清单，例如 `角色：A、B`、`场景：X`、`道具：Y`；这些信息必须被自然写进定场描述，组底 YAML 才保留短名单形式。
- 不把 `画面构图：` 写成摄影参数清单；它必须回答主体如何占据画面区域。摄影参数、光学语言和审美维度仍归属 `画面属性：`。
- 六字段完整是硬门槛，但不为了填满左/中/右/前景/中景/背景而编造上游没有的主体或位置；无人区域必须写本组真实可见的环境锚点、遮挡、光影、材质、空场压力或画面边界，不能省略，也不能写泛化占位句。
- 不添加人物与道具的无互动关联镜头，例如无必要的杯水涟漪、影子压住文件、餐具碰撞等。
- 不为了制造氛围新增未在本组出现的道具、环境事件或角色动作。
- 不省略同场景连续组的定场镜头；每组都要重新交代角色姿态和朝向。
- 不让第 N+1 组定场镜头与第 N 组结尾状态断裂，例如角色突然换边、坐站状态突变、朝向反转或未解释的距离变化。
- 不把旧示例中的某类道具、房间、动作或情绪反应硬编码为固定写法；所有定场内容必须由当前上游分镜和本组事实推出。

## Example

```markdown
定场镜头：教室前排课桌区内，冷白日光灯压低整体色温，黑板区域偏暗，窗侧微弱日光让第二排靠窗座位形成更清楚的边缘轮廓，教室内只有压低的呼吸声和粉笔灰静止在空气里。
画面构图：以前排课桌形成横向分割，林寂与苏红叶占据中景右侧的对峙焦点，讲台和黑板压在背景。
左侧：讲台左缘和黑板暗面压在画面左侧，粉笔槽的冷灰线条把陈大雷所在区域与课桌区隔开。
中间：林寂坐在第二排靠窗座位，身体朝向讲台但脸偏向右侧的苏红叶，右肩微转，神情紧绷。
右侧：苏红叶站在林寂课桌右前方半步处，左手停在桌沿上方，俯身面向林寂。
前景：前排课桌边缘横切画面下方，桌角遮住林寂下半身，让两人的对峙集中在上半身和视线。
中景：林寂与苏红叶处在主要中景关系里，二人隔着半张课桌形成低声压迫。
背景：陈大雷站在讲台左侧，身体朝向全班，黑板在他身后形成压迫性的暗色平面。

定场镜头：上一组末帧中，林寂仍坐在第二排靠窗座位，右肩微微转向站在课桌右前方半步处的苏红叶，冷白日光灯继续压住两人的脸部阴影；本组从同一画面状态改用更近的过肩观看距离进入。
画面构图：以苏红叶肩部形成右侧近景遮挡，林寂处在中景焦点，讲台左侧的陈大雷退入背景压迫位。
左侧：讲台左侧的陈大雷仍在背景偏左位置，身体面向全班，黑板暗面贴在他身后。
中间：林寂仍坐在第二排靠窗座位，脸偏向苏红叶，身体没有离开座位，是本组中景焦点。
右侧：苏红叶仍保持俯身面向林寂的位置，肩线靠近镜头形成遮挡。
前景：苏红叶靠近镜头的肩部和发丝形成右前景遮挡，压缩林寂的可见空间。
中景：林寂与苏红叶隔着课桌维持上一组尾帧的近距离对峙，视线仍锁在彼此之间。
背景：陈大雷仍站在讲台左侧面向全班，三人的左右关系、站位和朝向不重置。
```
