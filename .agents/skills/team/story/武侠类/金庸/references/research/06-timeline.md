# 06-timeline｜金庸关键时间线

| 时间 | 事件 | 影响 |
|---|---|---|
| 1924-03-10 | 出生于浙江海宁查氏家族 | 传统文化与家学底色 |
| 1930s-1940s | 少年时期阅读武侠、历史与外国文学；战乱中辗转避难 | 形成历史感、漂泊感与冒险叙事趣味 |
| 1948 | 调派至香港《大公报》工作 | 香港成为其创作与报业事业根基 |
| 1950s 初 | 以多个笔名写影评、剧评、专栏，并参与电影编剧与导演工作 | 形成电影叙事节奏与跨媒介意识 |
| 1955-02-08 | 以“金庸”为笔名在《新晚报》连载《书剑恩仇录》 | 正式进入武侠连载阶段 |
| 1957 | 发表《谈批评武侠小说的标准》 | 明确提出武侠也要按文学标准写 |
| 1959-05-20 | 与沈宝新创办《明报》 | 小说、社评、报业经营三线合一 |
| 1959-1972 | 高强度连载 15 部武侠作品 | 建立金庸宇宙与华语读者共同记忆 |
| 1960s | 《明报月刊》《明报周刊》《明报晚报》等陆续创办 | 报业版图扩张，公共影响力增强 |
| 1972 | 完成《鹿鼎记》后封笔 | 以反英雄作品结束主要武侠创作 |
| 1972-1980s | 全面修订旧作约十年 | 从报纸连载稿提升为长线作品工程 |
| 1981 | 首次以较高规格访问内地 | 报人/公共知识分子身份进一步显形 |
| 1985-1990 | 参与香港基本法相关委员会工作 | 从小说与报评进入制度层公共事务 |
| 1991 | 明报企业上市 | 报业事业高峰 |
| 1993-1994 | 辞去主席并正式退休 | 逐步淡出经营一线 |
| 2000-2006 | 新修版《金庸作品集》再度全面修订 | 对文本持续精修，显示极强作者工程意识 |
| 2010 | 剑桥大学获博士学位 | 学术身份与读书人形象强化 |
| 2018-10-30 | 在香港逝世 | 结束一生，但作品传播进入更广跨媒介阶段 |
| 2024 | 金庸百年诞辰纪念活动、香港文化机构相关展览继续推进 | 作品与作者身份作为公共文化符号延续 |

## 最近时效校验

- 金庸已于 2018-10-30 去世，不存在“近 12 个月本人新动态”。
- 最近一年与其相关的有效时效信息，主要是：
  - 2024 年百年诞辰纪念活动延续影响
  - 2024 年香港文化机构继续举办相关纪念展
  - 影视改编与学术讨论仍持续
- 结论：本 skill 的时效敏感部分，不在“本人观点变化”，而在“纪念活动、改编与研究生态变化”。
