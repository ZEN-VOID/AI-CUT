# Template: 修图 / 美颜美体

Template ID: `photoGPT.修图.美颜美体.v1`

## Input Roles

- 图一：待修人像或人物图，提供身份、姿态、构图和真实体态基础。

## Prompt Frame

```text
在保持图一中人物身份、五官比例、发型、肤色、服装、姿态、构图和场景不变的前提下，执行自然克制的美颜美体修图。

美颜目标：清理轻微痘印、暗沉、眼下阴影和不均匀肤色，保留真实皮肤纹理、妆面质感和面部立体结构；适度优化精神状态与肤色通透度。

美体目标：只做自然姿态与轮廓优化，保持真实人体比例、服装受力、肢体结构和透视合理，不夸张拉伸、不改变人物身份。

补充要求：<user_desc>
```

## Negative Constraints

- 不换脸、不改变五官身份、不改变年龄气质。
- 不过度磨皮、不液化失真、不拉长肢体。
- 不改变服装、背景、姿态关系和场景叙事。
