# 侦探剧核心要素模板 (Detective Core Elements)

> **核心功能**: 侦探推理题材的核心创作定律，定义推理小说的基石规范和进阶套路。

---

## 1. 推理类型学

### 本格推理
- **核心**: 公平游戏——所有线索对读者公开，读者理论上可以和侦探同时破案
- **关键要素**: 诡计、逻辑链条、公平性

### 社会派推理
- **核心**: 案件反映社会问题——凶手不是"坏人"，是"被逼到绝路的人"
- **关键要素**: 社会议题、人物动机、制度批判

### 硬汉派
- **核心**: 私家侦探在灰色地带游走，世界不是黑白的
- **关键要素**: 氛围、人物、街道感

### 倒叙推理
- **核心**: 先告诉你凶手是谁——观众享受"看侦探怎么抓到他"
- **关键要素**: 猫鼠博弈、心理对抗

### 叙述性诡计
- **核心**: 利用叙述视角欺骗读者——"叙述者就是凶手"
- **经典方法**: 不可靠叙述者、时间错位、叙述视角偷换

---

## 2. 推理十诫修订版

1. **所有线索对读者公开** — 不能藏着读者不知道的关键信息
2. **推理过程可复现** — 读者可以回溯验证每一个推理步骤
3. **真相不能凭空出现** — 不能靠"突然想到""灵光一现"
4. **反转必须有伏笔** — 至少3处可回溯的伏笔
5. **侦探不能"刚好"在现场** — 必须有人叫他去的合理理由
6. **凶手必须是出现过的人** — 不能是"从来没提过的角色"
7. **动机必须可理解** — 不能是"因为他疯了"
8. **科技不能是万能解** — DNA/监控不能突然出现解决问题

---

## 3. 线索设计

### 三层线索法
```
明线索 (20%): 直接可见，指向"表面真相"
暗线索 (50%): 藏在字里行间，需要分析才能发现
误导线索 (30%): 故意设置的歧途——但符合逻辑
```

---

## 4. AI 写作专项指南

### 常见陷阱
1. **线索不公平**: 真相靠"侦探突然想到" → 所有线索必须在前文出现过
2. **反转硬来**: 最后说"其实他才是凶手" → 至少3处可回溯伏笔
3. **动机不能穿越**: 凶手的动机必须和时代背景匹配
