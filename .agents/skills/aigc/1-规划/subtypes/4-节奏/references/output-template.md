# 节奏输出契约

## 固定区块

`第N集.md` 至少应包含以下固定区块：

1. `主驱动裁决`
2. `七步节奏蓝图`
3. `峰值账本`
4. `节奏执行策略`
5. `下游加载提示`

## 峰值类型最小集

`峰值账本` 至少允许从以下类型中选择主峰值或辅峰值：

- `PLOT_TWIST`
- `ACTION_IMPACT`
- `EMOTIONAL_BREAK`
- `MIND_GAME`
- `WORLD_REVEAL`
- `ENSEMBLE_RESONANCE`

## 硬规则

1. `主驱动裁决` 必须先写 `original_adherence` 布尔门与“是否保留原作节奏”的执行结论。
2. 输出只处理节奏蓝图，不得直接改写剧情事实、人物命运或核心对白。
3. 七步节奏蓝图是幕后支架，不得把所有标签机械地显性贴进下游正文。
4. `节奏执行策略` 必须明确哪些组该前置、压缩、留白、交叉或回收，以及为何这样裁。
5. `下游加载提示` 必须明确告诉 `2-组间/导演意图` 与 `3-明细` 哪些节奏法则不可稀释。
