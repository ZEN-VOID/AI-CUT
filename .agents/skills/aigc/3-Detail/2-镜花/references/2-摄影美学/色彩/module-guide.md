# 色彩 模块说明

## 定位

- 本叶子负责确定色温、色彩对比和情绪色气候。
- 它不负责替代布景、美术或剧情判断，只负责给镜头写出可承接的色彩关系。
- 它不是只给一个“冷暖感受”，而是要把色相、明度、饱和度和色彩心理一起收束成可执行色板。

## 使用方法

- 先锁冷暖或混合对照，再判断主导色相和明度/饱和度该怎么压。
- 再判断哪组色彩关系最值得强调，哪些只适合点到为止。
- 色彩选择要回指类型元素、全局风格、导演意图和当前情绪。
- 输出应让布景、服装或光线都能承接，而不是停在抽象赞美。

## 失真与修正

- 若只说“高级、克制、梦幻”，说明没有真正给出色彩关系。
- 若只有冷暖判断，没有明度或饱和度安排，说明色板还不够可执行。
- 若色彩和项目风格走向相反，回看 `Init / Global` 的风格基线。
- 若色彩压过了动作和人物主线，收掉次级色彩修辞。
- 若色彩心理说不清，回到当前组主情绪，重新确认哪种关系最有叙事收益。
