# 推理悬疑

## 核心冲突引擎

- 这一类题材的核心是“线索链晚于危险感，但早于最终揭晓”。读者要一直处在能猜、但猜不全的状态。
- 推进顺序最好是：异样先出现 -> 线索被看见但未被解释 -> 新证据改变旧理解 -> 真相只揭一半 -> 下一页压力继续。
- 真相必须能回溯前文细节，否则再大的反转也像临时编出来的。

## 角色原型

- 观察者负责发现异样，行动者负责闯入危险，隐瞒者负责制造信息差，嫌疑者负责扰乱判断。
- 主角最好不是万能侦探，而是有盲区、有偏见或有私人代价的人，这样线索回收才有戏。
- 配角的每次说谎、沉默、错记，都应该兼具人物动机和叙事功能。

## 漫画页面语法

- 页尾比页中更值钱，翻页点是这类题材最稳定的杠杆。
- 细节特写、半张脸、门缝、监控画面、空镜和物证局部都比解释段落更重要。
- 留白必须服务怀疑，不要用台词把疑点一次性说破。

## 典型桥段库

- 一个很小的物证推翻整套口供。
- 读者先看到危险，角色后知后觉。
- 同一画面第二次出现时含义完全改变。
- 页尾只露半个真相，下页先给反应，再给解释。
- 嫌疑人看似洗白，实际上把怀疑转移到了更危险的方向。

## 翻页机制

- 翻页点是这一类题材的核心杠杆，优先放“未完成动作”和“只露一半的证据”。
- 页尾高价值触发点：门将开启、视线突然移向画外、证据细节被看见、口供出现裂缝。
- 翻页后不要立刻讲透，先让读者看到后果、反应或第二个疑点。

## 信息主载体

- 主信息优先靠物证局部、视线方向、空间异常和重复场景的细小变化。
- 门缝、监控、录音、手机画面是稳定的辅助载体。
- 阴影和半张脸只能放大不确定感，不能代替证据链。

## 对白与旁白

- 对白宜短，负责试探、否认、错记和补一小格逻辑。
- 旁白或说明不能抢在证据前面。
- 推理说明最好拆到场景复盘和物证回收里，而不是一口气讲完。

## 母题系统

- 门缝、半张脸、录音、重复场景再看，是高价值回返母题。
- 母题二次出现时必须改变读者解释路径，否则只是重复气氛。

## 禁写模式

- 真相比疑问先到。
- 角色不断解释，画面没有参与推理。
- 反转只求“没想到”，不能回扣前文。
- 把恐怖氛围和推理线索混成一团，导致两头都不成立。
