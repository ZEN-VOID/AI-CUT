# 06 - 人物时间线

调研对象：梁朝伟（Tony Leung Chiu-wai）  
调研时间：2026-04-15

| 时间 | 事件 | 表演系统意义 |
|---|---|---|
| 1962 | 出生于香港。 | 城市商业文化、电视工业和香港电影黄金期共同构成职业土壤。 |
| 1980s 初 | 进入 TVB 训练班和电视系统，成为香港观众熟悉的青年演员。 | 形成职业纪律、镜头熟悉度和大众亲和力。 |
| 1989 | 凭侯孝贤《悲情城市》进入华语作者电影重要语境。 | 训练观察、静默和历史压力中的表演。 |
| 1990 | 在王家卫《阿飞正传》结尾出现，开启长线合作。 | 明星存在可以只靠一段无台词动作成立。 |
| 1994 | 《重庆森林》《东邪西毒》等作品扩大国际影迷认知。 | 孤独、等待、内心独白与碎片叙事进入核心方法。 |
| 1997 | 《春光乍泄》获得重要表演认可。 | 亲密关系、迁徙、疲惫与情感依赖成为可持续模型。 |
| 2000 | 《花样年华》获戛纳最佳男演员。 | “少演一点”的国际可读性得到确认。 |
| 2002 | 《无间道》成为香港类型片代表作之一。 | 克制表演可服务高强度警匪类型。 |
| 2007 | 《色，戒》进入李安式高心理压力表演环境。 | 欲望、权力和身体风险需要严格边界意识。 |
| 2013 | 《一代宗师》中的叶问成为晚期代表角色之一。 | 武术、礼法、隐忍和时代失落合并。 |
| 2021 | 出演 Marvel《尚气与十环传奇》。 | 跨入全球商业片，同时将反派父亲复杂化。 |
| 2023 | 获 Venice Golden Lion for Lifetime Achievement。 | 国际影史层面确认其长期贡献。 |
| 2024 | 香港演艺学院授予荣誉奖项；《金手指》等商业角色继续传播。 | 晚期影响进入教育与行业传承。 |
| 2025-2026 | 《Silent Friend》等国际项目进入影展与放映资讯。 | 晚期仍在跨语境合作中拓展表演环境。 |

## 最新动态（截至 2026-04-15）

- 已获得 2023 年威尼斯终身成就金狮，这一荣誉是其国际影史位置的重要锚点。
- 2024 年香港演艺学院荣誉奖项进一步确认其对表演教育和行业的影响。
- 2025-2026 年公开片单中可见《Silent Friend》等跨国作者项目相关信息；具体上映与地区发行状态需要在回答事实问题时再次核对。
