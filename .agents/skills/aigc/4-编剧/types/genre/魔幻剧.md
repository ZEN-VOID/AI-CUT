# 魔幻剧 Genre Type

本类型卡定义 `魔幻剧` 在 `4-编剧` 中的题材策略。它不是子技能，不拥有输出路径、节点、gate 或最终审查权。

## Identification

| field | value |
| --- | --- |
| `type_id` | `genre-mohuan` |
| `axis` | `genre_axis` |
| `aliases` | 魔幻、奇幻、魔法、诅咒、神谕、异族、魔物 |
| `primary_signals` | 魔法规则、诅咒、神谕、王国/部族、异族、魔物、圣物、黑暗力量、契约、祭坛 |

## Strategy

| strategy_slot | rule |
| --- | --- |
| `dramatic_pressure_bias` | 把冲突落到诅咒代价、信仰/神谕压力、契约束缚、族群边界、黑暗力量侵蚀和选择代价。 |
| `rhythm_bias` | 适合“预兆/异象 -> 角色不信或误判 -> 代价显现 -> 契约/诅咒逼近 -> 尾钩异变”的节奏。 |
| `controlled_supplement_bias` | 可补符文、圣物状态、环境异象、身体变化、群体恐惧、仪式痕迹；不得新增魔法规则或诅咒结果。 |
| `field_material_bias` | 城堡、森林、祭坛、石碑、圣物、符文、烛火、黑雾、钟声、乌云、裂纹、异族服饰、群体退避。 |
| `climax_hook_bias` | 高潮优先落到诅咒或契约的可见代价、圣物状态变化、异象扩散或身份/预言被确认。 |

## Prohibited Drift

- 不得新增 source 不支持的魔法能力、神谕内容、契约条款、异族关系或诅咒结果。
- 不得把魔幻写成抽象氛围；必须落到可见异象、物件状态、身体变化、声音和现场反应。
- 不得为了奇观改变事件结果或角色选择。

## Combination Notes

- `正剧 x 魔幻剧`：可用仪式、圣物、诅咒代价和角色选择强化戏剧压力。
- `解说剧 x 魔幻剧`：旁白承接神谕、关系和背景，画面用符文、圣物、异象、群体反应和环境变化承托。
