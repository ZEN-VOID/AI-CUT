# 反镜头说明腔 Module Spec

## 目标

- 清理小说正文中残留的影视分镜语法、镜头切换提示词和角色报幕段。
- 把“像在看脚本”的句法重新收束成“像在读小说”的段落推进。

## 重点清扫对象

1. 镜头切换词：如“画面骤碎 / 画面猛断 / 镜头一转 / 蒙太奇交叉闪现”。
2. 角色报幕段：人物名单独成段，像出场卡或字幕条。
3. 剪辑说明句：靠“切回 / 拉远 / 推近 / 定格”推进，而不是靠场面本身推进。
4. 只承担镜头功能、不承担叙事功能的过桥句。

## 修正原则

1. 优先把镜头切换改成物象、动作、声音或人物感知的自然过渡。
2. 若人物名单独成段，只保留在正文里确实承担语气突刺的极少数情况；其余默认并回前后场面。
3. “切场”应改成“前一场怎么自然带出后一场”，而不是作者直接喊 cut。
4. 若一句话删掉后情节并无损失，但脚本感大减，默认优先删除。
