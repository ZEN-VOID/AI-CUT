# Floor Plan Default Type Package

默认类型包用于所有 `分镜平面图` 任务。

## Fixed Context

- `references/floor-plan-sheet-contract.md`
- `review/review-contract.md`
- `templates/output-template.md`

## Gate

- 组源必须可追溯。
- 必须先建立 `source_spatial_comprehension`，理解本组叙事功能、动作链、空间锚点、角色状态、道具状态和禁止补写项。
- `floor_plan_panels` 必须从空间关系、位置变化、动作路径、摄影机方向或视觉节拍中裁决；不得机械等同原始 `分镜N`。
- 整体图面必须是黑白建筑平面图标准的顶视图，不得退化为故事板画面、透视场景、概念图或气氛图。
- 角色必须使用同集一致的彩色几何图标，配黑色文本角色名。
- 颜色只允许用于角色图标和标注层，且标注语义固定：红色身体运动、蓝色摄影机、绿色取景/构图、橙色灯光方向、紫色情绪/声音/叙事强调、黑色文本。
- 同一集相邻组必须审查空间连续性，记录不变锚点、变化位置、移动逻辑和叙事空间理由。
- 输出根必须位于项目 `12-图像/分镜平面图/`。
