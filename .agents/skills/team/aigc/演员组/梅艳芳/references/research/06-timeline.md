| 时间 | 事件 | 表演意义 |
|---|---|---|
| 1963-10-10 | 梅艳芳出生。 | 后来被公众视为香港流行文化黄金时代的重要象征。 |
| 1960s-1970s | 幼年开始与姐姐登台谋生。 | 早期训练来自真实观众和商业舞台，而非学院系统。 |
| 1982 | 赢得第一届新秀歌唱大赛。 | 从民间舞台进入香港流行音乐工业。 |
| 1983 | 发行早期唱片，并获东京音乐节相关奖项。 | 建立低沉声线与国际舞台识别。 |
| 1985 | 红馆演唱会创下新人级别的重要纪录。 | 舞台掌控力成为核心资产。 |
| 1986 | “坏女孩”等形象引发争议并扩大影响。 | 将女性危险感、独立感与流行商业成功结合。 |
| 1987-1988 | 出演《胭脂扣》并获得重要电影奖项认可。 | 证明她不是只靠歌坛形象，也能承载电影悲剧情绪。 |
| 1989 | 参与大型公共义演。 | 明星身份开始与公共行动绑定。 |
| 1990-1993 | 出演《川岛芳子》《审死官》《东方三侠》等类型片。 | 形成跨喜剧、动作、历史、漫画类型的强人格表演。 |
| 1990s | 与刘培基等创作者持续打造阶段性舞台形象。 | “百变”成为完整视觉-声音-身体系统。 |
| 1997 | 出演许鞍华《半生缘》。 | 从舞台式传奇气场转向现实女性命运。 |
| 2002 | 出演许鞍华《男人四十》。 | 晚期电影表演更贴近现代家庭与情感伦理。 |
| 2003 | SARS 相关公益演出；9 月公开病情；11 月完成最后演唱会；12 月 30 日去世。 | 公共担当、病中舞台与终章形象合并成身后神话。 |
| 2023-2024 | 香港文化博物馆举办“绝代芳华·梅艳芳”纪念展；国际媒体持续回顾她的电影。 | 身后影响继续被博物馆化、影史化与演员训练化。 |

## 最新动态（截至 2026-04-15）

- 无本人新动态；只有身后展览、传记片、评论与资料再整理。
- 2023-2024 年香港文化博物馆展览是最近较重要的机构级纪念资料。
