# 06 时间线

## 1967-1990：出发

- **1967-01-12**：生于日本鹿儿岛县。  
  来源：Japan Media Arts Festival profile

- **1988**：以《楓パープル》获手冢赏。  
  来源：Japan Media Arts Festival profile

## 1990-1999：体育漫画爆发与转向

- **1990**：`SLAM DUNK` 开始连载。  
  来源：Japan Media Arts Festival profile

- **1996**：`SLAM DUNK` 连载结束。  
  来源：官方站、公开作品信息

- **1997-06-04**：出版 `INOUE TAKEHIKO ILLUSTRATIONS`。  
  来源：官方站作品页

- **1998**：`バガボンド` 开始连载。  
  来源：Japan Media Arts Festival profile

- **1999-10**：`REAL` 开始在《Young Jump》间歇连载。  
  来源：Japan Media Arts Festival profile

## 2000-2010：奖项、画册、展览与纸面扩张

- **2000**：获讲谈社漫画奖与文化厅媒体艺术祭漫画部门大奖。  
  来源：Japan Media Arts Festival profile

- **2005-01-20**：`SLAM DUNK 10 DAYS AFTER` 相关 `SWITCH` 发行。  
  来源：官方站作品页

- **2006**：启动 `スラムダンク奨学金`。  
  来源：2017 官方随笔倒推出 12 年历程 + 公开项目史

- **2006-10-24 / 2006-11-25**：`WATER`、`DRAW` 等《バガボンド》相关画集与制作纪录推出。  
  来源：官方站作品页

- **2008-05-24 至 07-07**：上野之森美术馆举办“最后的漫画展”。  
  来源：官方站展览页

- **2008-06-01 / 06-09**：`OTHER HAND`、`漫画がはじまる` 推出。  
  来源：官方站作品页

## 2010-2019：公共表达与跨界对象研究

- **2010 冬 - 2011 春**：`Smile` 系列开始公开，并在东日本大震灾后加速、发起捐助。  
  来源：官方 `Smile` 项目页

- **2014**：高迪相关项目、展览与旅途推进。  
  来源：官方 topics 与 `pepita` 项目页

- **2015-10-20**：出版《円空を旅する》。  
  来源：官方作品页

- **2017-10-30**：官方发表 `SD奨学金12期生募集開始!`，完整回顾奖学金初心。  
  来源：官方随笔

- **2019-05-22**：官方随笔提示 `REAL` 85 话刊载。  
  来源：官方随笔目录页

## 2020-2024：重新进入《SLAM DUNK》，转向导演

- **2020-01-16**：官方随笔“スラムダンク奨学生探訪2020①”。  
  来源：官方随笔目录页

- **2022-10-20**：官方随笔 `THE FIRST` 发布。  
  来源：官方站

- **2022-12**：`THE FIRST SLAM DUNK` 上映，井上担任原作、脚本、导演。  
  来源：官方相关访谈与机构资料

- **2023-12-27**：官方站记录 `THE FIRST SLAM DUNK` 获日刊体育电影大奖相关荣誉。  
  来源：官方 topics

- **2024-03-12**：文部科学省举行艺术选奨贈呈式，井上以漫画家、动画导演身份获文部科学大臣赏。  
  来源：MEXT

- **2024-08-19**：`REAL` 16 卷发售。  
  来源：官方 topic

## 2025-2026：`REAL` 再次成为最新窗口

- **2026-04-17**：`REAL` 17 卷发售。  
  来源：官方 topic

- **2026-04-18**：`REAL × NEW ERA` 官方联名公布。  
  来源：官方 topic

## 时间线观察

- 井上的职业轨迹不是一条单线，而是三条主轴并行：**体育漫画 / 人物成长 / 纸面之外的图像实践**。
- 到 2026 年，最新公开动态仍然把 `REAL` 放在最前线，这对判断“现在的井上雄彦”非常重要。
