# 06 · 时间线

## 生平主线

| 时间 | 事件 | 对思想的影响 |
|---|---|---|
| 1844 | 生于 Röcken | 牧师家庭背景使他后来对基督教的反转更具内部性 |
| 1849–1850 | 父亲去世、弟弟早夭 | 早期创伤与对 suffering / fate 的敏感度有关 |
| 1858–1864 | 就读 Schulpforta | 严格古典教育、语言训练、早期写作与音乐兴趣成形 |
| 1864–1865 | Bonn 学神学与古典语文学，后转 Leipzig | 从宗教教育转入古典学与哲学 |
| 1865 | 读到 Schopenhauer | 悲观主义、艺术救赎、反形而上学张力的重要起点 |
| 1868 | 结识 Wagner | 文化重建、艺术-哲学联动的早期激情 |
| 1869 | 赴 Basel 任教 | 年轻教授身份确立，也开启与学院的不适 |
| 1870 | 普法战争中任医疗勤务，染病 | 身体脆弱与战争经验加深其“文化症状”视角 |
| 1872 | *The Birth of Tragedy* 出版 | 酒神 / 日神模型、文化重建野心登场 |
| 1873–1876 | *Untimely Meditations* | 形成反合时宜姿态与文化批判者身份 |
| 1876–1878 | 与 Wagner 关系恶化，*Human, All-Too-Human* 出版 | 写法与思想双重转向 |
| 1879 | 因健康辞职 | 从教授转为流亡式写作者 |
| 1881 | *Daybreak* | 道德心理拆解加深 |
| 1882 | *The Gay Science*；与 Lou Salome 关系破裂 | “上帝死了”、欢快科学与私人挫败并行 |
| 1883–1885 | *Thus Spoke Zarathustra* | 超人、自我超克、诗化宣告达到高峰 |
| 1886 | *Beyond Good and Evil* | 视角、等级、价值批判全面成熟 |
| 1887 | *On the Genealogy of Morality* | 谱系学方法定型 |
| 1888 | 密集完成数部晚期著作 | 重估、诊断、自我神话集中爆发 |
| 1889 | Turin 崩溃 | 结束创造性写作 |
| 1900 | 于 Weimar 去世 | 死后影响远大于生前 |

## 1880–1888 的游牧结构

- 冬天多在 Nice 等地中海城市
- 夏天多在 Sils-Maria
- 中间穿梭于意大利、瑞士、德国诸城

这个结构不是旅行趣味，而是身体与写作条件共同逼出来的生存方案。

## 身后命运

| 时间 | 事件 | 含义 |
|---|---|---|
| 1890s–1900s | Elisabeth 接管遗产与档案 | 为后续误读埋雷 |
| 1901 | *The Will to Power* 编辑出版 | 形成“遗稿拼装即总体系”的误导 |
| 20 世纪中叶后 | 重新被严肃哲学界阅读 | 从纳粹污名与流行误读中逐步剥离 |
| 20–21 世纪 | 深刻影响哲学、文学、心理学与文化理论 | 后效力远强于生前声望 |

## 2026 边界说明

- 尼采是历史人物，不存在“最近 12 个月动态”。
- 需要更新的不是其本人新动态，而是当代接受史、译本、研究争议与挪用方式。
