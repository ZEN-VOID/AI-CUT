# 无限流题材增强包

> 补充维度：套路拆解 / 情节模式 / 人物原型 / 反套路策略 / 质量过滤

## 一、核心套路与行规

### 1.1 副本设计铁律
- **每个副本有独立规则**：规则必须清晰、内洽，且与主线世界观有或明或暗的联系
- **规则有不完整性**：副本规则不是说明书——主角需要通过试错、观察、推理来发现完整规则
- **规则可被利用**：最精彩的无限流是主角利用规则而非蛮力通关

### 1.2 副本类型图谱
| 类型 | 特征 | 通关核心 |
|------|------|----------|
| 生存类 | 在限定时间内/条件下存活 | 资源管理+规则利用+心理博弈 |
| 解谜类 | 收集线索→推理真相→破解谜题 | 逻辑推理+信息拼图 |
| 对抗类 | 与其他玩家/势力竞争 | 博弈论+不对称优势 |
| 恐怖类 | 灵异/怪谈场景，规则不明 | 恐惧管理+发现规则+逃生 |
| 任务类 | 完成特定目标 | 策略规划+团队协作+突发应对 |

### 1.3 副本间连接
- 副本不是孤立单元——副本之间应有主题呼应、因果关联或伏笔串联
- 通关奖励应在后续副本中有作用（不是一次性消耗品）
- 副本难度应渐进上升，且上升有逻辑

### 1.4 常见套路陷阱
- 副本规则说明书式展开（开场就全部讲清楚）
- 所有副本都是同一模式（全战斗或全解谜）
- 通关方案永远最优（看不到主角试错过程）
- 主角专属道具/能力刚好克副本（巧合过多）

## 二、情节模式与节奏规则

### 2.1 副本节奏模板
```
规则发现期(15%)：进入副本→探索→初步发现规则→小意外
深度探索期(25%)：深入场景→验证规则→发现隐藏规则→首次重大危机
策略形成期(20%)：分析局势→形成策略→资源调配→团队分歧/联合
高潮对决期(25%)：执行策略→遭遇反制→调整→最终对决/通关
余波反思期(15%)：奖励结算→复盘→发现副本与主线的联系→新线索
```

### 2.2 主线推进节奏
- 主线世界的故事与副本交替推进，不能全部塞到副本之间的"休息时间"
- 每个副本揭示一部分世界观真相，逐步拼出大图
- 大副本（3-5章）和小副本（1-2章）交替出现

### 2.3 人物成长节奏
- 每个副本后主角有可感知的成长（能力/认知/经验，不限于数值）
- 副本中的关键时刻塑造主角性格
- 团队角色的专属高光副本（每个关键队友至少有一个）

## 三、反套路策略

### 3.1 反"上帝视角"
- 主角不是全知的——他不知道所有规则、不了解所有参与者、不能预判所有意外
- 智力体现为"在有限信息下做出最优决策"而非"全知全能"

### 3.2 反"副本孤岛"
- 副本中的事件对主线有真实影响——失去的东西就是失去了，获得的信任就是获得了
- 副本中的NPC/场景有超出副本的意义——它们可能是主线世界的投影/碎片/记忆

### 3.3 反"数值碾压"
- 通关靠的是策略而非能力数值
- 信息/规则/心理/人际——这四个维度的博弈至少与力量博弈同等重要

## 四、质量过滤规则

### 4.1 副本规则自洽检查
- [ ] 副本规则前后一致（无临时新增规则"恰好"帮主角）
- [ ] 副本难度与当前团队实力匹配（有挑战但可战胜）
- [ ] 通关方案有试错过程（不是一次想到最优解）
- [ ] 奖励与难度成正比

### 4.2 AI腔高发区警示
- 规则说明书式展开：用叙事展示规则，不要旁白解释
- 主角心理全透明：让读者自己推理，不要主角把所有推理过程"说"出来
- 通关过程流水账：剪辑节奏，省略非关键行动

### 4.3 失败码
| 失败码 | 症状 | 返工目标 |
|--------|------|----------|
| `FAIL-INF-RULE-EXPO` | 规则说明书式展开 | 将规则融入场景和行动 |
| `FAIL-INF-COPY-MODE` | 副本模式单一/模板化 | 变化副本类型和通关方式 |
| `FAIL-INF-NO-LINK` | 副本与主线无关联 | 增加世界观揭示/伏笔串联 |
