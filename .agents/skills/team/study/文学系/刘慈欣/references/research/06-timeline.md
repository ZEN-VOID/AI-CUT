# 06 人物时间线：刘慈欣

## 来源概况

本维度记录可核验关键节点，尤其覆盖近 12 个月动态。出生年份等基础资料使用权威媒体/出版页交叉；私人细节不做过度延伸。

## 时间线

| 时间 | 事件 | 对创作系统的影响 |
|---|---|---|
| 1963 | 刘慈欣出生于北京，后随家庭到山西阳泉一带生活 | 工业城市、能源系统、工程经验构成硬科幻底色 |
| 1980s | 接触凡尔纳等经典科幻，形成早期科幻想象来源 | 黄金时代科幻、探索冲动和技术神奇感成为长期坐标 |
| 1985 前后 | 毕业后进入山西娘子关电厂工作 | 工程系统、能源、巨构和技术组织成为作品真实质地 |
| 1999 | 在《科幻世界》发表早期短篇，包括《鲸歌》等 | 正式进入中文科幻核心发表场域 |
| 2000 | 《流浪地球》等作品陆续发表 | 行星工程、文明迁徙、宏观灾难成为标志性叙事 |
| 2006 | 《三体》在《科幻世界》开始连载 | “科学设定 + 文明史 + 宇宙社会学”进入成熟形态 |
| 2008-2010 | 《三体》三部曲陆续完成中文出版 | 黑暗森林、降维打击、面壁者等概念扩散 |
| 2014 | 《三体》英文版由 Tor 出版，Ken Liu 翻译 | 中文科幻进入英语主流科幻奖项与读者系统 |
| 2015 | 《三体》获 Hugo Award Best Novel | 成为中文科幻国际传播的标志性节点 |
| 2019 | 《流浪地球》电影上映 | 中国科幻电影工业化和巨构视觉的重要节点 |
| 2023 | 《流浪地球2》上映；腾讯剧版《三体》播出；成都 Worldcon 举办 | 刘慈欣作品成为跨媒介、跨产业现象 |
| 2024-03 | Guardian 长访谈随 Netflix《3 Body Problem》播出发表 | 强调科幻跨文化成功不只因“中国性”，而因人类整体尺度 |
| 2024 | China Daily HK 报道其关于 AI、新质生产力、刘慈欣科幻馆等动态 | 技术跃迁与个人作品公共化继续绑定 |
| 2025-03 | 央视网报道中国科幻大会相关观点：长期看 DeepSeek 可能取代科幻小说作家 | AI 写作威胁进入其公开判断 |
| 2025-09 | 中新网“东西问”采访：讨论科幻最大挑战、科技已来到身边等议题 | 最新公开访谈继续围绕技术现实化与科幻挑战 |
| 2026-04 | 本次调研截止 | 未发现 2026 年 4 月前后可核验的新长篇小说正式发布 |
