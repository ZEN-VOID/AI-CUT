# Template: 风格化 / 风格迁移

Template ID: `photoGPT.风格化.风格迁移.v1`

## Input Roles

- 待编辑图：提供主体、构图、场景事实。
- 风格参考图或文字：提供绘画/摄影/材质/色彩/时代风格。

## Prompt Frame

```text
在保持待编辑图中主体身份、构图布局、场景叙事、姿态关系和主要物体位置不变的前提下，将画面视觉语言迁移为指定风格。

风格迁移要求：吸收风格参考中的色彩体系、笔触/颗粒/材质语言、光影对比、镜头质感、时代气质和整体氛围；不要复制风格参考图中的具体人物、文字、logo 或无关场景。

保真锁定：主体五官、发型、服装轮廓、关键道具、场景结构和画面构图严禁漂移。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变主体身份和构图事实。
- 不复制风格参考图中的无关主体。
- 不产生过度重绘、五官漂移或文字乱码。
