# 北京电影学院 Agent Source Map

## 使用原则

- 优先使用你已经点名的十份资料，不额外扩写成泛化网页清单。
- 每条判断都尽量对应一个“知识用途”，而不只是列书名。

## 缺口披露

- 当前这套 source map 主要服务“导演阶段审校与返工”。
- 若未来任务扩展到完整拍摄执行、现场调度或镜头设备参数精算，需要补更细的器材与曝光层证据，不应假装现有十份材料已经覆盖全部细节。
