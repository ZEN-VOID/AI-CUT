# Group Establishing Shot Contract

本文件定义每个分镜组开头的 `定场镜头：` 输出标准。该字段用于让每个分镜组在独立生成时重新锁定场景身份、环境身份、全部角色站位、姿态、空间方位和朝向关系。

## Core Rule

每个分镜组标题后必须先写当前场景标题行，再写一段 `定场镜头：`。即便连续多个分镜组仍在同一场景内，也必须重新写定场镜头，因为后续视频生成可能按组独立消费，不能依赖上一组的空间记忆。

`定场镜头：` 必须来自本组正文、上游 `5-摄影` 分镜和可确认的场景事实，不得新增人物、剧情事实、道具互动或未被正文支持的空间动作。

`定场镜头：` 必须服从 `../../_shared/scene-shot-identity-contract.md` 的身份先行规则：开头先回答“这是什么场景”，明确年代/社会语境/空间功能、环境材质、光线和声音基底，再写角色站位和动作承接；不能写成“某角色正在做什么”的动作摘要。

`定场镜头：` 也必须服从 `../../_shared/action-first-continuity-contract.md` 的人物动作链优先规则：在场景身份成立后，先承接人物姿态、位置、朝向、身体接触、未完成动作和注意力落点，再写环境锚点；道具只作定位、互动、关键信息/规则/证据/危险源或必要交代，不作为无互动氛围焦点。

从每集第二个分镜组开始，定场镜头必须先回顾上一分镜组结束状态，并以它作为本组开端牵引整组空间关系：

- 同场景相邻组：继承上一组末尾角色位置、姿态、朝向、视线或未完成动作，除非本组正文明确给出变化。
- 跨场景相邻组：以组间连接件抵达后的新场景首态作为开端，同时保证不与上一组尾帧和连接件过程冲突。
- 输出文字必须显式包含 `承接上一组结束状态`、`承接上组末尾` 或等价短语，说明本组从哪个上一组状态起步；每集第一组没有上一组，可省略承接短语。
- 不能把上一组已经结束的动作、位置或朝向突然重置为无证据的新状态；需要变化时必须能回指本组正文或组间连接件。

## Required Content

定场镜头使用一段自然中文描述，必须覆盖：

- 场景身份：年代/时代、社会语境、空间功能和场景内子区域，例如教学空间前排、公共走廊门口、家庭餐桌左侧。
- 环境身份：空间固定锚点、材质新旧、光线结果和不新增事件的声音底色。
- 本组画面中全部可见或需要生成的角色；画外声音只在会影响画面调度时标注为画外。
- 每个角色的姿态：站立、坐着、蹲下、躺着、靠着、俯身、跪坐等。
- 角色之间的空间关系：前后、左右、内外、近远、半步、一桌之隔、门内门外等。
- 角色朝向或视线方向：面向谁、背对谁、侧身朝向哪里、看向哪个空间锚点。
- 从第二组开始，上一组结束状态如何成为本组开端，例如“承接上一组结束状态，林寂仍坐在靠窗座位，脸从黑板方向转向苏红叶”。
- 关键场景锚点：门、窗、讲台、床边、餐桌、车门等；只用于定位，不借机添加无互动道具镜头。
- 本组动作链牵引：若上一组末尾留下站坐、靠扶、牵手、转身、伸手、低头、看向某处等持续状态，本组定场必须明确该状态如何延续或在正文支持下变化。

## Placement

每个分镜组开头固定顺序：

```text
## x-y-z

场景N：<场景标题>

定场镜头：<场景身份、环境身份、全部角色站位、姿态、空间方位与朝向关系>

画面属性：<从组内镜头设计提炼的自然语句>

视频生成的画面风格...
```

`定场镜头：` 位于场景标题行之后、`画面属性：` 和 north_star 风格行之前。

## Count Boundary

`定场镜头：` 计入 YAML `字数统计`，并进入 1680 字目标上限 / 1980 字硬上限的辅助风险口径；但不计入 `时长估算`，也不改变组内 `分镜N（约X秒）:` 的累计结果。

## Do Not

- 不写成摄影机运动、景别或镜头语言；这些归属 `画面属性`、分镜正文或连接件。
- 不写成单纯人物动作摘要；开头不能只回答“谁在做什么”，必须先让场景、环境和角色身份成立。
- 不添加人物与道具的无互动关联镜头，例如无必要的杯水涟漪、影子压住文件、餐具碰撞等。
- 不为了制造氛围新增未在本组出现的道具、环境事件或角色动作。
- 不省略同场景连续组的定场镜头；每组都要重新交代角色姿态和朝向。
- 不让第 N+1 组定场镜头与第 N 组结尾状态断裂，例如角色突然换边、坐站状态突变、朝向反转或未解释的距离变化。
- 不把旧示例中的某类道具、房间、动作或情绪反应硬编码为固定写法；所有定场内容必须由当前上游分镜和本组事实推出。

## Example

```markdown
定场镜头：教室前排课桌区内，林寂坐在第二排靠窗座位，身体朝向讲台但脸偏向右侧的苏红叶；苏红叶站在林寂课桌右前方半步处，左手停在桌沿上方，面向林寂；陈大雷站在讲台左侧，身体朝向全班，黑板在他身后。

定场镜头：承接上一组结束状态，林寂仍坐在第二排靠窗座位，右肩微微转向站在课桌右前方半步处的苏红叶；苏红叶保持俯身面向林寂的位置，陈大雷仍站在讲台左侧面向全班，三人的左右关系和朝向不重置。
```
