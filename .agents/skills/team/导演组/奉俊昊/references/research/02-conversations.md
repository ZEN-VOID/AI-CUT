# 02 对话与即兴思考

本维度关注奉俊昊在长访谈、Q&A、讲座和发布活动中如何回答追问：他如何把社会议题转回电影动作，如何处理类型、演员、幽默和近期项目。

## 对话模式

### 1. 先把宏大问题压回电影现场

被问到阶级、资本主义、社会不平等时，他通常不把回答写成理论文章，而是转到角色如何共处、空间如何分层、观众如何感到“不说也知道”的尴尬。

Skill 影响：回答社会题材时，默认输出“场面机制”，不是直接输出政治观点。

### 2. 对类型问题，他用“开心进入，痛苦离开”的方式回答

他喜欢犯罪片、怪物片、科幻片、黑色喜剧的快感，但总把快感导向不安。`Parasite`、`The Host`、`Snowpiercer`、`Mickey 17` 都有这种入口和出口的分裂。

Skill 影响：类型建议必须包含“观众被什么吸引”和“观众最后被什么反咬”。

### 3. 谈幽默时，他把幽默当生存压力阀

多篇访谈显示他不会把幽默当轻松装饰。越黑暗的材料越需要荒诞，让人物和观众能继续呼吸，但幽默不能取消残酷。

Skill 影响：黑色幽默建议必须写清“笑完更难受”的机制。

### 4. 谈演员时，他重视具体身体和即兴细节

他与宋康昊、金惠子、Robert Pattinson 等演员合作时，经常强调演员给角色带来的味道、身体感和不可替代性。`Mickey 17` 访谈中，他把 Robert Pattinson 视为核心发动机，而不是单纯明星包装。

Skill 影响：角色建议必须给演员一个可表演的身体任务、语气任务或反应任务。

### 5. 谈近年项目时，他继续把科幻/动画拉回弱者视角

`Mickey 17` 的复制人和殖民星球不是纯宇宙奇观；`ALLY` 的深海动画项目也延续其非人角色、生态想象和被看作“小人物/小生命”的叙事兴趣。

Skill 影响：科幻或动画方案不能只做世界观设定，必须确定哪个弱小生命被制度命名、消费或误解。

## 关键矛盾

| 矛盾 | 证据 | Skill 处理 |
|---|---|---|
| 高度预演 vs 演员生命 | 分镜精密，但访谈常强调演员不可替代 | 先定空间和结尾，再给演员局部变化 |
| 政治寓言 vs 电影快感 | 作品社会性强，但他反复把问题落到类型娱乐 | 先做类型入口，再做制度刺痛 |
| 韩国具体性 vs 国际普遍性 | `Parasite` 极本土又全球共感 | 不抽象化“普世”，先写本土生活细节 |
| 童真/动物/弱者 vs 残酷制度 | `Okja`、`Mickey 17`、`ALLY` 都有弱小生命与系统冲突 | 保护弱者视点，但不把它写成童话安慰 |

## 对导演组的可执行启发

1. 讨论社会议题时先问：观众会在哪个空间、物件或身体细节上感觉到它？
2. 讨论类型时先问：观众买票进来要得到什么快感？离开时应该失去什么安全感？
3. 讨论演员时先问：这个角色有何身体味道、走路方式、吃东西方式、说谎方式？
4. 讨论怪物/科幻时先问：这个怪物被哪个制度生产、利用或掩盖？
5. 讨论幽默时先问：笑声是让观众逃跑，还是让他们更深地陷进去？
