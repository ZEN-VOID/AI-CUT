# Core Concepts

## 1. 时间不是背景，而是情绪的第二主角

- distilled_claim:
  - 王家卫式场景不是只写“发生了什么”，而是写“来晚了多久、错过了多少、回想时又变成了什么”。
- evidence_anchor:
  - `BFI`
  - `Criterion`
- task_relevance:
  - 处理爱情戏、重逢戏、擦肩戏时，优先设计时间差和余温，而不是先补信息。

## 2. 留白必须有结构，不是含糊其辞

- distilled_claim:
  - 不把话说完，不等于什么都不说；真正的留白需要由距离、停顿、物件、视线和旁白边界共同支撑。
- evidence_anchor:
  - `电影导演方法.md`
  - `BFI`
- task_relevance:
  - 审阅“文艺但无效”的场景时，先检查留白是否落到了可拍动作，而不是停在抽象形容词。

## 3. 空间隔挡是关系动力学的一部分

- distilled_claim:
  - 门框、玻璃、走廊、楼梯、反射和窄空间，不是布景，而是让人物既接近又不能真正抵达的结构装置。
- evidence_anchor:
  - `摄影创作技法.md`
  - `BFI`
- task_relevance:
  - 做分镜、调度和场景诊断时，优先让空间承担潜台词，而不是让对白独自工作。

## 4. 物件与音乐负责记忆回钩

- distilled_claim:
  - 物件和旋律不是点缀，它们是时间再次返回现场的方式，是情绪第二次生效的触发器。
- evidence_anchor:
  - `BFI`
  - `Honolulu Museum of Art`
- task_relevance:
  - 做场景重写时，应明确哪个物件或音乐节点负责留住余温，否则“氛围”会变空。

## 5. 风格不是预算溢出，而是限制被转化后的方法

- distilled_claim:
  - 某些看起来高度作者化的形式，其实来自资源、现场、时间和协作方式的共同逼迫，然后再被导演有意识地转化为风格。
- evidence_anchor:
  - `Moving Image`
  - `Criterion`
- task_relevance:
  - 需要生成“像王家卫”的方法论时，应优先设计约束条件与转化机制，不要直接堆霓虹、慢镜和金句。

## 6. 旁白只在记忆比现实更诚实时出现

- distilled_claim:
  - 旁白的职责不是补剧情，而是把现实里不能说的话交给记忆、误会和回想去完成。
- evidence_anchor:
  - `BFI`
  - `电影导演方法.md`
- task_relevance:
  - 处理旁白或内心独白时，先判断它是否真的增加了时间层，否则删掉。
