# 道具提取

从 BYKJ `02-剧本处理` 输出中提取关键叙事功能道具，并为每个主道具生成 JSON 设计规格。主清单不收录所有物件，只保留对剧情、信息、关系、规则、危险或关键画面有功能的道具。

默认输入：

```text
output/[项目名]/02-剧本处理/
```

默认输出：

```text
output/[项目名]/05-资产提取/道具提取/
├── 道具清单.json
├── 道具提取报告.json
└── manifest.json
```

核心规则：主清单默认只收 `narrative_function_score >= 3` 的道具；普通物件进入剔除列表或不记录。设计规格参考 `aigc/7-设计/道具/2-设计`，但本阶段不默认生成 Markdown 设计稿。
