# 科幻题材模板 (Sci-Fi)

> **核心卖点**: 科技设定 + 伦理冲突 + 宏大想象。在星辰大海中探索人类的未来与边界。

## 1. 核心流派细分

### 赛博朋克流
- **特点**: 高科技低生活，义体改造，巨型企业统治
- **核心爽点**: 黑客入侵、义体升级、反抗体制
- **代表作**: 《赛博英雄传》
- **创作要点**: 赛博朋克的核心不是"科技多炫"——是"科技让贫富差距大到穷人连自己的身体都不再属于自己"

### 太空歌剧流
- **特点**: 星际航行、外星文明、宇宙战争
- **核心爽点**: 星舰战斗、外星探索、文明碰撞
- **代表作**: 《三体》《银河帝国》
- **创作要点**: 太空歌剧不是"太空版中世纪"——要体现星际尺度下的政治、经济、通信逻辑

### 硬科幻流
- **特点**: 科学原理严谨，技术细节真实
- **核心爽点**: 科学解谜、技术突破、逻辑推演
- **代表作**: 《三体》《火星救援》
- **创作要点**: 硬科幻的爽感来自"用科学方法解决问题"——读者学到东西

### 末日科幻流
- **特点**: 科技灾难，人类存亡，文明重建
- **代表作**: 《流浪地球》

### 机甲/战争流
- **特点**: 巨型机甲、星际战争
- **代表作**: 《师士传说》《机动风暴》《超神机械师》

---

## 2. 科技代价（重要）
- **能源消耗**: 高科技需要大量能源
- **副作用**: 义体排斥、基因崩溃
- **伦理问题**: 克隆、意识上传的道德困境
- **技术失控**: AI叛变、武器失控

---

## 3. 追读心理

| 驱动力 | 执行手段 |
| --- | --- |
| 科技想象 | 每个新卷引入一个"令人震撼"的新科技概念 |
| 文明碰撞 | 外星文明必须"真正的外星"——不是"外国人穿外星皮" |
| 伦理张力 | 每个科技突破都引出道德两难——"能做的事不代表该做的事" |

## 4. AI 写作专项指南

### 常见陷阱
1. **科技万能化**: 技术可以解决一切 → 每项技术必须有代价和副作用
2. **外星人=人类换皮**: 外星文明就是"蓝皮肤的美国人" → 设计真正不同的生理/社会/认知结构
3. **科学错误**: 曲率驱动=超光速（实际不是）→ 要么做硬科幻要核实，要么做软科幻不解释细节
