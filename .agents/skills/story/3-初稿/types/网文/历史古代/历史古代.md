# 历史古代题材模板 (Historical)

> **核心卖点**: 大时代背景 + 权谋博弈 + 格局升级。在历史的洪流中，书写个人的传奇与时代的变迁。

- 通用叠加：Pack U01

## 1. 核心流派细分

### 争霸权谋流
- **特点**: 乱世争霸，逐鹿天下
- **核心爽点**: 战略碾压、收服人才、统一天下
- **代表作**: 《覆汉》《三国之最风流》《汉乡》《秦吏》
- **创作要点**: 历史大势不可改，但细节和路径可以重塑——读者知道结局，享受的是"过程"

### 仕途升迁流
- **特点**: 官场沉浮，步步高升
- **核心爽点**: 政治博弈、晋升打脸、权倾朝野
- **代表作**: 《官居一品》《宰执天下》《大明王侯》《上品寒士》
- **创作要点**: 官场规则是隐形的主角——懂规则的人活得好，不懂规则的死得快

### 科技种田流
- **特点**: 用现代知识改变历史
- **核心爽点**: 科技碾压、内政建设、富国强兵
- **代表作**: 《新宋》《唐砖》《赘婿》《回到明朝当王爷》
- **创作要点**: 科技输出有时代限制——不能跳过工业基础直接造蒸汽机

### 军事征战流
- **特点**: 沙场征战，名将之路
- **核心爽点**: 战术碾压、以少胜多、封狼居胥
- **代表作**: 《大唐双龙传》《窃明》《晚明》
- **创作要点**: 战争不止是打——后勤、情报、士气、地形都是变量

## 2. 追读心理与读者留存

| 驱动力 | 执行手段 |
| --- | --- |
| 历史好奇 | 在关键历史节点设置选择——主角选A改变了什么 |
| 身份成长 | 每个卷末必须有明确的"身份升级"或"地盘扩大" |
| 智力满足 | 谋略必须有"卧槽原来如此"的解释感——但前文要有线索 |

## 3. 平台优化建议

- **起点**: 权谋经得起推敲、配角丰满、考据扎实
- **番茄**: 压缩开篇到3章出第一个爽点、权谋适度简化、增加即时打脸

## 4. AI 写作专项指南

### 常见陷阱
1. **现代思维代入**: 主角动不动"人人平等" → 时代思维必须符合历史背景
2. **官职混乱**: 清朝官职用到唐朝 → 每个朝代的官制必须核实
3. **配角工具人**: 历史名人只为服务主角 → 每个历史人物要有独立行动逻辑
