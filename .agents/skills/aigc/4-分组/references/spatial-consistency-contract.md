# Spatial Consistency Contract

本文件定义 `4-分组` 的空间一致性规则，服务后续图像与视频阶段维持同一分镜组内的场面调度、人物方位和运动方向。

## Internal Spatial Anchor Reference

空间锚点是 `站位和位移：` 的内部参照系，不作为分镜组组头字段输出，也不作为最终提示词的独立段落出现。

执行时必须从当前分镜组对应的场景标题、画面字段、镜头语言、道具特写、环境持续物或补位画面中提取 1 个或多个内部空间锚点。优先选择能稳定指示人物位置、镜头方向、运动路径或场景几何关系的参照点，例如门口、窗边、讲台、黑板、走廊尽头、课桌边缘、楼梯口、车门、柜台、床沿、墙角、灯下区域等。

不得把抽象情绪、剧情主题、人物关系或纯风格词写成空间锚点。空间锚点不是道具百科，只保留对站位、位移和画面连续性有帮助的空间参照。

## Position And Movement Line

`站位和位移：` 必须关联到具体分镜明细，而不是只关联到整个 `镜头语言：` 容器。每个 `分镜N:` 下方必须紧跟或就近补入一条非空 `站位和位移：` 辅助行。默认格式为：

```text
分镜N：<原分镜描述>
站位和位移：<明确角色名><站在/坐在/靠在/蹲在/停在/走向/退到/绕过等表示位置关系或运动的动词><空间参照><方位>，<如有移动，说明方向、起点、终点或相对关系>
```

`站位和位移：` 的主语必须是明确角色名或上游已命名的稳定群体称谓。不得使用 `画面主体`、`主体`、`人物`、`角色`、`主角`、`该角色`、`该人物`、`此人`、`他`、`她`、`他们`、`她们` 等含混指代。若同一分镜有多名角色，应逐名说明其相对空间参照的位置或位移。

示例：

```text
分镜1：中景，林深站在车门外侧，苏清坐在车内。
站位和位移：林深站在车门外侧偏右，身体朝向巷口；苏清从车门内侧退到路灯阴影下。
```

若连续分镜的站位完全不变，也必须在对应 `分镜N:` 下方写明明确角色名，例如 `站位和位移：林深沿用分镜1站位，仍在车门外侧偏右，镜头由中景推近至近景。` 这样仍能保留分镜级归属。

`站位和位移：` 是 `4-分组` 新增的空间一致性辅助行；它必须补入对应 `分镜N:` 的明细块内或紧随该分镜明细之后，但不得删除、改写或重排上游 `3-摄影` 原有字段、对白、镜头语言、`分镜N:` 或场景顺序。

## Selection Priority

空间锚点优先级从高到低：

1. 角色发生站位变化、进出、转身、靠近、后退、绕行的真实位置。
2. 影响镜头构图和连续性的固定空间结构。
3. 多个画面反复出现的环境持续物。
4. 出场/入场补位画面使用的同一视觉锚点。
5. 重要叙事道具所在的具体位置。

若同组跨越多个强空间点，应列多个锚点，并在各个 `分镜N:` 对应的 `站位和位移：` 中说明角色与这些锚点的相对关系。

## Word Count Handling

`站位和位移：` 辅助行计入 1980 字限制和 `字数统计`。内部空间锚点不作为单独输出字段计入字数。若加入站位位移后超限，应缩小分镜组正文边界，而不是删除站位位移、摘要 north_star 或改写上游正文。
