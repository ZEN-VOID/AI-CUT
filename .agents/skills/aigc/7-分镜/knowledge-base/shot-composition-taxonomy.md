# Shot Composition Taxonomy

本文件是 `7-分镜` 的人工维护摄影构图知识库。它只提供可选构图词表、适用条件和误用边界；不替代 `SKILL.md` 的节点、gate、输出合同或 LLM 对叙事功能的判断。

## Selection Protocol

每条分镜先确定 `shot_narrative_function`，再选择 `composition_form`。

1. 选择 1 个 `primary_composition`，必须来自下表。
2. 可追加 1 个 `secondary_modifier`，用于前景、层次、视线、运动或空间压力。
3. 若确需使用表外构图，必须在报告或内部计划记录 `controlled_exception`：`term`、`reason`、`nearest_taxonomy_item`、`why_existing_item_not_enough`。
4. 成稿中的构图形式应使用中文可读名，例如 `三分法构图`、`对角线构图`、`框架式构图`；不要输出表格 ID。
5. 不得把情绪词、剧情动作、场景物件或摄影设备词当作构图形式。

## Primary Composition Taxonomy

| taxonomy_id | composition_form | best_for | avoid_when |
| --- | --- | --- | --- |
| `COMP-01` | 三分法构图 | 稳定叙事、人物与环境并读、视线留白、常规对话起点 | 高压中心凝视、极端制度感、需要强对称秩序 |
| `COMP-02` | 黄金比例构图 | 主体与陪体有优雅流向、空间层次渐进、情绪缓慢聚焦 | 快速冲击、信息必须直给、构图空间过窄 |
| `COMP-03` | 黄金螺旋构图 | 视线需要沿曲线进入关键主体、悬疑揭示、道具引向人物反应 | 群像布局、直线运动、空间缺少弧形或层级路径 |
| `COMP-04` | 中心构图 | 权力压迫、凝视、仪式感、规则显影、单主体强强调 | 需要环境信息并读或人物关系对照 |
| `COMP-05` | 对称构图 | 制度、秩序、神圣感、镜像关系、审判或仪式场面 | 情绪失控、混乱运动、自然主义偷拍感 |
| `COMP-06` | 对角线构图 | 动作推进、危险倾斜、冲突张力、空间切割 | 画面需要平稳陈述或静态规则展示 |
| `COMP-07` | 引导线构图 | 走廊、道路、光带、队列、建筑线条把视线带向主体 | 背景线条杂乱或会误导到非关键物 |
| `COMP-08` | 框架式构图 | 门窗、屏幕、镜子、栏杆形成窥视、隔离、身份困局 | 框架物无叙事意义或遮住关键表演 |
| `COMP-09` | 三角构图 | 三人关系、权力层级、稳定冲突结构、群像焦点 | 双人平行对话或需要弱化关系结构 |
| `COMP-10` | 前景遮挡构图 | 偷窥、压迫、危险临近、隐藏信息、空间纵深 | 关键动作/线索必须完全可读 |
| `COMP-11` | 留白构图 | 孤独、等待、未知压力、渺小感、信息延迟 | 快节奏动作或多信息必须同屏读清 |
| `COMP-12` | 深景分层构图 | 前中后景都有叙事任务、追踪关系、空间权力和危险源 | 下游视频模型易混淆多主体或单镜只需单一注意力 |
| `COMP-13` | 封闭式构图 | 困局、压迫、室内密闭、心理围困 | 需要开放前路、逃离感或广阔空间 |
| `COMP-14` | 开放式构图 | 未知延展、行动余地、场外威胁、人物将进入/离开 | 需要明确边界、规则或审判感 |
| `COMP-15` | 低角度仰拍构图 | 权势、威胁、英雄化、主体压迫观众 | 脆弱、细腻反应或需要环境俯视关系 |
| `COMP-16` | 高角度俯拍构图 | 脆弱、被控制、空间关系、群体调度、制度俯视 | 需要主体平等凝视或情绪亲近 |
| `COMP-17` | 过肩构图 | 对话关系、视线权力、监听、对峙中的主客位置 | 单人心理反应或需要完全隔离主体 |
| `COMP-18` | 反射构图 | 镜子、水面、玻璃、屏幕中的身份分裂、真相错位 | 反射来源无叙事意义或易造成空间误读 |
| `COMP-19` | 剪影构图 | 身份隐藏、命运感、强轮廓、情绪压缩 | 面部表演或道具细节必须可读 |
| `COMP-20` | S形曲线构图 | 视线柔性流动、道路/河流/人群动线、缓慢引导 | 紧张直冲、硬边冲突或空间没有曲线路径 |
| `COMP-21` | 放射/汇聚构图 | 所有线条指向中心事件、规则显影、爆点聚焦 | 需要自然主义或避免过强视觉设计感 |
| `COMP-22` | 不平衡构图 | 危机、失控、悬疑偏压、人物处于劣势 | 信息说明性镜头或需要稳定权力结构 |

## Secondary Modifiers

| modifier | use |
| --- | --- |
| `foreground_pressure` | 前景物压迫主体，但不遮挡关键表演 |
| `negative_space_direction` | 留白方向承接视线、威胁、出口或未知信息 |
| `layered_depth` | 前中后景分别承担主体、陪体、背景信息 |
| `eye-line_balance` | 通过视线方向和画面空白组织对话关系 |
| `motion_vector` | 让入画、退场、追逐、转身有明确画面方向 |
| `light_shape` | 用亮面、暗面、轮廓光或投影形成构图秩序 |
| `color_blocking` | 用色块区分阵营、空间或情绪，但不得替代画面基调 |

## Narrative Function Fit

| narrative_function | preferred_compositions |
| --- | --- |
| `reveal` | 引导线构图、框架式构图、深景分层构图、放射/汇聚构图 |
| `conceal` | 前景遮挡构图、框架式构图、反射构图、剪影构图、留白构图 |
| `emphasize` | 中心构图、对称构图、低角度仰拍构图、剪影构图 |
| `compare` | 三角构图、对称构图、过肩构图、深景分层构图 |
| `misdirect` | 反射构图、前景遮挡构图、不平衡构图、开放式构图 |
| `connect` | 三分法构图、黄金比例构图、S形曲线构图、引导线构图、过肩构图 |

## Invalid Composition Forms

以下写法不得作为 `composition_form`：

- `压迫感构图`，除非转写成中心构图、封闭式构图、前景遮挡构图或低角度仰拍构图。
- `电影感构图`、`高级构图`、`氛围构图`、`悬疑构图` 等泛化审美词。
- `主角站在门口`、`灯光打在脸上`、`镜头跟着移动` 等画面描述或运动描述。
- `长焦构图`、`广角构图`、`f/1.4` 等镜头/设备/参数词；这些可影响空间感，但不是本字段的构图方式。

## Output Examples

合法：

```text
中景，浅景深，框架式构图，主角被门框压在画面右侧三分之一，左侧冷光门缝作为前景压力，后景走廊虚化成灰白纵深
```

合法：

```text
全景，深景深，引导线构图，地面灯带从画面下方汇向远处审判台，主角站在汇聚点前，背景屏幕亮出规则文字
```

不合法：

```text
近景，浅景深，压迫感构图，主角很紧张，背景很高级
```
