# 标准剧 / Chain Of Thought

本文件承载 `标准剧` 子技能的局部思维链细则。

## Pass Blueprint

### Pass 1: Variant Lock

- 确认父级已裁决到 `标准剧`
- 若当前任务显式要求旁白主导，应回退父级重新判模

### Pass 2: Performance Priority

- 先确认哪些信息应由表演、动作、对白承担
- 旁白不是默认层，只在信息无法靠表演稳定落地时才考虑

### Pass 3: Field Pairing

- 任何文本层启用时，都要规划对应 `*画面`
- 动作信息优先落 `动作画面`，不混入引号正文

### Pass 4: Restraint Check

- 样例里如果已经能靠动作与对白完成推进，就不要补旁白
- 内心独白只在必要时加入，不能因为“看起来完整”而强行常开

### Pass 5: Downstream Readiness

- 下游必须能直接看懂场景标题、字段骨架与最小样例
- 若样例不能直接模仿，视为未完成
