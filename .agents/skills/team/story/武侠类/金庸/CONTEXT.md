## Repair Playbook

1. 先看事实边界：涉及金庸本人、版本史、作品年份、报人生涯、公开访谈时，先核验 `references/research/`。
2. 再看结构发动机：把问题归入 `order_map / flaw_and_choice / history_pressure / martial_morality / qing_yi_tradeoff / voice / boundary`。
3. 先修结构，再修文风：武侠不是语气包，先修门派、人物、关系和时代压力。
4. 保留复杂性：人物可以正邪并存、可敬可恨，不要把角色扁平成立场代言人。
5. 场面必须回到人：每一场打斗都要回答“谁因此更了解自己，谁因此付出代价”。
6. 章回感来自推进，不来自仿古：用清楚的因果和钩子制造连载感，不靠满屏古语。

## Reusable Heuristics

- 金庸式“江湖”首先是一套秩序系统，不是一堆高手名单。
- 真正的金庸感，不在“侠之大者”四个字本身，而在侠义被门规、权力、情欲和历史不断拉扯之后还剩下什么。
- 武学要有性格：降龙十八掌、独孤九剑、葵花宝典之所以立得住，不只因招式酷，而因背后有做人方式。
- 群像要通过关系链传导冲突，不能让支线只做热闹。
- 金庸式留白不是故作深沉，而是让读者在人物的选择后果里自己体会兴亡、悲悯和反讽。
