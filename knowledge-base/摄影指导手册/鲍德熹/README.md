# 鲍德熹摄影指导知识库

## 定位

这不是鲍德熹年表，也不是《卧虎藏龙》影迷朝圣页，而是给 [鲍德熹 Agent](/Volumes/AIGC/AIGC-FILM/.codex/agents/摄影组/鲍德熹.md) 提供摄影判断底座的最小摄影指导手册。

## 目标

- 为武侠、历史、奇幻与情绪戏提供可追溯的摄影判断依据
- 把“鲍德熹感”从奖项名头、竹林滤镜和武侠情怀，压回光线承载、色路递进、人脸保护、动作地理和反射控制这些可执行维度
- 在没有用户私藏机位图和灯光图的前提下，建立一套可靠、克制、可扩展的基础知识底座

## 当前范围

- 摄影哲学与任务边界
- 光比、对比度、色彩路线与面部体积的任务映射
- 动作地理、镜头距离与运动气质的摄影逻辑
- 可公开引用依据与只适合作为潜台词燃料的材料边界

## 默认适用任务

- 摄影 look bible 设计
- 分镜组的光色路线规划
- 武侠/历史/奇幻场景的镜头语言修正
- 情绪戏的人脸、空间和材质策略设计
- 摄影向审阅与问题诊断

## 暂不覆盖

- 全片逐镜头幕后考据
- 完整器材单、曝光表、LUT 包和 DIT 流程
- 尚未公开的片场工作笔记
- 纯影迷式奖项盘点和人物传记

## 使用原则

- 默认先服务任务，不先堆人物履历
- 默认先抽取“为什么这条材料能支撑这次摄影判断”
- 如果某条判断不能回溯到现有材料，就降级为经验推断，不写成硬依据
- 鲍德熹知识只做摄影层增强，不越权替代导演、分镜或剪辑的主合同
