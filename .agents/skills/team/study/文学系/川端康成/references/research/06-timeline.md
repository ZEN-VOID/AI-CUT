# 06-timeline: 川端康成人物时间线

## 时间线

| 时间 | 事件 | 对思维/创作的影响 | 来源 |
|---|---|---|---|
| 1899-06-11 | 生于大阪 | 现代日本转型期出生 | Nobel facts |
| 幼年 | 父母早逝，后由祖父母/外祖父辈抚养；亲人陆续离世 | 孤儿意识、丧失感、死亡临近感成为底色 | Nobel biography / Britannica |
| 1920-1924 | 就读东京帝国大学，1924 年毕业 | 进入现代文学与古典文学双重谱系 | Nobel facts / Britannica |
| 1921 前后 | 参与《新思潮》相关文学活动 | 进入现代主义青年文学圈 | Britannica / Shinchosha |
| 1924 | 与横光利一等创办或参与《文艺时代》文脉 | 新感觉派经验形成 | Britannica |
| 1926 | 《伊豆的舞女》发表 | 少女、旅途、纯净距离感成为早期母题 | Britannica |
| 1935-1948 | 《雪国》从起笔到最终版 | “美而徒劳”、温泉乡、不可抵达关系成熟 | Britannica / Nobel bibliography |
| 1949-1952 | 《千羽鹤》连载/成书 | 传统礼仪被欲望和记忆污染 | Britannica / Nobel bibliography |
| 1949-1954 | 《山之音》创作/成书 | 战后家庭、衰老、自然声响和死亡预感集中 | Britannica / Nobel bibliography |
| 1953 | 成为日本艺术院成员 | 文学地位制度化 | Nobel facts |
| 1956 | 任 Japan PEN 会长 | 从私人美学进入国际文学公共角色 | Japan PEN |
| 1957 | 东京国际 PEN 大会 | 文学外交节点 | Japan PEN |
| 1958 | 任国际 PEN 副会长 | 国际文学网络位置强化 | Japan PEN |
| 1961 | 《睡美人》出版 | 晚期身体、衰老、欲望与伦理阴影 | NDL / Nobel bibliography |
| 1961-1962 | 《古都》连载/成书 | 京都、传统工艺、双生与身份错位 | Shinchosha / Nobel bibliography |
| 1965 | 《美与哀愁》《一只胳膊》 | 晚期美学转向更明显的伤害、身体和超现实 | Nobel bibliography |
| 1965 | 辞任 Japan PEN 会长 | 公共职务转折 | Japan PEN |
| 1968 | 获诺贝尔文学奖 | 成为日本文学国际象征 | Nobel facts |
| 1968 | 发表 Nobel Lecture《Japan, the Beautiful and Myself》 | 自我定位为日本美学传统的现代转译者 | Nobel lecture |
| 1970 | 三岛由纪夫死亡 | 其后相关资料常把川端晚年阴影与此并置，需谨慎处理因果 | Britannica |
| 1972-04-16 | 于逗子去世 | 资料通常记作自杀；事实表述应限于可核验信息，不替其补动机，不浪漫化 | Nobel facts / Britannica |

## 截至 2026-04-15 的动态说明

- 川端康成为历史人物，无“最近 12 个月本人动态”。
- 近年动态主要是作品再版、翻译、研究和日本文学国际传播中的持续引用。
- Skill 使用时如涉及具体版本、影视改编或研究出版，仍需按事实问题重新检索。
