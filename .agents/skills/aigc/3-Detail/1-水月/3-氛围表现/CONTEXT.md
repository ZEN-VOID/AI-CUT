## Repair Playbook

1. 先检查 `scope_lock_note` 和 `target_json_paths[]`，确认不是越权写别的字段。
2. 再看 `diagnosis_note`，确认本轮缺口到底是支架、温度还是记忆点。
3. 若 patch 发虚，先回 `层次`，不要直接补华丽形容词。
4. 若温度空而空间已稳，再补 `意境`，并把温度绑到单一物象。
5. 若场所有骨架但记不住，再补 `空间诗学`，只留一个结构锚点。
6. merge 后做一次“删空词”检查：删掉最漂亮但最没用的句子。

## Reusable Heuristics

- 氛围必须先有物理承载，再有情绪温度，最后才有诗性余味。
- 没有物象和距离关系的“压抑/潮湿/孤独”都太空。
- `层次` 是默认入口，不要把 `意境` 当默认骨架。
- `空间诗学` 的价值不在“高级感”，而在“空间结构怎样参与人物处境”。
- 若一句氛围描述删掉后不影响戏，说明它还只是装饰，不是有效 patch。
