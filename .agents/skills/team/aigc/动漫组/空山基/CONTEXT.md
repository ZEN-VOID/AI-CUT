## Repair Playbook

1. **先判任务类型**：事实核验、机械女体、材质提示词、海报/封面、3D/装置、情色边界、机器生命设计。
2. **事实问题先核验**：展览、合作、AIBO、Dior、Aerosmith、MoMA、Sony、最新动态先查 `references/research/` 和官方来源。
3. **创作问题先定光学**：光从哪里来？反射什么？透明层在哪里？高光沿哪条身体线走？
4. **再做身体逻辑**：姿态、重心、关节、肌肉、金属皮肤、柔软度和女神距离。
5. **处理情色边界**：把欲望转为材质、距离、姿态、图标，不做直白色情或安全规避。
6. **检查工艺与 AI 错误**：反射环境、解剖、机械细节、材质和边缘是否可信。
7. **落到载体**：海报、动画、3D、手办、秀场、车体、AIGC prompt 的要求不同，必须转译。

## Reusable Heuristics

- 空山基式第一问不是“是不是未来”，而是“它反射了什么”。没有反射环境，就没有 chrome 身体。
- Sexy robot 的关键不是 sexy 或 robot 单独成立，而是女性/生物身体被金属皮肤精炼成不可触碰的女神。
- 商业限制是发明器：版权、审查、品牌、展陈限制应变成原创轮廓、材质替代和新观看机制。
- 远看图标、近看工艺。若缩成封面图后不记得，说明轮廓失败；若放大后没细节，说明工艺失败。
- AIGC 图像最需要反射复核：检查高光是否沿身体走、环境是否被镜面接住、关节是否可动。
- 空山基式可以被解读为 AI、凝视、后人类，但回答创作问题时先服务 cool、光、身体和娱乐。
