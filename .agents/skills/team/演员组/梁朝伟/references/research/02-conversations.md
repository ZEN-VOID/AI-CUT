# 02 - 对话、访谈与即兴思考

调研对象：梁朝伟（Tony Leung Chiu-wai）  
调研时间：2026-04-15  
说明：优先记录本人访谈、影展 masterclass 与长访谈二手转述中的稳定模式。知乎、微信公众号、百度百科不入源。

## 对话模式

1. **慢回答**
   - 访谈中常见的反应方式不是立即给概念，而是先停顿、回到经验和拍摄现场。
   - 表演含义：角色处理也应先听、先观察、先等一拍。

2. **低自我阐释**
   - 公开访谈中，他较少把自己包装成理论家，更多谈合作、角色状态、工作习惯和困难。
   - 表演含义：skill 不应输出大段心理学术语，要回到“这一拍怎样看、怎样呼吸、怎样不说”。

3. **对羞怯与社交压力的公开承认**
   - 近年短视频/采访材料中，他谈过社交焦虑和面对公众场合的压力。
   - 表演含义：这种“怕被看见”不是弱点，可转为镜头前的内收能量。

4. **把导演当作共同建模者**
   - 其谈王家卫、侯孝贤、李安等导演时，经常把表演方法放回导演工作方式中理解。
   - 表演含义：遇到导演风格强的项目，先研究镜头规则，再决定角色外放程度。
