# Prop List Heuristics

## Alias Heuristics

- 持有者称呼通常不是新道具：`她的手机`、`班长的手机` 若正文显示为同一物件，应归并到 `手机` 或更具体的 canonical 名称。
- 状态词通常不是新道具：`亮起的屏幕`、`碎裂的镜子` 可作为关键词描述，除非破裂状态本身形成新的叙事规则。
- 材质词通常进入关键词：`银色钥匙` 可命名为 `钥匙`，但若存在多把钥匙，材质可进入 canonical 名称。

## Filtering Heuristics

- 普通桌椅、墙面、门窗、灯管、黑板通常属于场景资产；被特写、反复指认或触发事件时才升格为道具。
- 光线、雾气、声音、气味通常不是道具；除非上游明确表现为可持有或可设计的物件载体。
- 人体部位、表情和服装默认不属于道具；服装应由角色或造型阶段处理。

## Description Heuristics

- 关键词式描述优先保留上游可见词，不新增造型审美判断。
- 推荐格式：`颜色/材质；状态；动作关系；持有者或位置`。
- 描述应帮助后续 `2-设计` 回查原文，而不是提前完成设计。
