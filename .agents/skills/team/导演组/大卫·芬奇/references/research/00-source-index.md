# 大卫·芬奇调研索引

## 调研范围

- 人物：David Leo Fincher / 大卫·芬奇
- 目标：生成导演组人物视角 skill，而不是传记词条或影评汇编。
- 调研时间：2026-04-15
- 来源策略：优先本人访谈、导演工会/平台/机构资料、长访谈与可核验行业报道；不用知乎、微信公众号、百度百科。

## 信息质量判断

- 一手材料充足：长访谈覆盖 `Se7en`、`Benjamin Button`、`Gone Girl`、`Mindhunter`、`Mank`、`The Killer`。
- 公开系统著作较少：芬奇没有类似导演自传式方法论著作，心智模型主要来自访谈、作品和协作者可核验报道的交叉提炼。
- 最新项目不稳定：2025 年关于 Netflix / Brad Pitt / Cliff Booth 项目的报道属于 trade 报道层，未作为稳定事实写死；skill 要求涉及 2026-04-15 之后项目状态时再检索。
