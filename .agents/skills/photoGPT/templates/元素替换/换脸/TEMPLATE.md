# Template: 元素替换 / 换脸

Template ID: `photoGPT.元素替换.换脸.v1`

## Input Roles

- 图一：待编辑原图，锁定服装造型、姿态动作、构图布局、背景环境和镜头关系。
- 图二：身份参照图，提供面部五官、发型发色、肤色、年龄气质和角色身份。

## Prompt Frame

```text
保留图一中的服装造型、姿态动作、构图布局、背景环境、镜头视角和光影关系完全不变，仅将图一中角色的面部五官、发型发色、肤色质感、年龄气质和身份特征替换为图二中角色的形象特征。

形象替换要求：完整还原图二角色的面部轮廓、五官比例、发型发色、肤色质感和整体气质；替换后的角色需自然融入图一的服装、姿态、光线和场景，不产生拼贴感。

服装与场景锁定：图一中的服装剪裁、面料质感、色彩配色、装饰细节、姿态、背景和构图严禁改变。

补充要求：<user_desc>
```

## Negative Constraints

- 不改变图一服装、姿势、背景、镜头和道具。
- 不混合两张脸造成五官漂移。
- 不改变年龄/性别/妆容，除非用户明确要求。
- 不产生面部边缘、肤色、光照不一致。
