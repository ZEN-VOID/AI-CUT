# Output Template

## Output Contract Alignment

### Required output

- `OUTPUT-GROUP-EPISODE`: `projects/aigc/<项目名>/5-分组/第N集.md`
- `OUTPUT-GROUP-REPORT`: `projects/aigc/<项目名>/5-分组/执行报告.md`

### Output format

- 逐集文件：Markdown 分镜组稿。
- 执行报告：Markdown 报告。

### Output path

```text
projects/aigc/<项目名>/5-分组/
├── 第1集.md
├── 第2集.md
└── 执行报告.md
```

### Naming convention

- 逐集分组稿：`第N集.md`
- 报告：`执行报告.md`
- 分镜组 ID：`x-y-z`

### Completion gate

- 输入摄影稿与 `north_star.yaml` 可回指；项目存在初始化综合时，已只读消费 `team.yaml.init_synthesis.stage_seed_summary."5-分组"`、`init_handoff.grouping_seed` 或 `north_star.yaml.创作阶段不变量.分组`，形成 `init_team_synthesis_context`，且未触发 team 身份、旧 stage profile 或伪顾问问答。
- 每组标题后先写当前场景标题行，例如 `场景1：外景 扶桑战船外舷与黑礁 - 夜`；即便未切换场景，新的分镜组也必须重复同一个场景标题。场景标题行下方立即输出字段标题 `全局风格：`，字段内三行内容分别为：第 1 行以固定前置词 `视频生成的画面风格，光影和氛围与场景参照图保持一致。需要生成现场物理互动音效、氛围感音效、环境声、自然现象声、动作声，不要生成任何字幕，不要生成背景音乐。` 开头，后接根据当前分镜组从全局风格母稿抽取的匹配风格句，300 字以内；第 2 行直引 `类型元素.类型元素提示词`；第 3 行直引 `细分风格.画面风格`。
- `全局风格：` 之后写 `画面风格：`，从组内镜头设计提炼构图布局核心选择、构图方式关键子维度、光源效果、色彩基调、关键摄影参数和观看位置；`画面风格：` 必须位于第一个原画面性字段标题上方，不再输出旧字段 `画面属性：`、`画面构图：`、`分镜画面：` 或 `左侧：` / `中间：` / `右侧：` / `前景：` / `中景：` / `背景：` 等位置细节字段。
- `画面风格：` 之后进入原正文第一个原画面性字段标题，该字段下方第一行必须直接是普通 `[0-N秒]` 时间码分镜行；首组自然整理本组开始画面，第二组起按回龙帧口径完整代入上一组结尾状态画面点内容，只调整景别和镜头视角进入本组；若该画面点来自对白画面、独白画面、旁白画面或音效画面，必须同步带入对应声音内容；不得输出 `分镜画面：`、`增补首帧：`、`回龙帧：`、来源说明或规则说明。
- YAML `字数统计` 必须计入分镜组标题后的场景标题行、`画面风格：` 和分镜剧本正文；这些组头描述不计入 `时长估算`；字段标题 `全局风格：` 及其三行 north_star 内容、YAML fenced block 本身均不计入 `字数统计`。
- 每组以连续原画面性字段总时长为主：裁决边界时每个上游画面字段取最后一个 `[起始秒-结束秒]` 的结束秒，组内相加后优先接近 15 秒；落盘到分镜组后，原分散画面字段的时间码必须整合为当前分镜组基准下连续递增的 `[N-N秒]`，后一个时间段起点必须等于前一个时间段终点，YAML `时长估算` 取最后结束秒。通常约 12-18 秒可接受；单组时长不得超过 18 秒，超过 18 秒必须拆分、重组或回退 `4-摄影` 修复，不能写完整性例外理由放行；字数和对白只作辅助风险复核。
- 不输出 `## <上一个分镜组ID>~<下一个分镜组ID>` 组间连接件，也不输出 `连接类型：`、`连接方法：`、`变化过程：`、`主体运动：`、`运镜设计：`、`透视适应：`、`避免元素：` 等连接件字段；相邻组承接只通过下一组第一个普通 `[0-N秒]` 分镜行完成。
- 正文同步 `4-摄影` 原换行且未改写。
- 执行报告同时记录初始化综合消费来源与采纳点、每组 `时长估算`、回龙帧结尾状态画面点完整代入、镜头调整和声音承托同步检查、validator error/warning 与 `boundary_review`、`global_style_review`、`visual_style_review`、`first_storyboard_continuity_review`、`connector_removal_review`、`faithfulness_review`、`statistics_evidence_review` 语义结论。
