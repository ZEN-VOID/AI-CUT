# 04 他者视角与批评：程小东

调研时间：2026-04-15

## 外部共识

来源类型：以机构材料和外部评论分析混合；二手评论仅用于识别行业共识与批评风险。

### 1. 香港 wire-fu / 新武侠动作美学的关键人物

AFA 与 BAFTA 等机构资料都把程小东放在香港动作电影与武侠动作设计核心位置。他的代表性贡献不是传统拳脚写实，而是把威亚、舞蹈、京剧身段、剪辑速度和奇幻空间融合成“飞翔的武侠”。

### 2. 与徐克系统共同塑造 1980-1990 年代新武侠

《倩女幽魂》《笑傲江湖》《东方不败》《新龙门客栈》等作品常被视为香港新武侠/神怪浪漫的重要样本。徐克访谈也说明《倩女幽魂》片场中两人经常讨论场面如何成立，显示程小东不只是“执行打斗”，而是参与视听场面构成。

### 3. 可接入张艺谋式色彩与仪式系统

《英雄》《十面埋伏》《满城尽带黄金甲》显示程小东能够把香港武指的速度和威亚系统转译进张艺谋的颜色、仪式、舞蹈、群体阵列与国际大片语境。

### 4. 动作风格影响国际观众对华语武侠的想象

外部评论常把其动作与“飘逸”“浪漫”“高飞”“视觉诗性”绑定。这个优点同时带来误读风险：观众可能只记住飞翔和慢镜，而忽略其动作背后的节奏、落点、安全和人物关系。

## 批评与风险

| 批评类型 | 具体表现 | Skill 处理方式 |
|---|---|---|
| 飞翔空转 | 只剩威亚奇观，人物关系不变 | 每个飞行动作必须绑定关系变化和情绪拍 |
| 过度飘逸 | 危险感被美化掉 | 在关键打击处加入重量、停顿、落地和受伤反馈 |
| 现场不可拍 | AIGC 或分镜只写漂亮运动，不写执行 | 输出必须包含威亚点、替身、安全、镜头和剪辑护栏 |
| 风格套用 | 所有项目都拍成《倩女幽魂》或《英雄》 | 先判断项目是神怪、写实、战争、喜剧还是寓言，再调动作物理 |

## 可运行提炼

1. 程小东视角的优势：把动作变成诗性空间、情绪节奏和奇幻物理。
2. 程小东视角的盲点：容易被使用者误解成“只要飞得漂亮”。
3. 使用该 Skill 时，必须同时启动 `flight_physics` 与 `landing_weight`，让飘和重同时存在。
