# Field Routing And Audio-Visual Contract

## Audio-Visual Pairing

声画一体性是强制门禁。文本与声音条目必须就近配对同命题画面字段：

| 声音字段 | 必须配对的画面字段 |
| --- | --- |
| `对白（角色）` | `对白画面` |
| `独白（角色）` | `独白画面` |
| `内心独白（角色）` | `内心独白画面` |
| `旁白（主体）` | `旁白画面` |
| `音效（来源）` | `音效画面` |

字段纯度：

- `对白/独白/内心独白/旁白/音效` 只写可听见、可说出或可被声音设计呈现的文本/声音本体。
- 对应 `*画面` 只写可见画面、表演、空间或承托。
- 画面字段不得重复解释声音文本。
- 声音字段不得塞入动作描写。
- `*画面` 字段不得写抽象概念、心理结论、主题解释、比喻结论或任务说明；必须落到可见的身体、空间、道具、文字、光线、运动或表情。
- `表演提示` 也必须可执行：写演员可做的身体任务、目光任务、呼吸、停顿、动作力度和对手关系；不得写“他似乎拿到底牌”“幸运还是诅咒”这类小说式判断。

## Required Subject Marking

- `对白（林寂）：“这什么情况？”`
- `独白（林寂）：“国运？副本？SSS级？”`
- `内心独白（林寂）：“她发现了。”`
- `旁白（系统提示）：“天道系统·全球公告。”`
- `音效（高跟鞋）：“嗒、嗒、嗒。”`

所有对白、内心独白、旁白都必须显式带主体。音效必须显式带来源；来源不可见时可写 `音效（不可见来源）`。

## Sound Literal Rule

声音字段必须写声音本体，而不是事件说明、时间说明或叙述概括。

| 错误 | 原因 | 修正 |
| --- | --- | --- |
| `音效（铃声）：“八点四十分铃声。”` | 这是时间说明，不是声音 | `音效（铃声）：“叮铃铃——”`，时间写入 `音效画面` 或上下文 |
| `音效（走廊外）：“一声短促的闷响。”` | 这是描述性句子，不是可听声音本体 | `音效（走廊外）：“呃。”` 或 `音效（走廊深处）：“咚……咚。”` |
| `音效（苹果）：“尖叫声。”` | 声音类别，不是声音表现 | `音效（苹果）：“咿——！”` |
| `对白（某人）：“他沉默了。”` | 叙述被误作对白 | 若上游不是直接引号对白，改为 `动作画面` 或 `表演提示` |

对白字段只承接上游明确引号对白，或用户明确授权新增的对白。不得把非引号叙述、概括性说明、作者判断转为对白。若上游原文已有引号，例如 `“普通反应。”`，则按对白冻结保留。

## Field Routing Table

| 原文内容类型 | 优先字段 | 处理要求 |
| --- | --- | --- |
| 角色可见动作、位移、手部动作、姿态变化 | `角色动作` / `动作画面` | 改写成可拍摄的短句，避免“他意识到/他记起/他感觉”等小说内视 |
| 教室、走廊、光线、窗帘、空气压迫、空间状态 | `环境描写` | 服务氛围与后续美术，不混入人物心理解释 |
| 高跟鞋、铃声、苹果尖叫、机械提示、走廊异响 | `音效` + `音效画面` | `音效` 写声音来源、质感、节奏和空间方位；`音效画面` 写声音触发时的可见源头、人物反应、空间承托或不可见来源的画面处理 |
| 苹果、教鞭、课本、黑板文字、血字、白纸 | `道具特写` / `规则显影` | 让道具成为规则信息载体 |
| 天道系统、真理之眼、好感度、死亡通报 | `系统画面` / `规则显影` / `旁白（系统提示）` | 文字信息可由旁白承载，画面字段负责可视化呈现 |
| 群体反应、全班凝固、集体低头、哭声扩散 | `群像画面` | 用可见行为表达群体心理 |
| 角色主观判断、恐惧、推理、记忆回闪 | `独白` / `心理反应` / `表演提示` | 可转独白或表演提示；不要塞进 `动作画面` |
| 角色服装、色彩构成、危险气质、非动作造型信息 | `角色造型` / `表情特写` | 服务造型、美术和表演，不作为动作句处理 |
| 国运灾难、新闻碎片、现实世界同步后果 | `现实灾难画面` / `系统画面` | 作为短促压力插针，不挤占主叙事 |

## Concrete Field Rewrite Examples

| 不合格字段 | 问题 | 合格投影 |
| --- | --- | --- |
| `道具特写：字迹确实是他的...他从来不这样写字。` | 后半句是主观判断 | `道具特写：扉页写着“林寂”；“寂”字最后一撇异常拉长。` + `心理反应：林寂盯住那一撇。` |
| `系统画面：规则测试开始。` | 任务说明，不是画面 | `系统画面：黑板中央浮出“课堂纪律条例”标题前的冷白光点。` |
| `环境描写：第二节课是课堂作业。` | 课程说明，不是环境 | 删除，或写 `动作画面：苏红叶把白纸发到每张课桌。` |
| `系统画面：没有字数限制。` | 规则文本被误放进系统画面 | `旁白（系统提示）：“没有字数限制。”` + `旁白画面：白纸题目下方留着大片空白。` |
| `对白画面：陈大雷语气压低，却有一种正面摊牌的爽快。` | 感受性判断 | `对白画面：陈大雷身体前倾，手掌摊开压在桌面，眼睛直视林寂。` |
| `表演提示：他似乎拿到一张别人没有的底牌...` | 小说式抽象 | `表演提示：林寂不抬头，指腹压住课本边角，眼神保持余光警戒。` |

## Allowed Field Titles

允许但不限于以下正式字段标题：

- `环境描写`
- `角色动作`
- `动作画面`
- `群像画面`
- `表情特写`
- `角色造型`
- `道具特写`
- `系统画面`
- `规则显影`
- `现实灾难画面`
- `对白（角色）`
- `对白画面`
- `独白（角色）`
- `独白画面`

- `内心独白画面`

## Bad Patterns

- `动作画面：林寂意识到自己拿到了底牌。`
- `系统画面：规则测试开始。`
- `环境描写：永夜中学的第一节课，在压抑的气氛中继续。`
- `心理反应：因为没有人敢在课堂时间缺席。`
- `对白（苏红叶）：“欢迎来到这里。”她微笑着看向全班。`
- `对白画面：苏红叶说欢迎来到永夜私立中学。`
- `音效：大家害怕得不敢说话。`
- `音效（铃声）：“八点四十分铃声。”`
- `场景2：苏红叶入场震慑全班`

## Good Patterns

```md
音效（高跟鞋）：“嗒、嗒、嗒。”
音效画面：教室前门外，红色细高跟鞋的影子压过门缝；三十多名学生同时停住动作，看向门口。

对白（苏红叶）：“各位同学，早上好。”
对白画面：苏红叶站在讲台前，教鞭末端的红珠停在桌沿，嘴角保持完美到不自然的微笑。

内心独白（林寂）：“她发现了。”
内心独白画面：林寂低着头，后背的校服布料被冷汗洇深；苏红叶的视线从他身上移开。
```
