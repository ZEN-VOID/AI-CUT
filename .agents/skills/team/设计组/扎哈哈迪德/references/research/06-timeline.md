# 06 人物时间线

| 时间 | 事件 | 方法影响 |
|---|---|---|
| 1950 | 出生于 Baghdad, Iraq | Middle East 地景、河流、沙丘、现代化城市经验成为后来的空间记忆 |
| 1960s-1971 | 在 Baghdad、Switzerland、England 等地受教育；后在 American University of Beirut 读数学 | 数学和多地经验形成几何、抽象和非单一文化视角 |
| 1972-1977 | 进入 London Architectural Association 学建筑 | 接触激进建筑教育、Koolhaas、Zenghelis、Tschumi 等思想环境 |
| 1977-1979 | 短暂加入 OMA，并在 AA 任教 | 将理论、竞赛和教学结合 |
| 1979 | 在 London 创办 Zaha Hadid Architects | 建立个人方法和长期工作平台 |
| 1982 | Eaton Place 室内项目获 Architectural Design 金奖 | 早期从室内和小尺度验证空间语言 |
| 1983 | 赢得 Hong Kong `The Peak Leisure Club` 竞赛，方案未建 | “纸上建筑”声名建立，绘画成为空间研究核心 |
| 1988 | 入选 MoMA `Deconstructivist Architecture` 展 | 国际前卫建筑谱系定位确立 |
| 1993 | Vitra Fire Station 完成 | 第一个重要已建作品，证明锐角构成可进入建筑实体 |
| 1994 | 赢得 Cardiff Bay Opera House 竞赛但项目被取消 | 形成与英国建筑建制、公众接受度和政治流程的长期张力 |
| 2003 | Rosenthal Center for Contemporary Art, Cincinnati 完成 | 美国首个重要已建项目，城市地面和“urban carpet”思路落地 |
| 2004 | 获 Pritzker Architecture Prize，首位女性 laureate | “不可建”叙事开始转向行业认可 |
| 2005 | Phaeno Science Center, Wolfsburg 完成 | 地面抬升、结构孔洞、公共流线与展览空间结合 |
| 2009-2010 | MAXXI Rome 完成并获 2010 Stirling Prize | 流线、走廊、展览和城市片段的成熟范例 |
| 2010 | Guangzhou Opera House 完成 | 大型文化建筑、地景流线和中国城市语境结合 |
| 2011 | London Aquatics Centre 完成；Philadelphia Museum 展 `Form in Motion` | 运动、身体、产品设计与建筑连续性被放大 |
| 2012 | 被授予 Dame Commander；Heydar Aliyev Center 完成 | 国际声誉高峰，同时政治委托争议加深 |
| 2014 | Heydar Aliyev Center 获 Design Museum Design of the Year | 形式成就与伦理争议同场出现 |
| 2015 | 被宣布为 2016 RIBA Royal Gold Medal 获得者 | 首位以个人身份获得该荣誉的女性 |
| 2016-02 | 正式领取 RIBA Royal Gold Medal | 生前最后重要行业荣誉之一 |
| 2016-03-31 | 在 Miami 去世，享年 65 岁 | 本人创作停止，ZHA 与 Zaha Hadid Design 继续运行 |
| 2017 | ArtisTree 展 `There Should Be No End To Experimentation` | 事务所用展览继续梳理其方法和遗产 |
| 2018 | Thames & Hudson 出版 expanded `The Complete Zaha Hadid` | 作品集补入最终项目和职业全线 |
| 2024 | ZHA 与 Zaha Hadid Foundation 名称授权诉讼进入公开报道 | 遗产、品牌和机构治理成为后续议题 |
| 2026-02/03 | London appeals court 判决使 ZHA 可结束旧授权条款并与 Foundation 重谈 | 最新动态属于遗产/事务所治理，不代表扎哈本人新观点 |

## 最新动态（截至 2026-04-15）

- Zaha Hadid 本人已于 2016 年去世，无新个人观点。
- Zaha Hadid Architects 仍以其名字和方法延续全球实践。
- 2026 年 ZHA 与 Zaha Hadid Foundation 的名称授权争议出现新判决；使用本 Skill 时应区分“扎哈本人方法”“ZHA 后续实践”“基金会遗产治理”。

## 可信度备注

- 时间线由 Pritzker、ZHA、TIME、Guardian、Wallpaper、博物馆展览资料交叉整理。
