# Scripts Contract

本目录只承载机械辅助脚本或说明，不能替代 LLM 完成角色归并、别名裁决、代称识别或 `原文描述（关键词式）` 写作。

Allowed:

- 读取 `projects/aigc/<项目名>/5-分组/第N集.md`。
- 抽取分镜组 ID 和组底 YAML `角色` 原值。
- 检查 `角色清单.md` 是否存在三列表头。
- 提示重复名称、疑似空字段、分镜组 ID 形态错误。

Forbidden:

- 自动生成 canonical `角色清单.md` 正文。
- 用字符串相似度直接裁决同一角色。
- 自动补写角色描述关键词。
- 从非 YAML 来源创建角色候选。
