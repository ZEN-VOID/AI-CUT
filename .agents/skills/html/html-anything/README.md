# Open Design / HTML Anything Codex Adapter

Codex-adapted wrapper for the Open Design / HTML Anything template library from `nexu-io/html-anything`.

## Source

- Upstream: `https://github.com/nexu-io/html-anything`
- Installed commit: `b54a0fbac2942bd82b113199712d488d4b3b3cfa`
- Upstream template copy: `references/upstream/templates/skills/`

## Usage

Use this package to choose a suitable HTML template family and map source material into a designed HTML artifact.

The local `SKILL.md` is the authoritative Codex runtime contract. Upstream template files are reference modules loaded selectively.
