# Novel To Screen Language Contract

## Purpose

本细则处理小说原文中“作为小说成立、但不能直接作为剧本画面成立”的表述。它不是改写剧情的授权，而是把小说句子的叙事功能二次转译为可见、可听、可执行的画面语言。

核心原则：

- 不逐句翻译小说文采；先判断原句在叙事中的功能，再选择影视表达策略。
- 保留上游事实、对白、事件顺序、信息释放和关系结果；允许在不改变剧情条件的前提下进行合理且艺术化的画面承托。
- 任何二次加工都必须落入既有字段：`环境描写`、`角色动作`、`动作画面`、`对白画面`、`独白画面`、`旁白画面`、`音效画面`、`道具特写`、`群像画面`、`表演提示`、`场面调度` 等；不得新增摄影方案字段或第二套解析体系。
- 主角内心想法、内心独白和主角视角下对他人行为的主观判断必须保留为主角主观层：优先进入 `独白（主角）` / `内心独白（主角）` 与配套画面，不得改成客观第三方概括。
- 从第三人称小说叙述转为 `内心独白（主角）` 时，主角自指必须从“他/她/其/角色名”转换为“我/我的/自己”或角色当下自然口吻；画面承托字段仍按剧本画面使用第三人称。第三人称只可保留给其他角色、引用文本或有留证的自我疏离表达。
- 描写人物动作时只保留镜头可实拍的客观动作、神态、语气、生理反应和空间关系；删除“试图、想要、打算、意图”等主观预判或心理意图词。
- 抽象概括、无法实拍的总结句，尤其是“反复如此、早已熟悉、往日总是、一直以来、每次都”这类体现重复、熟悉和往日常态的句子，不得直接进入终稿；必须删除、转成当前场景的一个可拍当下 beat，或转入主角内心独白。
- 禁止为补解释新增与当前主线剧情无关的人物过往背景、物品来历补充或回忆性信息；只保留当下可直接实拍、无需额外镜头铺垫的核心画面与情节。
- 若原文信息只能由小说叙述承担，允许保留短旁白；但旁白必须被画面证据承托，不得把长段解释当成默认主路径。

## Dialogue Lock Boundary

- 小说表述二次画面化不处理引号内对白；上游已有对白必须逐字冻结，不润色、不同义替换、不调整语序、不拆改语气词。
- 可加工的是对白外部的剧本承托：`对白（真实角色名，语态/状态短语）` 的第二项、对白前后的 `对白画面`、角色动作、视线、停顿、身体距离、道具动作、群体反应和环境余波。
- 不得把小说叙述句、作者判断、概括说明或心理解释改写成新对白；上游没有的台词不进入 canonical 编导稿。
- 当小说原文用“他说得很轻松/她似乎并不在意”这类叙述修饰对白时，只能转成说话状态标签或对白外画面承托，不能改写引号内台词。

## Expression Type Matrix

| novel expression type | novel function | screen-language strategy | forbidden output |
| --- | --- | --- | --- |
| 作者评论 / 主题判断 | 告诉读者意义、时代感、道德判断或命运感 | 转成场内压力、空间秩序、群体反应、静物状态、短旁白加画面证据 | `画面字段` 直接写“这说明/象征/预示/命运如此” |
| 主角视角判断他人 | 主角解读他人的动作、语气、停顿、沉默或动机 | 转成第一人称 `内心独白（主角）` + 主角观察到的可见证据，或主角的可感知反应 | 客观第三方写“她在试探他”“他早有准备”“对方明显撒谎”；或在内心独白中继续用“他/她/角色名”指代主角自身 |
| 心理内视 / 自由间接引语 | 交代角色害怕、犹疑、信任、嫉妒、羞惭、醒悟 | 转成呼吸、避视、停顿、手部动作、身体距离、道具处理、声线变化、独白或可感知心理反应 | `动作画面` 或 `心理反应` 写“他意识到/她觉得/他们明白”这类抽象内视 |
| 主观情绪感受 | 感到恶心、难受、愤怒、害怕、崩溃等 | 转成微表情、肢体动作、生理反应、声线变化，或主角内心独白 | `心理反应：他感到恶心/难受/愤怒` |
| 文学比喻 / 象征句 | 制造意象、气质、情绪温度 | 抽取可见物：光、影、风、雪、落叶、尘、血迹、空座、门槛、器物磨损、衣角湿痕；变成构图关系和材质状态 | 原样输出“仿佛命运的齿轮”“像一场审判” |
| 概括叙述 / 省略叙述 | 压缩时间、跳过过程、概括多人行为 | 拆成 2-4 个可拍 beat、群像画面、道具状态变化、短转场旁白；不补新事件结果 | 用一句“众人纷纷...”替代所有画面，或新增未发生桥段 |
| 往日常态 / 重复熟悉总结 | 说明某事反复发生、角色早已熟悉或过去总是如此 | 若当前剧情必须承接，转成当下一个可拍现象、道具磨损、动作熟练度或主角内心独白；非主线必要则删除 | 直接写“他早已习惯这一切”“这种场面每天都在重复” |
| 背景说明 / 历史叙述 | 交代时代、制度、前史、规则 | 只保留当前主线必要且可由当下画面/短旁白承托的信息；人物过往、物品来历和回忆补充若非当前情节必需，删除 | 大段旁白独立承担所有关键信息，或新增与当前主线无关的前史/来历 |
| 因果解释 | 说明为何如此、所以怎样 | 转成行动-反应链、阻碍、选择压力、场面调度和信息递送顺序 | 画面字段写“因为...所以...”的推理论文 |
| 关系结论 | 概括谁压制谁、谁信任谁、谁背叛谁 | 用站坐高低、远近、道具归属、谁先说/谁不接话、谁退半步等可见关系表达 | 直接写“二人关系发生改变”“他掌控全场” |
| 感官散文 | 写气味、寒意、燥热、潮湿、疼痛、寂静 | 转成可拍/可听效果：雾气凝在发梢、衣料贴住皮肤、火光发白、杯沿震动、虫声断掉、尘土呛住呼吸 | 仅写“空气中弥漫着恐惧/压抑” |
| 武侠/玄幻抽象 | 写杀气、剑意、内力、身法、气势 | 转成身体力学、速度节奏、空间压迫、衣摆/尘土/水面/器物反应、声响和对手迟滞 | 画面字段只写“杀气逼人”“剑意纵横” |
| 回忆 / 认知补叙 | 交代角色想起、发现、误会或理解 | 若上游明确可视，转成道具停点、目光锁定、短独白、闪回候选说明；未授权不新增闪回事实 | 自由扩写未出现的回忆画面 |
| 规则文字 / 系统说明 | 传达制度、任务、限制、倒计时或名单 | 用 `系统画面`、`规则显影`、`道具特写`、`旁白（系统提示）` 和角色反应承托 | 把规则解释塞进环境或动作字段 |

## Strategy Palette

- `externalize`: 把心理、关系、权力转成可见反应、动作和空间关系。
- `embody`: 把抽象状态交给演员身体、呼吸、声线、停顿和手部动作。
- `materialize`: 把信息或情绪压到物件、痕迹、文字、服饰、伤势、门窗、桌面和承载面。
- `spatialize`: 用距离、高低、站坐、出入口、座位、围合关系表现压力。
- `sonify`: 用声音本体、声音消失、环境声变化和人群声浪承托信息。
- `rhythmize`: 用快慢、停顿、重复、打断和余波处理小说概括句。
- `montage`: 对时间压缩和过程概括采用短 beat 串联；仅承接上游已经给出的过程或结果。
- `keep_as_narration`: 只有历史、规则、前史或小说原文本身承担不可替代信息时，保留短旁白，并配套画面证据。
- `leave_unsaid`: 让观众通过动作、沉默和环境承托理解，不把所有心理都解释出来。
- `split_fields`: 将同一句小说表达拆到多个字段，例如环境承托气氛、动作承托选择、道具承托信息、对白画面承托潜台词。
- `subjectivize_to_protagonist`: 将主角视角下对他人的判断改入主角内心独白，并配主角观察到的客观证据。
- `strip_to_present_action`: 删除无关前史、来历和往日常态，只保留当前场景可拍动作、状态和情节必需信息。

## Artistic Transformation Boundary

允许：

- 为原文没有细写的氛围、自然景物、静置道具、群体停顿、声音余波和空间压力补足画面层次。
- 把一句概括性叙述拆成多个可拍画面 beat。
- 把文学比喻转成画面母题、材质、色彩、光影、天气、静物和空间关系。
- 把人物心理转换为表演任务、身体行为和对白附近反应。
- 把主角视角判断改写为主角内心独白，并用当前场景可见/可听证据承托。
- 把直接情绪感受改写为微表情、肢体动作、生理反应或主角内心独白。

禁止：

- 新增对白、桥段、因果、线索、规则、事件结果或人物动机。
- 新增与当前主线无关的人物过往、物品来历、回忆性信息或解释性补充。
- 用摄影机位、景别、镜头运动替代剧本画面化。
- 把“合理艺术加工”当成脱离上游的自由创作。
- 将小说抽象句原样放进 `环境描写`、`角色动作`、`动作画面`、`道具特写` 或 `对白画面`。
- 在 `角色动作` / `动作画面` 中写“试图、想要、打算、意图”等主观预判或心理意图词。
- 把主角对他人的主观解读写成客观第三方总结。

## Weak And Strong Examples

弱：

```text
动作画面：他意识到自己已经无路可退，命运的阴影压了下来。
```

强：

```text
角色动作：他退到门槛前，脚跟碰上木槛后停住，握刀的手指一点点收紧。
环境描写：门外的风卷着细尘扑进屋内，烛火贴着墙面发抖，地上的影子被拉得很长。
```

弱：

```text
环境描写：众人都明白这是一场权力审判。
```

强：

```text
群像画面：堂下众人先后收声，最前排的人把视线垂到地面，后排有人迟半拍才放下手里的杯盏。
场面调度：主位仍坐在高处，案上的文书压在灯影中央，其他人只能站在阶下回话。
```

弱：

```text
道具特写：这封信证明了他早已背叛。
```

强：

```text
道具特写：信纸边角被水浸皱，落款处的私印缺了一角，封蜡上还压着未擦净的指痕。
```

弱：

```text
角色动作：林寂试图压住怒火，打算装作没听见。
```

强：

```text
角色动作：林寂把纸角抚平两次，第二次指节压得发白；他抬眼时下颌绷住，没有接那句话。
```

弱：

```text
对白画面：她是在试探林寂，这种手段他早已见过很多次。
```

强：

```text
内心独白（林寂）：“她在等我先补后半句。”
内心独白画面：她的问题停在半截，视线落到林寂握紧的手指上；林寂没有立刻开口，只把纸条往灯影外推了半寸。
```

## Review Checklist

- 是否识别了小说原文中的作者评论、主角视角判断、心理内视、直接情绪感受、比喻象征、抽象概括、往日常态句、背景说明、因果解释和关系结论。
- 每条高风险小说表述是否有 `source_function` 与 `screen_strategy`。
- 画面字段是否回答“看见什么 / 听见什么 / 演员做什么”，而不是解释“这意味着什么”。
- `角色动作` / `动作画面` 是否只写可实拍客观动作、神态、语气和生理反应，没有主观意图词。
- 主角视角下对他人行为的判断是否进入主角内心独白或主角观察证据，而不是客观第三方概括。
- 主角内心独白中指代主角自身的人称是否已从小说第三人称转成第一人称；第三人称是否只用于其他角色、引用文本或有留证的自我疏离。
- 直接情绪感受是否已转成微表情、肢体动作、生理反应或主角内心独白。
- 艺术加工是否只增强画面密度、节奏、氛围、材质、空间压力和表演可执行性。
- 若保留旁白，是否有画面证据承托，且没有让旁白成为默认信息堆放处。
- 是否无新增剧情事实、对白、因果、规则、线索、无关前史/物品来历/回忆性补充和摄影越权。
