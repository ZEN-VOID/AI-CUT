# Physical Atmosphere Index

本文件是 `6-氛围` 的本地物理氛围包索引，用于优先提供现场物理/舞台特效层面的可选手段。它不是执行经验层，不替代 `SKILL.md` 的触发机制、输出路径或 review gate。

## Default Selection Library

执行 `N4-ATMOSPHERE-PACK` 时，先从本节 12 类默认选择库中选型，再按题材、场景条件、触发类型、`3-美学` 风格协议和保真边界裁剪。默认选择库不是必选清单；无 source anchor、无触发理由或会改变剧情事实的选项不得使用。

## Combination Rules

- `薄雾 + 逆光`：让空间和光束可见，适合进入场景、仪式、梦境、悬疑。
- `风 + 织物/纸张/树叶`：让静态空间活动，适合离别、危机预兆、超自然冲击。
- `湿地面 + 冷光/霓虹`：增强夜景反光，适合城市、雨后、赛博、孤独。
- `声景消失 + 细小静物变化`：适合惊悚和压迫，比直接大声效更有效。
- `尘土/灰烬 + 低频声`：适合战场、废墟、爆发后的余波。

## Safety And Boundary Notes

- 本索引只提供画面化创作选项，不提供真实设备操作、安全距离、施工参数或危险特效执行指南。
- 任何可能构成真实危险操作的火、爆破、烟火、强风、密闭空间烟雾等内容，正文只写观众可感知的画面效果，不写操作方式。
- 若用户要求真实舞台执行或法律/安全合规，必须联网检索权威资料并将本技能降级为创作参考，不输出操作规范。
