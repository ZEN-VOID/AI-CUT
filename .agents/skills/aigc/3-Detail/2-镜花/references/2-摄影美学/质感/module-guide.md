# 质感 模块说明

## 定位

- 本叶子负责把材料、湿度、颗粒和表面反射等触感线索压入画面。
- 它不负责抽象赞美，只负责让观众“看得见摸得着”的材料感成立。
- 它默认服从当前组已经锁定的视觉控制线，而不是把所有可见材料平均铺满。

## 使用方法

- 优先找最先被感到的材料，再决定其表面行为如何帮助场感。
- 质感必须回指世界观和设计元素，而不是独立漂浮。
- 再判断哪些质感该被收住，避免材质描写抢走人物、动作和观看重心。
- 输出时保留最有效的材料焦点，不必面面俱到。

## 失真与修正

- 若质感只剩“高级、细腻、厚重”，说明没有真正可见。
- 若表面信息抢走了人物和动作主线，说明质感写得过满。
- 若材料线索和项目世界观不一致，优先回到设计元素约束。
- 若什么都想摸到，说明没有先锁“最先被感到的材料”。
