# Acceptance: Review Gate

用于 repair 方案和结果的最终验收。

## Fixed Context

- 默认加载 `review/review-contract.md`。
- 审查 source rule review、impact map、canonical owner、豆包 evidence、asset action 和 residual risks。
- 可使用 `code-reviewer` 口径做辅助结构审查；被不可用时降级为本地 checklist 并报告。

## Review Gate

- 没有 impact map、source owner 或 provider evidence/degradation，不得 PASS。
