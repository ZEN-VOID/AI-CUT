# 新闻热搜来源类型包

## Match Signals

- 新闻事件、网络热搜、社会议题、真实人物事件、纪实素材、平台争议话题。

## Fixed Context

- 先写 `truth_boundary`，再起戏。
- `faithful`：保留事实时间线，只用叙事组织提升可读性。
- `inspired_by`：保留热点议题与公共情绪，用虚构角色承载故事。
- `free_reimagining`：只借议题核、冲突核或情绪核，不伪装成现实事件。
- 必须区分：已知事实、公共情绪、未解问题、虚构承载线。

## Review Gate

- 读者能分清事实锚点与剧情扩写。
- 涉及现实受害者、未成年人、极端暴力或高度敏感身份信息时，必须收紧虚构和画面刺激。
