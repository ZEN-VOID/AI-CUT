## Repair Playbook

1. 先判断每个关键场景承担什么功能，再决定该补哪类感官信息。
2. 再补四个槽位：空间怎样限制人、主感官锚点是什么、关系温差在哪里、环境怎样改变行动或判断。
3. 若描写美但没用，优先删减形容，保留能影响人物感受和行动的部分。
4. 若静场发空，优先补“主物象 + 距离关系 + 沉默后果”。
5. 若亲密、禁忌、熟年或医疗场发飘，优先补“身体事实 + 代价余震”。
6. 若氛围空泛，优先补“空间阻力”和“身体反应”，而不是继续追加修辞。

## Reusable Heuristics

- 最有效的场景描写通常不是“描得多”，而是让空间本身成为压力、诱惑或遮蔽的一部分。
- 氛围感最怕空话，感官锚点越具体，读者越容易进场。
- 静场的高杠杆通常不在多写景，而在一个主物象、一段距离和一句没有说出口的话。
- 亲密场面的质感，不在“更香艳”，而在身体真实、羞耻余震和社会代价同时在场。
- 传统空间、旧城街巷、旅馆、病房、车厢这些地方，只有真正改变了人物选择，才配叫场景，而不只是背景。
