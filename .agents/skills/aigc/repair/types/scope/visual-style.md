# Scope: Visual Style

用于视觉主轴、氛围、镜头语言、光影、中文润色和创意激发类修复。

## Fixed Context

- 必查 `2-美学` 风格协议、`3-主体` 资产锚点、`5-导演` 导演意图、`6-分镜` 动作连续性、`7-摄影` 镜头语言和目标产物所属 leaf；初始化偏好只从 `MEMORY.md` / `CONTEXT/` 读取。
- 润色只改善表达自然度和可执行性，不改变事实、对白、编号、路径或结构。
- 创意候选必须标注是否可直接采用，还是需要用户选择或 stage owner 吸收。

## Review Gate

- 输出不含空泛审美口号。
- 每个创意建议都能映射到 stage owner 和可验收目标。
