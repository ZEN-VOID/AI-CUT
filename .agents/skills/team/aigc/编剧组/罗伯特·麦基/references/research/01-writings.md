## 书目与持续主张

### 1. `Story`

- 核心：故事工艺可被分解为结构、场景设计、节拍、转折、价值变化、高潮与结局。
- 对本 skill 的价值：这是麦基整套方法的母本，后续 `Dialogue / Character / Action` 都像是对其局部放大。

### 2. `Dialogue`

- 核心：对白不是“说得漂亮”，而是言语动作、潜台词、节拍与沉默的组织。
- 重复信念：角色说话时一定在做事；好对白一定牵连欲望与阻力。

### 3. `Character`

- 核心：人物塑造不是设定表，而是矛盾、维度、压力、选择与弧线。
- 重复信念：性格说明不会创造人物，选择才会。

### 4. `Action`

- 核心：动作类型并不低于“深刻电影”；动作故事同样需要角色、因果、价值升级与情感回报。
- 重复信念：动作场面不是填充物，而是故事推进器。

### 5. `Storynomics`

- 核心：叙事原则可迁移到商业表达、品牌与营销。
- 对本 skill 的价值：说明麦基不是只关心电影文本，而是把故事视作更广泛的说服与注意力结构。

## 反复出现的真信念（跨 3 本以上书或官方页面）

1. **故事是价值变化，不是事件堆砌。**
2. **结构是原则系统，不是公式模板。**
3. **人物通过压力下的选择显形。**
4. **场景必须发生转折，否则就是 exposition。**
5. **对白是带目的的言语行动。**
6. **结尾必须证明控制性观点。**

## 可直接迁移到编剧组的写法

- 先写终点价值，再倒推中段结构。
- 先找人物在最大压力下的选择，再补前史。
- 先写场景起终点的价值状态，再写对白。
- 先用剧情证明主题，再决定是否保留金句。

## 边界提醒

- 官方书页以营销语言为主，概念准确但案例密度不足；具体语气与争议需要配合访谈材料读取。
- 公开书页不会主动呈现外界对其“过度结构化”的批评，这部分要交给 `04-external-views.md`。
