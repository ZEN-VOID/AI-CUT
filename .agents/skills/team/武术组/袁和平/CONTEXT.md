## Repair Playbook

1. 先判断失败类型：泛化、招式清单、武侠口号化、花哨无力、环境缺席、技术空转、不可拍，或事实错误。
2. 回读 `SKILL.md` 的字段中心映射，定位对应 `field_id`。
3. 对动作输出，按“身体 -> 动机 -> 性格 -> 环境 -> 减法 -> 镜头/安全”顺序重写。
4. 对武侠输出，先补行动伦理：守护谁、反抗谁、代价是什么，再补动作。
5. 对奇观输出，先确认真人动作根基，再决定威亚、CG、AI 是否只是增强。
6. 对事实输出，回读 `references/research/0X-*.md`；无来源时降级为推断。
7. 若修复属于稳定规则，更新 `SKILL.md`；若属于运行经验，更新本 `CONTEXT.md`；不要把过程流水写入这里。

## Reusable Heuristics

- 袁和平式动作建议必须先有“真身托幻想”：没有身体训练和动作逻辑，威亚、CG、AI 都只是遮羞布。
- 打戏的最小闭环是：角色为何出手、身体怎么出手、环境如何改变出手、镜头如何证明这一下是真的。
- 处理武侠时，不要先写飞檐走壁；先写“看见什么不平，为什么必须管”。
- 处理 AIGC 武侠视频时，把每个漂亮动作问一遍：演员/角色能不能做，动作改了什么关系，环境是否参与，技术是否只做增强。
- 功夫喜剧不是乱搞笑；喜剧节奏仍要建立在身体准确性和动作因果上。
- 旧武侠要有新东西，但新东西不能只靠包装；最好从新的环境规则、身体限制或动作伦理里长出来。
