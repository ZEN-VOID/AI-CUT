# 少年战斗冒险

## 核心冲突引擎

- 这一类题材的核心不是“会打架”，而是少年主角被逼着在更大的世界规则里升级、立誓、结盟、翻盘。
- 单场戏最稳的推进顺序通常是：压制现状 -> 主角立场显形 -> 新招式/新理解/新代价出现 -> 关系或胜负被改写。
- 冒险感必须真的推动版图扩张：新敌人、新地域、新秩序、新门槛，而不是一直在同一训练场原地换皮。

## 角色原型

- 主角要有主动赢的欲望、可见的短板和一句能喊出口的立场，不适合纯被动卷入。
- 宿敌不是血包，而是主角的镜像压力源：天赋更强、规则更懂、信念更冷，逼主角长出新的姿态。
- 同伴团需要各自承担功能位：破局脑、情绪锚、肉身前排、价值观纠偏者，避免只做背景喝彩。

## 漫画页面语法

- 大招、冲刺、抬头宣言、宿敌对看和首次反杀，必须能独立撑页。
- 页面节奏要有“紧 -> 爆 -> 停 -> 再抬”的波形，不能全篇同速猛冲。
- 台词宜短、硬、可复述；长段解释要拆进姿态、动作轨迹、受伤结果和旁观反应里。

## 典型桥段库

- 初败后重新理解规则。
- 宿敌第一次露出惊讶。
- 同伴替主角扛一次代价，迫使主角完成立场升级。
- 新地域首次显露压倒性的世界尺度。
- 招式命名、誓言和禁忌代价一起出现，让突破不只是特效，而是人格声明。

## 翻页机制

- 最值钱的翻页不是“打出去”本身，而是“打出去前最后半拍的蓄压”。
- 页尾常见高价值触发点：宿敌第一次惊讶、招式即将命名、代价开始上身、世界门槛突然显形。
- 翻页后优先给姿态和结果，再补局部解释，避免把爆点切成说明书。

## 信息主载体

- 主信息优先落在身体姿态、冲刺轨迹、对视关系和受伤结果。
- 道具、符号印记和术式纹样只做辅助，不应取代动作本身。
- 旁观者反应是重要的二级信息位，用来证明这次突破真的改变了局势。

## 对白与旁白

- 对白以短句宣言、挑衅、应答为主，最好一口气能喊出来。
- 旁白只补必要门槛，不宜替角色完成燃点。
- 一旦进入爆点页，说明性台词必须减量，把位置让给动作和结果。

## 母题系统

- 誓言、宿敌镜像、代价印记、破碎地貌，适合做持续回返的题材母题。
- 母题不能只在初登场时华丽，必须在败北、突破、再起时反复变义。

## 禁写模式

- 打很久，但阶段关系没有变化。
- 解释能力系统的篇幅大于动作和后果。
- 只会堆“燃词”，人物却没有具体目标和短板。
- 所有爆点都压在最后，前面只剩平推。
