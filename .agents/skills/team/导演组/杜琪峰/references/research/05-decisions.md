# 05 · 决策记录与行动

## 关键决策

### 1. 从电视工业进入电影，再回电视磨炼

- 背景：早年电影处女作后，他曾回到电视系统继续工作。
- 决策逻辑：经验不足时回到工业训练，而不是强行维持电影作者姿态。
- 提炼：杜琪峰方法的底层不是“天才灵感”，而是长期工业训练。

### 2. 1996 年创办银河映像

- 背景：香港电影工业环境变化，片厂/明星/制片权力影响导演自主。
- 决策逻辑：建立自己的生产平台，以团队和类型片获得作者自由。
- 提炼：团队是一种作者性；自由必须有生产组织承载。

### 3. 商业片与作者片双线运作

- 背景：银河映像早期既拍爱情/喜剧/明星片，也拍犯罪片和个人化作品。
- 决策逻辑：用市场片维持团队和现金流，用作者片保留风格与锋利度。
- 提炼：商业不是作者性的敌人，失去商业自保才是。

### 4. 《枪火》式低预算空间美学

- 背景：预算有限，无法靠大规模动作场面堆效果。
- 决策逻辑：把紧张集中到站位、静止、等待和短促动作。
- 提炼：限制可以逼出风格；预算小要保护核心调度。

### 5. 面对合拍和内地市场

- 背景：香港电影市场收缩，合拍提供资源和市场，也带来审查与表达边界。
- 决策逻辑：进入但承认妥协，不把妥协包装成完全个人表达。
- 提炼：跨市场项目必须提前计算锋利度损耗。

### 6. 创办 Fresh Wave

- 背景：香港电影需要新导演和本土活力。
- 决策逻辑：不只拍自己的片，也搭建青年电影人才入口。
- 提炼：导演身份扩展为行业生态维护者。

### 7. 持续想拍《黑社会3》

- 背景：公开访谈中多次谈到对《黑社会3》的兴趣和风险。
- 决策逻辑：题材与香港现实关联过强，创作欲望与政治/团队风险并存。
- 提炼：有些电影不是“想拍就拍”，要等环境、团队和风险同时允许。

## 决策模式总表

| 决策模式 | 案例 | 可迁移规则 |
|---|---|---|
| 退回工业训练 | 早期回电视 | 能力不足时先练系统，不要硬装作者 |
| 建平台 | 银河映像 | 自由需要组织载体 |
| 双线现金流 | 商业片/作者片切换 | 先让团队活着，再让风格锋利 |
| 限制生美学 | 《枪火》 | 把限制变成调度规则 |
| 承认妥协 | 《毒战》合拍 | 不要把市场妥协伪装成纯作者表达 |
| 培育生态 | Fresh Wave | 个人影响力要回流行业 |
