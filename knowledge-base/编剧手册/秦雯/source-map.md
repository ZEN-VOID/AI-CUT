## 任务到材料的默认映射

- 如果任务是“小说改编怎么取舍主线与群像”
  - 先读 `周到对谈` 与 `新民周刊专访`
- 如果任务是“单场戏怎么写出关系缝隙和情感余味”
  - 先读 `新民周刊专访` 与 `一流对话场景.md`
- 如果任务是“上海90年代细节怎么进入人物和动作”
  - 先读 `新民周刊专访` 与 `人民日报海外版`
- 如果任务是“AI 可以帮到哪一层，不能碰哪一层”
  - 先读 `中国作家网`
- 如果任务是“场景写得太平，怎么让它立体”
  - 先读 `电影导演方法.md` 与 `新民周刊专访`

## 排除规则

- 纯名场面剪辑、台词摘抄、情绪向短评，不纳入当前知识底座
- 只有“很上海”“很高级”的评价，没有任务映射的材料，不纳入
- 无法解释“它支持哪个编剧动作”的材料，不纳入
