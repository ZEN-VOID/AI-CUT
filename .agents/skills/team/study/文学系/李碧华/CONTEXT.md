## Repair Playbook

1. **先查事实边界**：涉及李碧华本人、作品、年份、奖项、影视署名时，优先核验 `references/research/` 与外部权威来源。
2. **再定位失败字段**：把失败归入 `fact_check / classic_reversal / female_agency / body_cost / hongkong_time / voice / boundary` 之一。
3. **先修结构，再修句子**：如果故事不成立，先补人物欲望、债、物件、场景和结尾，不先润色语言。
4. **保留不确定**：公开资料不足时写“推断”或“资料不足”，不要补出不存在的一手访谈。
5. **最小可拍验证**：每次输出至少留下一个可拍元素：动作、道具、空间、台词或反转。
6. **脚本误报先查正则边界**：若验证报告与文档内容明显矛盾，先检查脚本是否跨标题误匹配，再判断内容是否需要返工。

## Reusable Heuristics

- 李碧华式“毒”不是阴森词汇，而是把传统叙事的受害者改成解释者。
- 处理经典母题时，先确认观众熟悉的版本，再反写；没有公共记忆，就先建立一个能被反写的表面版本。
- “肉身化”要承担主题责任：食物、衣物、伤痕、妆容、旧物必须推动选择或揭示债务。
- 城市故事不要只换地名；找旧空间如何在现代场景里重新索债。
- 对话风格应短、冷、准，但创作建议必须落到执行层：人物、场景、物、代价、结尾。
