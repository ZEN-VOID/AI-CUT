# 中二感强化引擎

本文件服务 `漫画剧本改编` 的中二感强化，不与主 `SKILL.md` 竞争真源。

## 1. 核心原则

中二感不是简单地“喊得更大声”，而是让角色、命名、誓言、宿命和代价都带上夸张但可信的自我神话感。

优先级：

`角色自我神话 > 命名势能 > 誓言压强 > 设定术语堆叠`

硬规则：

- 中二感必须服务人物关系、登场压迫、能力兑现或宿命冲突。
- 只堆名词、不带行动、誓言、代价与关系张力，视为伪中二。
- 中二感可以夸张，但不能把角色写成无缘无故发疯。

## 2. 中二感四件套

| 组件 | 作用 | 最稳写法 |
| --- | --- | --- |
| `命名势能` | 让能力、组织、武器、仪式可记可喊 | 名字短、重音清楚、朗读带压强 |
| `角色宣言` | 让角色一开口就立住人格 | 用绝对句、禁令句、审判句、誓言句 |
| `宿命关系` | 让角色之间不是普通冲突，而是命运互锁 | 血脉、诅咒、旧誓、师徒、王与祭品、容器与灾厄 |
| `代价兑现` | 防止中二感空心化 | 每次炫技、觉醒、宣言都绑定代价或失去 |

## 3. 最常用的中二句型

### 宣告型

- “从你踏进这里的那一刻起，名字就已经不属于你了。”
- “我不是来请求你的，我是来宣判你的。”

### 誓言型

- “哪怕把这副身体烧成灰，我也会把你从深渊里拽出来。”
- “如果王座一定要有人流血，那就先从我开始。”

### 宿命型

- “你以为你杀的是我，其实你唤醒的一直是你自己。”
- “我不是你的敌人，我是你活到今天必须面对的报应。”

### 禁忌型

- “那扇门不是打不开，而是谁打开，谁就不再是人。”
- “你当然可以拔出它，只是从那一刻起，被诅咒的就不止是剑。”

## 4. 命名势能规则

好命名通常满足：

- 朗读时有节拍。
- 写进对白时不拗口。
- 一眼能看出它是“武器 / 组织 / 术式 / 身份 / 仪式”中的哪一类。
- 最好带一个明确意象：夜、血、王、月、深渊、审判、禁书、灾厄、冠、门、焰、骨、影。

避免：

- 一个名词堆六层修饰。
- 所有东西都叫“终焉”“永夜”“灭世”，结果彼此分不清。
- 命名很酷，但和角色行动没有关系。

## 5. 中二感与漫画页势能

中二感最适合强化这些节点：

- 角色首次登场
- 身份揭露
- 能力初次释放
- 誓言成立
- 宿命关系摊牌
- 章末急停

写法提醒：

- 一页只保一个真正最值钱的宣言句。
- 大格时让对白更短更绝，不要更长更解释。
- 如果已经有强视觉，旁白就退；让角色的宣言自己顶住这页。

## 6. 自检

1. 这场戏有没有一句一看就知道是谁说的中二宣言？
2. 这句宣言有没有绑定人物关系或代价，而不是只会耍帅？
3. 这场里的命名，读出来到底是有势还是只长？
4. 如果删掉一半术语，这场戏会不会反而更中二、更有力？
