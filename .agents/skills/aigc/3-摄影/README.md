# aigc 3-摄影

`3-摄影` 将 `projects/aigc/<项目名>/2-编导/第N集.md` 的逐集编导稿升级为带大师级分镜、摄影、运镜、转场、光影和色彩设计的摄影镜头语言稿。

## 目录树

```text

├── references/
├── scripts/
├── templates/
├── review/
├── steps/
├── knowledge-base/
├── types/
├── agents/
│   └── openai.yaml
```
