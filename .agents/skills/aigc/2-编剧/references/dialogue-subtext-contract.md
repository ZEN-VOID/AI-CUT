# Dialogue Subtext Contract

## Purpose

在对白标签层增加潜台词行为标注：每句关键台词不只是"谁说的、以什么状态说的"，更是"这句话在执行什么戏剧动作"。潜台词是影视对白的灵魂——角色说出口的话和他真正想做的事，往往是两件事。

## Ownership

- 2-编剧 在 N4-FIELD 阶段识别对白的戏剧动作，建立 `dialogue_subtext_map`。
- 关键对白（转折、试探、揭示、博弈）必须标注戏剧动作。
- 4-表演 在 N4-PERF-ACTOR-CONTROL 阶段直接消费戏剧动作作为台词表演的潜台词锚点。
- 对白标签增强不改写引号内原对白。

---

## Theatrical Action Categories

每句关键对白在执行以下9种戏剧动作之一（或组合）：

| category | 创作含义 | 对白表面可能在说 | 对白实际在做 |
| --- | --- | --- | --- |
| `probing` | 试探、摸底 | 闲聊、问路、日常寒暄 | 探测对方知道多少、底线在哪里 |
| `deflecting` | 转移、回避 | 反问、讲笑话、岔开话题 | 不想回答、保护秘密、争取时间 |
| `concealing` | 隐瞒、伪装 | 说反话、平淡叙述、故意示弱 | 掩盖真实意图、伪装情绪、降低警觉 |
| `pressuring` | 施压、威胁 | 提条件、暗示后果、沉默等待 | 逼迫对方表态、让步、暴露 |
| `seducing` | 引诱、拉拢 | 示好、给好处、共情 | 让对方放下防备、建立信任、靠近 |
| `confessing` | 坦白、泄露 | 话多、犹豫、反复解释 | 不自觉说出真相、暴露弱点、寻求认同 |
| `testing` | 考验、设陷阱 | 故意透露假信息、抛出问题 | 观察对方反应、确认猜测、判断立场 |
| `bonding` | 亲近、共鸣 | 分享经历、共情、沉默陪伴 | 建立亲密感、修复关系、寻求同盟 |
| `asserting` | 宣示、划界 | 明确表态、下命令、拒绝 | 划定底线、宣示权力、建立边界 |

## Dialogue Tag Enhancement

### 当前格式
```
对白（林寂，语气平淡）："你来得很早。"
对白画面：林寂站在门口，看着教室里唯一的身影。
```

### 增强格式（关键对白可选）
```
对白（林寂，语气平淡）："你来得很早。"
表演提示：试探——他不确定对方为什么也来这么早
对白画面：林寂站在门口，目光在对方和空教室之间来回。
```

### 使用规则

- **不是所有对白都需要标注**。日常寒暄、信息传递型对白不需要戏剧动作。
- **必须标注的对白类型**：
  - 关键转折对白（改变关系/局势的对白）
  - 博弈对白（角色之间在试探/施压/隐瞒的对白）
  - 揭示对白（释放关键信息的对白）
  - 潜台词明显的对白（表面说一套实际做一套）
- **标注位置**：在 `对白（角色，状态）` 和 `对白画面` 之间，以 `表演提示：戏剧动作——具体语境` 形式
- **标注必须具体**：不能只写"试探"，要写"试探——她想确认对方是否知道那件事"

## Multi-Layer Subtext

高级对白同时运作在多个层级：

| layer | 描述 | 示例 |
| --- | --- | --- |
| 表面层 | 角色说出口的话 | "今天的咖啡不错。" |
| 行为层 | 这句话在做什么（戏剧动作） | 试探——她想看他是否还记得昨天的事 |
| 内心层 | 角色真正在想什么 | "他还会记得我们第一次喝咖啡的地方吗？" |

**创作规则**：
- 表面层由对白文本承担（逐字冻结）
- 行为层由 `表演提示` 承担
- 内心层由 `内心独白` 或 `心理反应` 承担
- 三个层级在创作时必须同时成立，但不必全写入终稿——行为层是必选项，内心层是可选项

## Subtext & Visual Support

每种戏剧动作有其对应的画面承托方向：

| theatrical_action | 优先画面承托 |
| --- | --- |
| `probing` | 视线在对方和某个信号之间来回、手指无意识动作、停顿后追问 |
| `deflecting` | 突然改变姿势、拿起杯子/物件、看向窗外/手机 |
| `concealing` | 表情控制过于完美（不自然的平静）、手指在桌面下收紧、呼吸刻意放慢 |
| `pressuring` | 前倾、降低声音、沉默等待、占据空间 |
| `seducing` | 缓慢靠近、声音变轻柔、为对方倒水/开门、目光停留 |
| `confessing` | 不看对方、语速加快后突然停住、手无处安放、声音变小 |
| `testing` | 故意给出一个停顿等待反应、抛出信息后观察眼神 |
| `bonding` | 降低音量、分享物件、沉默但不尴尬、微笑但不说话 |
| `asserting` | 站起来、声音提高或降低、直视对方、放下手中物件 |

## Failure Modes

| symptom | root_cause | fix |
| --- | --- | --- |
| 对白只有语气标注没有戏剧动作 | dialogue_subtext_map 缺失 | 回到 N4-FIELD 补充 |
| 潜台词太明显失去张力 | 表演提示写得太直白 | 改为暗示性更强的语境描述 |
| 潜台词标注变成心理解释 | 把角色内心想法写进表演提示 | 表演提示只写"在做什么"，不写"在想什么" |
| 所有对白都标注了戏剧动作 | 标注过度，日常对白不需要 | 只标注关键转折/博弈/揭示对白 |
| 对白画面与戏剧动作不一致 | 画面承托方向与潜台词矛盾 | 调整画面承托匹配戏剧动作 |

## Reusable Heuristics

- 对白不只是"谁说什么"，更是"这句话在执行什么戏剧动作"——试探、隐瞒、施压、引诱、逃避。
- 最好的对白是表面和行为之间有裂缝：角色说"我很好"的同时手指在桌面下攥紧。
- 潜台词的力量在于不说出口。表演提示写清了戏剧动作，演员就知道"要达成什么、不能暴露什么、用什么行为绕过去"。
- 不是每句对白都需要潜台词。信息传递型对白（"三点开会""门在那边"）直接说就好——给它们加潜台词反而显得做作。
- 潜台词张力来自"观众隐约感觉到但不确定"。太明确就变成明台词，太隐晦观众就错过了。
