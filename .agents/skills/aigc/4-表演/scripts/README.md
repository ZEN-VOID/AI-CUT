# 4-表演 Scripts

本目录只承载机械辅助脚本和说明。`4-表演` 的心理反应可感知化、演员演技五层控制、潜台词行为化、场面调度和沉默反应判断必须由 LLM 直接完成。

允许脚本职责：

- 检查输出路径、frontmatter、必需证据字段和报告字段是否存在。
- 扫描常见主观意图词、场景末尾总结块、摄影越权词和占位泄露。
- 统计 `心理反应`、`内心独白`、`场面调度` 等字段的基础覆盖情况。

禁止脚本职责：

- 生成或改写表演正文。
- 自动创造微表情、身体联动、潜台词行为或场面调度。
- 替代 `review/review-contract.md` 的人工/LLM 质量判断。
