# 青春甜宠题材模板 (Youth Sweet Romance)

> **核心卖点**: 纯爱心动 + 校园日常 + 双向奔赴。在最美好的年纪，遇见最美好的你。

## 1. 核心流派细分

### 校园恋爱流
- **特点**: 校园背景下的纯爱故事——教室、操场、图书馆
- **核心爽点**: 青春悸动、暗恋成真、毕业告白
- **创作要点**: 校园恋爱不是"谈恋爱"——是"在学习的间隙偷偷想你"

### 青梅竹马流
- **特点**: 从小一起长大，日久生情
- **核心爽点**: 默契满分、家人认可、理所当然的偏爱
- **创作要点**: 青梅竹马的挑战——太熟了没有新鲜感→需要"重新发现"对方的契机

### 欢喜冤家流
- **特点**: 从互怼到互爱——嘴上说讨厌，心里全是对方
- **核心爽点**: 斗嘴日常、口是心非、确认心意的反转
- **创作要点**: 斗嘴要有"甜"的底色——不能真伤到对方

### 治愈暖甜流
- **特点**: 无明显反派，以温暖日常为主——两个人互相治愈
- **核心爽点**: 被宠爱的幸福感、"看到就想谈恋爱"
- **创作要点**: 没有反派不代表没有冲突——内在冲突（自卑/过往/梦想）也是戏

## 2. 甜宠互动设计

### 心动时刻类型
- **意外接触**: 不小心碰到手——两人都僵硬了
- **专属特权**: 只对ta做的某件事——"只有你能..."
- **默默关注**: 偷偷记住ta的喜好——ta不经意说起，你默默记下
- **当众维护**: 在所有人面前为ta说话——"我相信ta"

## 4. AI 写作专项指南

### 常见陷阱
1. **甜过头无聊**: 全程没有冲突 → "甜"需要对比才能凸显——偶尔的小误会小别扭让甜更甜
2. **对话像念台词**: "我喜欢你""我也喜欢你" → 用行动和细节代替直接告白
