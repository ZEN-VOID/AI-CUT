# 1-分镜表现 模块说明

## 定位

- 本分支负责先形成稳定的 `shot_spine_patch`，锁定切镜数、插入点和构图骨架。
- 它是 `2-镜花` 的第一阶段，拥有分镜骨架判断权，不拥有摄影、运镜和转场主导权。

## 使用方法

- 每组先确认一条 `水月承接`，再严格按 `切换 -> 插入 -> 构图` 串行执行。
- 先稳住镜头脊柱，再让后续模块叠加光色、运动和转场。
- 汇流时优先保留主冲突节拍、动作连续和可读的分镜标记。

## 失真与修正

- 若还没形成 shot spine 就开始写摄影词，说明越序了。
- 若插入点一多 prose 立刻变碎，回退到 `切换` 重新估算镜头数。
- 若构图开始发明新关系或新空间，说明脱离了 `1-水月` 的事实层。
