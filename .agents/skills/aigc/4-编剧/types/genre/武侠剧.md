# 武侠剧 Genre Type

本类型卡定义 `武侠剧` 在 `4-编剧` 中的题材策略。它不是子技能，不拥有输出路径、节点、gate 或最终审查权。

## Identification

| field | value |
| --- | --- |
| `type_id` | `genre-wuxia` |
| `axis` | `genre_axis` |
| `aliases` | 武侠、江湖、武林、门派、侠义、刀剑恩仇 |
| `primary_signals` | 江湖规矩、门派身份、师承关系、武功招式、兵器、恩怨、名声、义气、镖局、擂台、客栈、山门 |

## Strategy

| strategy_slot | rule |
| --- | --- |
| `dramatic_pressure_bias` | 把冲突落到江湖名声、门派规矩、师承义务、恩怨债、公开见证和侠义选择。 |
| `rhythm_bias` | 适合“压迫 -> 出手前停顿 -> 见证者反应 -> 反制/留招 -> 尾钩余波”的节奏。 |
| `controlled_supplement_bias` | 可补站位、兵器状态、门派称呼、围观反应、规矩压力、出手前身体细节；不得新增武功结果或门派事实。 |
| `field_material_bias` | 客栈、山门、练武场、擂台、雨夜街巷、兵器、衣袍、血迹、茶盏、门匾、掌印、剑鞘、群侠沉默。 |
| `climax_hook_bias` | 高潮优先落到出手结果、兵器/伤痕状态、众人反应、名号暴露或规矩被挑战后的可感余波。 |

## Prohibited Drift

- 不得新增 source 不支持的武功、师门、江湖身份、恩怨关系或胜负结果。
- 不得把武侠写成只有招式名；动作必须可见、可听、可演。
- 不得用“侠义感、江湖气、高燃”替代具体人物选择和场面压力。

## Combination Notes

- `正剧 x 武侠剧`：可以强化对白交锋、动作前后停顿、见证者压力和出手余波。
- `解说剧 x 武侠剧`：旁白承接江湖关系和规矩，画面用兵器、门匾、站位、群像和现场痕迹承托，不新增戏剧回合。
