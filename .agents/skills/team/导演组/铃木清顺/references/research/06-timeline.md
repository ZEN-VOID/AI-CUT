# 06 人物时间线

| 时间 | 事件 | 说明 |
|---|---|---|
| 1923 | 铃木清顺出生于东京 | 原名 Seitaro Suzuki，后以 Seijun Suzuki 活动 |
| 1940s | 战争服役，战后复员 | 战争、死亡和制度荒诞成为后续作品底层经验 |
| 1948 | 进入松竹任副导演 | 获得制片厂现场训练 |
| 1954 | 转入日活 | 日活重启制片业务，清顺进入类型片流水线 |
| 1956 | 导演处女作 `Victory Is Mine` | 开始日活导演生涯 |
| 1950s-1960s | 高频拍摄日活类型片 | 黑帮、犯罪、青春、动作等类型命题累积 |
| 1963 | `Youth of the Beast` 等作品 | 风格化暴力、色彩和类型拆解显著增强 |
| 1964 | `Gate of Flesh` | 战后身体、欲望和色彩空间结合 |
| 1965 | `Story of a Prostitute`、`Tattooed Life` | 战争、欲望、身体和造型化空间继续推进 |
| 1966 | `Tokyo Drifter`、`Fighting Elegy` | 日活时期形式高峰之一 |
| 1967 | `Branded to Kill` | 杀手片被推至高度断裂和感官化 |
| 1968 | 被日活解雇 | 成为其作者神话和职业断层的核心事件 |
| 1970s | 诉讼后长期难以拍摄院线长片 | 转向电视、写作、广告等工作 |
| 1980 | `Zigeunerweisen` | 复出并进入 Taisho 梦幻/仪式美学阶段 |
| 1981 | `Kagero-za` | 延续梦境、欲望和古董空间 |
| 1991 | `Yumeji` | 后期三部曲完成 |
| 2001 | `Pistol Opera` | 晚期回访 `Branded to Kill` 式杀手等级神话 |
| 2005 | 晚期歌舞幻想作品 | 舞台化、色彩和人工性进一步明亮化 |
| 2017 | 逝世，享年 93 岁 | 国际影响和回顾展继续扩散 |
| 2022 | `Branded to Kill` 4K 修复获 Venice Classics 最佳修复片 | 影片从争议问题片进入修复经典语境 |
| 2023 | 诞辰百年前后多地回顾 | BFI、Japan Society、Harvard Film Archive 等机构继续重估其谱系 |
