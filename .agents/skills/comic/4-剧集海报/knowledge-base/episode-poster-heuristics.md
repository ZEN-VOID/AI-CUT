# 剧集海报知识库

本文件保存可检索的稳定经验。执行时按需读取，不替代 `SKILL.md`、`steps/`、`types/` 或 `review/` 的强制合同。

## High-Value Heuristics

- 海报最值钱的是“一眼钉住本集最贵的悬念”，不是把角色画得完整。
- 如果主体正确但海报仍发空，通常缺少前景压力物、背景后果或明确动作瞬间。
- `第N集` 是索引，不是卖点；卖点必须落在 hook title 和代表性画面上。
- 标题可以更传播，但必须从本集事实里抽尖，不能凭空制造另一个故事。
- 高光点发现要先列候选再筛选，避免第一眼选中漂亮但不值钱的镜头。
- 颜色应服务情绪命题：威胁、诱惑、逃亡、崩坏、秘密、翻脸各自需要不同光线策略。
- 生图 prompt 不是字段堆砌；它必须能让 `.agents/skills/cli/imagegen` 理解比例、主体、瞬间、文字、层次和风格。

## Title Mechanism Cheatsheet

| title type | best for | pattern |
| --- | --- | --- |
| 秘密型 | 门后、身份、真相未揭 | `原来她早就知道` |
| 代价型 | 伤害、牺牲、偿还 | `这一刀要有人还` |
| 关系型 | 背叛、翻脸、回头 | `他终于慌了` |
| 威胁型 | 危机逼近、逃不掉 | `今晚别开门` |
| 反差型 | 表面平静、内部崩坏 | `笑着的人先崩了` |

## Prompt Completeness Checklist

- 海报比例与画幅。
- 主体人物及关系。
- 代表性动作或瞬间。
- 前景遮挡、压力物或钩子物。
- 背景空间、威胁或后果。
- `第N集` 小字与大标题双轴居中。
- 氛围、主色、强调色和光线策略。
- 漫画风格连续性和禁止项。
