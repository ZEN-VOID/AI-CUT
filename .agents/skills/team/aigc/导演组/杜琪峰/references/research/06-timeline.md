# 06 · 人物时间线

| 时间 | 事件 | 影响 |
|---|---|---|
| 1955 | 杜琪峰生于香港。 | 香港城市经验成为后期电影身份和空间气质底层。 |
| 1970s | 进入 TVB 工作。 | 获得电视工业、演员调度和快速生产训练。 |
| 1980 | 执导电影《碧水寒山夺命金》。 | 早期电影尝试后继续回到电视系统积累。 |
| 1988 | 《城市特警》。 | 开始在动作/警匪类型中建立经验。 |
| 1989 | 《阿郎的故事》。 | 商业情节剧与明星片能力获得证明。 |
| 1993 | 《东方三侠》等类型片。 | 女性动作、漫画式类型和商业奇观经验扩展。 |
| 1996 | 与韦家辉创办银河映像。 | 进入团队作者性和自主生产阶段。 |
| 1999 | 《枪火》。 | 低预算、空间调度、静止动作美学成为标志。 |
| 2003 | 《PTU》。 | 夜间香港、警队与城市空间调度进一步成熟。 |
| 2004 | 《大事件》。 | 七分钟开场长镜头成为控制力代表。 |
| 2005-2006 | 《黑社会》《黑社会2》《放逐》。 | 江湖制度、权力继承和宿命群像成为国际识别点。 |
| 2005 | Fresh Wave 项目启动。 | 从个人创作扩展到香港青年电影生态。 |
| 2007-2008 | 《神探》《文雀》。 | 类型混合与城市轻盈感并存。 |
| 2011 | 《夺命金》。 | 金融压力、社会制度与多线叙事结合。 |
| 2012 | 《毒战》。 | 内地合拍机制下的类型实验与妥协。 |
| 2015 | 《华丽上班族》。 | 音乐剧、舞台化空间和商业世界实验。 |
| 2016 | 《三人行》。 | 医院封闭空间与数字/类型实验。 |
| 2019 | 《我的拳王男友》。 | 近年已上映导演长片之一，混合体育、爱情、音乐等类型。 |
| 2024 | Asian Cinema 发表长访谈。 | 公开回顾创作方法、香港电影现状、新片和《黑社会3》愿望。 |

## 最新动态（截至 2026-04-15）

- 2024 年访谈记录其仍在拍新片，但未找到已正式上映的新导演长片。
- 仍公开表达对《黑社会3》的创作愿望，同时将其与香港政治、团队风险和个人处境联系起来。
- Fresh Wave 和香港青年电影生态仍是其行业角色的一部分。

## 待更新观察点

- 新片是否正式公布片名、演员、发行年份。
- 《黑社会3》是否进入正式开发或拍摄。
- 杜琪峰是否继续在影展、教育或香港电影治理活动中发表新观点。
