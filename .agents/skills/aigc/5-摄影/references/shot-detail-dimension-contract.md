# Shot Detail Dimension Contract

本文件定义每条 `分镜N` 在自然语句中应覆盖的扩展维度。当前 `5-摄影` 的分镜明细侧重运镜、构图、光影、色彩等摄影语法，本合同补充角色表演、陪体/前景/背景动态、镜头视角变化、光影反射、动态焦点和节奏同步等维度，使分镜明细成为信息点密集、可被下游图像/视频阶段完整消费的自然中文镜头文字。

## Core Rule

每个 `分镜N` 的内部裁决必须审视以下六个维度族群，判断当前画面句子中哪些维度自然存在、值得被摄影机表达。存在则写入，不存在则跳过——不为凑数而硬塞。成稿时将覆盖的维度压缩为信息点密集的自然语句，不输出维度标签、参数清单或模板填空。

维度选取遵循"应有则有、没有不必强制"原则：一个只建立空间的环境镜没有角色在场，就不需要角色表演维度；一个只拍门板推开的道具特写没有陪体和前景，就不需要陪体动态和前景动态。维度数量完全取决于画面句子实际承载的信息，不设下限也不设上限。

## Dimension Families

### 1. 角色表演维度 character_performance

#### 角色情绪表现 emotion_expression

刻画角色内心变化与外在表情层次。

| dimension | description | execution cue |
| --- | --- | --- |
| 微表情变化 | 瞳孔收缩、嘴角微颤、眉心紧锁、鼻翼扇动 | **正面双眼特写**：用特写或近景捕捉瞬间变化，只拍正面双眼（眉骨到鼻尖），禁止侧面单眼；用特写捕捉瞬间变化，不用旁白解释情绪 |
| 呼吸节奏 | 呼吸加速/屏息/深呼吸/叹息 | 肩膀起伏、胸腔扩张、鼻翼张合作为可见指标 |
| 沉默压力 | 无台词时的情绪张力 | 停顿的时长、眼神回避、手指动作承托沉默 |
| 表情层次 | 多层情绪叠加（如表面平静+眼底恐惧） | 用景深分层：嘴部镇定在焦点内，眼底慌乱在浅景深中可读；**情绪转折类节拍优先正面近景/特写** |
| 情绪词转译（非独白场景） | 抽象情绪词→可见动作 | **强制规则**：不得写"紧张""愤怒""压抑"；必须转译为咬肌收紧鼓起、眉心竖纹加深、鼻翼微张、嘴角下拉、手指抓紧衣角、肩膀内收等可见物理动作 |
| 眼部特写 | 泪光、瞳孔变化、眼睑颤动 | **正面双眼特写**：瞳孔、泪光、眼睑颤动必须以双眼对称呈现；禁止单眼侧面特写（AI 视频易畸形） |

#### 角色肢体语言 body_language

姿态与走位设计。

| dimension | description | execution cue |
| --- | --- | --- |
| 姿态语义 | 前倾=压迫、后仰=防御、侧身=回避、僵直=恐惧 | 让姿态变化成为叙事动作而非静态摆拍 |
| 走位路径 | 入场方向、停留位置、退场路径 | 绑定空间锚点：从门到讲台、从座位到窗边 |
| 手部动作 | 握拳、松手、指尖触碰、手指颤抖 | 手部动作与镜头焦点同步：先给手，再给动作结果 |
| 身体距离 | 人物间物理距离变化 | 距离缩短=压迫/亲密，距离拉远=隔离/逃离 |

#### 角色语气和语速 dialogue_pace

对白节奏与情绪节拍。

| dimension | description | execution cue |
| --- | --- | --- |
| 语速变化 | 加速=紧张/愤怒、减速=强调/威胁、停顿=转折/犹豫 | 语速变化必须对应镜头运动速度变化 |
| 语气层次 | 低语、正常、提高、嘶吼、气声 | 语气强度决定景别尺度：低语用特写/近景，提高用中景 |
| 对白停顿 | 台词间的沉默 | 停顿时长影响镜头停留：短停=标准镜，长停=held 镜+微表情 |
| 画外音节奏 | 旁白/画外音的语速和节奏 | 画外音节奏决定镜头运动速度和剪辑点 |

#### 角色镜头意识 camera_awareness

角色与镜头互动方式。

| dimension | description | execution cue |
| --- | --- | --- |
| 直视镜头 | 打破第四面墙、挑衅、求助、宣告 | 只在有明确戏剧动机时使用，保持稀缺 |
| 视线回避镜头 | 被观察感、隐瞒、不安 | 角色目光从镜头方向滑开，暗示被监控或内心不安 |
| 镜头跟随意识 | 角色移动时对镜头距离的感知 | 角色朝镜头走来时身体占比逐渐增大，或背对镜头远离时逐渐缩小 |
| 主观镜头锚定 | 角色的视线成为镜头运动的起点 | 先给角色眼神方向，再沿视线运动到目标物 |

### 2. 非角色动态维度 non_character_dynamics

#### 运动特征 non_bio_motion

非生物主体的动态表现。

| dimension | description | execution cue |
| --- | --- | --- |
| 道具运动 | 门被推开、窗帘被风吹、粉笔字自行改变 | 让道具运动有起点、过程和落点，不只写"动了" |
| 机械运动 | 日光灯闪烁、钟表指针、投影仪光束 | 机械运动的节奏可与角色情绪形成同步或反差 |
| 自然运动 | 风、雨、灰尘、光影漂移 | 自然元素的运动方向和速度暗示空间状态 |
| 规则/异常运动 | 文字生长、颜色渗透、物理违反 | 异常运动的速度和方式决定惊悚/奇幻基调 |

#### 陪体动态 companion_dynamics

次要元素或群体的活动。

| dimension | description | execution cue |
| --- | --- | --- |
| 群体统一动作 | 全班低头、同时转头、集体起立 | 统一动作的同步程度暗示制度压迫强度 |
| 个体分散活动 | 不同学生各自的微动作 | 分散注意力时用深焦保持多点可读 |
| 陪体反应 | 旁观者的表情、姿态变化 | 陪体反应镜作为主事件的余波承托 |
| 陪体与主体互动 | 传递物品、身体接触、遮挡 | 互动动作可成为视线引导和焦点转移的触发点 |

#### 前景动态 foreground_motion

前景中运动元素。

| dimension | description | execution cue |
| --- | --- | --- |
| 前景遮挡运动 | 人物走过、门板开关、窗帘摆动 | 前景运动可创造"窥视"感或临时信息遮蔽 |
| 前景引导运动 | 前景元素向画面深处运动引导视线 | 前景运动方向决定观众注意力跟随路径 |
| 前景干扰运动 | 不相关元素在前景晃动 | 制造不安或分散注意力，配合手持微晃 |

#### 背景动态 background_motion

背景中流动元素。

| dimension | description | execution cue |
| --- | --- | --- |
| 背景人物活动 | 走廊行人、窗外人影、远处学生 | 背景运动速度与前景主体形成节奏对比 |
| 背景光影变化 | 云层移动导致光线变化、灯管闪烁 | 背景光变可暗示时间流逝或异常入侵 |
| 背景环境流动 | 树叶、窗帘、旗帜、水面 | 持续的背景流动维持空间活力或不安定感 |

### 3. 镜头技术维度 camera_technique

#### 景别变化 shot_scale_change

叙事节奏驱动的景别调整。

| dimension | description | execution cue |
| --- | --- | --- |
| 景别推近 | 大远景→远景→中景→近景→特写 | 推近对应信息聚焦、情绪加压、注意力收窄 |
| 景别拉远 | 特写→近景→中景→全景 | 拉远对应空间揭示、情绪释放、处境展示 |
| 景别跳切 | 直接从远景跳到特写 | 跳切对应惊吓、突然揭示、认知翻转 |
| 景别保持 | 同一景别内完成动作 | 保持对应观察、冷静压迫、制度不可抗拒 |

#### 镜头运动 camera_movement

镜头位移与跟随方式。

| dimension | description | execution cue |
| --- | --- | --- |
| 推/拉 | 镜头向主体靠近/远离 | 速度决定情绪：极慢推=悬疑逼近，快速推=冲击 |
| 摇/移 | 镜头水平/垂直转动或平移 | 摇镜跟随视线，移镜跟随走位 |
| 升降 | 镜头垂直位移 | 升起=释然/宏观，下降=压迫/聚焦 |
| 跟拍 | 镜头跟随运动主体 | 保持主体在画面中的相对位置不变 |
| 旋转/倾斜 | 镜头绕光轴旋转 | 少量使用表示眩晕、失衡、异常 |
| 手持/稳定 | 摄影机支撑方式 | 手持=临场感/不安，稳定=冷静/制度 |

#### 镜头视角 camera_angle

视线与透视变化。

| dimension | description | execution cue |
| --- | --- | --- |
| 视角切换 | 平视→仰视→俯视 | 视角变化必须对应权力关系或空间认知变化 |
| 视角固定 | 全程同一视角 | 固定视角对应冷静观察或制度碾压 |
| 主观↔客观切换 | 从角色主观视角切到旁观客观视角 | 切换点对应信息揭示或情绪转折 |
| 窥视视角 | 门缝、窗缝、遮挡物之间的视线 | 窥视视角增强被观察感和空间层次 |

### 4. 光影精细维度 light_refinement

#### 光影变化 light_change

随情绪的光线与阴影变化。

| dimension | description | execution cue |
| --- | --- | --- |
| 光线渐变 | 亮→暗、暗→亮、色温变化 | 渐变速度对应情绪变化节奏 |
| 光线突变 | 突然开灯/关灯/闪光 | 突变对应惊吓、揭示、规则入侵 |
| 阴影移动 | 阴影随人物或光源移动 | 阴影的运动方向和速度暗示威胁来源 |
| 光线质感变化 | 从硬光变柔光或反之 | 质感变化对应情绪基调转换 |

#### 光影与反射 reflection_refraction

反射/折射强化象征与情感。

| dimension | description | execution cue |
| --- | --- | --- |
| 镜面反射 | 玻璃、水面、黑板、瞳孔中的倒影 | 反射内容可承载信息揭示或情绪映射 |
| 折射变形 | 透过水杯/玻璃的变形 | 变形暗示感知扭曲或空间不可信 |
| 光斑/高光 | 镜头眩光、水面光斑、金属反光 | 光斑位置引导注意力或暗示危险 |
| 色彩反射 | 红色物体在白色表面的反射色 | 色彩反射可把危险母题渗透到安全区域 |

### 5. 焦点精细维度 focus_refinement

#### 动态焦点与景深变化 focus_depth_change

焦点转移增强情绪表达。

| dimension | description | execution cue |
| --- | --- | --- |
| 焦点拉移 | 从 A 拉到 B | 拉移速度决定信息揭示节奏：慢拉=悬疑，快拉=冲击 |
| 失焦再合焦 | 整体失焦后合到新焦点 | 失焦对应意识模糊/眩晕，合焦对应清醒/发现 |
| 景深变化 | 浅景深→深景深或反之 | 景深变化对应信息从集中到分散或反之 |
| 焦点呼吸 | 焦点在近处和远处之间微调 | 微调对应犹豫、注意力不稳定、感知异常 |

### 6. 节奏同步维度 rhythm_sync

#### 节奏同步 rhythm_synchronization

镜头运动与声画节奏一致性。

| dimension | description | execution cue |
| --- | --- | --- |
| 声画同步 | 镜头运动与声音节拍对齐 | 推镜落在鼓点上，摇镜跟随音乐旋律线 |
| 呼吸同步 | 镜头运动与角色呼吸节奏同步 | 呼气时推近，吸气时停顿 |
| 动作同步 | 镜头运动与角色动作节拍同步 | 推门瞬间甩镜，举手瞬间急停 |
| 情绪同步 | 镜头运动与情绪曲线同步 | 紧张段加速运镜，释放段减速或静止 |
| 剪辑节奏同步 | 镜头内部运动与切换频率协调 | 快切段落内运镜精简，长镜内运镜丰富 |

## Output Rule

1. 维度覆盖是内部裁决，不出现在正文标签中。
2. `shot_design_plan` 必须包含 `dimension_coverage` 字段，说明每条计划分镜覆盖了哪些维度以及为什么（画面中确实存在该信息且值得被摄影机表达）。
3. 成稿时把覆盖的维度压缩为信息点密集的自然语句，融入 `分镜N` 的正文中；不输出"角色情绪：紧张"等标签形式。
4. 维度选取遵循"应有则有、没有不必强制"：画面中没有角色就不写角色表演维度，没有运动的非生物就不写运动特征，没有前景就不写前景动态。维度数量完全取决于画面句子实际承载的信息。
5. 同一 `visual_unit` 内不同 `分镜N` 的维度覆盖可以不同：第一镜可能侧重空间建立（景别+运动+背景），第二镜可能侧重表演承托（情绪+肢体+语气+焦点）。
6. 维度选取不得违背 `natural-shot-detail-writing-contract.md`：不输出参数清单、不机械套用维度标签、不连续同构。
7. 禁止为了满足维度数量目标而虚构画面中不存在的信息：如果画面句子只描述了空间环境，不要硬加角色情绪或肢体语言。
8. **情绪维度补充规则**：情绪转折类画面应覆盖"眼部特写"维度（正面双眼）和"情绪词转译"维度；正面双眼（眉骨到鼻尖）用于捕捉瞳孔、泪光、眼睑颤动等对称可见信息；抽象情绪词必须被转译为可见肌肉变化和身体动作。
9. **心理变化维度补充规则**：涉及剧烈心理变化（震惊、崩溃、醒悟）的画面，维度覆盖应包含节奏同步（slow_burn/hold），用慢镜头或多角度正面切换实现，不在此时使用快速运镜或复杂环绕。

## Dimension Coverage Examples

### 示例 1：低信息环境镜（画面中无角色，只覆盖存在的维度）

```markdown
分镜1（约2秒）: 镜头从日光灯管的嗡鸣白光下降，经过黑窗帘的褶皱边缘，停在空荡课桌的第一排；桌面铅笔灰在冷白光下闪一下。
```

覆盖维度：镜头运动（下降）、景别变化（从光源到桌面）、背景动态（铅笔灰闪光）。画面中没有角色，不硬加角色表演维度；没有运动的非生物主体，不硬加运动特征——应有则有，没有不写。

### 示例 2：高信息表演镜（画面中自然存在的维度都覆盖）

```markdown
分镜1（约2秒）: 长焦压缩后排课桌，镜头贴近苏红叶停在桌沿的手指，红指甲在冷白灯下压住木纹；背景黑板字退成一片冷白。
```

覆盖维度：角色肢体语言（手指停在桌沿）、景别变化（贴近特写）、镜头运动（推近）、光影变化（冷白灯下的红指甲）、动态焦点（焦点在手指，背景虚化）。这 5 个维度都是画面中自然存在的信息，不为凑数补节奏同步或镜头意识。

### 示例 3：高点镜（画面中信息密度高，维度自然丰富）

```markdown
分镜1（约3秒）: 极慢推轨向黑板表面，镜头从右上角粉笔字的粗粝质感贴近，白色粉笔灰在灯光下闪烁；笔画末端开始延伸，新笔画从旧笔画里挤出来。
```

覆盖维度：运动特征（粉笔字自行生长）、镜头运动（极慢推轨）、景别变化（推向特写）、动态焦点（焦点跟随笔画末端）、光影与反射（粉笔灰在灯光下闪烁）、节奏同步（极慢推轨配合诡异气氛的慢节奏）。画面中没有角色在场，不硬加角色情绪；但画面信息密度高，6 个维度自然存在。
