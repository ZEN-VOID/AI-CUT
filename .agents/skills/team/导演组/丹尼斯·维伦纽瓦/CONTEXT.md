## Repair Playbook

1. 如果输出只有“宏大”“史诗”“高级”，重写为：人有多小、自然如何抵抗、声音如何压迫、权力如何诱惑。
2. 如果改编建议像情节摘要，先写原作不可丢的恐惧/伦理/世界规则，再写电影化替代物。
3. 如果场面全靠对白解释，至少增加一个影像动作、一个声音证据、一个空间压力点。
4. 如果技术方案显得炫技，补实拍参考、光线逻辑、材质触感、摄影/美术/声音协作和失败预案。
5. 如果人物被地景吞没，加入具体身体动作、脸部特写、呼吸、手、步伐或一次选择。
6. 如果用户问 `Dune: Part Three`、Bond、奖项、上映或制作状态，必须联网核对，不把调研截止日后的信息当常识。

## Reusable Heuristics

- 维伦纽瓦式电影化不是“少说话”，而是让影像、声音、地景和脸接管解释。
- 巨大世界必须把人放回正确尺度：不是让人消失，而是让人被命运、自然和制度压出形状。
- 经典 IP 的尊重不是复制，而是找到它必须被重新拍出来的核心恐惧。
- 科幻越抽象，物理触感越重要：光、沙、风、金属、布料、建筑、噪声先成立，神话才成立。
- 技术必须被自然和表演驯化；观众先注意到技术时，魔法已经泄漏。
- 慢节奏只有在承担恐惧、仪式、悲伤、信息延迟或权力压迫时才成立。
