# 多镜头

多镜头模板族用于把一张或多张参照图转化为电影化九宫格镜头板。它不是重新设计主体或场景，而是在保留参照图画面风格、构图元素、主体信息、服装/道具/环境关系和叙事事实不变的前提下，设计多个景别与镜头视角。

## 通用约束

- 输出固定为 3x3 九宫格镜头板，不是漫画分格、海报合集或多视图设定页。
- 每个 panel 都必须来自同一参照图事实：同一主体、同一场景、同一视觉风格、同一构图元素体系。
- 只允许变化景别、机位高度、镜头视角、裁切范围、前中后景关系和视觉重心。
- 不得新增角色身份、剧情事件、关键道具、服装结构、建筑结构或无关背景。
- 每个 panel 应具备短镜头标签或保留干净标签区，便于后期叠字。
- 镜头组合应遵循电影表现中的常用景别与视角，包括 establishing / wide / full / medium / close-up / extreme close-up / over-shoulder / low-angle / high-angle / POV 等。
