# 修仙题材模板 (Xianxia/Cultivation)

> **核心卖点**: 逆天改命 + 长生久视 + 残酷法则。修仙是与天争命，也是与人争利。

---

## 创意约束推荐（可选）
- 推荐约束包：Pack M01 / Pack M02
- 通用叠加：Pack U03

## 1. 核心流派细分

### 凡人流 (Mortal)
- **特点**: 主角资质平平，靠算计、谨慎和金手指逆袭。
- **核心爽点**: 智商碾压，积少成多，扮猪吃虎。
- **代表作**: 《凡人修仙传》——定义了"谨慎修仙"范式；《仙逆》——情绪驱动型凡人流。
- **创作要点**: 主角每一步都要有"不得不"的理由；每次越级击杀必须有合理消耗。

### 无敌流 (Invincible)
- **特点**: 开局即巅峰（或拥有无敌背景/师父）。
- **核心爽点**: 一路横推，专治各种不服，打破规则。
- **代表作**: 《帝霸》——极致无敌流的范式。
- **创作要点**: 无敌不等于无聊——敌人要从"实力压制"转向"规则/道德/情感困境"来制造张力。

### 家族流 (Clan)
- **特点**: 主角带领家族崛起，种田经营。
- **核心爽点**: 集体飞升，资源滚雪球，势力争霸。
- **代表作**: 《仙府之缘》《修真家族崛起》。
- **关键**: 家族内部必须团结（读者不接受家族内斗消耗）。

### 苟道流 (Cautious)
- **特点**: 绝不沾因果，绝不立危墙之下，把防御点满。
- **核心爽点**: 敌人死光了我还在，熬死对手也是一种胜利。
- **代表作**: 《苟在修仙世界》——极度谨慎的范式。
- **反转**: 被迫出手时，展现出碾压级的实力（因为准备太充分了）。

### 剑修流 (Sword Cultivator)
- **特点**: 以剑为道，一剑破万法，极致的攻击力。
- **核心爽点**: 纯粹的力量美学，一剑斩断一切规则。
- **代表作**: 《剑来》——剑道与世界观并重的标杆。
- **创作要点**: 剑意不仅是战斗力，更是人生态度和道心的外在表达。

### 阵法师/炼丹师流 (Specialist)
- **特点**: 主角以辅助职业为切入点，靠技术立足。
- **核心爽点**: 技术垄断带来的话语权——别人需要你的丹药/阵法，不敢动你。
- **代表作**: 《最强炼丹师》《超级阵法宗师》。

## 1.5 经典金手指与敌手压力

### 常用金手指
| 类型 | 核心机制 | 典型爽点 | 风险控制 |
|------|---------|---------|---------|
| 吞噬系统 | 吞噬敌人获得能力 | 越级击杀后实力暴涨 | 限制吞噬上限（不能跨大境界） |
| 签到系统 | 在地点/时机签到获得奖励 | 捡到同阶无法企及的资源 | 签到地点需要冒险 |
| 戒指老爷爷 | 强者残魂指导修炼 | 传授绝学后当场打脸天才 | 残魂终将消散或反噬 |
| 重生流 | 带着前世记忆重开 | 先知先觉地截胡机缘 | 蝴蝶效应导致信息过期 |
| 特殊体质 | 天生修炼速度与上限更高 | 短期突破震惊宗门 | 体质有代价（寿命/代价） |
| 小世界/洞天 | 随身携带独立空间 | 种植灵药、加速修炼 | 空间需要资源升级 |
| 时间加速 | 修炼速度是常人的N倍 | 同辈还在练气，你已金丹 | 加速有限制（每日/每月次数） |

### 常用桥段
- `退婚流`：悔婚羞辱 -> 主角崛起 -> 三年后打脸 -> 对方后悔。**变体**：对方退婚的理由从"你太弱"升级为更复杂的利益权衡。
- `打脸路人流`：轻视 -> 展示底牌 -> 群体震惊 -> 旧秩序失声。**变体**：路人不是一次性消耗品，而是后续剧情的铺垫。
- `秘境夺宝流`：线索出现 -> 试炼争夺 -> 获宝 -> 当场转化为战力。
- `宗门大比流`：报名被嘲 -> 连胜晋级 -> 决赛逆转 -> 身份跃迁。
- `上古遗迹流`：探索陨落大能遗迹 -> 破解阵法/考验 -> 获得传承 -> 引出上古恩怨线。

### 常见反派功能位
| 类型 | 特征 | 打脸方式 | 适用阶段 | 升级版设计 |
|------|------|---------|----------|-----------|
| 纨绔少爷 | 嚣张跋扈，仗势压人 | 当众击败 | 前期 | 不要只写蠢——给他一个"有恃无恐"的靠山 |
| 伪君子长老 | 表面正派，暗地算计 | 当众揭露 | 中期 | 伪君子被揭穿后，应有"道心崩塌"的行为变化 |
| 同门师兄 | 嫉妒天赋，处处设卡 | 反杀或反超 | 前期 | 嫉妒不是天生——写他曾经的期待和失落 |
| 敌对宗门 | 集体敌意，资源争夺 | 歼灭小队或夺走机缘 | 中后期 | 敌对宗门应有独立于主角存在的战略目标 |
| 魔道巨擘 | 实力碾压式的对手 | 智取/借力/代价交换 | 后期 | 魔道强者应有"为什么入魔"的背景故事 |

## 2. 世界观：修仙社会学

### 黑暗森林法则
- **资源有限**: 灵气、灵药、矿脉都是稀缺的。
- **杀人夺宝**: 在野外遇到陌生修士，第一反应是"他会不会杀我"，第二反应是"我能不能杀他"。
- **怀璧其罪**: 没有实力却拥有重宝，就是原罪。
- **信息不对称**: 高阶修士掌握更多规则信息（比如哪个秘境有重宝），低阶修士靠命去试。

### 阶级壁垒
- **散修**: 修仙界的底层，为了几块灵石拼命。**进阶写法**：散修之间也有互助网络和小团体文化。
- **宗门弟子**: 有组织，有靠山，看不起散修。**进阶写法**：宗门内部也有残酷的淘汰机制。
- **世家子弟**: 垄断资源，血脉传承。**进阶写法**：世家不是铁板一块——分支间的竞争往往比对外更激烈。
- **圣地/仙朝**: 金字塔顶端，制定修仙界的基本规则。

## 3. 机缘获取四步法 (Opportunity Arc)

不要直接让主角捡到宝物。要经历：

1. **传闻 (Rumor)**: 酒馆听说、拍卖会线索、古籍记载。**进阶**：传闻可能是陷阱——有人故意放出去钓鱼。
2. **探索 (Exploration)**: 进入秘境/遗迹，遭遇机关、阵法、妖兽。
3. **争夺 (Conflict)**: 宝物出世，多方势力混战。主角浑水摸鱼或强势镇压。
4. **收获 (Reward)**: 获得宝物，并立即转化为实力（如：当场突破）。

**五步进阶版**（增加竞争代价）：
5. **代价 (Cost)**: 获取机缘后立刻产生连锁反应——被追杀、宝物被觊觎、暴露底牌。

## 4. 境界与战力体系

### 境界划分（标准版）
1. **练气**: 引气入体，法术初显。（寿命 100）
2. **筑基**: 铸就道基，御剑飞行。（寿命 200）
3. **金丹**: 一粒金丹吞入腹，我命由我不由天。（寿命 500）
4. **元婴**: 碎丹成婴，肉身毁坏可夺舍。（寿命 1000）
5. **化神**: 领悟法则，神识化形。（寿命 2000）
6. **炼虚** → **合体** → **大乘** → **渡劫**。

### 境界变体
| 体系 | 特色 | 代表 |
|------|------|------|
| 武者体系 | 后天→先天→宗师→大宗师→天人→武圣 | 偏武侠向修仙 |
| 星辰体系 | 星徒→星师→星主→星王→星皇→星帝 | 偏玄幻向 |
| 神魔体系 | 以神魔血脉为核心等级 | 偏西幻向 |

### 战力崩坏控制
- **大境界压制**: 筑基杀练气如杀鸡，不可逾越（除非有逆天金手指）。
- **小境界差距**: 初期打后期很难，但可以用装备/符箓/阵法弥补。
- **人海战术**: 低阶修士数量再多，也堆不死高阶修士（AOE 技能存在）。
- **丹药/秘术爆发**: 可以暂时突破境界限制，但事后有严重代价。
- **境界瓶颈设计**: 每两个大境界间设置一个"瓶颈"——需要特定的领悟或机缘才能突破，避免连续升级。

## 5. 大纲结构 (200 万字标准)

### 卷一：宗门风云 (1-100 章)
- **新手村**: 外门弟子竞争，筑基之前。
- **核心冲突**: 资源争夺，小人陷害。
- **高潮**: 宗门大比第一，筑基成功。

### 卷二：血色试炼 (101-250 章)
- **地图**: 秘境/遗迹。
- **核心冲突**: 多宗门混战，魔道入侵。
- **成长**: 结丹，获得本命法宝。

### 卷三：海外/中州 (251-500 章)
- **换地图**: 遭遇追杀逃亡，或主动游历。
- **核心冲突**: 散修的艰难生存，结识新伙伴。
- **高潮**: 结婴，建立自己的势力。

### 卷四：界域战争 (501-1000 章)
- **格局**: 人妖两族大战，或正魔大战。
- **地位**: 成为一方大佬，左右战局。

## 6. 追读心理与读者留存

### 追读驱动力
| 驱动力 | 心理机制 | 执行手段 |
| --- | --- | --- |
| 升级期待 | 读者想看主角突破到下一个境界 | 每章结尾暗示"即将突破"的进展 |
| 打脸期待 | 读者想看欺负主角的人被反杀 | 给反派足够的"欠揍"积累，再引爆 |
| 机缘好奇 | 读者想知道下一个秘境有什么 | 提前泄露秘境的"传说"和"历史" |
| 身份悬念 | 读者想知道主角的隐藏身份/背景 | 分阶段释放身世线索，不要一次交代完 |
| 战斗期待 | 读者想看主角展示新获得的能力 | 获得能力后3章内必须有一个展示场景 |
| 复仇驱动 | 读者想看主角对仇人实施报复 | 设置一个"大仇"作为长线钩子 |

### 常见弃书点及防御
| 弃书点 | 原因 | 防御策略 |
| --- | --- | --- |
| 换地图后失去方向 | 读者熟悉的世界被清空 | 换地图前埋好"新世界的引路NPC"和"与旧世界的连接线索" |
| 战力崩坏 | 境界体系失去约束力 | 建立"战力档案"，每次升级前检查是否有逻辑矛盾 |
| 重复套路 | 打脸->升级->打脸无限循环 | 每次打脸更换变量：敌手身份、冲突类型、解决方式 |
| 女主缺失/扁平 | 女角色沦为花瓶 | 给主要女性角色独立于主角的目标和行动线 |
| 水文灌水 | 章节信息密度过低 | 每章至少包含1个信息增量（新线索/新能力/新角色/关系变化） |

## 7. 平台优化建议

### 番茄小说适配
- 开篇3章内必须让主角获得第一个金手指
- 宗门大比提前到50章以内
- 每3章至少1次打脸或突破
- 对话比例 > 35%

### 起点中文网适配
- 境界体系必须有创新点（不能照搬标准版9境）
- 每卷末尾必须有史诗级战斗或重大揭秘
- 配角群要有独立支线和读者记忆点
- 设定文档要经得起考据

## 8. 跨类型融合配方

| 融合类型 | 融合方式 | 关键创新点 |
| --- | --- | --- |
| 修仙 + 悬疑 | 修炼规则 = 推理规则；渡劫 = 案件 | 修仙世界中的"规则推理"——为什么这个秘境只能进3人？规则背后是谁在制定？ |
| 修仙 + 种田 | 宗门经营 + 灵田种植 | 修仙版模拟经营——灵药种植、弟子培养、宗门经济 |
| 修仙 + 科幻 | 灵气 = 暗能量；修炼 = 维度跃迁 | 科技修仙——飞剑是纳米材料，阵法是量子纠缠 |
| 修仙 + 都市 | 都市灵气复苏 | 现代人突然能修仙，社会秩序重构 |

## 9. 可读性工程

### 段落控制
- 打斗场景段落 < 80字
- 信息说明段落 < 150字
- 对白单句 < 30字
- 每3-5段必须有一次视觉焦点切换

### 节奏分布（单章）
- 前300字: 承接上章 + 抛出新信息
- 中间: 展开冲突或推进升级
- 后300字: 留下钩子——"即将突破""对手出现""秘境开启"

### 信息释放原则
- 境界体系: 不在第一章灌完——通过"突破"场景逐步解锁
- 世界观: 通过人物对白和冲突展示，不写设定说明书
- 金手指: 第一章给核心能力，附加功能在后文解锁

## 10. AI 写作专项指南

### 常见陷阱及修复
1. **打斗变成技能播报**: "张三使出飞剑术，李四用金刚罩挡下" → 改写为动作路径、空间关系、身体反震
2. **突破场景缺乏仪式感**: "轰的一声他突破了" → 写身体变化、天地异象、围观者反应
3. **敌人全都是智商负数**: 给每个反派一个"如果他不惹主角，他在做什么"的独立行为逻辑
4. **秘境探索缺乏探索感**: 写空间感（远近、高低、明暗）、材质（墙壁纹理、地面湿度）、声音（回声、水滴、呼吸）

### 自检清单（每章起草后）
- [ ] 本章是否有至少1次"读者会兴奋"的时刻？
- [ ] 主角是否比上一章更接近"突破"？
- [ ] 本章结尾是否有"必须看下一章"的钩子？
- [ ] 是否有超过200字的连续描写段落？有则拆分
- [ ] 打斗/突破场景是否有具体的身体感受？
- [ ] 反派是否有独立于主角存在的理由？
