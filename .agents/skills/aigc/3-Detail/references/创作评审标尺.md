# 3-Detail Creative Review Rubric

## 审核维度

### 1. 结构成立

- `分镜构图` 是否先行
- `分镜数 / 时间 / 剧本正文 / 主体锚定` 是否完整

### 2. 字段边界清楚

- `角色表现` 不抢 `分镜构图`
- `摄影表现 / 运镜手法` 不反向改结构
- `转场特效` 不抢人物和画面主轴

### 3. 具像程度

- 是否能看见动作、空间、物件、光气、关系
- 是否存在抽象评语替代具体内容

### 4. 连续性

- 前后镜是否接得住
- 时间是否连续
- 每镜正文是否与当前镜事实匹配

### 5. 艺术性

- 画面是否有抓手
- 语言是否有节奏
- 美学是否建立在剧情与动作之上，而不是飘在空中

### 6. 知识转译证据

- 若本轮调用了 `knowledge-base/电影学院派/*`，是否能在 `validation-report.md` 里看到：
  - `knowledge_mode`
  - `knowledge_domain`
  - `selected_bundles`
  - `applied_passes`
  - `translation_targets`
- 报告中的知识证据是否真的能回链到当前字段，而不是只写“已参考某书/某包”
- 若本轮没有实际采用学院派知识，是否显式写出 `unused_with_reason`

## 不通过信号

- 先写别的，后补 `分镜构图`
- 字段高度重复，只是换标题
- 一镜里没有清晰观看任务
- 读完整条镜头仍看不出谁在干什么、在哪里、为什么这样拍
- `validation-report.md` 没有知识证据槽位，或只有 bundle 名称没有转译落点
- 把 `180° / 30° / 三点布光 / 焦段参数` 之类教材词直接当字段值
