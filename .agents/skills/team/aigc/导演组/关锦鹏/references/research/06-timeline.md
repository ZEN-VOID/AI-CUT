# 06-timeline · 人物时间线与近年动态

## 时间线

| 时间 | 事件 | 影响 |
|---|---|---|
| 1957 | 生于香港，祖籍广东南海。 | 香港城市经验与华南文化背景成为其身份底色。 |
| 1976 | 进入 TVB 艺员训练班，同时在香港浸会学院修读传播。 | 从电视、表演训练和传播教育进入影像工业。 |
| 1979 | 成为电影全职副导演，参与 `点指兵兵`、`投奔怒海`、`花城` 等新浪潮相关作品。 | 学会现场工业、导演调度和新期香港电影语言。 |
| 1985 | 导演处女作 `女人心`。 | 女性心理与商业明星资源成为早期方法。 |
| 1986 | 导演 `地下情`。 | 都市关系、欲望与创作人焦虑进一步成形。 |
| 1987/1988 | 导演 `胭脂扣`，获香港电影金像奖最佳影片、最佳导演等。 | 幽灵、旧梦、明星文本与女性命运成为代表性方法。 |
| 1989/1990 | 导演 `人在纽约`，获金马奖多项荣誉。 | 离散女性与大中华身份寓言进入作品谱系。 |
| 1991/1992 | 导演 `阮玲玉`，张曼玉获柏林电影节最佳女演员银熊。 | 档案、纪录、重演、明星身体和历史女性互照。 |
| 1994 | 导演 `红玫瑰白玫瑰`。 | 张爱玲文本、欲望秩序和男性自欺进入其叙事。 |
| 1996 | 导演 `男生女相：中国电影之性别`。 | 以性别和 sexuality 重读华语电影，并公开自身观看位置。 |
| 1998 | 导演 `愈快乐愈堕落`。 | 回归前后身份、欲望和城市不安转为更直接的酷儿/都市叙事。 |
| 2001 | 导演 `蓝宇`，获金马奖最佳导演等。 | 同性亲密、公共风险、时间断裂与北京空间成为高峰。 |
| 2005 | 导演 `长恨歌`。 | 上海记忆、女性一生和城市历史相互缠绕。 |
| 2013 | 在北京成立制作公司，监制/参与内地项目，如 `致我们终将逝去的青春`。 | 转向创意监制、跨地域制作与新导演扶持。 |
| 2018 | 导演 `八个女人一台戏`。 | 回到香港、剧场后台、女演员和职业记忆。 |
| 2021 | HKIFF45 选为 Filmmaker-in-Focus，回顾 13 部作品并出版纪念书。 | 其作者谱系被香港国际电影节系统化确认。 |
| 2023 | `但愿人长久` 作为 HAF Goes to Cannes WIP 项目推介，关锦鹏任制片/监制之一。 | 后期工作重点转向扶持香港新导演、项目开发和国际路径。 |
| 2024 | `但愿人长久` 在香港院线/电影节传播中成为近年香港家庭移民题材的重要作品之一。 | 关锦鹏作为监制的“本土经验 + 普遍情感”路线继续可见。 |
| 2026-04-15 | 本次调研未核到 `八个女人一台戏` 之后已正式公映的关锦鹏导演新长片。 | Skill 的“最新动态”应聚焦监制、教学、回顾展和香港新导演支援。 |

## 近年动态判断

- 2021 年 HKIFF45 的 Filmmaker-in-Focus 是最近一次机构级系统回顾。
- 2023-2024 年可核对的近年创作相关动态主要是 `但愿人长久` 的监制/制片路径，以及他在 HK New Talents 访谈中谈到的年轻导演扶持。
- 未找到可靠来源证明截至 2026-04-15 已有 `八个女人一台戏` 之后的新导演长片正式公映。
