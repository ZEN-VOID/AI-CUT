# 06 人物时间线

## 关键节点

| 时间 | 事件 | 对导演系统的影响 |
|---|---|---|
| 1969 | 出生于韩国大邱，后成长于首尔。 | 韩国城市化、家庭结构和社会转型成为长期背景。 |
| 1980s-1990s | 延世大学社会学背景，参与电影社团和短片制作。 | 社会观察、影迷文化和自制影像经验融合。 |
| 1994 | 韩国电影艺术学院学习，完成短片。 | 形成更工业化的影像训练和类型意识。 |
| 2000 | 长片 `Barking Dogs Never Bite`。 | 城市公寓、黑色幽默、日常残酷和普通人荒诞初现。 |
| 2003 | `Memories of Murder`。 | 犯罪片与制度无能、地方暴力和历史空气结合。 |
| 2006 | `The Host`。 | 怪物片、家庭营救、污染政治和黑色喜剧融合，国际能见度提升。 |
| 2009 | `Mother`。 | 母爱、犯罪、执念和最后画面结构成熟。 |
| 2013 | `Snowpiercer`。 | 进入跨国英语制作，用列车压缩阶级系统。 |
| 2017 | `Okja`。 | Netflix/全球平台、动物寓言、食品资本与弱者视点结合。 |
| 2019 | `Parasite` 获 Cannes Palme d'Or。 | 房屋、地下空间、气味和阶级寓言达到国际突破。 |
| 2020 | `Parasite` 获奥斯卡 Best Picture、Best Director、Original Screenplay、International Feature 等。 | 非英语片和韩国电影在奥斯卡史上取得标志性突破。 |
| 2021 | 任第 78 届威尼斯电影节主竞赛评审团主席。 | 从作者导演进入国际影展公共判断角色。 |
| 2024 | 参与韩国电影界关于演员李善均事件调查与媒体伦理的公开呼吁。 | 其公共立场与行业伦理、媒体/警察权力边界相连。 |
| 2025 | `Mickey 17` 公映。 | 复制人、殖民、政治讽刺和英语片科幻工业磨合。 |
| 2025-03-23 至 2027-01-10 | Academy Museum 举办 `Director's Inspiration: Bong Joon Ho` 展览。 | 其档案、分镜、研究材料、道具和影响谱系被机构化呈现。 |
| 2026-04-11 至 2026-04-12 | Academy Museum 举办 `A Weekend With Bong Joon Ho`。 | 截至 2026-04-15 的最新公开活动之一，包括 `Zodiac` 与 `Mickey 17` 相关放映对谈。 |
| 2026-04 | CJ ENM 官方确认动画长片 `ALLY` 制作，目标 2027 年上半年完成并全球上映。 | 首部长篇动画项目，将深海/非人生命/冒险类型纳入其导演系统。 |

## 最新动态（截至 2026-04-15）

- Academy Museum 的 `Director's Inspiration: Bong Joon Ho` 展览仍在进行，展期为 2025-03-23 至 2027-01-10。
- Academy Museum 2026-04-11 至 2026-04-12 的 `A Weekend With Bong Joon Ho` 已在当前日期前结束；其公开页列出 `Zodiac` with David Fincher and Bong Joon Ho 与 `Mickey 17` with Bong Joon Ho。
- CJ ENM 于 2026-04 发布 `ALLY` 官方制作确认：该片是奉俊昊首部长篇动画，CJ ENM、Penture Invest 和 Pathe Films 共同投资/发行，目标 2027 年上半年完成制作并全球上映。
- `Mickey 17` 是其截至 2026-04-15 最近已公映长片，2025 年围绕该片的访谈集中讨论科幻、复制人、政治讽刺、Robert Pattinson 的表演和英语片工业。

## 需要持续核对的未来项

- `ALLY` 的正式完成、首映、发行日期、配音阵容和美学方向。
- 传闻或开发中的韩国恐怖/惊悚项目是否进入正式制作。
- `Mickey 17` 的后续奖项、流媒体/家庭发行与长期评价。
