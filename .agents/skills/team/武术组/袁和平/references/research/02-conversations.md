# 02 长对话与访谈

## NYAFF / NextShark 访谈

NextShark 对 2019 NYAFF 期间袁和平活动进行整理，能直接看到几条动作原则：

- 袁和平偏好能真实完成动作的演员，CG 更像最后增强。
- 他把导演和武术指导视为相关但不相同的工作；动作指导有更大的动作玩耍空间，导演则要承担更长周期。
- 他认为香港动作片从 1980/1990 年代的大数量，转向更需要高质量和新想法。
- 他谈到《黑客帝国》时，重点不在炫耀技术，而在训练演员、细化每个动作，并把东方武术元素带入好莱坞。
- 他区分不同片种：如古装功夫、现代格斗、超现实喜剧，都应有不同动作风格。

来源：https://nextshark.com/yuen-woo-ping-interview
可信度：中高。访谈经翻译转述，需注意语境。

## *The Matrix* 制作口述

SYFY Wire 采访袁和平、Chad Stahelski 与 Glenn Boswell，提供了动作系统的生产细节：

- 袁和平团队使用 storyboards、stunt viz、dojo 预演和持续精修。
- 主要演员进行了数月训练；袁和平为每位主角安排武术队成员做体能与动作教练。
- 制作中既要追求演员本人可见动作，又要根据伤病、威亚安全和政府规范调整执行。
- 访谈显示“演员成为可信动作身体”是《黑客帝国》动作风格成立的重要条件。

来源：https://www.syfy.com/syfy-wire/the-matrixs-stunt-coordinators-and-choreographers-reveal-how-the-iconic-fight-scenes-were
可信度：中高。多方口述，有制作回忆偏差，但可交叉验证。

## 2026《镖人》访谈

Global Times 和中国新闻网分别围绕《镖人：风起大漠》采访袁和平与吴京，呈现其晚近动作观：

- Global Times 记录袁和平选择《镖人》的原因之一是原漫画人物和行动有“落地”的质感。
- 袁和平将武侠精神归到见不平、帮弱抗强的行动冲动。
- 他在《镖人》动作设计中强调减少花哨，让刀、枪、剑每次出手有实感。
- 中国新闻网采访中，袁和平强调动作好看要贴合人物性格、剧情发展和环境特质。
- 《镖人》的沙暴戏、城墙决战和兵器切换都被解释为环境与人物关系共同决定动作。

来源：

- https://www.globaltimes.cn/page/202602/1355389.shtml
- https://www.chinanews.com.cn/cul/2026/02-18/10573956.shtml
- https://www.scmp.com/lifestyle/entertainment/article/3343880/director-yuen-woo-ping-revives-wuxia-blockbuster-blades-guardians

可信度：中高。宣传期采访，但与其长期作品逻辑一致。

## 访谈提炼

袁和平式即兴回答常见结构：

1. 先从片种或动作风格区别入手。
2. 很快落到演员身体、训练、动作细节或现场系统。
3. 对技术持实用主义：可以用，但不能替代动作根基。
4. 对武侠持朴素伦理观：看见不平，身体要站出来。
