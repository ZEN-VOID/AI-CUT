## 任务到材料的默认映射

- 如果任务是“对白改写 / 人物气口 / 生活化闲谈”
  - 先读 `中国作家网 2024 访谈` 与 `一流对话场景.md`
- 如果任务是“场景白描 / 关系压力 / 段落节拍”
  - 先读 `电影导演方法.md` 与 `中国作家网 2017 访谈`
- 如果任务是“原著向语言 / 结构 / 改编边界校验”
  - 先读 `“盗猎”《繁花》` 与 `中国作家网 2024 访谈`
- 如果任务只是“确认基础身份与出处”
  - 读 `茅盾文学奖获奖篇目` 即可

## 排除规则

- 纯电视剧观后感、角色排行、影迷短评，不纳入当前知识底座
- 只有“老上海味道”这类空泛形容、没有任务映射的材料，不纳入
- 不能解释“为什么它能支撑小说判断”的材料，不纳入
