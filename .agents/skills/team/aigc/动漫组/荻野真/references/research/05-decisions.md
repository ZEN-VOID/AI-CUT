# 05 决策记录与行动

## 关键决策

1. **以《孔雀王》将密教传说改造成青年漫画动作**
   - 1985 年出道即以密教退魔题材进入《周刊 Young Jump》。
   - 决策含义：不是画宗教漫画，而是把密教图像转成黑暗动作娱乐。

2. **晚期重启《孔雀王ライジング》**
   - 决策来源：重新复习密教，发现“佛像创造的意义”；同时要描写少年孔雀如何成为无印孔雀。
   - 决策含义：旧 IP 重启不只重演，而是补起源和价值观变化。

3. **接受《孔雀王戦国転生》企划**
   - 决策来源：编辑 S 的强烈来访、对荻野真卖点的刺耳判断、纸媒发行渠道差异。
   - 决策含义：编辑碰撞可成为创作恢复的燃料。

4. **把いい加減点子转成资料密集连载**
   - 起初是聊天里的战国穿越退魔点子；读者反响推动他认真查战国资料。
   - 决策含义：ノリ启动后必须靠资料和辻褄把它变成作品。

5. **在历史题材中挑战权力叙事**
   - 《戦国転生》第三卷短文里批评公文书和御用学者式历史观。
   - 决策含义：历史奇想的核心不是炫考据，而是把被遮蔽的活人戏找出来。

6. **病中完成最终回的 name 与下描き**
   - 家属说明称住院期间仍完成《ライジング》《戦国転生》最终回的 name 和下描き，ペン入れ由工作人员完成。
   - 决策含义：最终交付和作者性确认成为遗作处理的核心。

## 行为与声称的一致性

- 他常自称随意、ノリ、行き当たりばったり，但实际行为显示强烈资料调查和收束意识。
- 他重视编辑与读者热度，但不把责任推给市场；反复说要让读者满意、让整体结束。
- 他把宗教/历史资料娱乐化，但又不断回到现实痛感：灾后社会、孩子、历史被权力叙述的问题。
