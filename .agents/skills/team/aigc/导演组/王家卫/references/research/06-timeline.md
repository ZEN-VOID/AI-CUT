# 06 人物时间线

| 时间 | 事件 | 对导演方法的影响 |
|---|---|---|
| 1958 | 生于上海 | 上海成为后续记忆母题 |
| 1963 左右 | 随家人迁居香港 | 上海/香港双城身份形成 |
| 1980 | 进入 TVB 第一届编导训练班 | 学会电视工业和剧作基础 |
| 1982-1987 | 参与多部商业电影编剧 | 熟悉类型片、明星和工业节奏 |
| 1988 | 导演首作《旺角卡门》入选 Cannes Critics' Week | 从类型片切入作者风格 |
| 1990 | 《阿飞正传》 | 1960s、时间、漂泊、明星气质成为核心语言 |
| 1991 | 与刘镇伟创办 Jet Tone | 增加创作自主权 |
| 1994 | 《东邪西毒》《重庆森林》 | 同时展示慢炖与快炒两种方法 |
| 1995 | 《堕落天使》 | 城市夜行、边缘人物、霓虹和广角风格推进 |
| 1997 | 《春光乍泄》获 Cannes 最佳导演 | 离散空间、亲密关系和香港临界时刻合一 |
| 2000 | 《花样年华》获 Cannes 男演员奖和技术大奖 | 克制、走廊、旗袍、音乐和删减成为代表性语法 |
| 2004 | 《2046》 | 记忆迷宫和旧作互文扩大 |
| 2006 | 任 Cannes 主竞赛评审团主席 | 国际作者地位稳固 |
| 2007 | 《蓝莓之夜》 | 英语语境实验，检验方法迁移 |
| 2013 | 《一代宗师》 | 武术被转译为时间、谱系和身体训练 |
| 2018 | 获 Harvard 荣誉艺术博士 | 国际文化影响被制度认可 |
| 2023-2024 | 《繁花》在中国大陆播出 | 首部剧集，重返出生地上海 |
| 2024 | 《繁花》获白玉兰等多项认可 | 电视剧工业内的王家卫方法被验证 |
| 2025 | 《繁花》推进 MUBI、Filmin、Criterion Channel 等国际发行；同时出现署名/劳动争议 | 全球扩散与制作伦理复核同时发生 |
| 2026-03-05 | 上海市英文网站报道《繁花》进一步扩展国际发行 | 其最新公开动态仍围绕《繁花》的国际生命周期 |

## 最新动态摘要

- 《繁花》已从中国大陆、港澳台逐步扩展到欧洲、北美、韩国、印度、拉美等市场。
- 王家卫近年公开访谈主要围绕《繁花》的上海记忆、剧集体量、怀旧风险和城市活力。
- 2025 年后《繁花》署名/劳动争议成为使用王家卫方法时必须纳入的伦理边界。
