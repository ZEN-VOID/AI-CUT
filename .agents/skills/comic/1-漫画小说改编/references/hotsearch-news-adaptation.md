# 新闻事件与网络热搜改编

本文件处理“新闻事件 / 网络热搜 -> 漫画小说”的特殊边界。

## 1. 先判模式，不先起戏

### `faithful`

- 用途：需要尽量贴近现实事实。
- 写法：保留事实时间线，用叙事组织提升可读性，不虚构关键事实。

### `inspired_by`

- 用途：保留热点议题与情绪核，但用虚构角色承载故事。
- 写法：事实是背景，人物线是正文。

### `free_reimagining`

- 用途：只借用议题核、冲突核或社会情绪。
- 写法：不再伪装成现实事件本身。

## 2. 热搜素材最值钱的不是“热”，而是三件事

1. 公众为什么停下来。
2. 讨论里最大的情绪裂口是什么。
3. 事件里还有什么没解释完。

## 3. 热搜改编选题公式

可直接套用：

`热点事件 / 话题 -> 公众情绪极点 -> 单个虚构主角 -> 个人代价 -> 未解问题 -> 章末钩子`

示例：

- 城市文旅热搜 -> 集体狂欢与身份焦虑 -> 返乡创业女主 -> 为保住老街必须直播真相 -> 谁在借热搜洗掉旧案
- AI 工具热搜 -> 效率崇拜与替代恐惧 -> 被边缘化的编剧 -> 用 AI 逆袭接单 -> 发现系统在偷写她的人生
- 品牌危机热搜 -> 舆论围猎与资本操盘 -> 公关新人 -> 被迫删证据保饭碗 -> 她删掉的第一条记录恰好是命案线索

## 4. 近期可参照的热搜结构信号

下面两条是我本轮为这个技能补的近时参考，不是硬编码题库：

- 2026 年 1 月 1 日，搜狐转载的《2025 年百度热搜影响力报告》提到，百度热搜内容中新闻热点约占 55%，生活兴趣约占 28%，娱乐热点约占 17%，说明热搜改编不能只盯文娱八卦，现实议题和生活兴趣也很适合做故事入口。
  - 参考链接：[搜狐转述](https://www.sohu.com/a/971396821_122531443)
- 2026 年 3 月 27 日，远瞻慧库文章转述艾瑞《2026 年互联网平台热搜价值研究报告》，强调热搜内容结构正从娱乐主导转向社会民生与垂直议题，多平台月均产生约 5 万条热搜内容。
  - 参考链接：[远瞻慧库文章](https://www.baogaobox.com/insights/260326000026537.html)

把这两条转成创作启发：

- “新闻 / 民生 / 垂直领域”比单纯明星八卦更适合写成现实悬疑、都市成长、行业黑幕、城市传奇。
- “生活兴趣”类热搜更适合写成甜宠、轻喜、都市成长或身份逆袭。
- “娱乐热点”仍然能写，但最好绑定身份错位、塌房、隐藏关系、旧案重启等更强戏剧芯。

## 5. 热搜改编的风险门

以下情况必须降级处理：

- 事实来源明显单薄，无法确认。
- 议题涉及现实受害者、未成年人、极端暴力或高度敏感身份信息。
- 用户要求把未经证实的传闻写成确定真相。

降级策略：

- 从 `faithful` 改为 `inspired_by`
- 从现实事件本体改成虚构承载线
- 明确删除未经证实的人名、机构、细节

## 6. 热搜改编钩子启发

热搜类故事尤其适合以下钩子：

- `证据失效式`
- `身份错位式`
- `代价升级式`
- `危险逼近式`

不建议默认通篇都用“纯情感余韵式”，因为议题类内容更需要“信息缺口 + 代价提升”来带动追更。
