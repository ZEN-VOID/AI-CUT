## Dialogue-Preserving Performance Contract

1. 对白、独白、旁白原文逐字不可变；本层只允许补写其周围的可见动作与节奏支点。
2. 若某轮冲突需要更强推进，优先补 `打断 / 停顿 / 视线落点 / 道具动作 / 空间逼近或退让`，而不是改句。
3. 一轮台词只保留一个主动作，避免同一轮同时承担 `施压 + 解释 + 自证 + 转场`。

## 回合型微表演部署表

| 回合任务 | 更稳的外化准心 | 建议锚点组合 |
| --- | --- | --- |
| 施压 | 让对方先感到空间或视线压力 | `目光 + 姿态 + 交互反馈` |
| 躲闪 | 让观众先看到回避，再听到台词 | `目光 + 呼吸 + 手部/道具` |
| 试探 | 先抛一个轻动作，再看对方接不接 | `停顿 + 目光 + 交互反馈` |
| 反击 | 让节奏突然变硬或突然变轻 | `呼吸节拍 + 面部 + 姿态` |
| 暴露 | 让失守先落在身体，再落到句义 | `面部 + 手部/道具 + 呼吸` |
| 收口 | 用静场、回视或撤步结束回合 | `目光 + 姿态 + 空间关系` |

## Lens-Aware / Subtext Notes

- `特写级` 优先看眼神回收、嘴角顿挫、咬肌与手指细节。
- `中景级` 优先看肩背张力、前倾后仰、谁在抢空间主位。
- `全景级` 优先看谁逼近、谁后退、谁被桌椅门框或走位压住。
- 高级潜台词常来自言行错位：语气越平，动作越可能露出真正温度；但反差必须和人物目标有关，不能只为“高级感”而反着写。
