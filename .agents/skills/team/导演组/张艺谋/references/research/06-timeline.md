# 06-timeline · 张艺谋人物时间线与最新动态

## 时间线

| 时间 | 事件 | 对导演方法的影响 |
|---|---|---|
| 1950/1951 | 出生于西安；出生年份在部分英文资料中有 1950/51 差异 | 早年身份和时代经历进入其历史/制度压力叙事 |
| 文化大革命时期 | 下乡、工厂劳动，后接触摄影 | 普通人、劳动、时代压力成为其现实主义底层经验 |
| 1978 | 进入北京电影学院摄影系 | 建立摄影/构图/色彩作为第一语言 |
| 1982 | 毕业，进入第五代电影人核心代际 | 与陈凯歌、田壮壮等共同构成中国电影新浪潮 |
| 1984 | 担任 `Yellow Earth` 摄影 | 黄土、高地平线、形式突破影响第五代视觉 |
| 1987/1988 | 执导 `Red Sorghum`，获柏林金熊 | 以红色、女性生命力和民间激情建立国际声誉 |
| 1990-1991 | `Ju Dou`、`Raise the Red Lantern` | 以女性、父权空间、仪式和色彩形成早期作者签名 |
| 1992 | `The Story of Qiu Ju` | 从浓烈写意切到现实主义求告结构 |
| 1994 | `To Live` 获 Cannes 重要奖项并引发审查争议 | 家庭命运成为大历史的承压面 |
| 1999 | `Not One Less` 获 Venice Golden Lion；`The Road Home` | 乡村、教育、普通人和朴素情感进入更柔和路径 |
| 2002-2004 | `Hero`、`House of Flying Daggers` | 将色彩叙事和武侠动作推入全球商业类型片 |
| 2004 起 | `Impression Liu Sanjie` 等山水实景演艺 | 电影调度扩展到实景、文旅、群众和长期演出 |
| 2008 | 北京奥运开闭幕式总导演；Peabody Award 相关表彰 | 大型仪式、直播镜头、人海调度成为其能力侧面 |
| 2011-2016 | `The Flowers of War`、`The Great Wall` 等跨国大制作 | 国际合作与工业大片路径，伴随争议和风险 |
| 2018 | `Shadow` 获关注；CGTN 采访谈水墨、雨、阴阳、人性灰度 | 从强色彩转向低彩度水墨系统 |
| 2020 | `One Second` | 回到胶片、记忆、观影初心与时代伤痕 |
| 2021-2022 | `Cliff Walkers`、`Snipers`；2022 冬奥开闭幕式 | 谍战/战争类型与低碳仪式并行 |
| 2023 | `Full River Red` 票房 45.4 亿元级，成为其生涯最高票房 | “悬疑+喜剧+封闭空间+历史符号”进入春节档 |
| 2024 | `Article 20`；凭该片获第 37 届大众电影百花奖最佳导演 | 司法现实议题通过喜剧/家庭/职场方式大众化 |
| 2024 | SIFF 期间确认执导《三体》电影版 | 进入高风险中国科幻 IP 和大规模 VFX 工业 |
| 2025 | 央视网金鸡奖专访谈 AI 时代与电影初心 | 对技术变化的回应转向“电影初心”和代际传承 |
| 2026-02-17 | `Scare Out / 惊蛰无声` 春节档上映 | 当代国安题材与谍战商业片结合 |
| 2026-04-15 | 当前调研截止 | 最新主要动态：`Scare Out` 上映后、`The Three-Body Problem` 电影仍为待观察项目 |

## 最近 12 个月动态（截至 2026-04-15）

- 2025-11：央视网发布金鸡奖相关专访，张艺谋以评委会主任身份谈中国电影 120 年、AI 冲击和电影初心。
- 2026-01：China.org.cn 报道 `Scare Out / 惊蛰无声` 定档 2026 春节档，聚焦当代国家安全，主演包括易烊千玺、朱一龙、宋佳、雷佳音、杨幂等。
- 2026-02-17：`Scare Out / 惊蛰无声` 上映，进入春节档竞争。
- 2026-03：AP 报道该片与国家安全部指导/背书关系，并称上映两周票房约 1.1 billion yuan / 160 million USD 级别。
- 2024-06 至 2026-04：`The Three-Body Problem` 电影版仍处于公开报道所称的开发/前期阶段；未核到正式上映日期。

## 最新动态风险

- `Scare Out` 的票房、海外片名和平台数据在不同来源中可能有差异；使用时应以最新票房数据库或官方通稿核对。
- 《三体》项目的“无限投资”“忠于原著”属于片方公开表述，不能等同于最终成片质量或最终制作规模。
- 张艺谋仍在高频创作，2026-04-15 之后的新片、新访谈、新奖项必须重新检索。
