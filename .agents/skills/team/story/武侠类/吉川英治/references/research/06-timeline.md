# 06 时间线调研：吉川英治

## 调研结论

吉川英治的时间线清晰地支持当前技能定位：艰难出身 -> 大众连载成名 -> 历史长篇定型 -> 战后沉潜 -> 以《新・平家物語》再起 -> 国民文学地位确立。

## 关键时间线

| 时间 | 事件 | 影响 |
| --- | --- | --- |
| 1892-08-11 | 生于神奈川县久良岐郡中村根岸 | 艰难出身与大众视角形成底色 |
| 1925 | 《剣難女難》首次使用“吉川英治”笔名 | 确立大众长篇写作者身份 |
| 1926-1927 | 《鳴門秘帖》连载 | 快速建立读者基础 |
| 1935-1939 | 《宫本武藏》连载 | 成长型英雄与国民读者影响力形成 |
| 1939-1944 | 继续写《新書太閤記》《三国志》等 | 历史人物与长篇群像规模扩大 |
| 1944-1953 | 青梅生活，战后约三年停笔 | 沉潜与重估阶段 |
| 1950-1957 | 《新・平家物語》连载七年 | 群像历史长篇与史料踏查方法成熟 |
| 1953 | 获菊池宽奖 | 主流文学认可 |
| 1960 | 获文化勋章 | 成为首位获此荣誉的大众小说家 |
| 1962-09-07 | 去世 | 国民文学地位定型，后续以纪念馆和奖项延续影响 |

## 时效性说明

- 吉川英治已去世，不存在作者本人“近 12 个月新动态”。
- 当前可更新的动态主要是：纪念馆展览、奖项活动、作品传播与改编再传播。
