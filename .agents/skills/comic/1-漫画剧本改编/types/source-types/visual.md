# 图片视频来源类型包

## Match Signals

- 单张图片、角色照、海报、分镜图、短视频摘要、直播片段、采访片段、镜头序列。

## Fixed Context

- 图片先抽：视觉奇点、人物关系、动作瞬间、空间压力、前后因假设。
- 视频先抽：时间序列、镜头切点、动作节点、台词/字幕信号、节奏峰值。
- 不把视觉素材写成空泛美术描述；必须转成“谁想要什么、被谁阻拦、下一秒会发生什么”。
- 单帧可用“画面之前刚发生了什么、之后最可能发生什么”补剧情发动机，但不得伪造现实新闻事实。

## Review Gate

- `source_digest` 必须同时包含可见物、事件链、关系位和未解点。
- 至少锁 2-3 个可做大格、特写或翻页停笔的视觉锚点。
