# 高武题材增强包

> 补充维度：套路拆解 / 情节模式 / 人物原型 / 反套路策略 / 质量过滤

## 一、核心套路与行规

### 1.1 力量体系升维
- **等级阶梯**：学徒→武士→武师→宗师→大宗师→超凡→传奇→神话→不朽。每级有明显的能力质变节点。
- **能力觉醒**：血脉觉醒 / 精神觉醒 / 元素亲和 / 法则领悟，能力觉醒通常伴随代价（寿命/理性/情感/记忆）。
- **武学流派**：内家/外家/气功/剑道/拳法/暗器，不同流派有鲜明视觉特征和战斗哲学。

### 1.2 资源竞争
- **修炼资源**：灵石、丹药、天材地宝、秘境、传承。资源永远稀缺，竞争永远残酷。
- **势力博弈**：宗门/世家/散修/朝廷/异族，多势力交错，利益冲突是推进主线的核心引擎。
- **信息不对称**：高级功法/秘闻/宝藏信息是核心筹码，掌握信息的角色天然拥有优势。

### 1.3 常见套路陷阱
- 开局废材→意外获得→一路升级→打脸→收后宫（最老套模式）
- 长辈/师父必死定律（为给主角腾出压力空间）
- 学院/宗门段的千篇一律测试/比试
- 等级压制永远刚好（对手永远比主角高一两级）
- 宝物/功法刚好匹配（巧合过多）

## 二、情节模式与节奏规则

### 2.1 标准三段弧
```
觉醒期(1-3卷)：废材→发现天赋→初步变强→走出新手村
征战期(4-8卷)：闯荡江湖→加入势力/建立势力→对抗区域Boss→声名鹊起
封神期(9+卷)：触及天道/法则→对抗世界级Boss→封神/守护/超脱
```

### 2.2 节奏铁律
- **升级节奏**：前50章每5-8章升1级，中段每10-15章升1级，后期每20+章升1级
- **战斗密度**：每3-5章至少1场有实质意义的战斗（非碾压局），每卷至少1场生死战
- **爽点频率**：每2-3章1个小爽点（突破/打脸/获得），每10章1个中爽点（越级战胜/关键揭露），每卷1个大爽点（Boss战/势力易主）

### 2.3 情节模式图谱
| 模式 | 频率 | 说明 |
|------|------|------|
| 突破升级 | 高频(每5-8章) | 力量体系推演的核心驱动器 |
| 机缘奇遇 | 中频(每卷1-2次) | 秘境/遗迹/传承/奇遇，打开新地图 |
| 比试/擂台 | 中频 | 展现个人实力、获取声望、建立关系 |
| 势力对抗 | 持续 | 宗门战争、世家斗争、异族入侵 |
| 秘境探索 | 低频(每卷1次) | 团队协作、资源获取、世界观展开 |
| 复仇/归来 | 关键节点 | 被打倒→隐忍→归来复仇，经典高潮模式 |

## 三、人物原型

### 3.1 主角原型
| 原型 | 特征 | 适用场景 |
|------|------|----------|
| 废材逆袭型 | 开局极弱，靠意志/热血/外挂崛起 | 最广泛，需反套路化 |
| 重生归来型 | 前世强者的记忆带回来，知道未来走势 | 中后期为主，前期需隐藏 |
| 血仇觉醒型 | 全家/宗门被灭，一人踏上复仇之路 | 黑暗向，情感驱动力强 |
| 草根崛起型 | 毫无背景的底层少年，全靠自己拼出来 | 接地气，代入感强 |

### 3.2 核心配角
- **导师/引路人**：不要写成纯粹的工具人，给独立的目标和代价
- **对手/宿敌**：不是单纯的反派，而是"另一个可能成为的主角"，让读者理解他
- **伙伴/战友**：各自有独立的能力线和成长弧，不是主角的附属品
- **红颜/知己**：避免成为"花瓶"或"奖励"，应有独立人格和行动力

### 3.3 关系动态
- 师徒之间：传承不仅是能力，更是责任和代价
- 对手之间：竞争中有尊重，立场不同但人格可以平等
- 伙伴之间：背靠背的信任来自共同经历，不是设定

## 四、反套路策略

### 4.1 废材反套路
- **不靠外挂靠特性**：主角的特殊不在"获得金手指"，而在"别人没有的认知/经历/思维方式"
- **代价真实**：每次突破都有真实代价——寿命缩短、记忆丧失、情感迟钝、被追杀的代价
- **升级不是目的**：变强是为了保护某人/某物/某个信念，不是为变强而变强

### 4.2 对手反套路
- **不是工具人**：对手有自己的正义观、复仇理由和追随者，他的行为从自己的角度看是"正确"的
- **多维度对抗**：不只是战斗力对抗，还有智力博弈、信息博弈、心理博弈、资源博弈
- **对抗不是终点**：与重要对手的关系可以演变（宿敌→亦敌亦友→联合对抗更大威胁）

### 4.3 世界观反套路
- **力量不是一切**：在高武世界中展示力量无法解决的东西（政治、人性、命运）
- **世界的代价**：高武世界的存在本身就造成伤害（普通人被碾压、环境被破坏、世界规则被扭曲）
- **多重力量体系**：不只有一条修炼路径，不同的力量体系有各自的逻辑和代价

## 五、质量过滤规则

### 5.1 等级系统自洽检查
- [ ] 所有角色的等级与表现一致（无"等级虚标"）
- [ ] 每次升级有真实的铺垫、契机和代价
- [ ] 越级战斗有合理的"不对称"解释（不是纯开挂）
- [ ] 前期伏笔的能力在后期有合理兑现

### 5.2 战斗场面质量检查
- [ ] 战斗有空间感（方位、距离、地形、环境在变化）
- [ ] 战斗有节奏变化（试探→对攻→受伤→觉醒→逆转）
- [ ] 战斗结果有真实代价（不仅是"赢了"）
- [ ] 战斗推动剧情/人物/关系发展（不是为打而打）

### 5.3 AI腔高发区警示
- 高武题材最易出现：过度招式描述（流水账式拳来脚往）、等级数字堆砌、打脸模板化、靠对话解释战斗
- 应有：空间感战斗、代价驱动的战斗选择、让读者"看到"的战斗画面

### 5.4 失败码
| 失败码 | 症状 | 返工目标 |
|--------|------|----------|
| `FAIL-GAO-WU-LEVEL` | 等级系统矛盾或虚标 | 回源到角色卡/能力体系设计 |
| `FAIL-GAO-WU-FIGHT` | 战斗无空间感/无节奏/无代价 | 返工到战斗段落重写 |
| `FAIL-GAO-WU-POWER-ONLY` | 纯力量驱动、缺智力/心理/关系博弈 | 回补非力量维度的冲突 |
