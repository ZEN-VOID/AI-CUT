# Psychological Reaction Contract

## Purpose

本细则定义 `2-编导` 中 `心理反应` 的唯一业务语义：它不是小说内心、作者解释或主角抽象想象的收纳字段，而是影视剧本中把心理信息落实为可见、可听、可表演反应的字段。主角内心想法和内心独白必须保留，但应进入 `独白（主角）` / `内心独白（主角）` 或执行报告证据，不得伪装成客观第三方总结。

影视剧本的功能是让执行工具、演员、导演、声音部门和观众通过画面或声音接收信息。因此，`心理反应` 必须回答：

- 谁在反应。
- 上游触发点是什么。
- 身体、表情、呼吸、声音、道具、空间或对手关系发生了什么可感知变化。
- 观众不读解释文字时，能从哪里 GET 到这层心理。

## Ownership

- 本文件拥有 `心理反应` 的详细字段语义、类型化转译、演员表演口径、证据结构、正反例和 review 判断。
- `field-routing-and-audio-visual-contract.md` 只保留字段入口、声画纯度和简短跳转，不复制本细则全文。
- `performance-and-scene-craft-contract.md` 消费本细则，把心理变化进一步纳入演员任务、场面调度、沉默反应和内嵌目标。
- `actor-performance-control-contract.md` 消费本细则，把关键心理反应进一步细化为微表情、情绪动机、非面部生理联动、环境声音和微动态限制。
- `novel-to-screen-language-contract.md` 负责识别小说心理内视；本文件负责决定这些内视如何进入 `心理反应` 或改入其他声画字段。

## Protagonist Inner Voice And POV Rule

- 上游明确属于主角内心想法、主角判断、主角对他人行为的解读或主角视角下的怀疑，不应被删除；应优先保留为 `内心独白（主角）`，或在不需要语言化时转成 `心理反应`、`对白画面`、`表情特写`、`角色动作` 和对应画面。
- `内心独白（主角）` 是主角当下可被听见或可被声音设计呈现的心声，不是第三人称旁白。把小说第三人称叙述转入该字段时，指代主角自身的“他/她/其/角色名”必须改成“我/我的/自己”或角色当下自然口吻；`内心独白画面` 才使用第三人称可拍描述。
- 主角视角下对他人行为的判断不得写成客观第三方概括。例如“她是在试探他”“他看起来早有准备”不能作为客观叙述直接落入画面字段；应改为主角内心独白，或写主角观察到的可见证据和反应。
- 其他角色的心理不得被全知旁白直接断言。除非上游有该角色直接内心或对白证据，否则只能写主角看见/听见了什么，以及主角如何在内心判断。
- “感到恶心、难受、愤怒、害怕、崩溃”等主观情绪感受不得直接写成终稿心理结论；必须转成微表情、肢体动作、生理反应、声音变化，或主角内心独白。
- 与当前主线无关的人物过往、物品来历、额外回忆和解释性补充不得为了说明心理而新增；只能使用当前场景可见/可听信息和上游已经给出的必要信息。

## Source Scope

可进入本 pass 的上游信号：

| source_signal | examples | allowed output |
| --- | --- | --- |
| 惊疑 / 恐惧 | 意识到危险、发现异常、听见威胁 | 呼吸、停顿、吞咽、手指、身体后撤、道具停点 |
| 羞惭 / 心虚 | 被戳中、遮掩、不能承认 | 避开正面、声线断掉、整理无关物、错开距离 |
| 试探 / 隐瞒 | 不想说破、想套话、保留底牌 | 半句停住、反问前停顿、先看对方手再接物 |
| 信任变化 | 接受、怀疑、关系松动或变冷 | 物件接过/不接、距离变化、身体朝向变化 |
| 权力压迫 | 被凝视、被命令、被规则压住 | 站坐高低、手离开桌面、群体安静、对手不接话 |
| 推理 / 记起 | 从物件、文字、声音或场面得出判断 | 目光停点、道具特写、短内心独白、动作中断 |
| 未出口对白 | 想说但不能说、说到一半改口 | 喉结、吸气、改拿道具、把话咽回去、沉默延长 |
| 主角视角判断 | 主角解读他人动作、语气、沉默或回避 | 主角内心独白、主角观察到的可见证据、主角反应 |
| 主观情绪感受 | 恶心、难受、愤怒、恐惧、羞耻 | 微表情、肢体动作、生理反应、声线变化、内心独白 |

不可进入本字段的内容：

- 作者直接解释，例如“他终于明白了命运的安排”。
- 抽象想象，例如“她的灵魂坠入黑暗”。
- 因果论文，例如“因为没有人敢缺席，所以大家都沉默”。
- 关系结论，例如“两人的信任发生了变化”。
- 上游未给出的新幻象、新回忆、新台词、新答案或新因果。
- 主观意图预判，例如在动作字段写“试图、想要、打算、意图、准备借此”等不可拍心理。
- 与当前主线无关的过往背景、物品来历或回忆性信息。

## Getability Standard

`心理反应` 必须至少满足以下三类中的一类，关键心理 beat 建议满足两类以上：

| channel | must answer | examples |
| --- | --- | --- |
| 可见 | 摄影机能看见什么 | 指腹压纸、杯面晃动、肩背僵住、脚尖停在门槛内 |
| 可听 | 声音部门能听见或设计什么 | 吸气断掉、吞咽声、尾音收住、沉默后的纸张声消失 |
| 可演 | 演员能执行什么任务 | 想遮掩但不能逃开，于是先整理课本再抬眼；想追问但压住问题 |

失败判断：

- 只能靠字段标题理解心理，正文没有可见/可听/可演载体。
- 只写“意识到、觉得、明白、崩溃、震惊、害怕、不信任、被压迫”。
- 用比喻或抽象名词代替演员动作。
- 把需要明确语言的内容塞进 `心理反应`，却没有改入 `独白/内心独白`。

## Projection Ladder

处理心理信息时按以下顺序决策：

1. 先锁定上游触发点：文字、道具、动作、声音、对手一句话、群体反应、空间压力或规则显影。
2. 判定心理功能：恐惧、推理、遮掩、试探、权力压迫、信任变化、羞惭、未出口对白。
3. 选择主载体：
   - 能让观众从行为看懂：优先 `心理反应`、`角色动作`、`对白画面`、`表情特写`。
   - 需要文字或语言明确，尤其是主角对他人行为的主观判断：使用 `独白（主角）` / `内心独白（主角）`，并补对应画面。
   - 重点是物件信息：使用 `道具特写`，心理只写反应，不写推理结论。
   - 重点是关系空间：使用 `场面调度` 或拆入 `对白画面`、`角色动作`、`群像画面`。
4. 给演员任务：在规划证据中可判断角色即时目标、阻碍和策略；进入 `角色动作` / `动作画面` 时只保留客观可拍动作、神态、语气和生理反应，不写“试图、想要、打算、意图”等主观预判词。
5. 对关键心理 beat 执行 `actor_performance_control_pass`：至少选择一个微表情变量和一个非面部身体变量；强情绪、压抑情绪或情绪切换场景还应判断表层情绪、压制情绪、隐藏动机、环境声音和微动态限制。
6. 写成紧贴 beat 的正文，不在场景末尾总结，不新增无关前史或回忆。

## Actor Performance Model

`心理反应` 可使用以下最小结构，但终稿不必输出字段名以外的结构标签：

```text
心理反应：<主体>因<上游触发点>产生<心理功能>；外显为<身体/表情/呼吸/声音/道具/空间/对手反应>。
```

写作时删除解释层，只保留可演结果：

```md
心理反应：林寂盯住黑板上最后一行规则，呼吸停了一拍，压在课本边缘的手指慢慢收紧。
```

不要写成：

```md
心理反应：林寂意识到规则真正开始生效，心里生出强烈恐惧。
```

## Field Selection Matrix

| psychological_need | prefer | avoid |
| --- | --- | --- |
| 角色听到威胁但不能回应 | `心理反应` / `音效画面` / `对白画面` | 新增一句解释性对白 |
| 角色完成推理 | `道具特写` + `心理反应`，或 `内心独白` + `内心独白画面` | `心理反应：他明白了真相` |
| 角色不信任对方 | `对白画面` / `角色动作` / `心理反应` | `心理反应：他不信任她` |
| 主角判断对方在撒谎或试探 | `内心独白（主角）` + `内心独白画面`，或主角观察到的可见证据 + 主角反应 | 客观叙述“她在撒谎”“她是在试探他” |
| 角色被权力压住 | `场面调度` / `群像画面` / `心理反应` | `环境描写：压迫感笼罩全场` |
| 角色想说但不能说 | `心理反应` / `角色动作` / `音效画面` | 新增未授权台词 |
| 角色出现恶心、难受、愤怒等主观感受 | `心理反应` / `表情特写` / `角色动作` / `内心独白` | `心理反应：他感到恶心/难受/愤怒` |
| 角色心理必须被明确听见 | `独白` / `内心独白` + 对应画面 | 把完整想法藏进 `心理反应` |

## Evidence Contract

在执行报告或节点证据中，关键心理反应应能等价覆盖：

```yaml

    source_anchor: ""
    subject: ""
    reaction_function: "fear | recognition | suspicion | concealment | shame | trust_shift | power_pressure | withheld_speech | reasoning | memory"
    pov_owner: "protagonist | other_character | external"
    trigger: ""
    performance_task: ""
    getability_channels:
      visible: []
      audible: []
      performable: []
    target_fields:
      - "心理反应"
    risk_check:
      abstract_inner_life: false
      new_dialogue: false
      new_event: false
      over_explaining: false
      field_summary_block: false
      subjective_intent_in_action: false
      unrelated_backstory: false
```

该证据不是终稿正文块。它只证明 LLM 投影前已经完成心理可感知化判断。

## Rewrite Patterns

| weak | issue | strong |
| --- | --- | --- |
| `心理反应：林寂意识到自己拿到了底牌。` | 抽象认知，没有可演载体 | `心理反应：林寂盯住纸角那道红印，呼吸停住，指腹把课本边缘压出一道浅痕。` |
| `心理反应：她的内心像坠入无边黑暗。` | 文学比喻，不能执行 | `心理反应：她吸了一口气没有吸满，手背贴住杯壁，杯中水面轻轻晃了一下。` |
| `心理反应：因为没有人敢在课堂时间缺席。` | 因果解释，不是反应 | `群像画面：后排的椅脚声停住，几名学生同时把脚收回课桌下。` |
| `心理反应：两人的信任发生变化。` | 关系结论 | `心理反应：阿真把纸条推回桌线外，停了半拍，又用两根手指把它压住，没有收进袖中。` |
| `心理反应：他不想暴露自己知道规则。` | 目标正确但仍是解释 | `心理反应：他没有立刻追问，先低头把课本摆正，把真正的问题压到最后两个字才抬眼。` |
| `角色动作：林寂试图掩饰自己的愤怒。` | 主观意图和情绪结论混入动作 | `角色动作：林寂把纸角抚平两次，第二次指节压得发白，抬眼时下颌绷住。` |
| `对白画面：她是在试探林寂。` | 主角视角判断被写成客观概括 | `内心独白（林寂）：“她在等我先补后半句。”` + `内心独白画面：她的问题停在半截，视线落到林寂握紧的手指上。` |
| `心理反应：林寂感到一阵恶心。` | 直接书写主观感受 | `心理反应：林寂喉头轻轻一滚，嘴角压住，手背抵到唇边又放下。` |

## Review Checklist

交付前检查：

- 每条 `心理反应` 都有明确主体。
- 每条都能回指上游触发点。
- 每条至少包含一个可见、可听或可演通道；关键心理 beat 至少包含两个。
- 没有“意识到/觉得/明白/内心/灵魂/象征/说明/因为所以/关系发生变化”等抽象解释作为字段主体。
- 需要明确语言化的内容已进入 `独白/内心独白`，并配对画面。
- `内心独白（主角）` 引号内的主角自指已统一为第一人称；第三人称只指向其他角色、引用文本，或有明确自我疏离创作留证。
- 主角视角下对他人行为的主观解读已进入主角内心独白或主角反应，没有被写成客观第三方总结。
- 动作字段只写可实拍的动作、神态、语气和生理反应，没有“试图、想要、打算、意图”等主观预判词。
- “感到恶心/难受/愤怒”等主观感受已转成微表情、肢体动作、生理反应或主角内心独白。
- 没有新增对白、回忆、幻象、事件、因果或结论。
- 没有为解释心理新增无关人物过往、物品来历或回忆性信息。
- 没有在场景末尾集中总结心理状态；反应已内嵌到对应 beat。
