# 06-timeline.md

主题：罗伯特·麦基时间线与最新动态边界

## 时间线

### 早年与训练

- 1940s：公开二手资料普遍将其出生信息指向美国密歇根州 Detroit 区域。  
  可信度：中（二手）

- 1960s：进入戏剧/文学训练路径，后与密歇根大学、Old Vic 等经历相关联。  
  可信度：中（二手 + 访谈侧证）

### 职业形成

- 1970s-1980s：逐步累积写作、导演、教学与文本分析经验。  
  可信度：中

- 1980s：`Story Seminar` 成为其最核心的公共工作形式。  
  可信度：中高（官方长期自述）

### 书籍阶段

- 1997：`Story`
- 2016：`Dialogue`
- 2018：`Storynomics`
- 2021：`Character`
- 2023：`Action`

以上年份与书目顺序可由官方书页与公开书店资料交叉支持。  
可信度：高

### 最近 12-24 个月可确认动态

- 截至 2026-04-18，McKee Story 官网仍在运营，Books / Events / Webinar 结构完整。  
  来源：官网导航与活动页  
  可信度：高

- 2025-2026 的公开状态仍以 seminar / webinar / books 为主，而非新的大规模转型叙事。  
  来源：官网 About / Events  
  可信度：高

## 时间线用途提醒

- 用于本 skill 时，最近动态只可写到“官网仍在更新课程与书”，不要虚构具体城市巡讲、最新访谈观点或未核验的新书计划。
- 生平细节若需要高精度（出生年、具体学校年份、个人家庭细节），应再做专门核验。
