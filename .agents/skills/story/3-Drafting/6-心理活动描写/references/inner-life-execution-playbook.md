# Inner-Life Execution Playbook

本文件是 `6-心理活动描写` 的本地执行细则。

它只负责把“心理活动描写”映射成可执行重写动作，不再复写父级合同。

## 1. 先锁 POV，再写心理

- 先回答：
  - 这一段是谁在感？
  - 他/她现在最多能意识到哪里？
  - 此刻的心理运动是被什么触发的？
- 若回答不出来，先不要写内心句。

## 2. 抽象心理词的默认落地位

| 抽象词 | 优先落地位 |
| --- | --- |
| 害怕 / 紧张 | 呼吸、手心、视线、肌肉反应 |
| 羞耻 / 心虚 | 回避、嘴硬、动作变形、视线不敢停 |
| 欲望 / 心动 | 身体趋近、注意力被勾住、想说又压回 |
| 愤怒 / 不甘 | 力道、忍住、反击冲动 |
| 怀疑 / 动摇 | 判断翻转、记忆回闪、看法松动 |

## 2.5 压力类型速判

先给当前心理运动定型，再决定怎么写：

| 压力类型 | 优先写法 | 不能只写成 |
| --- | --- | --- |
| 生理性恐惧 | 身体先缩、呼吸乱、注意力被危险劫持 | “他很害怕” |
| 羞耻/心虚 | 回避、嘴硬、想遮掩、动作卡顿 | “她觉得很丢脸” |
| 欲望/吸引 | 趋近冲动、注意力黏住、想说又压回 | “她忽然心动了” |
| 关系风险 | 想失去什么、谁会受伤、代价影子逼近 | “这段关系很复杂” |
| 身份/伦理压力 | 角色职责、制度边界、自我约束 | “他职业道德感很强” |

## 3. 四种常用重写动作

1. `情绪词 -> 知觉`
   例：不是“她很慌”，而是“她听见门轴一响，后背先凉了一寸”。
2. `总结句 -> 半显性自辩`
   例：不是“他其实已经后悔”，而是“他把那句话在心里压回去，像压一口不该翻出来的旧火”。
3. `概念判断 -> 当下判断`
   例：不是“她终于明白自己爱上了他”，而是“她一听见那名字，先想的不是躲，是他今晚会不会伤着”。
4. `纯心理 -> 心理+动作影子`
   例：让人物的心理变化带出下一拍要靠近、回避、出手或沉默。

## 3.5 渡边淳一式可选镜片

仅在当前场面确实涉及亲密关系、禁忌关系、熟年情感、医疗观察或身体真实时启用；它是增益镜片，不是默认模板。

### 镜片 A：身体先承认

- 先问：这段心理运动首先落在身体哪里？
- 优先落点：呼吸、皮肤、视线停驻、吞咽、手指、步伐、坐姿变化。
- 用法：把“她对他有感觉”改成“她先注意到自己没有把视线移开”这一类临场事实。

### 镜片 B：羞耻比结论更早到

- 先问：人物最先想藏起什么？
- 优先落点：嘴硬、转移话题、回避触碰、故作平静、拿身份遮挡自己。
- 用法：不要先写“她知道自己越界了”，先写“她先把语气放冷，像这样就能遮过去”。

### 镜片 C：代价影子要提前入场

- 先问：如果顺着这股心理走下去，谁会付账？
- 优先落点：配偶、孩子、名誉、岗位、病人、时间、体面、旧承诺。
- 用法：关系还没爆炸时，也要让人物心里闪过一秒“这不是只有我们两个人的事”。

### 镜片 D：熟年人物先有时间感

- 先问：这个人已经不再年轻，什么现实正在压着他/她？
- 优先落点：疲惫、药物、旧伤、家庭账目、职业末期、体力下滑、对孤独的敏感。
- 用法：别把中年/熟年心理写成青春版自白，要让现实时间表压进句子里。

### 镜片 E：观察者视角不能只剩术语

- 先问：这个人是在看见别人，还是在把别人分析成对象？
- 优先落点：职业训练带来的观察习惯、职责边界、想靠近又自我约束的瞬间。
- 用法：医生、护理者、调查者可以冷静，但不能靠术语硬撑真实感。

## 3.6 高风险场面补写顺序

遇到亲密、禁忌、熟年或职业伦理场面，优先按这个顺序补：

1. 触发点：谁说了什么、碰到了什么、看见了什么。
2. 身体信号：身体先怎么变。
3. 自我约束或自辩：人物想怎么把这件事压住、改口或合理化。
4. 代价影子：他/她知道继续下去会伤到什么。
5. 下一拍动作影子：靠近、回避、沉默、转移、试探。

## 4. 禁写信号

- 像作者讲主题的总结句
- 一口气把人物想法解释完的整齐长句
- 比人物当前认知更早抵达的终局结论
- 只有“痛、难受、复杂、震惊”等抽象词，没有场面触点
- 只有欲望或痛苦，没有羞耻、自我约束和关系/身份代价
- 把禁忌关系、熟年情感或职业伦理写成悬浮抒情，没有现实压力
